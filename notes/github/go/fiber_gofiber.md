### [gofiber fiber](https://github.com/gofiber/fiber)  ![GitHub Repo stars](https://img.shields.io/github/stars/gofiber/fiber?style=social)

Fiber 是一个基于 Go 语言构建的 Express 风格 Web 框架，底层依托 Fasthttp 实现极致性能。项目采用零内存分配设计，旨在简化 Web 开发流程并保障高性能。核心功能涵盖健壮的路由系统、静态文件服务、丰富的中间件支持、模板引擎、WebSocket 及 Server-Sent Events 等。框架兼容标准库 net/http 及 Express 处理器，内置多种中间件并支持外部扩展，适用于快速构建高性能 Web 应用及微服务。
