### [evcc-io evcc](https://github.com/evcc-io/evcc)  ![GitHub Repo stars](https://img.shields.io/github/stars/evcc-io/evcc?style=social)

evcc 是一款可扩展的电动汽车充电控制器和家庭能源管理系统，致力于实现不依赖云服务的本地能源管理。

其核心功能包括：
1. 广泛兼容各类电动汽车充电桩、能源计量设备（光伏、电池、电表）、车辆及热泵/电加热器，支持 EEBus、OCPP 协议。
2. 提供简洁用户界面，支持 Modbus、HTTP、MQTT 等插件扩展，便于自定义集成设备。
3. 具备状态通知、日志记录功能，并通过 REST/MQTT API 与 Home Assistant、openHAB 等家庭自动化系统无缝集成。
