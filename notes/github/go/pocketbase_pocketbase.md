### [pocketbase pocketbase](https://github.com/pocketbase/pocketbase)  ![GitHub Repo stars](https://img.shields.io/github/stars/pocketbase/pocketbase?style=social)

PocketBase 是一个开源的 Go 语言后端解决方案，以单个可执行文件形式提供。其核心功能包括：

- 嵌入式 SQLite 数据库，支持实时订阅
- 内置用户认证和文件管理功能
- 提供便捷的 Admin 管理后台界面
- 提供简单的 REST-ish API

支持作为独立应用运行，也可作为 Go 库嵌入项目以自定义业务逻辑，支持 JavaScript 扩展，并提供 JavaScript 和 Dart SDK 客户端。
