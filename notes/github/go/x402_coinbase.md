### [coinbase x402](https://github.com/coinbase/x402)  ![GitHub Repo stars](https://img.shields.io/github/stars/coinbase/x402?style=social)

x402 是一个互联网原生支付的开放标准，支持加密与法币网络及各类价值形式。它通过 HTTP 402 状态码及 Header 无缝集成支付流程，无需额外请求，实现极简集成（服务器仅需 1 行代码）。系统通过客户端、资源服务器和 Facilitator 协作，支持可扩展的支付方案，遵循开放、信任最小化及易用原则，完成资源的支付访问与结算。
