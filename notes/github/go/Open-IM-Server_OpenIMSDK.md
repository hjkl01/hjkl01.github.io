### [OpenIMSDK Open-IM-Server](https://github.com/OpenIMSDK/Open-IM-Server)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenIMSDK/Open-IM-Server?style=social)

OpenIM 是一个面向开发者的开源即时通讯解决方案，包含 SDK 和 Server 两部分，用于在应用中集成即时通讯功能。

- **OpenIM SDK**：基于 Go 语言，支持跨平台，提供用户、好友、群组、会话管理及本地存储等核心模块。
- **OpenIM Server**：采用微服务架构，支持集群及多种部署方式，可支撑海量用户和消息并发。
- **业务扩展**：提供 REST API 和 Webhooks，方便后端系统对接及事件回调。
- **系统支持**：兼容 Linux、Windows、Mac 系统及 ARM、AMD 架构，采用 Apache License 2.0 协议。
