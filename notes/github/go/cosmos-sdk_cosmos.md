### [cosmos cosmos-sdk](https://github.com/cosmos/cosmos-sdk)  ![GitHub Repo stars](https://img.shields.io/github/stars/cosmos/cosmos-sdk?style=social)

Cosmos SDK 是一个模块化、开源的区块链开发工具包（SDK），旨在构建安全、高性能且可高度定制的 Layer 1 区块链。它提供涵盖治理、状态管理、账户抽象等功能的标准化模块，并支持自定义模块开发。该 SDK 原生集成 IBC 协议实现跨链互操作性，兼容任意共识引擎（推荐使用 CometBFT），同时提供适用于许可网络的企业级模块。目前该框架已被 200 多个区块链项目在生产环境中应用。
