### [canopy-network canopy](https://github.com/canopy-network/canopy)  ![GitHub Repo stars](https://img.shields.io/github/stars/canopy-network/canopy?style=social)

该项目是 Canopy 网络协议的 Golang 实现，旨在构建一个去中心化的区块链启动平台。其核心功能包括：

- 递归框架：支持区块链之间相互引导，形成安全、自主的网络。
- 种子链：启动递归循环的初始链。
- 控制器模块：协调区块链各部分通信。
- 状态机：管理交易如何改变区块链状态。
- 共识机制：拜占庭容错（BFT）机制确保节点间就新区块达成一致。
- 点对点网络：节点间安全加密通信。
- 数据持久化：管理区块链存储，确保数据快速可靠验证。

项目支持本地和 Docker 环境运行，提供测试、构建和部署工具，并欢迎社区贡献代码。
