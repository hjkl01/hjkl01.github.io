### [gin-gonic gin](https://github.com/gin-gonic/gin)  ![GitHub Repo stars](https://img.shields.io/github/stars/gin-gonic/gin?style=social)

Gin 是基于 Go 语言编写的高性能 HTTP Web 框架，提供类 Martini 的 API。其核心功能包括零分配路由、中间件支持（认证、日志、CORS 等）、自动 JSON 绑定与验证、内置错误恢复、路由分组及多格式渲染。该框架适用于构建高并发 REST API、微服务和 Web 应用，具备高性能和低内存占用特点，并拥有丰富的社区中间件生态。
