### [tulir whatsmeow](https://github.com/tulir/whatsmeow)  ![GitHub Repo stars](https://img.shields.io/github/stars/tulir/whatsmeow?style=social)

whatsmeow 是一个用于 WhatsApp Web 多设备 API 的 Go 语言库。支持收发私聊与群组的文本及媒体消息、管理群组及接收变更事件、创建和使用邀请链接、发送和接收输入状态及送达/已读回执、读写应用状态（如联系人、置顶/静音）并处理解密失败的重试。目前实验性功能支持发送状态消息，暂不支持广播列表消息和通话。
