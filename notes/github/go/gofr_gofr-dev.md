### [gofr-dev gofr](https://github.com/gofr-dev/gofr)  ![GitHub Repo stars](https://img.shields.io/github/stars/gofr-dev/gofr?style=social)

GoFr 是一个基于 Go 语言的微服务开发框架，旨在简化微服务构建，重点关注 Kubernetes 部署与开箱即用的可观测性。核心功能涵盖：简化的 API 语法、默认 REST 标准、配置管理、日志/追踪/指标监控、内置认证与中间件、gRPC 支持、带熔断器的 HTTP 服务、发布订阅、数据源健康检查、数据库迁移、定时任务、动态日志级别调整、Swagger 文档、抽象文件系统及 WebSocket 支持。
