### [atuinsh atuin](https://github.com/atuinsh/atuin)  ![GitHub Repo stars](https://img.shields.io/github/stars/atuinsh/atuin?style=social)

Atuin 是一款功能强大的魔法 Shell 历史记录工具。它用 SQLite 数据库替代原有 Shell 历史记录，并记录命令的退出码、执行时长、工作目录等额外上下文信息。支持跨终端、会话及机器间的完全加密同步，提供全屏历史搜索 UI 及统计功能。兼容 zsh、bash、fish 等多种 Shell，保留原有历史文件的同时增强历史管理能力。
