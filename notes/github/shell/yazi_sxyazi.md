### [sxyazi yazi](https://github.com/sxyazi/yazi)  ![GitHub Repo stars](https://img.shields.io/github/stars/sxyazi/yazi?style=social)

Yazi 是一款基于 Rust 语言编写的异步终端文件管理器。它利用非阻塞异步 I/O 和多线程实现高速操作，支持实时任务调度与取消。主要功能包括：内置多种终端的图片预览与代码高亮、Lua 并发插件系统、命令行包管理器、虚拟文件系统以及 Git 等工具集成。界面提供 Vim 风格操作、多标签页、批量重命名、可视模式及鼠标支持。项目目前为公共测试版，可作为日常驱动使用。
