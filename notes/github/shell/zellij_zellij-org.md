### [zellij-org zellij](https://github.com/zellij-org/zellij)  ![GitHub Repo stars](https://img.shields.io/github/stars/zellij-org/zellij?style=social)

Zellij 是一款面向开发者、运维人员及终端爱好者的终端工作区工具（终端复用器）。其设计理念是在保持简单性的同时提供强大功能，具备优秀的开箱即用体验。核心功能包括深度自定义、布局自动化、真多人协作、独特的浮动与堆叠分屏界面、支持任意编译语言的 WebAssembly 插件系统，以及内置 Web 客户端（无需本地终端）。
