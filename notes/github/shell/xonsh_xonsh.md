### [xonsh xonsh](https://github.com/xonsh/xonsh)  ![GitHub Repo stars](https://img.shields.io/github/stars/xonsh/xonsh?style=social)

Xonsh 是一个由 Python 驱动的跨平台功能型 Shell，作为 Python 3 的超集实现了 Shell 命令与 Python 语言的无缝融合。

主要功能包括：
1. **混合执行**：既可作为 Shell 使用（支持管道、变量、重定向），又支持直接运行 Python 代码。
2. **双向交互**：在 Python 中可直接调用系统命令，在 Shell 中可直接调用 Python 对象。
3. **插件扩展**：提供 xontribs 扩展系统，支持集成 Jupyter、AI 工具、包管理器及终端美化。
4. **跨平台部署**：支持 Docker、AppImage、Conda 等多种方式，兼容 Windows、Linux 和 macOS。
5. **AI 友好**：便于与 LLM 及数据分析工具结合，支持 AI 辅助工作流。
