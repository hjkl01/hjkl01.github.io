### [HelixDB helix-db](https://github.com/HelixDB/helix-db)  ![GitHub Repo stars](https://img.shields.io/github/stars/HelixDB/helix-db?style=social)

HelixDB 是一个基于 Rust 开发的开源图向量数据库，旨在为 AI 应用（Agent、RAG）提供统一后端平台，整合应用、向量与图数据存储。它主要采用图 + 向量数据模型，并兼容 KV、文档及关系型数据。核心特性包括内置 MCP 工具、自动文本向量化、RAG 搜索与图遍历、默认安全、基于 LMDB 的超低延迟及类型安全的 HelixQL 查询语言。项目提供 CLI 部署工具与 SDK，采用 AGPL 协议开源。
