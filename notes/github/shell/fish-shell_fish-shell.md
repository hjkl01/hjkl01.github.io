### [fish-shell fish-shell](https://github.com/fish-shell/fish-shell)  ![GitHub Repo stars](https://img.shields.io/github/stars/fish-shell/fish-shell?style=social)

fish 是一款智能且用户友好的交互式命令行 Shell，适用于 macOS、Linux 和 Windows 平台。它提供无需配置的语法高亮、输入时自动建议及高级选项卡补全功能。项目兼容 Bash 或 Zsh 的基本使用习惯，内置详细文档及配置工具，旨在提供开箱即用的友好 Shell 体验。
