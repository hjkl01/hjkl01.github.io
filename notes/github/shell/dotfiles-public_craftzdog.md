### [craftzdog dotfiles-public](https://github.com/craftzdog/dotfiles-public)  ![GitHub Repo stars](https://img.shields.io/github/stars/craftzdog/dotfiles-public?style=social)

该项目为开发者 dotfiles 配置集合，旨在提供跨平台开发环境配置，主要包含：

1. **Neovim 配置**：基于 LazyVim，需 Neovim 0.9.0+ (LuaJIT)，适配多种真彩终端，集成插件生态。
2. **工具配置**：包含 tmux 和 Git 配置文件。
3. **Shell 配置**：
   - macOS 和 Linux：Fish shell，含主题、插件管理及效率工具。
   - Windows：PowerShell，含提示符主题、终端图标及命令补全。

附带配置使用及工作流的视频教程与文章链接，使用前请自行评估风险。
