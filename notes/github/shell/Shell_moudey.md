### [moudey Shell](https://github.com/moudey/Shell)  ![GitHub Repo stars](https://img.shields.io/github/stars/moudey/Shell?style=social)

Shell 是一款功能强大的 Windows 资源管理器右键菜单管理工具。它允许用户自定义集成菜单项，创建访问网页、文件、文件夹及应用程序的快捷命令，同时支持修改或删除系统或第三方软件添加的菜单项。该软件具备轻量便携、界面高度可定制、支持复杂嵌套菜单与多列显示、表达式语法及丰富图标样式等特点，资源占用极低，兼容 Windows 7/8/10/11 系统。
