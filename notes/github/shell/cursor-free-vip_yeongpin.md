### [yeongpin cursor-free-vip](https://github.com/yeongpin/cursor-free-vip)

**项目核心内容总结：**  
Cursor Free VIP 是一款用于学习和研究的工具，支持重置 Cursor 应用的配置，适用于 Windows、macOS 和 Linux 系统，提供多语言（中/英/越）支持。  

**功能特点：**  
- 支持多系统运行，兼容主流操作系统；  
- 可自定义配置（如浏览器路径、等待时间、邮件验证等）；  
- 提供安装脚本，支持 Linux/macOS（curl 安装）和 Windows（PowerShell 安装）；  
- 支持通过 TempMailPlus 获取验证码，适配多种邮件服务。  

**使用方法：**  
1. 以管理员权限运行脚本；  
2. 安装前确保 Cursor 已关闭；  
3. 根据系统选择安装命令（如 Linux 使用 `curl` 安装脚本，Windows 使用 PowerShell 命令）。  

**注意事项：**  
- 仅限学习和研究，不生成虚假邮件或 OAuth 访问；  
- 遵守软件使用条款，避免使用临时邮箱导致账号被封；  
- 配置文件路径需根据系统手动调整（如存储路径、SQLite 路径等）。
