### [ghostty-org ghostty](https://github.com/ghostty-org/ghostty)  ![GitHub Repo stars](https://img.shields.io/github/stars/ghostty-org/ghostty?style=social)

Ghostty 是一款快速、功能丰富且原生的跨平台终端模拟器。它无需在速度、功能或原生 UI 体验之间做妥协，兼容所有现有终端标准与软件，可直接替代现有终端。项目支持现代可选特性以增强 CLI 工具交互性，提供多窗口、标签页和分屏功能，并针对渲染和 IO 性能进行深度优化。此外，项目提供 C 语言兼容的 `libghostty` 库供第三方嵌入使用，并内置本地崩溃报告功能。
