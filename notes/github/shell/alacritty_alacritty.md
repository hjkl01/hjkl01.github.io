### [alacritty alacritty](https://github.com/alacritty/alacritty)  ![GitHub Repo stars](https://img.shields.io/github/stars/alacritty/alacritty?style=social)

Alacritty 是一款快速、跨平台的 OpenGL 终端模拟器，支持 BSD、Linux、macOS 和 Windows 系统。它采用 GPU 加速渲染，提供高性能体验、合理默认设置及基于 TOML 的广泛配置选项。软件目前处于 Beta 阶段，不包含标签页或分屏功能（需配合窗口管理器或终端复用器使用）。
