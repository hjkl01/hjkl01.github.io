### [junegunn fzf](https://github.com/junegunn/fzf)  ![GitHub Repo stars](https://img.shields.io/github/stars/junegunn/fzf?style=social)

fzf 是一个通用的命令行模糊查找工具，通过模糊匹配算法快速过滤和选择文本列表（如文件、命令历史、进程、主机名等）。

主要功能：
1. **核心搜索**：支持输入省略字符即可精准匹配，适用于文件、命令历史、进程、Git 提交等多种数据列表。
2. **Shell 集成**：为 Bash、Zsh、Fish 提供键绑定和模糊补全功能，实现文件/目录快速选择、命令历史检索及路径补全。
3. **编辑器集成**：提供 Vim 和 Neovim 插件支持。
4. **增强特性**：具备预览窗口（支持文本高亮、图片显示）、动态刷新候选列表、外部程序执行及自定义界面样式。
5. **性能与平台**：单二进制文件分发，处理速度快，跨平台支持 Linux、macOS 及 Windows。
