### [fish2018 pansou](https://github.com/fish2018/pansou)  ![GitHub Repo stars](https://img.shields.io/github/stars/fish2018/pansou?style=social)

PanSou 是一款高性能网盘资源搜索 API 服务。核心功能包括支持 Telegram 频道及自定义插件的并发异步搜索，具备智能排序、网盘类型自动分类及多级缓存机制。支持百度网盘、阿里云盘、夸克网盘等多种资源类型及磁力链接。提供 Docker 与源码部署方案，支持 MCP 协议集成至 AI 应用。API 接口涵盖搜索、JWT 认证管理及健康检查，支持关键词过滤与结果定制。
