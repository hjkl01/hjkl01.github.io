### [cli cli](https://github.com/cli/cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/cli/cli?style=social)

GitHub CLI (`gh`) 是一款命令行工具，将 GitHub 核心功能（如拉取请求、Issues 等）集成至终端，支持与 `git` 协同工作。它适用于 macOS、Windows 和 Linux 系统，兼容 GitHub.com 及 GitHub Enterprise 服务。作为独立工具（非 `git` 代理），它支持多种安装方式，兼容 GitHub Codespaces 和 Actions 环境，并提供构建验证功能以确保安全性。
