### [josean-dev dev-environment-files](https://github.com/josean-dev/dev-environment-files)  ![GitHub Repo stars](https://img.shields.io/github/stars/josean-dev/dev-environment-files?style=social)

该项目提供一套 macOS 开发环境配置文件，旨在搭建高效编程工作流。核心功能涵盖：
1. **终端配置**：WezTerm 终端、Zsh 设置及 fzf、fd、bat 等 CLI 工具。
2. **窗口管理**：Yabai 和 Aerospace 平铺式窗口管理器，配合 skhd 快捷键。
3. **界面定制**：基于 Sketchybar 的自定义菜单栏。
4. **会话管理**：Tmux 配置，实现与 Neovim 协同操作。
5. **编辑器配置**：基于 lazy.nvim 的 Neovim，集成插件管理、Telescope 查找、LSP 补全、Treesitter 高亮及 Git 工具。
