### [LuaLS lua-language-server](https://github.com/LuaLS/lua-language-server)  ![GitHub Repo stars](https://img.shields.io/github/stars/LuaLS/lua-language-server?style=social)

lua-language-server 是一款 Lua 语言服务器，基于语言服务器协议（LSP）支持 Visual Studio Code、NeoVim 等多种编辑器。它涵盖 Lua 5.1 至 5.5 及 LuaJIT 版本支持，提供代码自动补全、动态类型检查、语法诊断、跳转定义、元素重命名、代码格式化及文档生成等功能，旨在提升 Lua 开发效率与体验。
