### [nvim-dap-ui](https://github.com/rcarriga/nvim-dap-ui) ![GitHub Repo stars](https://img.shields.io/github/stars/rcarriga/nvim-dap-ui?style=social)

nvim-dap-ui 是 Neovim 的 nvim-dap 调试界面插件，提供开箱即用的调试 UI。它以“元素”为单位组织窗口，可组成侧边栏/托盘布局或浮动窗，支持打开、关闭、切换与自动随调试事件显示。包含作用域变量、线程与堆栈、监视表达式、断点列表、REPL、控制台等元素，支持编辑/展开/跳转/切换等操作。还支持浮动显示元素与悬浮评估表达式。
