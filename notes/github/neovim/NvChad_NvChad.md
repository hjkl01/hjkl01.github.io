### [NvChad NvChad](https://github.com/NvChad/NvChad)  ![GitHub Repo stars](https://img.shields.io/github/stars/NvChad/NvChad?style=social)

NvChad 是一个基于 Lua 的 Neovim 配置文件，旨在提供极速启动（约 0.02-0.07 秒）和美观的 UI 界面。项目采用 93% 的延迟加载策略以优化性能，内置文件导航、模糊搜索、LSP、自动补全及语法高亮等核心功能插件。提供约 56 种主题及自定义状态栏、标签栏等 UI 组件，建议配合 Starter Config 作为基础配置进行二次开发。
