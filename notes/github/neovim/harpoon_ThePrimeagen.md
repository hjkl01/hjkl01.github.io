### [ThePrimeagen harpoon](https://github.com/ThePrimeagen/harpoon)  ![GitHub Repo stars](https://img.shields.io/github/stars/ThePrimeagen/harpoon?style=social)

Harpoon 是一款适用于 Neovim 0.5.0+ 的插件，旨在通过最少按键高效导航项目文件及终端。

核心功能：
1. **文件导航**：支持标记常用文件，通过快捷键或快速菜单快速跳转、排序和管理标记，支持以分割窗口或新标签页打开。
2. **终端管理**：支持创建和切换多个持久化终端，可导航至指定索引终端，支持向终端发送命令或执行预设命令。
3. **集成支持**：原生支持 Tmux（含 Pane 控制），集成 Telescope 扩展浏览标记，支持在 Tabline 显示标记状态。
4. **配置自定义**：支持设置保存策略、排除文件类型、按 Git 分支区分标记、菜单宽度及 UI 高亮样式等。
