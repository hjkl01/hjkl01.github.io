### [mfussenegger nvim-dap](https://github.com/mfussenegger/nvim-dap)  ![GitHub Repo stars](https://img.shields.io/github/stars/mfussenegger/nvim-dap?style=social)

`nvim-dap` 是 Neovim 的调试适配器协议（DAP）客户端实现。它支持启动或附加应用程序进行调试，提供断点管理、代码单步执行、程序状态检查（含 REPL 及 UI 扩展）及表达式求值功能。该插件支持多种语言（需配置对应调试适配器），具备高度可扩展性，可作为库供其他插件使用。仅支持 Neovim。
