### [CopilotC-Nvim CopilotChat.nvim](https://github.com/CopilotC-Nvim/CopilotChat.nvim)

**项目核心内容总结：**

**功能**：CopilotChat.nvim 是一个基于 AI 的代码补全工具，整合 GitHub Copilot 和 Kite 的功能，为 Neovim 提供智能代码建议，支持多种编程语言。

**使用方法**：  
1. 需要 Neovim 0.9+ 及 Lua 环境；  
2. 通过插件管理器（如 packer）安装，配置 Lua 脚本启用插件；  
3. 安装完成后，使用默认快捷键（如 `Tab` 或 `Enter`）触发补全功能。

**主要特性**：  
- AI 驱动的智能代码补全，支持 Python、JavaScript、TypeScript 等语言；  
- 可自定义快捷键和补全触发规则；  
- 优化性能，减少对编辑器的资源占用；  
- 兼容主流 Neovim 主题和插件生态。
