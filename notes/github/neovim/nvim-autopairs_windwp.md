### [windwp nvim-autopairs](https://github.com/windwp/nvim-autopairs)  ![GitHub Repo stars](https://img.shields.io/github/stars/windwp/nvim-autopairs?style=social)

nvim-autopairs 是一款适用于 Neovim 0.7+ 的强力自动字符配对插件。核心功能包括：支持多字符类型（括号、引号等）自动配对；可基于文件类型、正则表达式及 Tree-sitter 节点状态自定义配对规则与条件；兼容 nvim-cmp、coq_nvim 等补全插件以确保选中后正确配对；提供 FastWrap 功能实现文本快速包裹；集成自动标签支持（HTML/TSX）；提供动态 API 用于启用、禁用及规则管理。
