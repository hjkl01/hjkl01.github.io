### [MagicDuck grug-far.nvim](https://github.com/MagicDuck/grug-far.nvim)  ![GitHub Repo stars](https://img.shields.io/github/stars/MagicDuck/grug-far.nvim?style=social)

grug-far.nvim 是一个用于 Neovim 的强大查找和替换插件，支持使用 ripgrep (rg) 和 ast-grep 引擎进行高效搜索与替换操作。其核心功能包括：

- 使用 rg 或 ast-grep 的完整功能进行搜索和替换。
- 自动搜索或在插入模式退出时手动搜索。
- 将搜索结果以快速修复列表（quickfix list）形式打开。
- 支持跳转到匹配文件/行/列，支持自定义快捷键。
- 支持在结果中直接编辑并同步回原文件。
- 支持自动/手动保存搜索历史并重新加载。
- 支持多行搜索与替换。
- 支持在可视选择范围内搜索和替换。
- 支持预览搜索结果。
- 支持使用 Lua 或 Vimscript 脚本进行动态替换。
- 支持语法高亮和结果折叠。
- 支持将结果打开为浮动窗口或快速修复列表。
- 支持切换搜索引擎（如 ripgrep、astgrep、astgrep-rules）。
- 支持调试查询和显示完整命令行。
- 提供丰富的快捷键和自定义配置选项。
- 支持与 NeoTree、Oil、MiniFiles 等文件管理插件集成。
