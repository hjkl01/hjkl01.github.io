### [windwp nvim-ts-autotag](https://github.com/windwp/nvim-ts-autotag)  ![GitHub Repo stars](https://img.shields.io/github/stars/windwp/nvim-ts-autotag?style=social)

nvim-ts-autotag 是一个基于 Treesitter 的 Neovim 插件，用于自动闭合和重命名 HTML 标签。支持 HTML、Vue、JavaScript、TypeScript、XML 等多种语言。需 Neovim 0.9.5+ 及对应 Treesitter 解析器。提供自动闭合、重命名和跟随斜杠闭合配置，支持按文件类型覆盖设置、添加别名或扩展配置。
