### [jdhao nvim-config](https://github.com/jdhao/nvim-config)  ![GitHub Repo stars](https://img.shields.io/github/stars/jdhao/nvim-config?style=social)

这是一个跨平台（Linux、macOS、Windows）的 Neovim 个人配置项目，提供一套完整且文档丰富的编辑器方案。

核心功能包括：
- **插件与补全**：使用 Lazy.nvim 管理插件，支持 nvim-cmp、Ultisnips 代码补全及 LSP 服务。
- **代码效率**：集成 Git 操作、项目级模糊搜索、快速跳转、代码高亮（Treesitter）及 Markdown/LaTeX 预览。
- **界面增强**：包含自定义状态栏、文件树、按键提示、搜索索引高亮及动画通知。
- **其他工具**：支持终端 GUI 模式、异步代码执行、撤销管理及丰富的自定义快捷键与命令。

注意：配置仅适配最新版 Neovim，建议按需复制模块进行定制。
