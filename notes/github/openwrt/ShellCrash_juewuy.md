### [juewuy ShellCrash](https://github.com/juewuy/ShellCrash)  ![GitHub Repo stars](https://img.shields.io/github/stars/juewuy/ShellCrash?style=social)

ShellCrash 是一款在 Shell 环境中便捷部署和管理 mihomo 及 sing-box 内核的脚本工具。支持多内核切换、在线订阅配置管理、自动化定时任务、本地 Web 仪表盘、多种流量转发模式（如路由模式和本地模式）及一键在线维护。兼容 OpenWrt 路由器、Linux 服务器、第三方固件及 Docker 环境，通过命令行启动管理界面。
