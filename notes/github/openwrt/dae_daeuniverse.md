### [daeuniverse dae](https://github.com/daeuniverse/dae)  ![GitHub Repo stars](https://img.shields.io/github/stars/daeuniverse/dae?style=social)

dae 是一款基于 Linux 内核 eBPF 技术的高性能透明代理工具，作为 v2rayA 的继任者，通过内核级流量分流实现流量绕过应用层转发（Real Direct），极大降低性能损耗和资源消耗。其主要功能包括：支持按进程名（本地）或 MAC 地址（局域网）分流；支持基于延迟策略的自动节点切换（涵盖 TCP/UDP/IPv4/IPv6）；具备高级 DNS 解析能力；支持多种代理协议及 shadowsocks、trojan 等协议的 Full-cone NAT。
