### [Watfaq clash-rs](https://github.com/Watfaq/clash-rs)  ![GitHub Repo stars](https://img.shields.io/github/stars/Watfaq/clash-rs?style=social)

ClashRS 是一款基于规则的自定义网络代理软件。主要功能包括：支持基于 IP、域名及地理位置的灵活流量路由；提供本地防欺骗 DNS 服务（支持 UDP/TCP/DoH/DoT）；可作为 HTTP/Socks5 代理或 utun 设备网关运行；支持 Hysteria2、Shadowsocks、Trojan、Vmess 等多种出站协议及传输层；具备动态远程规则与代理加载能力，并集成 Jaeger 追踪。项目支持 Linux、macOS、Windows、iOS 平台及 Docker 部署。
