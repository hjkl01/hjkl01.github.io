### [ehang-io nps](https://github.com/ehang-io/nps)  ![GitHub Repo stars](https://img.shields.io/github/stars/ehang-io/nps?style=social)

NPS 是一款轻量级、高性能的内网穿透代理服务器，配备功能强大的 Web 管理终端。支持 TCP、UDP、HTTP(S)、SOCKS5、P2P 及 HTTP 代理等多种协议，兼容 Linux、Windows、MacOS 等系统，支持安装为系统服务。提供 Web 界面进行简单配置，支持 HTTPS 转换、多证书管理及流量带宽实时监控。具备缓存、压缩、加密、限流、端口复用等扩展功能，以及域名解析、URL 路由、自定义头、404 页配置等高级特性，同时支持服务端多用户注册与管理。
