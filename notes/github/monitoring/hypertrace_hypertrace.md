### [hypertrace hypertrace](https://github.com/hypertrace/hypertrace)  ![GitHub Repo stars](https://img.shields.io/github/stars/hypertrace/hypertrace?style=social)

Hypertrace 是一个云原生分布式追踪与可观测性平台，旨在提供分布式系统的可见性。核心功能包括将追踪数据转化为洞察以执行根本原因分析，识别性能瓶颈（如慢 API 调用、数据库查询）及服务过载，监控微服务依赖及应用程序状态，评估新版本发布影响及诊断请求失败原因。平台支持 Zipkin/Jaeger 集成，提供 Docker Compose 快速启动和 Kubernetes 生产部署方案，采用开源核心模型。
