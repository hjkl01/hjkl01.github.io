### [henrygd beszel](https://github.com/henrygd/beszel)  ![GitHub Repo stars](https://img.shields.io/github/stars/henrygd/beszel?style=social)

Beszel 是一款轻量级服务器监控平台，提供友好的 Web 界面，配置简单且开箱即用，支持自动备份、多用户、OAuth 认证及 API 访问。

主要功能包括：
- **资源监控**：追踪主机及 Docker/Podman 容器的 CPU、内存、磁盘、网络、负载、温度、GPU、电池及 S.M.A.R.T. 健康状态历史数据。
- **告警系统**：支持配置 CPU、内存、磁盘、带宽、温度、负载和状态警报。
- **架构设计**：由 Hub（Web 管理端）和 Agent（监控端）组成，用户可管理个人系统，管理员支持系统共享。
