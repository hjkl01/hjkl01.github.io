### [upptime upptime](https://github.com/upptime/upptime)  ![GitHub Repo stars](https://img.shields.io/github/stars/upptime/upptime?style=social)

Upptime 是一款完全由 GitHub Actions、Issues 和 Pages 驱动的开源运维监控及状态页面工具。它通过 GitHub Actions 定期检测端点在线状态并记录响应时间数据至 Git 仓库，实现长期趋势追踪。当检测到服务故障时，自动创建 GitHub Issue 进行管理，服务恢复后自动关闭。项目利用 GitHub Pages 托管状态页面，展示实时状态、响应时间及事故历史。其配置灵活，数据存储在 Git 中，无需外部服务器或订阅，对现有 GitHub 用户完全免费。
