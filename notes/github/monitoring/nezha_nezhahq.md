### [nezha](https://github.com/nezhahq/nezha) ![GitHub Repo stars](https://img.shields.io/github/stars/nezhahq/nezha?style=social)

哪吒监控（Nezha Monitoring）是一款可自托管、轻量级的服务器与网站监控及运维工具。它支持监控系统状态，以及 HTTP、TCP 和 Ping 检测，其中 HTTP 监控还可检查 SSL 证书变更、即将过期和已过期状态。

项目还提供告警推送、定时任务执行和 Web 终端功能，并配有用户前台与管理后台，支持多语言文档与社区协作翻译。
