### [highlight highlight](https://github.com/highlight/highlight)  ![GitHub Repo stars](https://img.shields.io/github/stars/highlight/highlight?style=social)

highlight.io 是一款开源的全栈监控平台，旨在帮助开发者发现、解决和预防 Web 应用问题，使其能自信地交付产品。

核心功能包括：
1. **会话回放**：提供高保真的 DOM 交互回放、网络请求查看及控制台日志分析，帮助理解 Bug 发生原因。
2. **错误监控**：支持自定义错误分组与警报规则，并能关联相关会话与日志以定位问题。
3. **日志与追踪**：提供强大的日志搜索、警报设置及服务器端操作性能追踪，支持嵌入会话、错误和日志信息。
4. **多语言 SDK 支持**：提供丰富的 SDK 便于集成至各种环境。

平台支持免费托管服务、简易 Docker 自托管及企业级自托管等多种部署方式，致力于构建现代化、一体化且易于使用的监控解决方案。
