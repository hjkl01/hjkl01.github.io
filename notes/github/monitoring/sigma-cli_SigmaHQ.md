### [SigmaHQ sigma-cli](https://github.com/SigmaHQ/sigma-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/SigmaHQ/sigma-cli?style=social)

Sigma 命令行界面（sigma-cli）是一个基于 pySigma 库的命令行工具，用于管理、列出和转换 Sigma 规则为特定查询语言。它支持通过插件扩展后端和处理管道，允许用户将 Sigma 规则目录或文件转换为不同后端（如 Splunk）的查询语句，并支持指定输出格式、处理管道及后端选项。项目还提供机制供开发者集成新的后端和处理管道。
