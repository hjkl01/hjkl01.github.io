### [mher flower](https://github.com/mher/flower)  ![GitHub Repo stars](https://img.shields.io/github/stars/mher/flower?style=social)

Flower 是一个开源 Web 应用程序，用于监控和管理 Celery 集群，提供 Celery 工人和任务的实时状态信息。主要功能包括：实时监控任务进度、详情和历史；远程管理工人（状态、重启、队列、池大小、任务终止等）；监控 Broker 队列统计；支持多种认证方式（Basic Auth/OAuth）；集成 Prometheus；提供 REST API 接口。
