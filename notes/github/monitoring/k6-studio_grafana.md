### [grafana k6-studio](https://github.com/grafana/k6-studio)  ![GitHub Repo stars](https://img.shields.io/github/stars/grafana/k6-studio?style=social)

Grafana k6 Studio 是一款适用于 Mac、Windows 和 Linux 的桌面应用程序，旨在通过交互式界面帮助用户生成 k6 性能测试脚本。项目包含三个核心组件：Recorder 用于在浏览器中记录用户流并生成 HAR 文件；Generator 基于 HAR 文件自动生成 k6 脚本，无需编写 JavaScript，支持通过规则定制脚本；Validator 用于执行测试运行以验证脚本、请求响应及日志是否符合预期。
