### [giampaolo psutil](https://github.com/giampaolo/psutil)  ![GitHub Repo stars](https://img.shields.io/github/stars/giampaolo/psutil?style=social)

psutil 是一个跨平台的 Python 库，用于获取运行进程和系统利用率（CPU、内存、磁盘、网络、传感器）的详细信息。它主要用于系统监控、性能分析、资源限制及进程管理。该库支持 Linux、Windows、macOS、FreeBSD 等多个操作系统平台，实现了 ps、top、netstat 等经典 UNIX 命令行工具的功能。
