### [netdata netdata](https://github.com/netdata/netdata)  ![GitHub Repo stars](https://img.shields.io/github/stars/netdata/netdata?style=social)

Netdata 是一款开源的实时基础设施监控平台，提供秒级指标采集、存储与可视化。支持 Linux、macOS、FreeBSD 及 Windows 系统，具备零配置自动发现能力，可监控系统资源、存储、网络、硬件、容器、虚拟机、日志及应用。采用边缘端机器学习模型自动检测异常，资源占用极低，支持分层存储与水平扩展（通过 Parent 节点或 Netdata Cloud）。数据默认本地存储，保障隐私安全，可选 Cloud 实现远程访问与企业级管理。
