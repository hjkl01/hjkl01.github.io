### [louislam uptime-kuma](https://github.com/louislam/uptime-kuma)  ![GitHub Repo stars](https://img.shields.io/github/stars/louislam/uptime-kuma?style=social)

Uptime Kuma 是一款易于使用的自托管监控工具。它支持 HTTP(s)、TCP、Ping、DNS、WebSocket、Steam 游戏服务器及 Docker 容器等多种类型的监控。该工具拥有美观快速的响应式界面，支持 20 秒监控间隔、多语言、状态页面管理及域名映射。同时具备丰富的通知功能（支持 Telegram、Discord、邮件等 90 多种服务）、证书信息查看、代理支持及双因素认证。
