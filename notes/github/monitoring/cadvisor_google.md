### [google cadvisor](https://github.com/google/cadvisor)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/cadvisor?style=social)

cAdvisor 是一个容器资源监控守护进程，用于收集、聚合、处理和导出运行容器的资源使用及性能数据。它原生支持 Docker 及其他容器类型，记录资源隔离参数、历史用量、直方图和网络统计等信息。项目支持 Docker 镜像、独立部署及 Kubernetes 等多种运行方式，内置 Web 界面，提供 REST API 和多种存储插件导出数据，旨在帮助用户理解并优化容器资源使用情况。
