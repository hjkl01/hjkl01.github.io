### [dgtlmoon changedetection.io](https://github.com/dgtlmoon/changedetection.io)

**项目核心内容总结：**  
ChangeDetection.io 是一个用于监控网页和 JSON API 变化的工具，支持通过 XPath、JSONPath、JQ 等过滤器提取和解析数据，可检测内容变化并触发通知（如邮件、Slack、Discord 等）。  

**主要功能：**  
- 支持多种数据提取方式（XPath、CSS、JSONPath、JQ），可解析 HTML 中嵌入的 JSON 数据。  
- 提供丰富的通知方式，支持自定义 API、邮件、企业通讯工具等。  
- 支持通过 Docker、Python Pip 安装，可配置代理（如 Bright Data、Oxylabs）。  
- 支持导入 Excel 文件管理监控列表，支持 REST API 管理监控任务。  

**使用方法：**  
- 安装方式：Docker 命令、docker-compose 或 Python Pip 安装。  
- 配置监控规则，设置过滤器和通知方式，通过界面或 API 管理任务。
