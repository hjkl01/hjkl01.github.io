### [komari](https://github.com/komari-monitor/komari) ![GitHub Repo stars](https://img.shields.io/github/stars/komari-monitor/komari?style=social)

Komari：轻量级自建服务器监控工具。

核心功能：
- 数据采集：通过轻量代理收集服务器性能数据。
- 状态展示：提供 Web 界面直观监控面板。
- 资源占用：极低能耗，适配各规模服务器。
- 数据隐私：完全自建托管，保证数据绝对控制权。
- 部署方式：支持一键脚本、Docker、二进制及云端快速部署。
- 自定义：支持前端主题开发与代理事件接口扩展。

安全警告：Komari 仅限用于您拥有或获授权管理的系统。严禁将其武器化、未经同意部署，或用于未授权访问、持久化、命令执行等滥用行为。在 Windows 环境下启用远程控制功能时，客户端将在每次用户登录时强制弹出通知提醒。
