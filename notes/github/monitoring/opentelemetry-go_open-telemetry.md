### [open-telemetry opentelemetry-go](https://github.com/open-telemetry/opentelemetry-go)  ![GitHub Repo stars](https://img.shields.io/github/stars/open-telemetry/opentelemetry-go?style=social)

OpenTelemetry-Go 是 OpenTelemetry 的 Go 语言实现，提供 API 用于测量应用性能与行为并将数据发送至可观测性平台。支持分布式追踪（稳定）、指标（稳定）和日志（Beta）信号。用户需对应用进行插桩，并配置 OTLP、Prometheus、stdout 或 Zipkin 等导出器发送遥测数据。兼容 Go 1.25/1.26 及主流操作系统环境。
