### [pawelmalak flame](https://github.com/pawelmalak/flame)  ![GitHub Repo stars](https://img.shields.io/github/stars/pawelmalak/flame?style=social)

Flame 是一款自托管服务器起始页应用，设计简洁易用，无需编辑文件即可搭建应用中心。

核心功能如下：
- **应用与书签管理**：内置 GUI 编辑器，支持直接创建、更新、删除应用和书签，并可置顶常用项。
- **搜索功能**：集成搜索栏，支持本地过滤、11 种预设搜索引擎及自定义引擎。
- **认证系统**：提供密码认证以保护设置、应用和书签。
- **界面定制**：支持自定义 CSS、15 种内置主题及自定义主题构建器。
- **天气组件**：显示当前温度、云覆盖及动画天气状态。
- **容器集成**：支持与 Docker 和 Kubernetes 集成，自动根据容器标签或 Ingress 注解发现并添加应用。
- **书签导入**：支持导入 HTML 书签文件（实验性功能）。
