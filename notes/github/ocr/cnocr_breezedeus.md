### [breezedeus cnocr](https://github.com/breezedeus/cnocr)  ![GitHub Repo stars](https://img.shields.io/github/stars/breezedeus/cnocr?style=social)

CnOCR 是一个基于 Python 3 的文字识别（OCR）工具包，支持简体中文、繁体中文、英文、数字及竖排文字。内置 20 余个预训练模型，覆盖场景、文档、纯数字及通用等多种应用场景，集成 CnSTD 实现文字检测与定位。支持单行文字快速识别、模型自定义训练及 FastAPI HTTP 服务，兼容 PyTorch 与 ONNX 格式，支持 CPU 和 GPU 环境部署。
