### [hiroi-sora Umi-OCR](https://github.com/hiroi-sora/Umi-OCR)  ![GitHub Repo stars](https://img.shields.io/github/stars/hiroi-sora/Umi-OCR?style=social)

Umi-OCR 是一款免费、开源、支持批量的离线 OCR 文字识别工具，适用于 Windows 7 x64 及 Linux x64 系统，无需安装与网络环境。

主要功能：
1. **截图识别**：支持快捷键截图，具备文本后处理与排版解析功能，支持公式识别。
2. **批量识别**：支持多张图片批量 OCR，可设置忽略区域排除干扰文字，支持任务完成后自动关机。
3. **文档识别**：支持 PDF 等格式，可对扫描件 OCR 并生成双层可搜索 PDF。
4. **二维码**：支持读取及生成二维码/条形码，兼容 19 种协议。
5. **接口调用**：支持命令行及 HTTP 接口，便于集成。

软件内置 PaddleOCR 和 RapidOCR 引擎，支持多语言界面、主题切换及全局设置。
