### [feishu-cli](https://github.com/riba2534/feishu-cli) ![GitHub Repo stars](https://img.shields.io/github/stars/riba2534/feishu-cli?style=social)

feishu-cli 是飞书开放平台命令行工具，核心实现 Markdown 与飞书文档双向无损转换，并支持 Mermaid/PlantUML 图表自动转可编辑画板、智能表格处理与大文档并发导入。它覆盖文档、知识库、表格、消息、群聊、文件、权限、日历、任务、评论、搜索等全平台操作，并提供 11 个 AI 技能文件让 AI Agent 直接操控飞书。
