### [leo9960 wechat-app-unpack](https://github.com/leo9960/wechat-app-unpack)  ![GitHub Repo stars](https://img.shields.io/github/stars/leo9960/wechat-app-unpack?style=social)

本项目提供微信小程序解包工具及源码还原方案。
1. 提供多种语言版本的解包脚本（Python、PHP、JAVA、NodeJS 等）。
2. 解析解包后的文件结构，包含样式文件、配置文件（app-config.json）、JS 源码（app-service.js）及 WXML 文件（page-frame.html）。
3. 提供 WXML 分析工具，支持还原混淆后的 page-frame.html 源码。
