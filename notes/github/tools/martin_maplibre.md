### [maplibre martin](https://github.com/maplibre/martin)  ![GitHub Repo stars](https://img.shields.io/github/stars/maplibre/martin?style=social)

- 基于 Rust 的高性能矢量瓦片服务器与工具集，面向大数据与高并发
- 即时从 PostGIS 自动发现表/函数并生成矢量瓦片
- 从 PMTiles（本地或 HTTP）与 MBTiles 直接提供瓦片
- 支持将多个瓦片源合并为一个数据源
- 可托管样式，并按需生成雪碧图与字体字形
- 批量生成：从任意受支持数据源导出为 MBTiles（martin-cp）
- MBTiles 工具：检查、复制、校验、比较与应用增量差异
