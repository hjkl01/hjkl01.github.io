### [xpzouying xiaohongshu-mcp](https://github.com/xpzouying/xiaohongshu-mcp)  ![GitHub Repo stars](https://img.shields.io/github/stars/xpzouying/xiaohongshu-mcp?style=social)

该项目是一个基于 Model Context Protocol (MCP) 的小红书自动化运营工具，支持通过 AI 客户端或命令行工具自动化操作小红书平台，实现内容发布、互动、搜索等功能。核心功能包括：

1. **登录与检查登录状态**  
   自动登录小红书并检查当前登录状态，支持保存 cookies。

2. **发布图文内容**  
   支持通过标题、描述和图片发布图文内容，支持本地路径或 HTTP 图片链接。

3. **发布视频内容**  
   支持本地视频文件的上传和发布，自动处理视频格式转换。

4. **搜索内容**  
   根据关键词搜索小红书内容，支持筛选和排序。

5. **获取推荐列表**  
   获取小红书首页的推荐内容列表。

6. **获取帖子详情**  
   获取帖子的详细信息，包括互动数据和评论，支持加载全部评论和子评论。

7. **发表评论**  
   支持自动发表评论到指定帖子。

8. **获取用户主页信息**  
   获取用户的基本信息、关注数、粉丝数及笔记内容。

9. **回复评论**  
   回复指定评论，支持通过评论 ID 或用户 ID 定位目标评论。

10. **点赞/取消点赞**  
    为帖子点赞或取消点赞，智能检测避免重复操作。

11. **收藏/取消收藏**  
    收藏或取消收藏帖子，智能检测避免重复操作。

12. **支持多种部署方式**  
    提供二进制文件、源码编译和 Docker 容器部署方式，支持 Windows、macOS、Linux 系统。

13. **集成多种 AI 客户端**  
    支持通过 MCP 协议接入多种 AI 客户端，如 Claude Code、Cursor、VSCode、OpenClaw 等，实现自然语言操作小红书。

14. **浏览器插件版（x-mcp）**  
    提供零配置的浏览器插件版本，适合非技术用户使用。

15. **运营建议与风险提示**  
    提供小红书内容运营的建议，包括标题、正文、标签限制、内容类型选择等，并提示账号风险与合规要求。

该项目开源、可扩展，适用于自动化内容发布和运营，提升小红书账号的管理效率。
