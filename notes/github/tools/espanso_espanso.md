### [espanso espanso](https://github.com/espanso/espanso)  ![GitHub Repo stars](https://img.shields.io/github/stars/espanso/espanso?style=social)

Espanso 是一款基于 Rust 编写的跨平台文本扩展器，支持 Windows、macOS 和 Linux。它通过检测输入关键词，自动将其替换为文本、脚本、表情符号或图片。核心功能包括系统级代码片段、自定义脚本与 Shell 命令执行、日期与正则触发、表单支持及内置包管理。软件坚持隐私优先，100% 本地运行无追踪，支持应用特定配置，可集成于任意程序中，通过文件化配置提升打字效率。
