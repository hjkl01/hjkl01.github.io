### [danielmiessler Fabric](https://github.com/danielmiessler/Fabric)  ![GitHub Repo stars](https://img.shields.io/github/stars/danielmiessler/Fabric?style=social)

Fabric 是一个开源框架，旨在通过 AI 增强人类能力，解决 AI 工具整合难的痛点。它通过将 AI 提示词（称为“模式”）按任务分类组织，使用户能统一创建、收集和集成 AI 解决方案。

主要特性：
1. **多模型兼容**：支持 OpenAI、Anthropic、Google Gemini、Ollama 及 Azure 等数十种 AI 提供商。
2. **多端交互**：提供命令行（CLI）、REST API 服务器及 Web 界面，便于集成至工作流。
3. **模式库与策略**：内置摘要、分析等预置模式，支持自定义私有模式及提示词策略（如思维链）。
4. **辅助工具**：包含 PDF 转换、代码上下文生成、视频转录等实用命令。
5. **部署与配置**：支持 Docker、Go 安装、Shell 补全及全平台国际化。

项目致力于让用户便捷地将 AI 融入日常任务，提升效率与创造力。
