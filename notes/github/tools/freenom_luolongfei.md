### [luolongfei freenom](https://github.com/luolongfei/freenom)  ![GitHub Repo stars](https://img.shields.io/github/stars/luolongfei/freenom?style=social)

这是一个基于 PHP 的 Freenom 域名自动续期工具，核心功能如下：
1. **自动续期**：支持自动续费 Freenom 免费顶级域名，支持多账户管理。
2. **消息通知**：支持邮件（Gmail、QQ 等）、Telegram Bot、企业微信、Server 酱、Bark 等多种渠道，在续期成功、失败或出错时发送通知。
3. **部署支持**：支持 Docker、Docker Compose、Koyeb、Railway、云函数及源码等多种部署方式。
4. **开源协议**：采用 MIT 协议开源。
