### [gotenberg gotenberg](https://github.com/gotenberg/gotenberg)  ![GitHub Repo stars](https://img.shields.io/github/stars/gotenberg/gotenberg?style=social)

Gotenberg 是一个容器化的 API，用于简化 PDF 转换。它基于 Chromium 和 LibreOffice 引擎，提供 `multipart/form-data` 接口，使开发者无需管理复杂的依赖、浏览器版本或字体。只需发送文件或 URL，即可生成 PDF，支持转换本地文件、网页、注入自定义 CSS、合并 PDF 等功能。
