### [gildas-lormeau SingleFile](https://github.com/gildas-lormeau/SingleFile)  ![GitHub Repo stars](https://img.shields.io/github/stars/gildas-lormeau/SingleFile?style=social)

SingleFile 是一款支持多浏览器（Chrome、Firefox、Edge 等）及命令行工具的 Web 扩展，旨在将完整的网页保存为单个 HTML 文件。主要功能包括：支持保存当前标签页、选中内容或帧，以及批量保存多个标签页；允许在保存时高亮文本、添加笔记或移除内容；支持自动保存、自定义快捷键，并可上传至 Google Drive 或 GitHub，提供区块链存证。生成的文件经过优化压缩，无需扩展即可直接打开查看，同时支持用户脚本集成。
