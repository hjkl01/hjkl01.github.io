### [aquasecurity trivy](https://github.com/aquasecurity/trivy)  ![GitHub Repo stars](https://img.shields.io/github/stars/aquasecurity/trivy?style=social)

Trivy 是一个全面且通用的安全扫描工具，支持扫描容器镜像、文件系统、远程 Git 仓库、虚拟机镜像和 Kubernetes。它可以检测操作系统包及软件依赖（SBOM）、已知漏洞（CVE）、基础设施即代码（IaC）问题与配置错误、敏感信息、密钥以及软件许可证。Trivy 支持大多数流行的编程语言、操作系统和平台。
