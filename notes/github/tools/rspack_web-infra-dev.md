### [web-infra-dev rspack](https://github.com/web-infra-dev/rspack)  ![GitHub Repo stars](https://img.shields.io/github/stars/web-infra-dev/rspack?style=social)

Rspack 是一个由 Rust 编写的高性能 JavaScript 打包工具，具有极快的构建速度和热更新能力。它完全兼容 Webpack 生态系统，支持插件和加载器，可无缝替换 Webpack。核心特性包括快速启动、极速 HMR、模块联邦支持、内置生产优化策略以及框架无关性。Rspack 作为 Rstack 统一工具链的核心组件，与 Rsbuild、Rslib、Rspress 等工具共同提供高性能的 JavaScript 开发体验。
