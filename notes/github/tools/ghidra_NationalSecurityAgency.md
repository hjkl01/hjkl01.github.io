### [NationalSecurityAgency ghidra](https://github.com/NationalSecurityAgency/ghidra)  ![GitHub Repo stars](https://img.shields.io/github/stars/NationalSecurityAgency/ghidra?style=social)

Ghidra 是由美国国家安全局（NSA）开发的开源软件逆向工程框架，提供全套高级软件分析工具。它支持 Windows、macOS 和 Linux 平台，能够分析多种处理器指令集和可执行格式的编译代码。功能包括反汇编、汇编、反编译、图分析及脚本编写，支持用户交互式和自动化模式。用户可使用 Java 或 Python 开发自定义扩展和脚本，用于解决复杂逆向工程中的扩展与协作问题，适用于恶意代码分析和系统漏洞研究。
