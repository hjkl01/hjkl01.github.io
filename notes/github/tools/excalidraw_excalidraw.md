### [excalidraw excalidraw](https://github.com/excalidraw/excalidraw)  ![GitHub Repo stars](https://img.shields.io/github/stars/excalidraw/excalidraw?style=social)

Excalidraw 是一款开源的虚拟手绘风格白板工具，提供无限画布、多种绘图工具（形状、箭头、手绘等）、图像支持及形状库，支持深色模式与本地化。项目具备实时协作、端到端加密、本地自动保存及离线使用（PWA）功能，支持导出为 PNG、SVG 或 JSON 格式及分享链接。并提供 npm 包与 VSCode 扩展，便于集成到应用中。
