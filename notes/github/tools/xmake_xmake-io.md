### [xmake-io xmake](https://github.com/xmake-io/xmake)  ![GitHub Repo stars](https://img.shields.io/github/stars/xmake-io/xmake?style=social)

xmake 是一款基于 Lua 脚本的跨平台轻量级构建工具，兼具构建后端、工程生成器与包管理功能。

- **平台与语言**：支持 Windows、Linux、macOS、Android、iOS 等平台，支持 C/C++、Rust、Swift 等多种语言混合编译。
- **包管理**：内置包管理器，支持官方及第三方仓库依赖自动下载、集成与版本锁定。
- **性能特性**：支持并行编译、增量编译、分布式编译及本地/远程构建缓存加速。
- **扩展性**：提供简洁的 xmake.lua 配置，支持插件生成 IDE 项目，集成编辑器及 CI/CD 环境。
