### [cadence-workflow cadence-java-samples](https://github.com/cadence-workflow/cadence-java-samples)  ![GitHub Repo stars](https://img.shields.io/github/stars/cadence-workflow/cadence-java-samples?style=social)

该项目提供 Java Cadence 客户端和服务器的功能示例，核心内容包括：

1. **HelloWorld**：演示活动工作流、重试、异步调用、子工作流、异常处理、查询、信号、周期性任务、搜索属性及 Cron 功能。
2. **文件处理**：演示任务路由特性，实现文件下载、处理及上传的特定主机绑定执行。
3. **行程预订**：演示基于 Cadence 的 SAGA 模式实现。
4. **Spring Boot**：展示使用 Spring Boot 框架启动 Cadence 工作器服务。

此外，项目文档包含基于 Gradle 的构建、Docker 部署 Cadence 服务器、域注册及运行示例的完整指南。
