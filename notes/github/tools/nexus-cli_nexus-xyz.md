### [nexus-xyz nexus-cli](https://github.com/nexus-xyz/nexus-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/nexus-xyz/nexus-cli?style=social)

Nexus CLI 是一个高性能命令行工具，用于向 Nexus 网络贡献计算证明。核心功能包括：

1. **节点证明**：用户注册钱包和节点后，启动 CLI 自动生成网络验证证明。
2. **自适应难度**：根据节点性能自动调整任务难度，支持手动指定难度级别以适配不同硬件资源或优化收益。
3. **灵活部署**：支持预编译二进制、Docker 容器化部署及非交互式安装，并提供无头模式。
4. **配置管理**：自动管理用户凭证与节点 ID，支持凭证实时清除。
