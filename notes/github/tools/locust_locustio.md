### [locustio locust](https://github.com/locustio/locust)  ![GitHub Repo stars](https://img.shields.io/github/stars/locustio/locust?style=social)

Locust 是一款开源的性能与负载测试工具，支持 HTTP 及其他协议。它允许用户使用原生 Python 代码编写测试场景，摆脱 GUI 或特定语言的限制。基于事件驱动架构，Locust 支持分布式部署，可处理数十万高并发用户。该工具提供实时 Web 界面以监控吞吐量、响应时间和错误，并支持测试运行中调整负载。其架构高度可扩展，允许导入 Python 库或编写自定义客户端以适配各种系统测试。
