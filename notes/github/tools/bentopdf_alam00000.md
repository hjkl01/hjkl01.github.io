### [alam00000 bentopdf](https://github.com/alam00000/bentopdf)  ![GitHub Repo stars](https://img.shields.io/github/stars/alam00000/bentopdf?style=social)

BentoPDF 是一款隐私优先、支持自托管的客户端 PDF 工具包，所有文件处理均在浏览器本地完成，无需上传至服务器，确保数据隐私。

核心功能涵盖：
- **组织管理**：合并、拆分、排序、旋转、提取页面及 PDF 装订。
- **编辑修改**：表单创建与填写、水印、裁剪、OCR 识别、文本/表格提取、内容移除。
- **格式转换**：支持 PDF 与图片、Office 文档、Markdown、JSON 及电子书等多种格式互转。
- **安全优化**：压缩、加密/解密、数字签名、权限控制、元数据清理及线性化。
- **自动化**：提供可视化节点编辑器构建自定义处理工作流。

支持 Docker、Podman 及静态托管部署，具备离线和局域网部署能力，采用双许可模式。
