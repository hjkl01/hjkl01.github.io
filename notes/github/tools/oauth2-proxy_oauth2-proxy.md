OAuth2 Proxy 是一款灵活的开源工具，可用作独立反向代理或集成在现有基础设施中的中间件组件。它通过 OAuth2 或 OIDC 认证机制，为 Web 应用程序提供简单且安全的安全保护。项目支持 Google、Microsoft、GitHub 等多种认证提供商，能够提取用户详情（如用户名和群组），并将其作为 HTTP 头转发至上游应用。
