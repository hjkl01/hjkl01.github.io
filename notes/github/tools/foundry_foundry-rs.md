### [foundry-rs foundry](https://github.com/foundry-rs/foundry)  ![GitHub Repo stars](https://img.shields.io/github/stars/foundry-rs/foundry?style=social)

Foundry 是基于 Rust 编写的用于以太坊应用开发的极速、便携且模块化工具箱，包含以下核心功能：

1. Forge：构建、测试、模糊测试、调试和部署 Solidity 合约。
2. Cast：与 EVM 智能合约交互、发送交易及获取链数据。
3. Anvil：提供快速本地以太坊开发节点。
4. Chisel：快速、实用的 Solidity REPL 工具。
