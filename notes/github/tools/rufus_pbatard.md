### [pbatard rufus](https://github.com/pbatard/rufus)  ![GitHub Repo stars](https://img.shields.io/github/stars/pbatard/rufus?style=social)

Rufus 是一款可靠的 USB 格式化和制作可启动 USB 闪存驱动器的工具。支持将 USB 设备格式化为 FAT、NTFS、exFAT 等多种文件系统；能从 ISO 或磁盘镜像创建 DOS、BIOS 及 UEFI 启动盘（含 UEFI NTFS 支持）；支持制作绕过 TPM 和安全启动的 Windows 11 安装盘及 Windows To Go 驱动；可将现有驱动器转换为 VHD、VHDX、FFU 镜像或创建持久化 Linux 分区。此外，提供官方 Windows ISO 下载、校验和计算、坏块及假冒闪存盘检测功能，支持优化 Windows 安装设置及验证 UEFI 启动媒体。支持 38 种语言，无需安装且便携，兼容安全启动，遵循 GPLv3 开源协议。
