### [kohler gifsicle](https://github.com/kohler/gifsicle)  ![GitHub Repo stars](https://img.shields.io/github/stars/kohler/gifsicle?style=social)

Gifsicle 是一款 GIF 图像处理工具，主要功能包括：合并多个 GIF 生成动画、将动画分解为帧、修改帧内容、控制图像交错、添加透明度、设置延迟与循环、添加或删除注释、翻转旋转图像、优化文件体积及更改调色板。项目附带 Gifview 用于展示 GIF 图像和动画，以及 Gifdiff 用于检查两个 GIF 文件的视觉一致性。
