### [ankitpokhrel jira-cli](https://github.com/ankitpokhrel/jira-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/ankitpokhrel/jira-cli?style=social)

JiraCLI 是一款面向 Atlassian Jira 的交互式命令行工具，旨在通过终端操作减少对 Web 界面的依赖，提升工作效率。

核心功能包括：
1. **问题管理**：支持搜索、创建、编辑、删除、克隆问题，以及状态转移、用户分配、评论和工作日志管理、问题链接与解绑。
2. **迭代与项目**：支持史诗（Epic）和冲刺（Sprint）的问题增减与管理，以及发布版本、项目和看板的列表查询。
3. **交互体验**：默认提供交互式终端界面（支持键盘导航），同时支持纯文本、JSON 和 CSV 格式输出，便于脚本处理。
4. **兼容支持**：兼容 Jira Cloud 和本地部署环境；支持 Linux、macOS、Windows 等系统；支持 Basic Auth、Bearer Token 及 mTLS 多种认证方式。
