### [IceWhaleTech CasaOS](https://github.com/IceWhaleTech/CasaOS)  ![GitHub Repo stars](https://img.shields.io/github/stars/IceWhaleTech/CasaOS?style=social)

CasaOS 是一款面向家庭场景的个人云操作系统。它提供直观友好的用户界面，支持在多种硬件（如树莓派、NUC、ZimaBoard）及主流 Linux 系统上快速部署。系统内置应用商店支持一键安装常用服务，并兼容海量 Docker 应用。具备文件管理和系统资源监控功能。旨在降低个人数据中心成本，帮助用户掌控数据所有权，实现本地化智能服务与数据协作。
