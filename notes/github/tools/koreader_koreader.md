### [koreader koreader](https://github.com/koreader/koreader)  ![GitHub Repo stars](https://img.shields.io/github/stars/koreader/koreader?style=social)

KOReader 是一款专为电子墨水屏设备设计的文档查看器，支持 Kindle、Kobo、PocketBook 等嵌入式阅读器以及 Android 和 Linux 系统。它兼容多种固定版面与流式排版格式（如 PDF、EPUB、DjVu），内置扫描文档重排功能。软件提供多语言界面、高度可定制的排版选项，并针对电子墨水屏优化了显示效果与翻页速度。此外，它还集成 Calibre、维基百科及翻译工具，支持插件扩展、字典查询及网络服务，由全球志愿者社区维护。
