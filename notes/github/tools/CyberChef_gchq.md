### [gchq CyberChef](https://github.com/gchq/CyberChef)  ![GitHub Repo stars](https://img.shields.io/github/stars/gchq/CyberChef?style=social)

CyberChef 是一款基于 Web 浏览器的“网络瑞士军刀”工具，用于在本地执行各类数据处理和网络分析操作。

**主要功能：**
*   **数据处理**：支持编码/解码（Base64、XOR 等）、加密/解密（AES、DES、Blowfish 等）、哈希计算、压缩/解压、二进制/十六进制解析、字符编码转换、日期转换及网络协议解析（IPv6、X.509）。
*   **交互特性**：采用拖拽式界面组合操作食谱，支持自动输出结果、自动编码检测（Magic）、断点调试、文本高亮显示及文件拖拽加载。
*   **部署与安全**：完全在客户端运行，数据不上传服务器；支持 Docker 部署和 Node.js API；支持食谱保存与 URL 分享。
