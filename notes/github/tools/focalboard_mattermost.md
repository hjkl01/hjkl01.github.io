### [mattermost focalboard](https://github.com/mattermost/focalboard)  ![GitHub Repo stars](https://img.shields.io/github/stars/mattermost/focalboard?style=social)

Focalboard 是一款开源、多语言、支持自托管的项目管理工具，可作为 Trello、Notion 和 Asana 的替代方案。它用于定义、组织、跟踪和管理个人或团队的工作。项目提供两种版本：个人桌面版（单用户独立应用，支持 macOS、Windows、Linux）和个人服务器版（多用户独立服务器）。此外还支持 Docker 部署、源码构建及提供 API 文档。
