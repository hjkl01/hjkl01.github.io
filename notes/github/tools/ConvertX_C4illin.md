### [C4illin ConvertX](https://github.com/C4illin/ConvertX)  ![GitHub Repo stars](https://img.shields.io/github/stars/C4illin/ConvertX?style=social)

ConvertX 是一个基于 TypeScript、Bun 和 Elysia 开发的自托管在线文件转换服务。它支持超过一千种文件格式，具备批量处理、多账户管理及密码保护功能。项目集成了 FFmpeg、LibreOffice、Pandoc 等多种转换引擎，涵盖图像、视频、文档、电子书及 3D 资产等类型。支持通过 Docker 一键部署，并提供丰富的环境配置选项。
