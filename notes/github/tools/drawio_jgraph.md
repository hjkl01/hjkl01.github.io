### [jgraph drawio](https://github.com/jgraph/drawio)  ![GitHub Repo stars](https://img.shields.io/github/stars/jgraph/drawio?style=social)

draw.io 是一款由 draw.io Ltd 与 draw.io AG 联合开发的可配置绘图及白板应用程序。

**核心功能**：
支持创建图表和白板内容，可通过 GitHub Pages、Docker、桌面版或 WAR 包运行，兼容主流浏览器。

**关键限制与说明**：
1. 非 SVG 编辑器，SVG 导出仅用于网页嵌入。
2. 当前版本不支持实时协作编辑。
3. 不接受外部代码贡献，由核心团队全权开发。
4. 源码采用 Apache 2.0 协议，图标资源禁止用于 Atlassian 产品。
5. draw.io 为欧盟注册商标。
