### [miroslavpejic85 call-me](https://github.com/miroslavpejic85/call-me)  ![GitHub Repo stars](https://img.shields.io/github/stars/miroslavpejic85/call-me?style=social)

Call-Me 是基于 WebRTC 的浏览器端一对一视频通话应用。支持端到端加密、用户名登录及多语言设置。核心功能涵盖视频通话、实时聊天、屏幕与文件共享、音视频控制。项目提供 REST API、Click-to-Call 及 Iframe 集成，支持 Docker 或 Node.js 部署及私有化自托管。
