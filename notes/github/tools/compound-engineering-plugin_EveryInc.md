### [EveryInc compound-engineering-plugin](https://github.com/EveryInc/compound-engineering-plugin)  ![GitHub Repo stars](https://img.shields.io/github/stars/EveryInc/compound-engineering-plugin?style=social)

该项目提供 Compound Engineering 插件及市场，旨在让工程工作逐次更易。核心功能如下：

1. **多工具集成**：支持安装并转换为 Claude Code、Cursor 及 OpenCode、Copilot、Windsurf 等数十种 AI 编码工具的插件格式。
2. **配置同步**：可将个人 Claude Code 配置（技能、命令、MCP 服务）自动同步至其他支持工具，确保跨平台一致性。
3. **核心工作流**：集成 `/ce:plan`、`/ce:work`、`/ce:review`、`/ce:compound` 命令，遵循“计划 - 执行 - 审查 - 沉淀”循环。
4. **设计理念**：通过充分规划、审查和知识沉淀消除技术债务，使后续工程工作更易进行。
