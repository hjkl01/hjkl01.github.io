### [ScottPlot ScottPlot](https://github.com/ScottPlot/ScottPlot)  ![GitHub Repo stars](https://img.shields.io/github/stars/ScottPlot/ScottPlot?style=social)

ScottPlot 是一款面向 .NET 的免费开源绘图库，旨在轻松交互式地展示大规模数据集。它支持通过少量代码快速创建折线图、柱状图、饼图、散点图等多种图表。该库兼容 Windows Forms、WPF、控制台、Blazor、Avalonia、Uno Platform 及 Notebooks 等多种开发与运行环境。
