### [ast-grep ast-grep](https://github.com/ast-grep/ast-grep)  ![GitHub Repo stars](https://img.shields.io/github/stars/ast-grep/ast-grep?style=social)

ast-grep 是一款基于 Tree-sitter 解析器和抽象语法树（AST）的命令行工具，支持代码的结构化搜索、代码检查和重写。它通过匹配语法结构节点而非文本来实现代码匹配，允许用户使用类似代码的模式进行查找与替换。该工具支持 YAML 配置编写 Lint 规则，具备多核处理能力，适用于轻量级静态分析、大规模代码重构及强制编码规范。
