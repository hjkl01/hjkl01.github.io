### [krau SaveAny-Bot](https://github.com/krau/SaveAny-Bot)  ![GitHub Repo stars](https://img.shields.io/github/stars/krau/SaveAny-Bot?style=social)

Save Any Bot 是一个 Telegram 机器人，支持将 Telegram 文件及其他网络资源保存到任意存储服务。功能包括绕过 Telegram 限制保存内容，支持文档、视频、图片、贴纸及 Telegraph 格式。具备批量下载、流式传输及多用户支持能力。可自动整理文件，监控指定聊天并自动保存消息。支持在不同存储后端（如 Alist、S3、WebDAV、本地磁盘、Rclone 等）间传输文件。集成 yt-dlp 和 Aria2，支持从千余网站下载媒体及通过 URL/磁力链接下载文件，支持编写 JS 插件扩展网页资源保存能力。
