### [micro-editor micro](https://github.com/micro-editor/micro)  ![GitHub Repo stars](https://img.shields.io/github/stars/micro-editor/micro?style=social)

micro 是一款基于终端的轻量级文本编辑器，旨在提供直观易用的编辑体验，并充分利用现代终端能力。它以内置静态二进制形式分发，无需外部依赖，支持跨平台运行。

主要功能包括：
1. **编辑特性**：支持多光标、分屏、标签页、撤销重做（持久化）、宏、行号、Unicode 及软换行。
2. **交互体验**：兼容常用快捷键（可自定义），具备优秀的鼠标支持（拖拽、双击选词、三击选行）。
3. **显示与高亮**：支持 130 多种语言语法高亮、多种配色方案、真彩色及系统剪贴板集成。
4. **扩展能力**：内置 Lua 插件系统与插件管理器，支持自动补全、语法检查及错误通知。
5. **部署方式**：单文件分发，无依赖，支持从命令行或标准输入启动。
