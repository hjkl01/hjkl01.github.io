### [usememos memos](https://github.com/usememos/memos)  ![GitHub Repo stars](https://img.shields.io/github/stars/usememos/memos?style=social)

Memos 是一款开源、自托管的隐私优先笔记服务，适用于个人笔记、团队维基及知识管理。基于 Go 和 React 构建，资源占用低且部署简便，支持 Docker 及多种数据库（SQLite、MySQL、PostgreSQL）。原生支持 Markdown 存储，无跟踪、广告或订阅费，保障用户数据自主权。提供 REST 和 gRPC API 接口，界面简洁且适配深色模式与移动端。
