### [biomejs biome](https://github.com/biomejs/biome)  ![GitHub Repo stars](https://img.shields.io/github/stars/biomejs/biome?style=social)

Biome 是一款面向 Web 项目的高性能工具链，集成了代码格式化与静态检查功能。它支持 JavaScript、TypeScript、JSX、JSON、CSS 和 GraphQL，格式化性能优异且与 Prettier 兼容度达 97%，提供 340 多条静态检查规则以输出详细诊断。Biome 零配置启动，无需依赖 Node.js 运行，具备一流的 LSP 支持，专为编辑器实时交互设计，旨在通过统一工具链提升开发者体验与项目健康度。
