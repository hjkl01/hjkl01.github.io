### [mulaRahul keyviz](https://github.com/mulaRahul/keyviz)  ![GitHub Repo stars](https://img.shields.io/github/stars/mulaRahul/keyviz?style=social)

Keyviz 是一款免费开源的实时按键与鼠标操作可视化工具。它能在屏幕任意位置动态显示键盘快捷键及鼠标动作（如点击、拖动、滚动），适用于教程、演示和协作场景。主要功能包括：全面自定义外观（颜色、大小、布局、动画等）、筛选特定按键、记录输入历史。支持 Windows、macOS 及 Linux (X11) 平台，基于 Rust 和 Tauri 开发。
