### [KOReader.patches](https://github.com/SeriousHornet/KOReader.patches) ![GitHub Repo stars](https://img.shields.io/github/stars/SeriousHornet/KOReader.patches?style=social)

该项目为 KOReader 的可视化改造补丁合集，适用于 Coverbrowser 与 Project: Title 插件。主要功能包括：先禁用默认封面进度/状态组件以便重绘；封面和文件夹封面圆角（可拉伸）、文件夹名称与数量显示；自定义阅读状态图标；封面页数徽章与进度百分比徽章；系列标识或系列编号徽章；新的圆角进度条；收藏星标；已读书籍淡化显示。补丁与图标分别放入 koreader/patches 与 koreader/icons。
