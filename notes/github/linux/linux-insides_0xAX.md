### [0xAX linux-insides](https://github.com/0xAX/linux-insides)  ![GitHub Repo stars](https://img.shields.io/github/stars/0xAX/linux-insides?style=social)

本项目是一套持续更新的 Linux 内核内部原理教程书籍，旨在分享 Linux 内核及底层技术知识。内容涵盖系统启动、初始化、中断、系统调用、内存管理、同步原语及数据结构等核心模块，当前正在适配现代内核版本（v6.18+）。项目要求读者具备 C 语言和汇编语言基础，支持多国语言翻译，并提供社区贡献指南及 Linux 内核学习邮件列表。
