### [somewm](https://github.com/trip-zip/somewm) ![GitHub Repo stars](https://img.shields.io/github/stars/trip-zip/somewm?style=social)

somewm 是基于 wlroots 的 Wayland 版 AwesomeWM，完整支持 AwesomeWM 的 Lua API，rc.lua、控件和主题可直接复用，支持 LuaJIT 与多版本 Lua。主要功能包括内置锁屏、IPC 命令行控制、分数缩放、输入设备配置、标签在显示器热插拔后保持、可配置窗口阴影、壁纸 GPU 缓存以及空闲/DPMS 电源管理。
