### [HyDE-Project HyDE](https://github.com/HyDE-Project/HyDE)  ![GitHub Repo stars](https://img.shields.io/github/stars/HyDE-Project/HyDE?style=social)

HyDE 是针对 Arch Linux 及其衍生发行版的高度定制化桌面配置项目。它提供自动化安装脚本，将最小化系统配置为完整的桌面环境，支持对主题、壁纸、启动器（Rofi、wlogout）及系统组件（SDDM、GRUB 等）进行自定义。项目支持通过外部仓库管理多种官方及社区主题，提供虚拟机测试脚本（HyDEVM）、NVIDIA 驱动兼容性及便捷的更新与配置备份恢复功能。
