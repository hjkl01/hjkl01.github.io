### [meefik linuxdeploy](https://github.com/meefik/linuxdeploy)  ![GitHub Repo stars](https://img.shields.io/github/stars/meefik/linuxdeploy?style=social)

Linux Deploy 是一款开源工具，用于在 Android 设备上快速安装和运行 GNU/Linux 操作系统。其核心功能如下：

1. **多方式安装**：支持通过磁盘镜像、目录、磁盘分区或 RAM 创建 Linux 环境。
2. **广泛兼容**：支持 Debian、Ubuntu、Arch、Alpine 等多种发行版，以及 arm、x86 等多种架构和 ext2/3/4 文件系统。
3. **丰富环境**：支持 LXDE、Xfce、MATE 等多种桌面环境，提供 CLI、SSH、VNC、X11 等控制接口。
4. **便捷管理**：通过图形界面管理安装过程、服务启停及远程配置，运行于 chroot 环境中，所有更改可逆。
