### [lxc incus](https://github.com/lxc/incus)  ![GitHub Repo stars](https://img.shields.io/github/stars/lxc/incus?style=social)

Incus 是一个现代、安全且强大的系统容器和虚拟机管理器，提供在容器或虚拟机中运行和管理完整 Linux 系统的统一体验。它基于简单的 REST API 构建，支持广泛的 Linux 发行版镜像，具备从单机实例到数据中心集群的弹性扩展能力，适用于开发和生产环境。Incus 帮助用户构建小型私有云，以高效优化的方式运行各类工作负载。该项目是 LXD 的社区分支，采用 Apache 2.0 协议开源，并提供便捷的 LXD 迁移工具。
