Cronboard 是一款终端应用，用于在本地或通过 SSH 管理定时任务（cron）。可查看任务状态，创建/编辑/删除任务，暂停与恢复，支持路径自动补全、校验并以可读形式反馈执行时间，支持 @daily 等特殊表达式，可切换用户（需 sudo）并按关键词搜索任务。
