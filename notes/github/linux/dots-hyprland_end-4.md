### [end-4 dots-hyprland](https://github.com/end-4/dots-hyprland)  ![GitHub Repo stars](https://img.shields.io/github/stars/end-4/dots-hyprland?style=social)

这是一个基于 Hyprland 的桌面环境配置文件项目，当前核心版本使用 Quickshell 小部件系统（illogical-impulse 风格）。

主要功能：
- 应用概览：显示应用实时预览，支持搜索、计算和命令运行。
- AI 支持：集成 Gemini API 和 Ollama 模型。
- 动态配色：根据壁纸自动生成 Material 风格主题。
- 透明安装：执行命令前显示指令。
- 快捷键：兼容 Windows 或 GNOME 习惯。

注：旧版 AGS 及 EWW 风格已停止支持。
