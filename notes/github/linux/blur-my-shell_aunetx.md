### [aunetx blur-my-shell](https://github.com/aunetx/blur-my-shell)  ![GitHub Repo stars](https://img.shields.io/github/stars/aunetx/blur-my-shell?style=social)

Blur my Shell 是一款 GNOME Shell 扩展，用于为界面组件添加模糊视觉效果。它支持对概览视图、顶部面板、Dash to Dock、应用程序文件夹、锁屏及窗口选择器等组件应用模糊。提供静态模糊（基于壁纸图像，可自定义效果管道如高斯模糊、像素化、圆角等）和动态模糊（实时高斯模糊，支持透明度）两种模式。用户可自定义颜色、背景样式及模糊触发条件（如全屏时自动禁用），并支持通过白名单或黑名单模式控制窗口模糊效果。此外，该扩展兼容 Dash to Panel、Window List 等多种第三方插件，并适配多个 GNOME Shell 版本。
