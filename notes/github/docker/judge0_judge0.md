### [judge0 judge0](https://github.com/judge0/judge0)  ![GitHub Repo stars](https://img.shields.io/github/stars/judge0/judge0?style=social)

Judge0 是一个稳健、快速、可扩展且带有沙箱机制的开源在线代码执行系统，用于构建 AI 代理、编程竞赛、在线学习及招聘评估等需要代码执行功能的应用。它支持 90 多种编程语言，提供 HTTP JSON API 和 Python SDK 以实现便捷集成，支持自行托管或 SaaS 部署。系统具备安全沙箱环境，可执行不受信任代码，支持多文件项目、自定义编译选项及资源限制，并提供详细执行结果与 Webhooks 支持。项目分为社区版（CE）和额外版（Extra CE），架构易于部署和扩展。
