### [abiosoft colima](https://github.com/abiosoft/colima)  ![GitHub Repo stars](https://img.shields.io/github/stars/abiosoft/colima?style=social)

Colima 是一款面向 macOS（Intel 及 Apple Silicon）和 Linux 的极简容器运行时工具。核心功能包括：支持 Docker、Containerd 和 Incus 等运行时环境，可选集成 Kubernetes；提供自动端口转发、卷挂载及多实例管理；支持 AI 工作负载的 GPU 加速容器（需 Apple Silicon）；允许通过命令行或配置文件自定义虚拟机资源（CPU、内存、磁盘）；具备简单的 CLI 接口。
