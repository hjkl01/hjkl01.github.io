### [casdoor casdoor](https://github.com/casdoor/casdoor)  ![GitHub Repo stars](https://img.shields.io/github/stars/casdoor/casdoor?style=social)

Casdoor 是一款开源的、AI 优先的身份与访问管理（IAM）系统、AI MCP 网关及认证服务器，配备 Web 管理界面。它支持 OAuth 2.1、OIDC、SAML、CAS、LDAP、SCIM、WebAuthn、TOTP、MFA、Face ID 等多种认证协议与标准，并兼容 Google Workspace 和 Azure AD。项目支持通过源代码、Docker 及 Kubernetes Helm 进行部署。
