### [portainer portainer](https://github.com/portainer/portainer)  ![GitHub Repo stars](https://img.shields.io/github/stars/portainer/portainer?style=social)

Portainer 是一款轻量级容器应用交付平台，用于管理 Docker、Swarm、Kubernetes 及 ACI 环境。它通过图形界面（GUI）和 API 提供对容器、镜像、卷、网络等编排资源的集中管理。该系统可部署为单个容器，兼容 Linux 或 Windows 集群。项目分为开源社区版（CE）和包含 RBAC 等高级功能的企业版（BE）。
