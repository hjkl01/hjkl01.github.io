### [base node](https://github.com/base/node)  ![GitHub Repo stars](https://img.shields.io/github/stars/base/node?style=social)

本项目提供 Base 网络节点的 Docker 构建方案，用于在基于 Optimism OP Stack 的以太坊 L2 上运行节点。核心功能包括：
1. 支持 Base 主网和测试网部署。
2. 支持多种客户端（默认 Reth，兼容 Geth、Nethermind）。
3. 通过 Docker Compose 快速搭建与管理。
4. 提供 L1 连接配置、硬件要求及性能调优选项。
5. 支持快照同步以加速节点初始化。
