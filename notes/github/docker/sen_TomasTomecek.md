### [TomasTomecek sen](https://github.com/TomasTomecek/sen)  ![GitHub Repo stars](https://img.shields.io/github/stars/TomasTomecek/sen?style=social)

sen 是一款支持 Docker 和 Podman 的终端用户界面（TUI）工具，支持交互式管理容器和镜像（启动、停止、重启、删除、检查），提供仪表盘视图、容器日志实时流式传输、搜索过滤、实时状态更新、Vim 风格快捷键、镜像树状视图及磁盘空间统计等功能。项目当前处于维护模式，仅提供 Bug 修复。
