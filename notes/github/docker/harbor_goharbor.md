**Summarizing Harbor项目功能**

Harbor是一个云原生注册中心，支持镜像签名和扫描，基于Docker分发扩展，提供安全、高效的镜像管理功能。

Harbor 是开源可信的云原生镜像/Helm 仓库，支持存储、签名、扫描与复制镜像与图表，提供基于角色的访问控制、策略化镜像复制、漏洞扫描、LDAP/AD 与 OIDC 单点登录、镜像删除与垃圾回收、内容签名（Notary）、审计日志、图形化管理界面、REST API，并可通过 Docker Compose、Helm 或 Operator 便捷部署。
