### [Kong kong](https://github.com/Kong/kong)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kong/kong?style=social)

Kong Gateway 是一款云原生、平台无关的高性能可扩展网关，专注于管理 API、LLM（大语言模型）及 MCP（模型上下文协议）流量。它提供代理、路由、负载均衡、健康检查及身份认证功能，支持跨多家 AI 提供商的 LLM 统一路由与 MCP 流量治理。项目具备强大的插件扩展能力（支持 Lua、Go、JavaScript），原生集成 Kubernetes Ingress，适用于云、裸金属及容器环境，支持多种部署模式与 AI 高级特性（如语义安全、缓存）。
