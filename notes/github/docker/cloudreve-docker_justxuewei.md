### [justxuewei cloudreve-docker](https://github.com/justxuewei/cloudreve-docker)  ![GitHub Repo stars](https://img.shields.io/github/stars/justxuewei/cloudreve-docker?style=social)

本项目为 Cloudreve V3 的 Docker 镜像，旨在提供纯净、小体积且易安装的私有云存储部署方案。支持多种架构，涵盖上传、配置、数据库及头像路径的映射管理，允许通过环境变量调整权限（PUID/PGID）与时区。提供初始密码获取方式，同时附带 Nginx 反向代理、Aria2 离线下载及 Docker Compose 部署教程，并包含容器升级指南。
