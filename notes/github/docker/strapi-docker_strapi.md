### [strapi strapi-docker](https://github.com/strapi/strapi-docker)  ![GitHub Repo stars](https://img.shields.io/github/stars/strapi/strapi-docker?style=social)

这是一个 Strapi v3 的 Docker 容器化项目，提供 `strapi/strapi` 和 `strapi/base` 两个镜像。`strapi/strapi` 用于快速创建新项目或挂载运行本地项目，支持通过环境变量配置数据库；`strapi/base` 用于构建生产环境的自定义镜像。项目旨在简化 Strapi 内容管理框架的部署，但仅支持 v3 版本，且更新镜像标签不会自动升级 Strapi 应用版本。
