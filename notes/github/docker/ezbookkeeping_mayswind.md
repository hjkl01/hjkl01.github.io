### [mayswind ezbookkeeping](https://github.com/mayswind/ezbookkeeping)  ![GitHub Repo stars](https://img.shields.io/github/stars/mayswind/ezbookkeeping?style=social)

ezBookkeeping 是一款轻量级、自托管的个人财务管理应用。它支持记录日常交易、导入多格式财务数据、搜索筛选账单，并通过图表或自定义维度分析消费趋势。应用界面适配移动端与桌面端，支持 PWA、深色模式及 AI 收据识别。提供多语言、多货币、多时区切换，以及定时交易、地图定位、图片附件等功能。具备双重认证、外部认证及应用锁等安全特性。资源占用低，支持 Docker 及二进制部署，兼容多种数据库和操作系统架构。
