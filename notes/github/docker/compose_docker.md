### [docker compose](https://github.com/docker/compose)  ![GitHub Repo stars](https://img.shields.io/github/stars/docker/compose?style=social)

Docker Compose 是一个用于定义和运行多容器 Docker 应用程序的工具。它基于 Compose 文件格式（YAML）配置应用所需的服务和容器，通过单一命令（docker compose up）即可创建并启动整个应用环境。该工具支持在 Windows、macOS 和 Linux 平台上使用，旨在简化多容器应用的部署与管理流程。
