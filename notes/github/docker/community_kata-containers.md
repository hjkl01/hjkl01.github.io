### [kata-containers community](https://github.com/kata-containers/community)  ![GitHub Repo stars](https://img.shields.io/github/stars/kata-containers/community?style=social)

Kata Containers 是一个开源项目，致力于构建兼具容器性能体验与虚拟机隔离安全性的轻量级虚拟机标准实现。该项目架构无关，支持多种 Hypervisor，兼容 OCI 规范和 Kubernetes，整合了 Intel Clear Containers 与 Hyper runV 技术。由 Open Infrastructure Foundation 管理，遵循开放治理原则，设有架构委员会负责技术决策，并提供社区贡献及安全漏洞管理机制。
