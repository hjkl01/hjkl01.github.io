### [glanceapp glance](https://github.com/glanceapp/glance)

Glance 是一个轻量级、高度可定制的仪表盘工具，支持多种信息源的小部件（如 RSS、Reddit、天气、市场数据、视频等），可自定义标题和布局。主要特性包括快速加载、低内存占用、跨平台支持（Docker、二进制文件）、移动设备优化、主题自定义。使用方法包括 Docker 安装、手动安装、源码构建。常见问题涉及 DNS 限速、浏览器扩展冲突、配置错误等。支持通过自定义小部件（iframe、HTML、API）扩展功能。
