### [lirantal dockly](https://github.com/lirantal/dockly)  ![GitHub Repo stars](https://img.shields.io/github/stars/lirantal/dockly?style=social)

Dockly 是一款基于 Node.js 的沉浸式终端界面工具，用于管理 Docker 容器、服务和镜像。

主要功能：
1. 自动连接本地 Docker 守护进程，或通过命令行自定义远程连接参数（Socket 路径、主机、端口、协议）。
2. 支持对显示的容器进行过滤。
3. 支持直接在 Docker 容器内运行（需挂载 Docker Socket）。

环境要求：
依赖 Node.js 环境，版本需在 v7.6 及以上。
