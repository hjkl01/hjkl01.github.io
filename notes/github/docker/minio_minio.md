### [minio minio](https://github.com/minio/minio)  ![GitHub Repo stars](https://img.shields.io/github/stars/minio/minio?style=social)

MinIO 是一款高性能、兼容 S3 API 的开源对象存储解决方案，适用于 AI/ML、分析及数据密集型工作负载。项目基于 AGPL v3.0 协议开源，支持源码编译、Docker 容器化及 Kubernetes (Helm/Operator) 部署。内置 Web 管理控制台和命令行工具 `mc`，可与现有 S3 工具无缝集成。社区版目前仅以源码形式分发，商业或专有用途推荐使用 AIStor Enterprise 版本。
