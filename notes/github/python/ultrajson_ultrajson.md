### [ultrajson ultrajson](https://github.com/ultrajson/ultrajson)  ![GitHub Repo stars](https://img.shields.io/github/stars/ultrajson/ultrajson?style=social)

UltraJSON (ujson) 是一个用纯 C 编写并为 Python 提供绑定的高速 JSON 编码器和解码器。它可作为 Python 标准 JSON 库的无缝替代品，支持 `dumps` 和 `loads` 操作，并提供 HTML 字符转义、ASCII 输出限制、斜杠转义及缩进等配置选项。项目目前处于维护模式，因架构存在安全风险，建议用户迁移至 orjson。此外，支持通过环境变量控制调试符号及 double-conversion 库的链接构建。
