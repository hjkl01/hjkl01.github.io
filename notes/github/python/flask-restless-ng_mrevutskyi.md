### [mrevutskyi flask-restless-ng](https://github.com/mrevutskyi/flask-restless-ng)  ![GitHub Repo stars](https://img.shields.io/github/stars/mrevutskyi/flask-restless-ng?style=social)

Flask-Restless-NG 是一个 Flask 扩展，用于自动生成符合 JSON API 规范的 URL 端点。它支持基于 SQLAlchemy 或 Flask-SQLAlchemy 模型构建 RESTful API。该项目是 Flask-Restless 的维护分支，支持 Flask 2.2+ 和 SQLAlchemy 1.4.x-2.0.x，并显著优化了 JSON 响应的序列化性能。
