### [epezent implot](https://github.com/epezent/implot)  ![GitHub Repo stars](https://img.shields.io/github/stars/epezent/implot?style=social)

ImPlot 是基于 Dear ImGui 的即时模式 GPU 加速绘图库，专为实时数据可视化和交互式图表设计，集成简单且无除 ImGui 外的外部依赖。

核心功能：
1. 支持多种图表类型：折线、散点、柱状、热力图、饼图、茎图、误差条、直方图及图像等，可混合展示。
2. 丰富的交互与控制：支持缩放、平移、框选、自动适配及查询范围；支持多轴、子图及线性/对数/时间轴配置。
3. 样式与数据支持：支持标记、颜色、图例、配色等样式定制；兼容浮点、整数等多种数据类型及数据步进。
4. 集成与扩展：API 简洁，支持多种语言绑定（C、Python、.NET、Java 等）。

注意：适用于实时交互可视化，非出版级绘图工具；高密度绘图需注意 16 位索引限制。
