### [jakevdp PythonDataScienceHandbook](https://github.com/jakevdp/PythonDataScienceHandbook)  ![GitHub Repo stars](https://img.shields.io/github/stars/jakevdp/PythonDataScienceHandbook?style=social)

该项目是《Python 数据科学手册》的开源代码仓库，提供全书的 Jupyter 笔记本资源。核心内容包括 IPython、NumPy、Pandas、Matplotlib 和 Scikit-Learn 等 Python 数据科学库的使用教程。支持在线阅读、本地运行代码，以及通过 Google Colab 和 Binder 启动在线交互式环境，适用于具备 Python 基础的数据科学学习者。
