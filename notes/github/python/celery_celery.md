### [celery celery](https://github.com/celery/celery)  ![GitHub Repo stars](https://img.shields.io/github/stars/celery/celery?style=social)

Celery 是用 Python 编写的分布式任务队列系统，用于在多线程或多台机器间异步分配和处理任务。它通过消息代理（如 RabbitMQ、Redis）在客户端与工作者进程之间传递消息，支持高可用性和水平扩展。项目特点为简单易用、高性能及高度灵活，支持多种消息传输、并发模型和结果存储后端。Celery 可集成于 Django、Flask、FastAPI 等 Web 框架，其协议支持 Node.js、Go、PHP 等其他语言开发。
