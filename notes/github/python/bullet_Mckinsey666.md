### [Mckinsey666 bullet](https://github.com/Mckinsey666/bullet)  ![GitHub Repo stars](https://img.shields.io/github/stars/Mckinsey666/bullet?style=social)

bullet 是一个用于 Python 命令行工具（CLI）的提示库，用于制作美观简洁的交互界面。支持自定义格式、颜色、背景色、样式及表情符号。提供多种输入提示类型，如项目列表、复选框、滚动条、密码、是/否问题、数字及滑动输入等，并支持丰富的预设及自定义样式主题。兼容 Python 3.6 及以上版本。
