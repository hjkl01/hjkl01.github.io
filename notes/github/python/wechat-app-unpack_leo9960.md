### [leo9960 wechat-app-unpack](https://github.com/leo9960/wechat-app-unpack)  ![GitHub Repo stars](https://img.shields.io/github/stars/leo9960/wechat-app-unpack?style=social)

本项目用于微信小程序（wxapkg）的解包与源码分析。提供 Python、PHP、JAVA、Node.js 等多种语言的解包工具；解析解包后的文件结构（pages 目录、配置文件、JS 源码、WXML 模板）；详细说明核心文件的格式与内容；提供 page-frame.html 中 WXML 源码的还原分析器及版本兼容性说明。
