### [jackzhenguo python-small-examples](https://github.com/jackzhenguo/python-small-examples)  ![GitHub Repo stars](https://img.shields.io/github/stars/jackzhenguo/python-small-examples?style=social)

本项目致力于提供 Python 经典小例子与案例，旨在帮助学习者通过精简实用的代码高效掌握 Python 编程技能。内容涵盖以下领域：

1. **基础操作**：算术运算、类型转换、内置函数及数据结构操作；
2. **核心编程**：函数模块用法、面向对象、正则表达式、装饰器及生成器；
3. **数据处理**：NumPy 与 Pandas 数据分析，Turtle、Matplotlib 等绘图可视化；
4. **常用工具**：文件读写、日期时间、多线程、JSON 及调试技巧；
5. **工作案例**：计算器、列表处理、字典操作等实战场景。

所有示例均标注难度与版本，提供详细代码链接。允许非商业转载，但需备注来源。
