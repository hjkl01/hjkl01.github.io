### [zauberzeug nicegui](https://github.com/zauberzeug/nicegui)  ![GitHub Repo stars](https://img.shields.io/github/stars/zauberzeug/nicegui?style=social)

NiceGUI 是一款基于 Python 的轻量级图形用户界面（GUI）框架，支持在浏览器或原生窗口中运行。它提供按钮、对话框、图表、3D 场景等丰富组件，适用于微 Web 应用、仪表盘、机器人及智能家居项目。核心功能包括代码自动重载、内置数据刷新定时器、数据绑定、用户持久化及 Pytest 测试支持。架构采用后端优先模式，后端基于 Python/FastAPI，前端基于 Vue/Quasar，通过 WebSocket 实现实时通信，并兼容 Jupyter Notebook 环境。
