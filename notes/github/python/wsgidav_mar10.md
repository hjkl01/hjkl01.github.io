### [mar10 wsgidav](https://github.com/mar10/wsgidav)  ![GitHub Repo stars](https://img.shields.io/github/stars/mar10/wsgidav?style=social)

WsgiDAV 是一个基于 Python 和 WSGI 协议的通用可扩展 WebDAV 服务器。它支持 SSL 加密，可跨 Linux、OSX 和 Windows 独立运行，提供 Docker 及 Windows 安装包，也可作为 Python 库集成至任何 WSGI 兼容的 Web 服务器后端。项目具备灵活的可配置中间件架构，支持多种认证方式，可拓展用于暴露虚拟文件系统或实现在线 Office 文档编辑等功能。
