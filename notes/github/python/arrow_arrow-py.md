### [arrow-py arrow](https://github.com/arrow-py/arrow)  ![GitHub Repo stars](https://img.shields.io/github/stars/arrow-py/arrow?style=social)

Arrow 是一个 Python 库，用于简化日期和时间的创建、操作、格式化与转换。它作为 datetime 模块的替代品，默认支持时区感知和 UTC，提供更人性化的 API。核心功能包括：简化常见输入场景的日期创建；支持 ISO 8601 标准解析与格式化；支持时区转换及相对时间偏移；生成时间跨度、范围及界限；支持时间人性化显示；兼容 Python 3.8+、多种时区库及类型提示。
