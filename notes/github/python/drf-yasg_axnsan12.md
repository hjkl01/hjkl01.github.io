### [axnsan12 drf-yasg](https://github.com/axnsan12/drf-yasg)  ![GitHub Repo stars](https://img.shields.io/github/stars/axnsan12/drf-yasg?style=social)

drf-yasg 是一款基于 Django Rest Framework 的 Swagger 生成工具，能够生成真实的 OpenAPI 2.0 规范（不支持 OpenAPI 3.0）。项目兼容 Django 4.0+、DRF 3.13+ 和 Python 3.10+。

核心功能：
- 支持嵌套序列化器和模式，提供响应描述及代码生成工具兼容的模型定义。
- 内置 Swagger UI 和 ReDoc 查看器，支持 JSON 和 YAML 格式。
- 提供 Schema 缓存、自动生成验证及 DRF 版本控制支持。
- 兼容第三方库（如 djangorestframework-camel-case）。
