### [plotly dash](https://github.com/plotly/dash)  ![GitHub Repo stars](https://img.shields.io/github/stars/plotly/dash?style=social)

Dash 是基于 Plotly.js、React 和 Flask 的 Python 框架，用于构建机器学习与数据科学 Web 应用。它将 UI 控件直接绑定到 Python 分析代码，支持声明式和反应式的交互逻辑，可构建包含多种图表类型及跨过滤功能的应用。项目包含开源版与企业版，企业版提供可扩展托管、部署、身份认证、低代码工具及 AI/ML 加速等企业级功能。
