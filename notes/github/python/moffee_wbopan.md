### [wbopan moffee](https://github.com/wbopan/moffee)  ![GitHub Repo stars](https://img.shields.io/github/stars/wbopan/moffee?style=social)

moffee 是一个开源幻灯片制作工具，可将 Markdown 文档转换为专业的演示文稿。它自动处理布局、分页和样式，支持简单的 Markdown 语法及扩展功能（如 Obsidian Flavored Markdown）。用户可通过标题或分隔符创建幻灯片，并使用特殊符号实现图文的横向或纵向布局。项目提供实时 Web 预览界面，支持幻灯片放映及导出为 HTML 或 PDF。内置多种主题，支持通过前导元数据自定义样式，基于 Python 开发，可通过 pip 安装。
