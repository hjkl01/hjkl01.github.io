### [python-trio trio](https://github.com/python-trio/trio)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-trio/trio?style=social)

Trio 是一款 Python 生产级异步并发与 I/O 库，用于编写同时进行多项任务及并行 I/O 的程序（如网络爬虫、Web 服务器）。它基于“结构化并发”设计，原生支持 async/await，相比 asyncio 和 Twisted 更简洁，极度注重可用性与正确性。该库成熟稳定，支持 Linux、macOS、Windows 和 FreeBSD，要求 Python 3.10 及以上版本。
