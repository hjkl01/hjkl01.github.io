### [selwin python-user-agents](https://github.com/selwin/python-user-agents)  ![GitHub Repo stars](https://img.shields.io/github/stars/selwin/python-user-agents?style=social)

`user_agents` 是一个 Python 库，用于解析 HTTP 用户代理字符串以识别设备信息。它基于 `ua-parser` 实现，可识别设备类型（手机、平板、PC），检测触摸屏能力，提取浏览器、操作系统及设备详情（品牌、型号、版本），并判断是否为机器人。库提供简洁的属性接口方便获取解析结果。
