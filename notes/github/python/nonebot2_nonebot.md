### [nonebot nonebot2](https://github.com/nonebot/nonebot2)  ![GitHub Repo stars](https://img.shields.io/github/stars/nonebot/nonebot2?style=social)

NoneBot2 是一个现代、跨平台、可扩展的 Python 聊天机器人框架。它基于 Python 的异步特性和类型注解，支持高效开发与类型安全。框架通过适配器支持多种聊天平台（如 QQ、Telegram、飞书、GitHub 等）及多种 Web 驱动（如 FastAPI、aiohttp）。提供 NB-CLI 脚手架简化项目创建，拥有活跃社区及丰富插件生态，采用 MIT 许可证。
