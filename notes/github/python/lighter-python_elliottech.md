### [elliottech lighter-python](https://github.com/elliottech/lighter-python)  ![GitHub Repo stars](https://img.shields.io/github/stars/elliottech/lighter-python?style=social)

Lighter Python 是 Lighter 协议的 Python SDK，支持 Python 3.8+ 异步编程。它封装了 Lighter 主网的 REST API 和 WebSocket 接口，主要功能包括：账户管理（查询账户信息、API Key、盈亏）、订单交易（创建、取消及查询）、市场数据（订单簿、成交记录、资金费率）以及链上交互（发送交易、充值提现历史、Nonce 获取）。支持 WebSocket 实时同步订单簿与账户状态，所有公开 API 端点无需授权即可访问。
