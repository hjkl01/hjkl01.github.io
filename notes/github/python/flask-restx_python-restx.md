### [python-restx flask-restx](https://github.com/python-restx/flask-restx)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-restx/flask-restx?style=social)

Flask-RESTX 是一个基于 Flask 的 Python 扩展，旨在帮助开发者快速构建 REST API。作为 Flask-RESTPlus 的社区驱动分支，它提供装饰器和工具来描述 API 并遵循最佳实践，支持通过 Swagger 自动生成 API 文档。项目要求 Python 3.9+，并兼容多种 Flask 版本。
