### [RamezIssac django-slick-reporting](https://github.com/RamezIssac/django-slick-reporting)  ![GitHub Repo stars](https://img.shields.io/github/stars/RamezIssac/django-slick-reporting?style=social)

django-slick-reporting 是一款功能齐全的一站式 Django 报表引擎。支持通过少量代码快速生成简单、分组、时间序列及交叉表报告，并能一键生成多种图表（如柱状图、饼图）。项目支持自定义复杂计算，具备高性能优化特性且易于扩展。开发者既可通过 Django 视图类（ReportView）使用，也可直接调用底层生成器类（ReportGenerator）进行交互。
