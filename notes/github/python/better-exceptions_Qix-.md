### [Qix- better-exceptions](https://github.com/Qix-/better-exceptions)  ![GitHub Repo stars](https://img.shields.io/github/stars/Qix-/better-exceptions?style=social)

`better-exceptions` 是一个 Python 库，用于自动美化并增强异常信息的显示效果。安装后设置 `BETTER_EXCEPTIONS` 环境变量即可全局生效，无需修改代码。支持 Python REPL 交互模式，兼容 `unittest` 测试框架及 Django 中间件集成。允许自定义输出截断长度，生产环境中应禁用环境变量以防敏感数据泄露。
