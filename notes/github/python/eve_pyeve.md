### [pyeve eve](https://github.com/pyeve/eve)  ![GitHub Repo stars](https://img.shields.io/github/stars/pyeve/eve?style=social)

Eve 是一个基于 Flask 的开源 Python REST API 框架，用于轻松构建和部署高度可定制的 RESTful Web 服务。它原生支持 MongoDB 数据库，并通过扩展支持 SQL 数据库。主要功能包括完整的 CRUD 操作、可自定义的资源与项端点、数据过滤排序分页、HATEOAS 支持、JSON/XML 渲染、条件请求、数据验证与完整性控制、批量插入、API 及文档版本控制、身份认证、CORS、速率限制、文件存储及 GeoJSON 支持等。
