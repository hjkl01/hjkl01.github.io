### [lk-geimfari mimesis](https://github.com/lk-geimfari/mimesis)  ![GitHub Repo stars](https://img.shields.io/github/stars/lk-geimfari/mimesis?style=social)

Mimesis 是一款用于 Python 的伪数据生成器，可生成多种语言的广泛假数据。核心功能包括：支持 46 种本地化语言；提供多种数据提供者（如人名、邮箱、电话等）；支持基于架构的复杂及关系数据生成；具备高性能、全类型提示及自定义扩展能力。用户只需导入对应提供者对象即可快速生成所需数据。
