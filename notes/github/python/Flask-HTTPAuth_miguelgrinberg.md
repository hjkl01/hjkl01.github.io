### [miguelgrinberg Flask-HTTPAuth](https://github.com/miguelgrinberg/Flask-HTTPAuth)  ![GitHub Repo stars](https://img.shields.io/github/stars/miguelgrinberg/Flask-HTTPAuth?style=social)

Flask-HTTPAuth 是一个简单的 Flask 扩展，为 Flask 路由提供 Basic、Digest 和 Token HTTP 认证功能。它支持密码哈希、自定义验证回调及登录权限装饰器，用于实现 Flask 应用的 HTTP 认证机制。
