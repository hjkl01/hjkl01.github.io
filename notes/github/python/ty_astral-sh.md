### [astral-sh ty](https://github.com/astral-sh/ty)  ![GitHub Repo stars](https://img.shields.io/github/stars/astral-sh/ty?style=social)

ty 是一款由 Astral 团队开发的基于 Rust 的 Python 类型检查器和语言服务器。它具有比 mypy 和 Pyright 快 10 至 100 倍的性能，提供代码导航、补全、自动导入及内联提示等语言服务器功能。支持细粒度增量分析、规则配置、文件覆盖、抑制注释及部分类型化代码，并提供交集类型、类型缩小等高级类型特性。兼容 VS Code、PyCharm 等编辑器，目前处于 Beta 版本。
