### [joke2k faker](https://github.com/joke2k/faker)  ![GitHub Repo stars](https://img.shields.io/github/stars/joke2k/faker?style=social)

Faker 是一个 Python 库，用于生成假数据，适用于数据库引导、文档创建、压力测试及数据匿名化。它提供丰富的数据生成器（如姓名、地址、文本等），支持多语言本地化及自定义 Provider。项目支持唯一值生成、随机种子控制、集成 Pytest 和 Factory Boy，并提供命令行工具。需 Python 3.8 及以上版本。
