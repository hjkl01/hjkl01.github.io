### [asweigart pyautogui](https://github.com/asweigart/pyautogui)  ![GitHub Repo stars](https://img.shields.io/github/stars/asweigart/pyautogui?style=social)

PyAutoGUI 是一个跨平台的 Python GUI 自动化模块，用于编程控制鼠标和键盘。它支持 Windows、macOS 和 Linux 系统，底层隐藏了各操作系统 API 的复杂性，提供统一的简单接口。

主要功能包括：
1. 鼠标控制：移动光标、点击、双击及相对移动。
2. 键盘控制：模拟打字、按键、组合键及按键状态控制。
3. 消息框交互：显示警告、确认、输入提示及密码输入框。
4. 截图与图像识别：截取屏幕图像，并能在屏幕上定位和查找特定图像元素的位置。
