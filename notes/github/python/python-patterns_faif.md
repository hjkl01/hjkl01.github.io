### [faif python-patterns](https://github.com/faif/python-patterns)  ![GitHub Repo stars](https://img.shields.io/github/stars/faif/python-patterns?style=social)

该项目是一个 Python 设计模式与惯用法的集合，提供了多种设计模式的 Python 代码实现。内容涵盖创建型、结构型、行为型、可测试性设计、基础模式及其他模式。此外，项目列出了 Python 中不推荐的反模式（如单例、上帝对象、过度继承）及其替代方案，并包含视频教程、贡献规范及测试指南，帮助开发者理解模式应用的权衡与最佳实践。
