### [dynaconf dynaconf](https://github.com/dynaconf/dynaconf)  ![GitHub Repo stars](https://img.shields.io/github/stars/dynaconf/dynaconf?style=social)

Dynaconf 是用于 Python 的配置管理工具。它支持多种配置文件格式（TOML, YAML, JSON, INI, PY）及环境变量覆盖，提供分层环境配置系统（默认、开发、测试、生产）。功能涵盖敏感信息保护（支持 Hashicorp Vault 和 Redis）、Django 与 Flask 框架扩展，以及配置验证、解析、模板替换、嵌套键访问和 CLI 操作（初始化、验证、导出等）。
