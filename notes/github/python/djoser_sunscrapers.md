### [sunscrapers djoser](https://github.com/sunscrapers/djoser)  ![GitHub Repo stars](https://img.shields.io/github/stars/sunscrapers/djoser?style=social)

djoser 是一个基于 Django Rest Framework 的 Django 认证系统 REST 实现库。它提供处理用户注册、登录、登出、密码重置和账户激活等操作的视图，支持自定义用户模型。项目支持 Token 认证、JWT 认证、社交认证和 WebAuthn，并对部分功能进行了重写，以更好地适配单页应用（SPA）架构。
