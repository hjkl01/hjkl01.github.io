### [gae-init gae-init](https://github.com/gae-init/gae-init)  ![GitHub Repo stars](https://img.shields.io/github/stars/gae-init/gae-init?style=social)

gae-init 是一个基于 Google App Engine 的 Python Web 应用启动模板。它集成了 Flask、RESTful、Jinja2、Bootstrap 等技术栈，支持 Google、Facebook 等 OpenID 登录。项目提供初始化、本地开发运行、自动化测试、一键部署到 Google App Engine 及项目重置等功能，旨在简化新应用的构建与部署流程。
