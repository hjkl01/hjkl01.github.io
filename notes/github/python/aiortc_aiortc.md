### [aiortc aiortc](https://github.com/aiortc/aiortc)  ![GitHub Repo stars](https://img.shields.io/github/stars/aiortc/aiortc?style=social)

aiortc 是一个基于 asyncio 的 Python 库，用于实现 WebRTC 和 ORTC 通信。支持音频、视频和数据通道的收发，涵盖 SDP、ICE、DTLS、SRTP 及纯 Python SCTP 等协议。支持 VP8/H.264 视频及 Opus/PCMU/PCMA 音频编解码，与 Chrome 和 Firefox 兼容。代码简洁，测试完善，便于理解 WebRTC 原理及与 Python 生态（如 OpenCV）集成。
