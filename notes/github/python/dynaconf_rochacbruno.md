### [rochacbruno dynaconf](https://github.com/rochacbruno/dynaconf)  ![GitHub Repo stars](https://img.shields.io/github/stars/rochacbruno/dynaconf?style=social)

Dynaconf 是 Python 配置管理库，遵循 12 因子应用指南。它支持 TOML、YAML、JSON、INI、PY 等多种文件格式，允许通过环境变量覆盖配置。功能包括设置验证、解析、敏感信息保护及分层环境管理（开发、测试、生产）。内置 Hashicorp Vault 和 Redis 集成，兼容 Django 和 Flask 框架，并提供 CLI 工具用于初始化与管理。
