### [graphql-python flask-graphql](https://github.com/graphql-python/flask-graphql)  ![GitHub Repo stars](https://img.shields.io/github/stars/graphql-python/flask-graphql?style=social)

Flask-GraphQL 是一个为 Flask 应用集成 GraphQL 支持的扩展库。它提供 `GraphQLView` 视图类，可快速配置 GraphQL 端点并启用 GraphiQL IDE 用于调试。项目支持批量查询、GraphQL 订阅、自定义上下文与中间件、错误格式化等灵活配置。此外，它兼容 Graphene v3，并允许通过子类化实现动态请求根值设置。
