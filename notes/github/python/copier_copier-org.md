### [copier-org copier](https://github.com/copier-org/copier)  ![GitHub Repo stars](https://img.shields.io/github/stars/copier-org/copier?style=social)

Copier 是一个用于渲染项目模板的 Python 库及命令行工具。它支持从本地路径或 Git URL 生成项目，能够动态替换文本文件中的变量，并默认避免覆盖现有文件。项目通过 `copier.yml` 配置问卷问题，利用 Jinja2 模板引擎生成代码结构。Copier 提供 CLI 命令和 Python API 接口，主要用于代码脚手架搭建、快速启动新项目以及管理模板更新和生命周期。
