### [celery kombu](https://github.com/celery/kombu)  ![GitHub Repo stars](https://img.shields.io/github/stars/celery/kombu?style=social)

Kombu 是一个 Python 消息传递库，旨在通过为 AMQP 协议提供高层接口来简化消息处理。它支持多种可插拔的消息服务器后端（如 RabbitMQ、Redis、MongoDB、Amazon SQS 等），具备消息负载自动编码、序列化和压缩功能。项目提供一致的异常处理、可靠的连接管理，支持生产者、交换机、队列及多种路由模式，并包含用于单元测试的内存传输方案。
