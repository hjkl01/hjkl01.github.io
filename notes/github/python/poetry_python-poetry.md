### [python-poetry poetry](https://github.com/python-poetry/poetry)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-poetry/poetry?style=social)

Poetry 是一款用于 Python 项目的打包与依赖管理工具。它通过统一的 `pyproject.toml` 格式取代了 `setup.py`、`requirements.txt` 等传统文件。Poetry 支持声明、管理和安装项目依赖，确保环境一致性，并提供版本约束、可选依赖、Git 依赖及依赖分组等管理功能。
