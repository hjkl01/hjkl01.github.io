### [learnpyqt 15-minute-apps](https://github.com/learnpyqt/15-minute-apps)  ![GitHub Repo stars](https://img.shields.io/github/stars/learnpyqt/15-minute-apps?style=social)

该项目包含数百个 Python GUI 示例，支持 PyQt6、PySide6、PySide2 及 PyQt5 框架。内容涵盖完整的演示应用（如网页浏览器、扫雷、记事本、画板等）和可复用的控件组件（如图形均衡器、调色板、密码编辑器等）。所有代码遵循 MIT 协议，允许自由重用和修改。用户安装依赖后运行 main.py 即可使用。
