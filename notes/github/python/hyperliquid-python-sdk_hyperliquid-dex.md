### [hyperliquid-dex hyperliquid-python-sdk](https://github.com/hyperliquid-dex/hyperliquid-python-sdk)  ![GitHub Repo stars](https://img.shields.io/github/stars/hyperliquid-dex/hyperliquid-python-sdk?style=social)

hyperliquid-python-sdk 是一个 Hyperliquid 去中心化交易所的 Python API 开发工具包（SDK）。其核心功能为封装交易指令执行与账户信息查询（如用户状态获取）接口，支持通过 pip 安装。使用需配置账户公钥与私钥进行身份验证，项目提供基础使用示例，遵循语义化版本规范，采用 MIT 协议开源。
