### [django channels](https://github.com/django/channels)  ![GitHub Repo stars](https://img.shields.io/github/stars/django/channels?style=social)

Django Channels 是 Django 官方项目，为 Django 代码提供 WebSocket、长轮询 HTTP、任务卸载及其他异步支持。它基于熟悉的 Django 设计模式，通过灵活底层框架支持行为自定义与新协议开发。依赖 Python 3.9+ 及 Django 4.2+。核心组件包括 Daphne（服务器）、channels_redis（后端）及 asgiref（基础库）。
