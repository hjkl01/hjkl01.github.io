### [python-gino gino](https://github.com/python-gino/gino)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-gino/gino?style=social)

GINO 是一个基于 SQLAlchemy 核心构建的轻量级 Python asyncio 异步 ORM，支持 PostgreSQL 和 MySQL 数据库。核心功能包括提供异步引擎与连接、异步 CRUD 对象模型、上下文连接与事务管理、复用 SQLAlchemy 核心构建查询、支持 Alembic 迁移，兼容 Starlette、FastAPI、Sanic 等主流 Web 框架，并提供丰富的 PostgreSQL JSONB 支持。
