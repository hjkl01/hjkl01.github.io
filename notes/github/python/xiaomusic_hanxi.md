### [hanxi xiaomusic](https://github.com/hanxi/xiaomusic)  ![GitHub Repo stars](https://img.shields.io/github/stars/hanxi/xiaomusic?style=social)

XiaoMusic 是一款运行于小爱音箱的音乐播放项目，支持通过语音指令控制播放本地或在线下载（使用 yt-dlp）的歌曲，具备播放控制、模式切换（循环/随机）、歌单管理及收藏功能。项目基于 Python + FastAPI 开发，支持 Docker、Docker Compose 及 Pip 部署，提供 Web 管理后台进行配置，兼容多种型号的小爱音箱及 mp3、flac 等常见音频格式。
