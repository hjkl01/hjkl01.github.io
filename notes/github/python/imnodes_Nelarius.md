### [Nelarius imnodes](https://github.com/Nelarius/imnodes)  ![GitHub Repo stars](https://img.shields.io/github/stars/Nelarius/imnodes?style=social)

imnodes 是 dear imgui 的一个轻量级、无依赖节点编辑器扩展，旨在提供简单的即时模式接口，以便在 ImGui 窗口中构建节点编辑器。

主要功能包括：

1.  **节点编辑**：支持创建、管理节点、链接（Link）及输入/输出引脚（Pin），所有状态由用户控制。
2.  **UI 扩展**：允许在节点内部嵌套 ImGui 组件及自定义标题栏。
3.  **交互检测**：提供节点悬停、选中状态及链接创建/删除的检测功能。
4.  **样式定制**：支持自定义节点、引脚和链接的颜色及样式。
5.  **导航辅助**：内置交互式小地图（MiniMap），支持缩放和滚动以辅助大型图导航。
6.  **便捷集成**：仅需复制核心头文件与源文件，无需外部依赖，支持通过配置文件扩展。
