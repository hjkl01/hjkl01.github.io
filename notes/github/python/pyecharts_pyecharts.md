### [pyecharts pyecharts](https://github.com/pyecharts/pyecharts)  ![GitHub Repo stars](https://img.shields.io/github/stars/pyecharts/pyecharts?style=social)

pyecharts 是一款基于 Python 与 Apache ECharts 的数据可视化库。它提供简洁的链式 API 设计，支持 30 多种常见图表类型及 400 多个地图文件。项目支持 Jupyter Notebook 环境，可轻松集成至 Flask、Django 等 Web 框架，支持生成交互式 HTML 或静态图片，当前主要维护 Python 3.7+ 版本。
