### [pywebio PyWebIO](https://github.com/pywebio/PyWebIO)  ![GitHub Repo stars](https://img.shields.io/github/stars/pywebio/PyWebIO?style=social)

PyWebIO 是一个 Python 库，支持使用脚本方式编写交互式 Web 应用程序及浏览器端 GUI 应用，无需掌握 HTML 和 JavaScript 知识。它通过命令式函数在浏览器中实现用户输入与输出，可将普通脚本快速转换为 Web 应用。PyWebIO 支持集成到现有 Web 服务框架（如 Flask、Django 等），兼容 asyncio 异步编程，并支持第三方数据可视化库。
