### [dpgaspar Flask-AppBuilder](https://github.com/dpgaspar/Flask-AppBuilder)  ![GitHub Repo stars](https://img.shields.io/github/stars/dpgaspar/Flask-AppBuilder?style=social)

Flask App Builder 是一个基于 Flask 的简单快速应用开发框架，提供自动 CRUD 生成、详细安全配置及图表集成等功能。

核心功能包括：
1. **数据库**：基于 SQLAlchemy，支持多种数据库（SQLite、MySQL 等）及 MongoDB，具备审计和多连接支持。
2. **安全**：自动权限查找，基于角色的权限控制，支持 OAuth、OpenID、LDAP、数据库等多种认证方式及用户自注册。
3. **视图与组件**：自动生成菜单和 CRUD 界面，提供多种过滤器、Google 图表、表单组件（Select2、日期选择器）及扩展系统。
4. **REST API**：自动生成 CRUD RESTful 接口，支持 JWT 保护、国际化及数据验证。
5. **表单**：根据数据库模型自动生成表单，支持文件上传、自定义验证器和字段集。
6. **国际化与 UI**：支持多语言（Babel），内置 Bootstrap、Font-Awesome 图标及前端交互库。
