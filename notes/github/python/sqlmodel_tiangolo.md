### [tiangolo sqlmodel](https://github.com/tiangolo/sqlmodel)  ![GitHub Repo stars](https://img.shields.io/github/stars/tiangolo/sqlmodel?style=social)

SQLModel 是一个基于 Pydantic 和 SQLAlchemy 的 Python 库，用于通过 Python 对象与 SQL 数据库进行交互。它设计简洁、易用且兼容性强，特别适用于 FastAPI 应用开发。该库允许将 SQL 表定义为 Python 类，通过类实例代表数据行，从而最小化代码重复。它支持数据的增删改查操作，并提供优秀的编辑器自动补全和类型检查支持。
