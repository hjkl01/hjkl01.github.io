### [carltongibson django-filter](https://github.com/carltongibson/django-filter)  ![GitHub Repo stars](https://img.shields.io/github/stars/carltongibson/django-filter?style=social)

django-filter 是一个可复用的 Django 应用，允许用户通过 URL 参数声明式地动态过滤 QuerySet。它提供类似 Django admin 的列表过滤接口，API 设计类似于 ModelForms，支持构建可定制的过滤集（FilterSet）。此外，该项目还支持 Django REST Framework (DRF) 集成，用于在 API 视图中实现数据过滤。
