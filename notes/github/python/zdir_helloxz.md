### [helloxz zdir](https://github.com/helloxz/zdir)  ![GitHub Repo stars](https://img.shields.io/github/stars/helloxz/zdir?style=social)

Zdir 是一款基于 Golang 和 Vue3 开发的多功能私有存储程序，适用于 NAS 或 VPS 部署。它提供文件索引、管理（上传、下载、删除、重命名、复制移动）、多格式在线预览（支持图片、文档、音乐、视频）以及私有文件分享功能。系统支持 WebDAV 服务、API 接口、全局搜索、文本编辑、音乐播放模式和多语言界面。自 4.x 版本起，该项目已转为闭源。
