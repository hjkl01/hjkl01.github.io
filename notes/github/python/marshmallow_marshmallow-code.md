### [marshmallow-code marshmallow](https://github.com/marshmallow-code/marshmallow)  ![GitHub Repo stars](https://img.shields.io/github/stars/marshmallow-code/marshmallow?style=social)

marshmallow 是一个与 ORM、ODM 及框架无关的 Python 库，用于简化复杂数据类型的序列化与反序列化。它能够将对象转换为原生 Python 数据类型，反之亦然。主要功能包括验证输入数据、将输入数据反序列化为应用层对象、将对象序列化为原始类型（如 JSON），适用于 HTTP API 等场景。
