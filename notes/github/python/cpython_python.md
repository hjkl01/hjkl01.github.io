### [python cpython](https://github.com/python/cpython)  ![GitHub Repo stars](https://img.shields.io/github/stars/python/cpython?style=social)

本项目是 Python 3.15.0 alpha 6 版本的 CPython 官方源代码仓库。主要提供 Python 解释器的跨平台构建与安装指南，支持 Unix、Linux、BSD、macOS、Cygwin 及 Windows 系统。包含编译优化选项（如 PGO 和 LTO）、测试验证脚本及多版本共存管理方法。提供官方文档、开发者指南及发布计划链接。项目采用 Python 软件基金会许可证，不含 GPL 代码，可用于专有项目。
