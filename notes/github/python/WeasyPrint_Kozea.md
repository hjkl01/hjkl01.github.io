### [Kozea WeasyPrint](https://github.com/Kozea/WeasyPrint)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kozea/WeasyPrint?style=social)

WeasyPrint 是一款基于 Python 的 HTML 和 CSS 视觉渲染引擎，用于将网页转换为 PDF 文档（如统计报告、发票、票据等）。它支持 Web 打印标准，不依赖 WebKit 或 Gecko 等完整渲染引擎，而是采用专为分页设计的 Python CSS 布局引擎。该项目遵循 BSD 许可证，是自由软件，兼容 Python 3.10+ 环境。
