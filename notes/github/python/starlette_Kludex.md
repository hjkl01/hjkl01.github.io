### [Kludex starlette](https://github.com/Kludex/starlette)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kludex/starlette?style=social)

Starlette 是一个轻量级 ASGI 框架/工具包，专为构建 Python 异步 Web 服务设计。它提供轻量级 HTTP 框架、WebSocket 支持、进程内后台任务、启动和关闭事件、基于 httpx 的测试客户端、CORS、GZip、静态文件、流式响应、会话和 Cookie 支持等功能。Starlette 代码 100% 类型标注，依赖极少，兼容 asyncio 和 trio 后端，既可作为完整框架使用，也可作为 ASGI 工具包独立调用组件。
