### [mingrammer diagrams](https://github.com/mingrammer/diagrams)  ![GitHub Repo stars](https://img.shields.io/github/stars/mingrammer/diagrams?style=social)

Diagrams 是一个基于 Python 的“架构即代码”工具，允许用户通过编写代码绘制云系统架构图。它支持 AWS、Azure、GCP、Kubernetes 等多种主流云服务及本地环境，适用于架构原型设计和现有系统可视化，并支持通过版本控制追踪架构变更。该工具仅用于绘图，不涉及云资源管理或部署代码生成。
