### [meilisearch meilisearch](https://github.com/meilisearch/meilisearch)  ![GitHub Repo stars](https://img.shields.io/github/stars/meilisearch/meilisearch?style=social)

Meilisearch 是一款极速且易于集成的搜索引擎，专为提升应用、网站和工作流程的搜索体验而设计。其核心功能包括：
- **混合搜索**：结合语义搜索与全文搜索，以获取最相关结果。
- **高性能**：提供“即时搜索”体验，响应时间通常少于 50 毫秒。
- **搜索优化**：支持拼写容错、同义词、过滤、排序、分面搜索及地理空间搜索。
- **多语言支持**：优化支持中文、日文、希伯来语及拉丁字母语言。
- **集成与安全**：提供 RESTful API 及多语言 SDK，支持多租户、API 密钥权限管理。
- **AI 就绪**：原生兼容 LangChain 和模型上下文协议 (MCP)。
- **灵活部署**：提供开源社区版（MIT 授权）和企业版，易于安装、部署和维护。
