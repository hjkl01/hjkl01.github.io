### [awesome-rust](https://github.com/rust-unofficial/awesome-rust) ![GitHub Repo stars](https://img.shields.io/github/stars/rust-unofficial/awesome-rust?style=social)

Awesome Rust 是一个精选的 Rust 生态资源导航项目，核心功能是按类别系统收录并维护大量 Rust **应用、开发工具、库、注册表与学习资源**，帮助开发者快速发现可用项目与技术方案。

项目内容覆盖：
- **应用**：终端工具、数据库、区块链、网络/安全、编辑器、Web、虚拟化、MLOps、生产力等众多领域；
- **开发工具**：构建、调试、部署、测试、静态分析、格式化、IDE、FFI、嵌入式等；
- **库（Crates）**：AI、异步、网络、数据库、图形/GUI、加密、数据处理、解析、文本处理、Web 编程等几乎全栈方向；
- **注册表与资源**：Rust 包注册表方案、教程、课程、书籍、性能与最佳实践资料。

总体上，它不是单一功能软件，而是一个面向 Rust 开发者的**高质量索引与知识入口**。
