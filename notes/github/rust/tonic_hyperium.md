### [hyperium tonic](https://github.com/hyperium/tonic)  ![GitHub Repo stars](https://img.shields.io/github/stars/hyperium/tonic?style=social)

Tonic 是一个基于 Rust 的高性能 gRPC 框架实现，基于 HTTP/2 协议。它专注于提供一流的异步 IO 支持、互操作性和灵活性，旨在作为 Rust 生产系统的核心构建块。核心包含通用 gRPC 实现、基于 Hyper 的 HTTP/2 实现以及由 Prost 驱动的代码生成工具，支持双向流、TLS 加密、负载均衡、自定义元数据、认证和健康检查等功能。
