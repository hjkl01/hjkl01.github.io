### [sunface rust-course](https://github.com/sunface/rust-course)  ![GitHub Repo stars](https://img.shields.io/github/stars/sunface/rust-course?style=social)

“Rust 语言圣经”是一个全面的 Rust 编程教程，覆盖从入门到精通的完整知识体系。主要功能包括：提供深入解析与高级专题（如性能优化、异步编程、标准库解析等）；提供实战代码片段（Cookbook）及配套练习题；帮助规避常见陷阱及解决编译器错误；具备优秀的内容索引能力。旨在为中国用户打造免费、持续更新的高质量 Rust 学习资源与工具书。
