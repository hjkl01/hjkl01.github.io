### [charlotte-os Catten](https://github.com/charlotte-os/Catten)  ![GitHub Repo stars](https://img.shields.io/github/stars/charlotte-os/Catten?style=social)

Catten 是 CharlotteOS 的通用单体操作系统内核，采用 Rust 编写，强调安全、清晰度与灵活性。其核心功能包括提供极简的底层系统调用接口，以及基于能力的类型安全 URI 风格系统命名空间，支持跨主机命名空间访问。Catten 支持 x86-64、Aarch64 和 RISC-V64 架构，依赖标准文档化的固件管理硬件资源，涵盖内存、存储、显示、输入及网络等基础硬件驱动支持。项目目前处于早期开发阶段。
