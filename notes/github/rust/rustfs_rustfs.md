### [rustfs rustfs](https://github.com/rustfs/rustfs)  ![GitHub Repo stars](https://img.shields.io/github/stars/rustfs/rustfs?style=social)

RustFS 是一款基于 Rust 构建的高性能分布式对象存储系统。它完全兼容 S3 协议，采用 Apache 2.0 开源许可证，具备内存安全性和高性能优势，适用于数据湖、AI 及大数据工作负载。系统支持分布式架构、版本控制、存储桶复制、生命周期管理及 OpenStack Keystone 认证。提供 Web 管理控制台，支持 Docker、Kubernetes Helm、Nix 及源码等多种部署方式，旨在为企业级应用提供安全、合规且易于管理的存储解决方案。
