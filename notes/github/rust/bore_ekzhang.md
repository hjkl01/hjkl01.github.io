### [ekzhang bore](https://github.com/ekzhang/bore)  ![GitHub Repo stars](https://img.shields.io/github/stars/ekzhang/bore?style=social)

bore 是一个基于 Rust 开发的现代轻量级 TCP 隧道工具，用于将本地端口暴露到远程服务器，从而绕过 NAT 防火墙。与 ngrok 和 localtunnel 类似，bore 专注于简洁高效的 TCP 流量转发，支持多种安装方式且易于自建部署。项目仅需客户端和服务器两个二进制文件，并提供可选的密钥认证功能以确保连接安全。
