### [rust-lang rust-clippy](https://github.com/rust-lang/rust-clippy)  ![GitHub Repo stars](https://img.shields.io/github/stars/rust-lang/rust-clippy?style=social)

Clippy 是一个 Rust 静态分析工具，旨在通过超过 800 种 lint 规则捕获常见错误并改进代码质量。它支持按类别（如正确性、风格、性能等）管理 lint 的默认检查级别。可用作 Cargo 子命令运行，支持自动应用修复建议，也可集成至 CI 环境或在非 Cargo 项目中使用。用户可通过配置文件自定义 lint 的允许/禁止策略及最小支持的 Rust 版本（MSRV）。
