### [sunface rust-by-practice](https://github.com/sunface/rust-by-practice)  ![GitHub Repo stars](https://img.shields.io/github/stars/sunface/rust-by-practice?style=social)

这是一个用于通过实战练习精通 Rust 语言的项目。它提供大量在线可读、编辑和运行的挑战性示例与练习，涵盖异步、线程、标准库、算法等 Rust 各个方面。项目采用章节制，每章包含示例、练习和实践，难度从简单到超难，并配有完整解决方案。支持在线访问 (practice.rs) 和 mdbook 本地运行，旨在填补 Rust 学习与实际项目开发之间的差距。
