### [RustPython RustPython](https://github.com/RustPython/RustPython)  ![GitHub Repo stars](https://img.shields.io/github/stars/RustPython/RustPython?style=social)

RustPython 是一个完全使用 Rust 编写的 Python 3 解释器（兼容 CPython >= 3.14.0），提供不依赖 CPython 绑定的 Python 环境。核心功能包括本地运行、WebAssembly (WASI) 编译及 Rust 应用嵌入。支持虚拟环境、pip 包管理（需 SSL 支持）及实验性 JIT 编译器。项目目前处于开发阶段，尚未完全投入生产，欢迎社区贡献完善兼容性。
