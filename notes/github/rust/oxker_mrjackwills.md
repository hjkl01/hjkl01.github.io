### [mrjackwills oxker](https://github.com/mrjackwills/oxker)  ![GitHub Repo stars](https://img.shields.io/github/stars/mrjackwills/oxker?style=social)

oxker 是一个基于 Rust 语言开发的轻量级终端用户界面（TUI）工具，利用 ratatui 和 Bollard 库实现 Docker 容器的查看与控制。其核心功能包括：容器管理（状态监控、启停及进入执行命令）、日志处理（浏览、筛选、搜索及保存）、交互配置（自定义快捷键、配色及运行参数）以及多平台部署支持（Cargo、Docker、Nix 等多种安装方式）。
