### [commonwarexyz monorepo](https://github.com/commonwarexyz/monorepo)  ![GitHub Repo stars](https://img.shields.io/github/stars/commonwarexyz/monorepo?style=social)

Commonware 是一个面向对抗环境的 Rust 库集合，提供构建分布式系统所需的核心基础原语。功能涵盖广播、数据编解码、容错编码、拜占庭共识、密码学、P2P 通信、并行计算、数据存储及异步运行时。项目包含多种示例应用（如区块链），并设有从 ALPHA 到 EPSILON 的稳定性分级制度以确保依赖安全。此外，支持 MCP 协议与大语言模型集成，采用 MIT 和 Apache 2.0 双重许可。
