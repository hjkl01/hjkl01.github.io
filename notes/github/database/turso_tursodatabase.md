### [tursodatabase turso](https://github.com/tursodatabase/turso)  ![GitHub Repo stars](https://img.shields.io/github/stars/tursodatabase/turso?style=social)

Turso Database 是一款使用 Rust 编写的进程内 SQL 数据库，完全兼容 SQLite。它支持 Go、JavaScript、Java、Python、Rust 和 WebAssembly 多语言绑定，提供异步 I/O、跨平台运行（Linux、macOS、Windows、浏览器）及向量搜索功能。核心特性包括变更数据捕获（CDC）、改进的架构管理、多版本并发控制（MVCC）、数据加密、增量计算和全文搜索。项目配备命令行工具及 MCP 服务器模式，便于 AI 助手与数据库交互。目前处于 Beta 阶段，不建议直接用于生产环境。
