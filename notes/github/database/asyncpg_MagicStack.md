### [MagicStack asyncpg](https://github.com/MagicStack/asyncpg)  ![GitHub Repo stars](https://img.shields.io/github/stars/MagicStack/asyncpg?style=social)

asyncpg 是专为 PostgreSQL 和 Python/asyncio 设计的高性能数据库客户端库。它基于 asyncio 框架，原生实现 PostgreSQL 服务器二进制协议，查询性能约为 psycopg3 的 5 倍。核心功能包括支持预编译语句、可滚动游标、查询结果部分迭代、复合类型与数组的自动编解码及自定义数据类型支持，适用于 Python 3.9+ 及 PostgreSQL 9.5 至 18 版本。
