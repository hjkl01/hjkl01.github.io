### [paradedb paradedb](https://github.com/paradedb/paradedb)  ![GitHub Repo stars](https://img.shields.io/github/stars/paradedb/paradedb?style=social)

ParadeDB 是一个基于 PostgreSQL 的现代搜索与分析引擎，作为 Elasticsearch 的替代品，专为实时及高频更新的工作负载设计。它增强了 PostgreSQL 的搜索与分析功能，支持通过 Docker 或 Kubernetes 快速部署，兼容 PostgreSQL 15+ 版本，并提供开源及商业授权选项。
