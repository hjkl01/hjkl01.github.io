### [readysettech readyset](https://github.com/readysettech/readyset)  ![GitHub Repo stars](https://img.shields.io/github/stars/readysettech/readyset?style=social)

Readyset 是一个面向 PostgreSQL 和 MySQL 的透明数据库缓存系统。它部署在应用与数据库之间，利用数据库复制流自动同步缓存查询结果，无需手动处理缓存失效或重写应用代码。Readyset 能将复杂 SQL 读取转换为极速查找，提供内存键值存储的性能与扩展性，兼容现有 ORM 和数据库客户端，支持 Docker、二进制文件及云服务部署。
