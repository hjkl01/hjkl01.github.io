### [Studio3T robomongo](https://github.com/Studio3T/robomongo)  ![GitHub Repo stars](https://img.shields.io/github/stars/Studio3T/robomongo?style=social)

Robo 3T 是一款基于 shell 的跨平台 MongoDB 图形化管理工具，其核心功能是在标签页界面中嵌入真实的 mongo shell，支持命令行访问与图形界面交互。该项目已停止开发，最终版本为 1.4.4（支持 MongoDB 4.2 shell），建议用户尝试 Studio 3T Free。软件支持 Windows、macOS 和 Linux 平台。
