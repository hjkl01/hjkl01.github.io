### [dbeaver dbeaver](https://github.com/dbeaver/dbeaver)  ![GitHub Repo stars](https://img.shields.io/github/stars/dbeaver/dbeaver?style=social)

DBeaver 是一款免费的跨平台数据库管理工具，面向开发者、数据库管理员及分析师。它支持超过 100 种数据库驱动（基于 JDBC/ODBC），提供 SQL 编辑、数据管理、ER 图、数据迁移、执行计划分析及 AI 代码生成等功能。社区版支持主流关系型数据库，商业版额外支持 NoSQL、云数据库及扁平文件。项目基于 Java 和 Eclipse RCP 架构构建，并提供基于 Web 的云端管理工具 CloudBeaver。
