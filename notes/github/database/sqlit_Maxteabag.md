### [Maxteabag sqlit](https://github.com/Maxteabag/sqlit)  ![GitHub Repo stars](https://img.shields.io/github/stars/Maxteabag/sqlit?style=social)

sqlit 是一款基于终端的轻量级 SQL 数据库客户端工具（TUI），旨在提供类似 lazygit 的便捷查询体验。

核心功能总结：

1.  **广泛数据库支持**：兼容 SQL Server、PostgreSQL、MySQL、SQLite 等主流数据库及 Snowflake、BigQuery、Supabase 等云数据库。
2.  **终端交互体验**：支持 Vim 风格快捷键、SQL 语法高亮、查询历史记录、结果集过滤及模糊搜索。
3.  **便捷连接管理**：支持自动发现 Docker 容器、保存并切换连接配置、通过 SSH 隧道安全连接及操作系统密钥存储密码。
4.  **命令行集成**：支持 CLI 模式执行 SQL 查询、导出 CSV/JSON 格式及管理数据库连接。
5.  **易用性特性**：无需配置文件即可启动，支持智能代码补全、多主题切换及缺失驱动的自动安装提示。
