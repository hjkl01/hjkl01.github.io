### [danvergara dblab](https://github.com/danvergara/dblab)  ![GitHub Repo stars](https://img.shields.io/github/stars/danvergara/dblab?style=social)

dblab 是一个基于 Go 语言编写的轻量级交互式终端数据库客户端，支持 PostgreSQL、MySQL、SQLite3、Oracle 和 SQL Server。它跨平台运行于 OSX、Linux 和 Windows，采用单二进制零依赖发布。主要功能包括支持 SSH 隧道和 SSL 加密连接、通过配置文件管理多数据库连接、执行 SQL 查询、查看表结构索引及约束、使用树状结构管理数据库，以及支持自定义快捷键。
