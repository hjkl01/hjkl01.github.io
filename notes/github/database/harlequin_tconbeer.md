### [tconbeer harlequin](https://github.com/tconbeer/harlequin)  ![GitHub Repo stars](https://img.shields.io/github/stars/tconbeer/harlequin?style=social)

Harlequin 是一款终端 SQL 集成开发环境（IDE）。通过插件适配器支持连接多种数据库（默认内置 DuckDB，可选 PostgreSQL、MySQL、SQLite 等），支持在命令行编辑和执行 SQL 查询、浏览数据目录及导出数据。支持自定义主题、快捷键和配置文件，兼容 Linux、MacOS、Windows 系统，并提供 Django 框架集成。
