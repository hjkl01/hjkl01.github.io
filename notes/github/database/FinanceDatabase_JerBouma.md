### [JerBouma FinanceDatabase](https://github.com/JerBouma/FinanceDatabase)  ![GitHub Repo stars](https://img.shields.io/github/stars/JerBouma/FinanceDatabase?style=social)

FinanceDatabase 是一个提供免费金融产品分类信息的 Python 项目，数据库包含 30 万余个金融符号，涵盖股票、ETF、基金、指数、货币、加密货币及货币市场。它旨在帮助用户获取全球 100 多个国家不同行业的产品概况，支持通过 Python 包按国家、行业、交易所等条件灵活筛选。项目不直接提供实时基本面数据，但可与 FinanceToolkit 集成进行财务分析。所有数据源自公开免费资源，完全免费，并依靠社区协作维护更新。
