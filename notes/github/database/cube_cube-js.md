### [cube-js cube](https://github.com/cube-js/cube)  ![GitHub Repo stars](https://img.shields.io/github/stars/cube-js/cube?style=social)

Cube Core 是一款开源的无头语义层，用于构建嵌入式分析、自定义商业智能（BI）工具或为 AI 代理提供数据上下文。它支持 REST、GraphQL 和 SQL 等多种 API，兼容所有 SQL 数据源，内置关系缓存引擎以确保低延迟和高并发。项目支持 Docker 本地部署或自托管，旨在解耦语义层与特定工具，实现业务指标定义的一次编写和跨平台复用。
