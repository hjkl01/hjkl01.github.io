### [pgmodeler pgmodeler](https://github.com/pgmodeler/pgmodeler)  ![GitHub Repo stars](https://img.shields.io/github/stars/pgmodeler/pgmodeler?style=social)

pgModeler 是一款专为 PostgreSQL 生态系统设计的开源跨平台数据库建模工具，支持 Linux、Windows 和 macOS 系统。核心功能包括：

1. **可视化建模**：通过拖拽界面创建和编辑复杂的 PostgreSQL 对象（如表、视图、函数等）。
2. **反向工程**：连接现有数据库实例，快速生成视觉模型，用于文档记录、审计或修改。
3. **智能同步（Diff）**：比较模型与实时数据库或模型文件，自动生成 SQL 脚本以同步结构差异，并自动处理依赖关系。
4. **集成管理**：内置数据库管理模块，支持执行自定义 SQL 命令、浏览表数据及管理服务器对象。

项目采用 GPLv3 协议开源，并由 Nullptr Labs 提供专业支持与服务。
