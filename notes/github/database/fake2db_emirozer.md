### [emirozer fake2db](https://github.com/emirozer/fake2db)  ![GitHub Repo stars](https://img.shields.io/github/stars/emirozer/fake2db?style=social)

fake2db 是一款用于测试目的的数据库生成工具，可生成包含符合常见模式的真实有效假数据的数据库。支持 SQLite、MySQL、PostgreSQL、MongoDB、Redis 和 CouchDB 等数据库类型。用户可通过命令行参数自定义数据行数、数据库名称、连接信息、语言环境及随机种子，并支持自定义数据列生成。
