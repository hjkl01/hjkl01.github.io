### [surrealdb surrealdb](https://github.com/surrealdb/surrealdb)  ![GitHub Repo stars](https://img.shields.io/github/stars/surrealdb/surrealdb?style=social)

SurrealDB 是基于 Rust 构建的多模型数据库，统一支持文档、图、关系型、时序、地理空间和键值数据类型。它集数据库、API 后端及服务端权限控制于一体，支持实时数据同步、事件驱动、全文/向量/混合搜索及行级访问控制。作为单一可执行文件，它可运行于嵌入式、浏览器（WebAssembly）、边缘、自托管或分布式集群环境。支持 SurrealQL 等多种查询语言及多语言 SDK，旨在简化架构，加速构建高性能、安全的数据密集型应用。
