### [blizzless blizzless-diiis](https://github.com/blizzless/blizzless-diiis)  ![GitHub Repo stars](https://img.shields.io/github/stars/blizzless/blizzless-diiis?style=social)

DiIiS 是《暗黑破坏神 III：夺魂之镰》的开源本地服务器项目。核心功能包括账号创建与认证、大厅及社交系统（聊天、好友、公会）、游戏机制（全职业技能、地牢与物品生成、套装、五章剧情、冒险模式及挑战魔域裂缝基础）、怪物 AI 及局域网支持。

技术实现基于 .NET 7，支持 PostgreSQL 或 Docker 部署，兼容客户端版本 2.7.4.84161，提供服务器命令系统与全局配置，已移除捐赠商店。后续规划聚焦于大秘境完善、物品合成、任务奖励及游戏平衡优化。
