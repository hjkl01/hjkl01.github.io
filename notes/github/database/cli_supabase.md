**Summarizing Supabase CLI 功能**

Supabase CLI 提供本地运行、管理数据库迁移、函数操作、生成类型、通过管理 API 进行认证请求和引导等功能。

Supabase CLI 提供命令行工具，可在本地运行 Supabase、管理数据库迁移、创建与部署 Supabase Functions、根据数据库 schema 生成类型，并对管理 API 发起认证请求，还支持引导式创建 Supabase 项目。
