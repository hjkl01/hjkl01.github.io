### [TaKO8Ki gobang](https://github.com/TaKO8Ki/gobang)  ![GitHub Repo stars](https://img.shields.io/github/stars/TaKO8Ki/gobang?style=social)

gobang 是一款基于 Rust 编写的跨平台 TUI 数据库管理工具，目前处于 Alpha 版本。它支持 macOS、Windows 和 Linux 操作系统，兼容 MySQL、PostgreSQL 和 SQLite 数据库。该工具采用纯键盘控制的操作界面（类似 Vim 风格快捷键），用户可通过 TOML 配置文件来管理数据库连接。
