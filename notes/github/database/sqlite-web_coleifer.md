### [coleifer sqlite-web](https://github.com/coleifer/sqlite-web)  ![GitHub Repo stars](https://img.shields.io/github/stars/coleifer/sqlite-web?style=social)

sqlite-web 是一个基于 Web 的 SQLite 数据库浏览器和管理工具，由 Python 开发。它允许用户在浏览器中直接浏览、查询和管理 SQLite 数据库，支持创建新库。主要功能包括：管理表结构（增删改列、索引）、数据增删改查、执行自定义 SQL 查询、导入导出 JSON 和 CSV 文件，以及运行时动态加载数据库。支持通过命令行、Docker 容器或高性能 gevent WSGI 服务器部署，并提供密码保护、SSL 加密、只读模式等安全配置选项。
