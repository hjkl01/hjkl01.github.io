### [aminalaee sqladmin](https://github.com/aminalaee/sqladmin)  ![GitHub Repo stars](https://img.shields.io/github/stars/aminalaee/sqladmin?style=social)

SQLAdmin 是一个基于 SQLAlchemy 模型的灵活管理界面。它支持 SQLAlchemy 同步与异步引擎，可无缝集成 Starlette 和 FastAPI 框架。项目使用 WTForms 构建表单，支持 SQLModel，界面基于 Tabler 设计。用户可通过定义模型视图快速构建和管理数据库内容的后台界面。
