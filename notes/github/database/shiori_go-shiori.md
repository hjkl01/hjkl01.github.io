### [go-shiori shiori](https://github.com/go-shiori/shiori)  ![GitHub Repo stars](https://img.shields.io/github/stars/go-shiori/shiori?style=social)

Shiori 是一款基于 Go 语言的简易书签管理器，支持命令行和 Web 界面。功能涵盖书签的增删改查、搜索，支持 Netscape 书签及 Pocket 数据的导入导出。采用单二进制文件分发，便携易用，支持多种数据库（如 SQLite3、PostgreSQL）。默认可解析网页内容并创建离线归档，并提供 Firefox 和 Chrome 浏览器扩展支持（Beta）。
