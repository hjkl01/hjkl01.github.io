### [NanmiCoder MediaCrawler](https://github.com/NanmiCoder/MediaCrawler)  ![GitHub Repo stars](https://img.shields.io/github/stars/NanmiCoder/MediaCrawler?style=social)

MediaCrawler 是一款基于 Playwright 的多平台自媒体数据采集工具，支持小红书、抖音、快手、B 站、微博、贴吧、知乎等平台。核心功能包括关键词搜索、指定帖子与创作者主页爬取、二级评论获取、IP 代理池支持及登录态缓存。采用浏览器自动化技术，无需 JS 逆向即可获取签名参数。支持 CSV、JSON、Excel、SQLite、MySQL 等多种数据格式存储。提供命令行及 WebUI 可视化操作界面。项目仅限学习研究使用，禁止非法用途。
