### [bytebase bytebase](https://github.com/bytebase/bytebase)  ![GitHub Repo stars](https://img.shields.io/github/stars/bytebase/bytebase?style=social)

Bytebase 是一款开源数据库 DevOps 工具，专注于数据库 CI/CD 流程。它提供 Web 协作平台，帮助开发者和 DBA 管理数据库 Schema 生命周期。核心功能包括：GitOps 集成、自动化迁移与回滚、SQL 代码审查、敏感数据脱敏、细粒度访问控制、审计日志及架构漂移检测。支持 PostgreSQL、MySQL、MongoDB 等多种数据库，适用于数据库版本控制、自动化部署、集中化管理及安全合规等场景。
