### [pingcap tidb](https://github.com/pingcap/tidb)  ![GitHub Repo stars](https://img.shields.io/github/stars/pingcap/tidb?style=social)

TiDB 是一款开源、云原生的分布式 SQL 数据库，具备高可用性、水平与垂直可扩展性、强一致性和高性能。其核心功能包括：支持分布式事务以确保 ACID 合规；采用计算存储分离架构，支持无缝扩展；内置 Raft 共识协议实现数据多副本及自动故障转移；结合 TiKV 和 TiFlash 引擎提供 HTAP（混合事务/分析处理）能力；高度兼容 MySQL 8.0 协议及工具生态；支持本地、Kubernetes 及 TiDB Cloud 等多种部署模式。项目遵循 Apache 2.0 开源协议。
