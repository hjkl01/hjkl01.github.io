### [dbx](https://github.com/t8y2/dbx) ![GitHub Repo stars](https://img.shields.io/github/stars/t8y2/dbx?style=social)

DBX 是一款轻量级（约 15MB）开源数据库工具，支持桌面端（macOS/Windows/Linux）、Docker 自托管和 Web 使用，内置 AI 助手与 MCP 能力。它可统一连接和管理 40+ 数据库（并可通过 JDBC 扩展更多），提供 SQL 编辑与执行、智能补全、历史与片段管理、数据表格浏览与编辑、导入导出、库间迁移、数据对比、ER 图、Schema 对比、执行计划、字段血缘等完整数据库开发与运维功能。
项目特色是“无额外运行时依赖”（无需 Java/Python）、内置 AI 生成/优化/纠错 SQL（含安全检查）、支持 Claude/OpenAI/Ollama、支持 Redis/MongoDB 专项浏览，以及通过 MCP/CLI 与 Claude Code、Cursor、Windsurf 等 AI 编程代理联动直接查库执行 SQL。
DBX 基于 Tauri + Vue3 + TypeScript + Rust 构建，采用 AGPL-3.0 开源协议，功能免费且默认不采集遥测数据。
