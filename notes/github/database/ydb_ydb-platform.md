### [ydb-platform ydb](https://github.com/ydb-platform/ydb)  ![GitHub Repo stars](https://img.shields.io/github/stars/ydb-platform/ydb?style=social)

YDB 是一款开源分布式 SQL 数据库，兼具高可用性、可扩展性与严格一致性及 ACID 事务。它支持行式和列式表及持久化队列，具备跨可用区容错能力和自动灾难恢复机制，存储和计算层支持独立水平扩展。YDB 支持多租户和 Serverless 模式，兼容 PostgreSQL 和 Kafka API，可通过 Ansible、Kubernetes 或手动方式部署，主要运行于 x86 64 位 Linux 平台。
