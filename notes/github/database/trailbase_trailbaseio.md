### [trailbaseio trailbase](https://github.com/trailbaseio/trailbase)  ![GitHub Repo stars](https://img.shields.io/github/stars/trailbaseio/trailbase?style=social)

TrailBase 是一个基于 Rust、SQLite 和 Wasmtime 构建的开源单可执行文件 Firebase 替代品。它提供亚毫秒级延迟、类型安全的 REST 及实时 API、多数据库支持、地理空间功能、SSR 及内置认证与管理后台。项目支持多种语言客户端库，易于自我托管，无需专用缓存即可确保数据一致性。
