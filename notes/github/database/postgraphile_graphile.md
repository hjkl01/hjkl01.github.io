### [graphile postgraphile](https://github.com/graphile/postgraphile)  ![GitHub Repo stars](https://img.shields.io/github/stars/graphile/postgraphile?style=social)

该项目是 Graphile 维护的 GraphQL 相关工具集（单仓库），核心功能包括：

1. **Grafast**：GraphQL.js 的高效计划与执行引擎，可替代 `execute` 方法，通过声明式计划解析器优化查询性能。
2. **PostGraphile**：基于 PostgreSQL 数据库自动构建高性能 GraphQL API 的工具，支持根据数据库模式生成 API 并允许定制。

项目还提供一系列辅助库，涵盖 Schema 构建系统、配置管理、安全 SQL 生成、数据库内省及高性能缓存等，旨在为 GraphQL 应用开发提供高效、灵活且可扩展的解决方案。
