### [centerofci mathesar](https://github.com/centerofci/mathesar)  ![GitHub Repo stars](https://img.shields.io/github/stars/centerofci/mathesar?style=social)

Mathesar 是一款基于 PostgreSQL 的开源 Web 应用，提供类电子表格的直观界面，使用户无需编写代码即可查看、编辑、查询和协作处理数据。

核心功能包括：
1. **自托管与安全**：支持 Docker 快速部署，数据保留在用户服务器，原生集成 PostgreSQL 角色权限控制。
2. **数据管理**：直接操作数据库模式、表结构及约束，支持导入导出、列迁移及自定义数据类型（如邮箱、URL）。
3. **查询与交互**：提供无 SQL 查询构建器、筛选排序分组功能，以及数据表单构建与分享。
4. **协作与权限**：支持多用户协作、表权限管理及外键关系可视化。

项目 100% 开源（GPLv3），处于公共 Beta 阶段，适用于生产环境。
