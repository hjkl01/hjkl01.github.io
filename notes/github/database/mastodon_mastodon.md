### [mastodon mastodon](https://github.com/mastodon/mastodon)  ![GitHub Repo stars](https://img.shields.io/github/stars/mastodon/mastodon?style=social)

Mastodon 是一个基于 ActivityPub 协议的免费开源社交网络服务器，属于 Fediverse（联邦宇宙）的一部分。它支持用户发布链接、图片、文本和视频，并实现不同服务器间的无缝互联互通。核心功能包括实时动态时间线、媒体附件处理、隐私设置及完善的安全审核工具（如屏蔽、拉黑、举报）。项目提供 OAuth2 和 REST API 支持第三方客户端接入，采用 Ruby on Rails、PostgreSQL 等技术栈构建，遵循 AGPLv3 开源协议。
