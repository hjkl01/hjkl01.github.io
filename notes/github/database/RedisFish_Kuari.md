### [Kuari RedisFish](https://github.com/Kuari/RedisFish)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kuari/RedisFish?style=social)

RedisFish 是一款跨平台的 Redis GUI 客户端。支持 Redis 基本数据类型处理及数据状态颜色区分；支持数据多选、表格双击编辑及右键放大编辑文本或 JSON；提供 JSON 自动格式化。键名列表支持分页查询、前缀过滤及列表/文件夹双模式；控制台支持单/多命令执行及日志查看；具备数据库监控（CPU、内存、客户端等）、发布/订阅图形化操作及 TLS 支持。此外还支持深色模式、多语言自动切换及 Mac 自动更新。
