### [okbob pspg](https://github.com/okbob/pspg)  ![GitHub Repo stars](https://img.shields.io/github/stars/okbob/pspg?style=social)

pspg 是一款专为表格数据浏览设计的 Unix 分页器，主要作为 psql 的增强分页工具，同时兼容 mysql、sqlite、pgcli 等多种命令行客户端。

主要功能包括：
1. **表格导航**：支持冻结首行或首列，按指定数值列排序，提供类似 vi/less 的键盘导航。
2. **界面交互**：支持鼠标操作，提供多种精美主题，支持 Unicode 边框。
3. **数据处理**：支持选定范围复制到剪贴板，支持行/列搜索，具备实时监控模式。
4. **数据导出**：支持 CSV、TSV、SQL 值等多种格式导出。
5. **环境支持**：兼容 Debian、RedHat、macOS 及 WSL2，支持通过环境变量或配置文件配置。
