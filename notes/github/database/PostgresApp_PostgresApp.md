### [PostgresApp PostgresApp](https://github.com/PostgresApp/PostgresApp)  ![GitHub Repo stars](https://img.shields.io/github/stars/PostgresApp/PostgresApp?style=social)

Postgres.app 是专为 Mac 设计的 PostgreSQL 数据库运行工具。它集成了 PostgreSQL 核心程序及多种扩展（如 PostGIS、pgvector、pldebugger 等），提供图形界面以管理服务器的启停，支持同时运行多个版本的 PostgreSQL。首次启动自动初始化数据库集群、创建用户及数据库，并以当前用户身份运行服务。项目还内置了丰富的命令行工具，涵盖 PostgreSQL、PostGIS 及 GDAL 等组件。
