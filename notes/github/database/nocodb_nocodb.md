### [nocodb nocodb](https://github.com/nocodb/nocodb)  ![GitHub Repo stars](https://img.shields.io/github/stars/nocodb/nocodb?style=social)

NocoDB 是一个快速在线构建数据库的无代码工具。它提供丰富的电子表格界面，支持表、列、行的增删改查，以及多种视图类型（网格、画廊、表单、看板、日历）和数据操作功能（排序、筛选、分组）。项目支持细粒度的访问控制、角色管理及公开或密码保护的共享。此外，它提供 REST API 和 SDK 用于编程访问，并通过集成服务（聊天、邮件、存储）实现工作流自动化。支持 Docker、Auto-upstall 及多平台二进制文件等多种部署方式。
