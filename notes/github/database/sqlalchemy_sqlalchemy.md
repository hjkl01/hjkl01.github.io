### [sqlalchemy sqlalchemy](https://github.com/sqlalchemy/sqlalchemy)  ![GitHub Repo stars](https://img.shields.io/github/stars/sqlalchemy/sqlalchemy?style=social)

SQLAlchemy 是一个 Python SQL 工具箱及对象关系映射器（ORM），提供高效、灵活的数据库访问能力。核心功能包括：工业级 ORM，支持声明式配置与对象透明持久化；关系导向查询系统，可在对象模型中直接使用连接、子查询等完整 SQL 特性；独立的 Core 层，提供 SQL 表达式语言、模式元数据及连接池；灵活的关联对象加载策略与缓存；以及数据库结构反射与生成。项目默认采用事务管理，使用绑定参数保障安全，并强调开发者对数据模型与架构设计的完全控制权。
