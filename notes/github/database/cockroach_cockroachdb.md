### [cockroachdb cockroach](https://github.com/cockroachdb/cockroach)  ![GitHub Repo stars](https://img.shields.io/github/stars/cockroachdb/cockroach?style=social)

CockroachDB 是一款云原生分布式 SQL 数据库，用于构建、扩展和管理现代数据密集型应用。它基于强一致性键值存储，支持水平扩展，具备抵抗磁盘、机器、机架及数据中心故障的高可用性，无需人工干预。支持强一致 ACID 事务和熟悉的 SQL API，兼容 PostgreSQL 协议，提供 CockroachCloud 托管服务或多种部署方式。
