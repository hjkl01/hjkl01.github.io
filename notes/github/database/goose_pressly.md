### [pressly goose](https://github.com/pressly/goose)  ![GitHub Repo stars](https://img.shields.io/github/stars/pressly/goose?style=social)

Goose 是一款数据库迁移工具，提供命令行（CLI）和库（Library）两种使用方式。支持多种数据库（如 Postgres、MySQL、SQLite 等）的模式管理，允许通过增量 SQL 文件或 Go 函数编写迁移脚本。核心功能涵盖迁移创建、执行、回滚、状态检查及版本控制，支持嵌入式迁移文件、SQL 环境变量替换、数据填充及混合版本控制策略。
