### [docker-mailserver docker-mailserver](https://github.com/docker-mailserver/docker-mailserver)  ![GitHub Repo stars](https://img.shields.io/github/stars/docker-mailserver/docker-mailserver?style=social)

Docker Mailserver 是一个生产就绪的容器化全功能邮件服务器。它集成 Postfix、Dovecot、Rspamd、ClamAV 等核心组件，提供 SMTP、IMAP、LDAP、反垃圾、反病毒、邮件身份验证及安全防御功能。项目仅使用配置文件管理数据，无需数据库，支持 Let's Encrypt 证书、OAuth2 认证及多种邮件处理工具，便于快速部署、升级和维护。
