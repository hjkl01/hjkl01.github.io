### [axllent mailpit](https://github.com/axllent/mailpit)  ![GitHub Repo stars](https://img.shields.io/github/stars/axllent/mailpit?style=social)

Mailpit 是一款专为开发者设计的轻量级、零依赖邮件测试工具，提供 SMTP 服务器、现代 Web 界面及 REST API。它用于捕获和测试邮件，支持查看邮件详情（HTML、源码、附件）、搜索过滤及实时通知。核心功能涵盖 SMTP/POP3 协议（支持 TLS 与认证）、邮件内容分析（HTML 兼容性、链接检查、垃圾邮件评分）、截图生成、消息标签化、SMTP 中继与转发、混沌工程（模拟 SMTP 错误）及 Webhook 通知。支持静态二进制或 Docker 部署，处理速度快，内存占用低。
