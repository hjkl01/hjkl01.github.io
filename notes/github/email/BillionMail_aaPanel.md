### [aaPanel BillionMail](https://github.com/aaPanel/BillionMail)  ![GitHub Repo stars](https://img.shields.io/github/stars/aaPanel/BillionMail?style=social)

BillionMail 是一款开源的邮件服务器及邮件营销平台，支持自托管以实现对邮件活动的完全控制。主要功能总结如下：

- **邮件营销**：支持发送新闻通讯、推广邮件和事务性消息，无发送数量限制。
- **数据分析**：提供送达率、打开率和点击率等高级分析指标。
- **管理功能**：内置客户管理系统及可复用的自定义营销模板。
- **WebMail**：集成 RoundCube，支持通过 Web 端收发邮件。
- **部署灵活**：支持 Docker 及 aaPanel 一键安装，配置简便。
- **隐私安全**：基于 AGPLv3 协议开源，数据私有化存储，无第三方追踪。
