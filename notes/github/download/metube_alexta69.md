### [alexta69 metube](https://github.com/alexta69/metube)  ![GitHub Repo stars](https://img.shields.io/github/stars/alexta69/metube?style=social)

MeTube 是一个基于 Web 的视频下载管理工具，后端使用 yt-dlp，支持从 YouTube 及其他数十个网站下载音视频和播放列表。项目支持 Docker 部署，可通过环境变量灵活配置下载并发、存储路径、文件命名、HTTPS 及反向代理设置。支持浏览器扩展（Chrome/Firefox）、iOS 快捷指令、Raycast 扩展及书签工具，方便将链接直接发送至下载。此外，还具备自定义目录、iOS 格式兼容转换、Cookie 支持（用于下载私有视频）及 yt-dlp 自动更新等高级功能。
