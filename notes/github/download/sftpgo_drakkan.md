### [drakkan sftpgo](https://github.com/drakkan/sftpgo)  ![GitHub Repo stars](https://img.shields.io/github/stars/drakkan/sftpgo?style=social)

SFTPGo 是一款全功能、高可配置的事件驱动文件传输解决方案。它支持 SFTP、HTTP/S、FTP/S 及 WebDAV 等多种服务器协议，兼容本地文件系统、加密存储、S3、Google Cloud Storage 和 Azure Blob 等多种存储后端。用户可利用本地和云存储后端进行内部或与合作伙伴间的文件交换与存储。项目提供功能完整的开源社区版和具备高性能、高可用及高级安全增强功能的企业版。
