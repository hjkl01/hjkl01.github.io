### [NickvisionApps Parabolic](https://github.com/NickvisionApps/Parabolic)  ![GitHub Repo stars](https://img.shields.io/github/stars/NickvisionApps/Parabolic?style=social)

Parabolic 是一款基于 yt-dlp 的视频下载前端应用。它支持从数百个网站下载视频，提供 mp4、webm、mp3、opus、flac 和 wav 等多种格式，支持并发下载任务以及元数据和字幕获取。该应用兼容 Linux、Windows、macOS 平台，并支持 Chrome 和 Firefox 浏览器扩展。
