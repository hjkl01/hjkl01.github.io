### [SeaHOH ykdl](https://github.com/SeaHOH/ykdl)  ![GitHub Repo stars](https://img.shields.io/github/stars/SeaHOH/ykdl?style=social)

YouKuDownLoader (ykdl) 是一款专注于中国大陆视频网站下载的视频工具。作为 you-get 的分支，该项目优化了代码结构，已不支持 Python 3.4 及以下版本，运行需依赖 FFmpeg 和 mpv。它支持通过 pip 等包管理器安装，致力于保持对国内视频站点的兼容性，持续进行功能增强、更新维护及 Bug 修复。
