### [cloudreve cloudreve](https://github.com/cloudreve/cloudreve)  ![GitHub Repo stars](https://img.shields.io/github/stars/cloudreve/cloudreve?style=social)

Cloudreve 是一款自托管的多云文件管理系统，支持本地及多种主流云存储（如 OneDrive、S3、阿里云 OSS 等）作为后端。核心功能包括客户端直传、后台下载（集成 Aria2/qBittorrent）、归档处理、WebDAV 支持、拖拽上传、元数据搜索、多用户多组管理、分享链接设置及在线预览/编辑多种文件。系统支持主题定制、PWA 应用与国际化的开箱即用方案，基于 Go 和 React 技术栈。
