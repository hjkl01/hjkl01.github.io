### [omniget](https://github.com/tonhowtf/omniget) ![GitHub Repo stars](https://img.shields.io/github/stars/tonhowtf/omniget?style=social)

OmniGet 是一款免费开源的多平台下载工具，主打“粘贴链接即可下载”。它可从 YouTube、Instagram、TikTok、Twitter/X、Reddit、Bilibili 等平台下载视频/图片内容，并通过 yt-dlp 支持 1000+ 网站。  
核心功能包括：链接解析与清晰度选择下载、课程平台整套课程下载（含课时/附件/描述）、内置 BT 下载（`.torrent` 与 magnet）、跨设备四词代码传文件。  
软件可自动管理并更新 yt-dlp 与 FFmpeg，无需额外配置；支持 Windows/macOS/Linux（含 Linux Flatpak、Windows 便携版）、多语言与多主题。  
还提供全局快捷键一键后台下载、Chrome 扩展（可传递页面链接与登录信息并检测网页视频流），以及插件系统（课程、Telegram、音视频格式转换等），并开放 Plugin SDK 供扩展开发。
