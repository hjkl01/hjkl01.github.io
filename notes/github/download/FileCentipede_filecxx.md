### [filecxx FileCentipede](https://github.com/filecxx/FileCentipede)  ![GitHub Repo stars](https://img.shields.io/github/stars/filecxx/FileCentipede?style=social)

File Centipede 是一款全能型互联网文件上传与下载管理器，支持 HTTP、HTTPS、FTP、WebDAV、SSH/SFTP 及 BitTorrent 等多种协议。软件配备浏览器扩展，支持从网页捕获音视频、图片和磁力链接。内置 HTTP 请求器、文件合并、编码/解码、翻译、校验和及 Torrent 管理等辅助工具。设计快速、可定制、支持多语言，完全免费且无广告。
