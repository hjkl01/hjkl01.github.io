### [yt-dlp yt-dlp](https://github.com/yt-dlp/yt-dlp)  ![GitHub Repo stars](https://img.shields.io/github/stars/yt-dlp/yt-dlp?style=social)

yt-dlp 是一个支持数千个网站的命令行音频/视频下载工具，由 youtube-dl 分支而来。它提供丰富的下载选项，包括格式选择与排序、字幕和缩略图下载、元数据修改、音频提取及章节分割等后处理功能。支持从浏览器导入 Cookie 认证、SponsorBlock 去广告、插件扩展以及嵌入 Python 使用。具备配置文件管理和自动更新机制，需 Python 3.10+ 环境，推荐使用 ffmpeg 以增强视频合并及处理功能。
