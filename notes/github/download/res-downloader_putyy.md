### [putyy res-downloader](https://github.com/putyy/res-downloader)  ![GitHub Repo stars](https://img.shields.io/github/stars/putyy/res-downloader?style=social)

爱享素材下载器是一款基于 Go 和 Wails 开发的跨平台资源下载工具，支持 Windows、macOS 和 Linux 系统。其核心功能是通过代理抓包技术，支持视频、音频、图片、m3u8 及直播流等多种资源类型的嗅探与下载。该软件兼容微信视频号、小程序、抖音、快手、小红书、酷狗音乐、QQ 音乐等多个平台，并支持代理设置以获取受限网络资源。界面简洁易用，用户只需开启代理并在外部资源页面加载内容，即可在软件内查看并下载目标素材。仅供学习与研究使用，禁止商业或违法用途。
