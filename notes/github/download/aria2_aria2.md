### [aria2 aria2](https://github.com/aria2/aria2)  ![GitHub Repo stars](https://img.shields.io/github/stars/aria2/aria2?style=social)

aria2 是一款高性能的命令行下载工具，支持 HTTP(S)、FTP、SFTP、BitTorrent 及 Metalink 协议。其核心功能包括通过分段和多源下载最大化带宽利用率，支持 HTTP 下载数据上传至 BitTorrent 集群，以及 Metalink 数据校验。项目提供 JSON-RPC 和 XML-RPC 远程接口，支持 daemon 模式。此外，还具备 BitTorrent 扩展（DHT、PEX、UDP Tracker 等）、下载速度控制、代理与 SSL 验证、Cookie 管理及 IPv6 支持等功能。
