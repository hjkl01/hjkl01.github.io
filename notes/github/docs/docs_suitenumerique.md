### [suitenumerique docs](https://github.com/suitenumerique/docs)  ![GitHub Repo stars](https://img.shields.io/github/stars/suitenumerique/docs?style=social)

La Suite Docs 是一款开源的实时协作文本编辑器，旨在作为 Notion 或 Google Docs 的替代方案。

**核心功能：**
*   **实时协作**：支持多人在线编辑、光标追踪、评论及细粒度访问控制。
*   **文档编辑**：支持富文本、Markdown、Slash 命令及区块系统，可选 AI 辅助写作。
*   **知识管理**：提供子页面层级结构及内容搜索功能。
*   **数据自主**：支持 Kubernetes、Docker 等多种方式自托管，强调数据所有权。
*   **互操作性**：支持 .docx、.md、.odt、.pdf 等格式的导入与导出。

该项目由法国和德国政府联合发起，采用 MIT 许可证，基于 Django、Next.js、ProseMirror 等技术构建。
