### [ducaale xh](https://github.com/ducaale/xh)  ![GitHub Repo stars](https://img.shields.io/github/stars/ducaale/xh?style=social)

xh 是一个基于 Rust 实现的快速命令行 HTTP 请求工具，复刻了 HTTPie 的语法与设计。它专注于性能优化，支持 HTTP/2。功能涵盖 HTTP 请求发送、JSON/表单/Multipart 数据处理、响应格式化与着色、会话管理、身份验证、代理、SSL 配置、文件下载、重定向及离线构建请求。支持 curl 命令翻译、单二进制文件分发，兼容 HTTPie 语法运行。
