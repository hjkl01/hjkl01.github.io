### [Kong insomnia](https://github.com/Kong/insomnia)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kong/insomnia?style=social)

Insomnia 是一款开源跨平台 API 客户端，支持 GraphQL、REST、WebSockets、SSE、gRPC 等 HTTP 兼容协议。核心功能包括 API 调试、基于 OpenAPI 的设计、测试套件运行、API 模拟（云或自托管）、CI/CD 集成及团队协作。支持本地 Vault、Git 同步和云端加密同步三种数据存储方式，并提供第三方插件扩展。软件兼容 Mac、Windows 和 Linux 系统，采用免费计划与付费订阅相结合的模式。
