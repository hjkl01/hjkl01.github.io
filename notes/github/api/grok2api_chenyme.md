### [chenyme grok2api](https://github.com/chenyme/grok2api)  ![GitHub Repo stars](https://img.shields.io/github/stars/chenyme/grok2api?style=social)

Grok2API 是基于 FastAPI 重构的 Grok 接口服务，全面适配最新 Web 调用格式。支持流式/非流式对话、工具调用、图像生成/编辑、视频生成/超分及深度思考功能。具备号池并发与自动负载均衡能力，提供 Token 管理、状态筛选、批量操作及 NSFW 模式开启等管理面板。兼容 OpenAI 接口标准，支持多种 Grok 模型，支持本地、Docker、Vercel 及 Render 等多种部署方式。
