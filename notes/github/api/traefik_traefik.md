### [traefik traefik](https://github.com/traefik/traefik)  ![GitHub Repo stars](https://img.shields.io/github/stars/traefik/traefik?style=social)

Traefik 是一个现代化的 HTTP 反向代理和负载均衡器，旨在简化微服务部署。它能够自动集成 Docker、Kubernetes、Swarm 等基础设施，通过监听服务注册中心 API 动态生成路由配置，无需手动维护或重启服务。核心功能包括配置自动更新、多种负载均衡算法、通过 Let's Encrypt 自动提供 HTTPS 支持、支持 WebSocket/HTTP/2/gRPC、提供指标监控和访问日志，以及内置 Web 管理界面。
