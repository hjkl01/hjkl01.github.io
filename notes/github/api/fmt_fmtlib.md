### [fmtlib fmt](https://github.com/fmtlib/fmt)  ![GitHub Repo stars](https://img.shields.io/github/stars/fmtlib/fmt?style=social)

`{fmt}` 是一个开源的 C++ 格式化库，旨在提供比 C stdio 和 C++ iostreams 更快、更安全的替代方案。它支持 C++20 std::format 和 C++23 std::print 标准，具有类似 Python 的格式字符串语法。核心特性包括高性能浮点格式化、便携式 Unicode 支持、类型安全、编译时格式检查、无外部依赖及跨平台兼容性。该库轻量、易用，广泛应用于各类大型项目中。
