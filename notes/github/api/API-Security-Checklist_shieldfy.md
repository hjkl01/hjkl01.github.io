### [shieldfy API-Security-Checklist](https://github.com/shieldfy/API-Security-Checklist)  ![GitHub Repo stars](https://img.shields.io/github/stars/shieldfy/API-Security-Checklist?style=social)

该项目提供了一份 API 安全清单，列出了在设计、测试和发布 API 时需要遵循的关键安全反制措施。内容涵盖认证授权、访问控制、输入输出验证、处理逻辑安全、CI/CD 流程、监控告警以及速率限制、GraphQL 安全、零信任架构等高级最佳实践，旨在帮助开发者全面保障 API 安全性。
