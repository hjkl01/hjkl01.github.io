### [vitalik django-ninja](https://github.com/vitalik/django-ninja)  ![GitHub Repo stars](https://img.shields.io/github/stars/vitalik/django-ninja?style=social)

Django Ninja 是一个基于 Django 和 Python 3.6+ 类型提示的快速 API 构建框架。其核心功能包括：

1. **高性能**：得益于 Pydantic 和异步支持，执行速度极快。
2. **快速开发**：利用类型提示和自动生成文档，让开发者专注于业务逻辑。
3. **标准兼容**：基于 OpenAPI (Swagger) 和 JSON Schema 标准。
4. **自动文档**：自动生成并展示交互式 API 文档（Swagger UI 或 Redoc）。
5. **Django 集成**：与 Django 核心及 ORM 深度集成，设计简单，适用于生产环境。
