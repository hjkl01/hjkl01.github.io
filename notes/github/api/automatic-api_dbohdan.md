### [dbohdan automatic-api](https://github.com/dbohdan/automatic-api)  ![GitHub Repo stars](https://img.shields.io/github/stars/dbohdan/automatic-api?style=social)

本项目是一个资源列表，收录了能够自动为数据库暴露 REST、GraphQL 或 HTTP API 的工具，以及自带 API 的数据库。列表详细列出了各工具支持的数据源、API 类型、实现语言、许可证及 GitHub 统计等信息，旨在帮助用户快速查找数据库 API 接入方案。
