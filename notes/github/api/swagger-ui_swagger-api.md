### [swagger-api swagger-ui](https://github.com/swagger-api/swagger-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/swagger-api/swagger-ui?style=social)

Swagger UI 是基于 OpenAPI 规范自动生成 API 可视化文档的工具，使开发团队和最终用户能够在无需后端实现逻辑的情况下查看和操作 API 资源。项目提供多种 NPM 模块及 React 组件版本，兼容多个 OpenAPI 规范版本，支持安装、配置、自定义布局及 OAuth2 集成等功能。
