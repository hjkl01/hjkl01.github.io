### [usebruno bruno](https://github.com/usebruno/bruno)  ![GitHub Repo stars](https://img.shields.io/github/stars/usebruno/bruno?style=social)

Bruno 是一款开源的 API 客户端 IDE，用于 API 的探索与测试。支持将 API 集合直接存储在本地文件系统，使用 Bru 标记语言保存，并通过 Git 进行协作。采用离线优先策略，无云端同步，注重数据隐私，支持 Mac、Windows 和 Linux 跨平台运行。大部分功能免费开源，同时提供付费商业版本。
