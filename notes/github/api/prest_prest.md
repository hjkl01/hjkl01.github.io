### [prest prest](https://github.com/prest/prest)  ![GitHub Repo stars](https://img.shields.io/github/stars/prest/prest?style=social)

pRESTd 是一个基于 Go 语言构建的轻量级生产就绪 RESTful API 服务器，可直接在现有或新的 PostgreSQL 数据库（9.5 及以上版本）之上运行，提供即时、实时且高性能的应用服务。其核心功能包括：轻量级易配置的服务器、支持通过 URL 模板进行直接 SQL 查询、针对高性能优化、内置身份认证和授权、以及支持可插拔的自定义路由和中间件。该项目旨在简化安全且高性能的 PostgreSQL RESTful API 开发流程。
