### [polarsource polar](https://github.com/polarsource/polar)  ![GitHub Repo stars](https://img.shields.io/github/stars/polarsource/polar?style=social)

Polar 是一款面向开发者的开源支付基础设施与变现平台。它支持销售 SaaS 和数字产品（如软件订阅、文件下载、GitHub 仓库访问及许可证密钥）。平台作为记录商，负责处理账单、收据、客户账户管理及销售税（VAT）等税务事务。提供公共 API 及多语言 SDK 供集成，采用 Python FastAPI 后端与 NextJS 前端架构。
