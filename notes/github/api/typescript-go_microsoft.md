### [microsoft typescript-go](https://github.com/microsoft/typescript-go)  ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/typescript-go?style=social)

该项目是 TypeScript 的新一代原生实现预览版（TypeScript 7），旨在最终合并至官方 TypeScript 仓库。支持通过 npm 安装并使用 `tsgo` 命令替代 `tsc`，同时提供 VS Code 扩展。目前已完成解析扫描、类型检查、JSX、构建模式及增量构建；声明生成、JS 输出及语言服务正在开发中；Watch 模式为原型，API 尚未就绪。当前版本功能与 TypeScript 6.0 尚未完全对齐，属于开发预览阶段。
