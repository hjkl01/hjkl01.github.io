### [cloudreve Cloudreve](https://github.com/cloudreve/Cloudreve)  ![GitHub Repo stars](https://img.shields.io/github/stars/cloudreve/Cloudreve?style=social)

Cloudreve 是一款自托管的云文件管理系统，支持多云存储（本地、OneDrive、S3 兼容及各大云厂商）。核心功能包括：客户端直传、拖拽上传及断点续传；集成下载工具实现后台任务；支持文件压缩解压、批量下载、在线预览与编辑、WebDAV 服务及媒体元数据搜索；具备多用户多分组管理及分享链接过期设置；界面支持主题定制、多语言及 PWA。系统提供开箱即用的部署方案。
