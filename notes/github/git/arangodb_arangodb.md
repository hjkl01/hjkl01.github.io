### [arangodb arangodb](https://github.com/arangodb/arangodb)  ![GitHub Repo stars](https://img.shields.io/github/stars/arangodb/arangodb?style=social)

ArangoDB 是一款可扩展的多模型数据库系统，集原生图数据库、JSON 文档存储和搜索引擎于一体，通过统一查询语言（AQL）实现数据操作。

*   **多模型存储**：支持键值、文档和图关系的混合存储，可灵活选择无模式或 Schema 验证。
*   **高可用与扩展**：支持水平分片、数据复制及自动故障转移，保障系统弹性与稳定性。
*   **查询能力**：支持事务处理、多种索引优化及 ArangoSearch 全文检索。
*   **部署与版本**：支持本地、云端及容器部署，提供 Web 界面和命令行工具，含免费社区版及无数据量限制的企业版。
