### [expr-lang expr](https://github.com/expr-lang/expr)  ![GitHub Repo stars](https://img.shields.io/github/stars/expr-lang/expr?style=social)

Expr 是一个专为 Go 语言设计的表达式求值器，专注于提供安全、快速且准确的动态配置能力。它具备内存安全、无副作用及保证程序终止的特性，支持静态类型检查以防止运行时错误，并能与 Go 类型无缝集成。Expr 内置丰富的运算符和内置函数（如 filter、map 等），采用优化编译器和字节码虚拟机实现高性能，广泛应用于业务规则引擎、动态策略配置及数据转换等场景。
