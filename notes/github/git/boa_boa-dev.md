### [boa-dev boa](https://github.com/boa-dev/boa)  ![GitHub Repo stars](https://img.shields.io/github/stars/boa-dev/boa?style=social)

Boa 是一个用 Rust 编写的实验性 JavaScript 引擎，具备词法分析、语法解析和解释执行能力，支持超过 90% 的最新 ECMAScript 规范。项目提供 AST、执行引擎、垃圾回收器等核心库模块，支持命令行运行脚本、REPL 交互、调试及流程图生成，并兼容 WebAssembly 环境。此外，项目还提供基准测试和性能分析工具。
