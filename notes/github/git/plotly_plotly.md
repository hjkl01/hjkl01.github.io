### [plotly plotly](https://github.com/plotly/plotly)  ![GitHub Repo stars](https://img.shields.io/github/stars/plotly/plotly?style=social)

基于 plotly.js 的 R 语言包，用于创建交互式网络图形。主要功能包括：
1. 通过 `ggplotly()` 将 ggplot2 静态图表转换为交互式网页图表。
2. 通过 `plot_ly()` 直接调用 plotly.js，支持平行坐标、地图、3D 曲面等多种 ggplot2 无法实现的专业图表。
3. 支持 Shiny 集成、客户端视图链接、静态图生成及自定义 JavaScript 事件。
