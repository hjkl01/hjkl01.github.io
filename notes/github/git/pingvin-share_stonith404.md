### [stonith404 pingvin-share](https://github.com/stonith404/pingvin-share)  ![GitHub Repo stars](https://img.shields.io/github/stars/stonith404/pingvin-share?style=social)

Pingvin Share 是一款自托管的文件分享平台（WeTransfer 替代品），目前已归档。核心功能包括：通过链接分享文件（大小仅受磁盘限制）、设置分享过期时间、密码及访客访问限制、邮件发送收件人、反向分享。系统支持 OIDC 和 LDAP 认证，集成 ClamAV 安全扫描，兼容本地存储及 S3 存储，推荐通过 Docker 部署。
