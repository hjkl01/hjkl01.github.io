### [yq](https://github.com/mikefarah/yq) ![GitHub Repo stars](https://img.shields.io/github/stars/mikefarah/yq?style=social)

yq 是一个轻量级、便携式命令行处理器，支持 YAML、JSON、XML、INI、properties、CSV、TSV 等多种数据格式。使用类似 jq 的语法，可读取、转换、合并、就地更新文件内容，支持多文档处理、格式互转、环境变量替换、注释保留等操作。用 Go 语言编写，提供无依赖二进制，支持 Homebrew、snap、Docker、Podman、Chocolatey、Scoop、winget 等多种安装方式。
