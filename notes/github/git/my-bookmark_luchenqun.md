### [luchenqun my-bookmark](https://github.com/luchenqun/my-bookmark)  ![GitHub Repo stars](https://img.shields.io/github/stars/luchenqun/my-bookmark?style=social)

本项目是一款在线书签管理工具，旨在解决本地书签易丢失、多浏览器整合难及搜索不便等问题。主要功能包括：
1. 用户与分类管理：支持注册登录，书签分类的增删改查及多维度排序（日期、点击量等）。
2. 书签操作：支持快速添加（自动抓取标题、快捷键操作）、Chrome 书签导入导出、公私有权限控制及转存他人书签。
3. 搜索与发现：支持按时间、分类、关键词多维度查询，提供热门书签展示。
4. 扩展功能：内置备忘录功能（快捷键添加与分享）、全局快捷键、Chrome 插件快速添加及手机平板移动端适配。
