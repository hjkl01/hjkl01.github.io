### [gopher64 gopher64](https://github.com/gopher64/gopher64)  ![GitHub Repo stars](https://img.shields.io/github/stars/gopher64/gopher64?style=social)

gopher64 是一款支持 Windows 和 Linux 平台的 Nintendo 64 游戏模拟器。主要功能涵盖在线联机对战（支持云端及自建服务器）、键盘与手柄的按键映射、以及便携模式（将数据保存在可执行文件同目录）。项目基于 Rust 语言开发，遵循 GPLv3 开源协议，核心代码源自 mupen64plus 和 ares。
