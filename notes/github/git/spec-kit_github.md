### [github spec-kit](https://github.com/github/spec-kit)  ![GitHub Repo stars](https://img.shields.io/github/stars/github/spec-kit?style=social)

Spec Kit 是一个开源工具包，支持规范驱动开发模式，旨在通过将规范变为可执行的代码实现来加速软件构建。它提供 Specify CLI 工具，支持集成多种主流 AI 编码代理（如 Claude、GitHub Copilot 等）。核心功能涵盖确立项目原则、创建功能规范、制定技术实施计划、分解任务列表及执行代码实现。该工具支持从零创建新项目或对现有项目进行功能迭代，帮助开发者专注于产品场景与预期结果，提高开发效率与代码质量。
