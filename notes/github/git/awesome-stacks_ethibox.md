### [ethibox awesome-stacks](https://github.com/ethibox/awesome-stacks)  ![GitHub Repo stars](https://img.shields.io/github/stars/ethibox/awesome-stacks?style=social)

Awesome Stacks 是一个基于 Docker Swarm 和 Traefik 的项目，允许用户通过一条 Docker 命令部署 150 多种开源 Web 应用。它支持零配置启动，利用 Traefik 和 Let's Encrypt 实现自动安全配置，并可通过环境变量自定义域名、存储路径及应用版本，同时兼容 Portainer 模板部署。
