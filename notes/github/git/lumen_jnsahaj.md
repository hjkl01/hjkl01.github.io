### [jnsahaj lumen](https://github.com/jnsahaj/lumen)  ![GitHub Repo stars](https://img.shields.io/github/stars/jnsahaj/lumen?style=social)

Lumen 是一款基于 Rust 的命令行工具，旨在为 AI 编程时代提供 Git 代码审查与辅助功能。

- **可视化 Diff 查看器**：支持语法高亮、行内批注、多主题定制、多分支/PR 对比及堆叠模式。
- **AI 代码助手**：自动生成规范提交信息、解释变更内容、根据自然语言生成 Git 命令。
- **多模型兼容**：支持 OpenAI、Claude、Ollama、Groq 等多种 AI 服务提供商。
- **高度可配置**：支持 CLI 参数、项目配置文件及环境变量管理。

通过 Cargo 或 Homebrew 安装。
