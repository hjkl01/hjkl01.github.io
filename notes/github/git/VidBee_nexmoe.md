### [nexmoe VidBee](https://github.com/nexmoe/VidBee)  ![GitHub Repo stars](https://img.shields.io/github/stars/nexmoe/VidBee?style=social)

VidBee 是一款基于 Electron 和 yt-dlp 构建的开源视频下载器，支持从全球 1000 多个网站（如 YouTube、TikTok、Instagram）下载视频和音频。软件提供简洁的桌面客户端界面，具备下载队列管理、断点续传和实时进度跟踪功能。核心特色支持 RSS 自动订阅，可自动在后台下载关注创作者的新视频。项目同时支持 Web 端和 API 部署，兼容 Docker，并提供浏览器扩展。
