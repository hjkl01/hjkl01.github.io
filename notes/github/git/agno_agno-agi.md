### [agno-agi agno](https://github.com/agno-agi/agno)  ![GitHub Repo stars](https://img.shields.io/github/stars/agno-agi/agno?style=social)

Agno 是一个用于构建、运行和管理大规模代理软件的运行时框架。

**主要功能：**
- **框架**：构建带记忆、知识库、护栏和 100+ 集成的代理、团队和工作流。
- **运行时**：无状态、会话隔离的 FastAPI 后端，支持快速部署为生产 API。
- **控制平面**：通过 AgentOS UI 进行测试、监控和管理。

**关键特性：**
- **生产就绪**：支持流式响应、原生追踪、审计及用户/会话数据隔离。
- **治理与信任**：内置审批工作流、人机回环、运行时护栏和评估集成。
- **自主部署**：在用户自身基础设施运行，拥有数据和系统控制权。
- **灵活架构**：支持单代理、多代理团队及结构化工作流，代码简洁。
