### [hynek structlog](https://github.com/hynek/structlog)  ![GitHub Repo stars](https://img.shields.io/github/stars/hynek/structlog?style=social)

structlog 是 Python 生产级的结构化日志解决方案。它具有简单、强大、快速的特点，基于函数处理字典，API 设计熟悉。支持 JSON、logfmt 及控制台等多种输出格式，也可将日志转发至 Python 标准库的 logging 模块。项目支持 asyncio、上下文变量和类型提示，自 2013 年以来稳定应用于生产环境。
