### [astral-sh ruff](https://github.com/astral-sh/ruff)  ![GitHub Repo stars](https://img.shields.io/github/stars/astral-sh/ruff?style=social)

Ruff 是一个用 Rust 编写的极速 Python 代码检查器和格式化工具。其速度比现有工具快 10 至 100 倍，可替代 Flake8、isort、Black 等多个代码质量工具。Ruff 内置 800 多条规则，支持自动修复、内置缓存及层级配置，兼容 Python 3.14。它提供编辑器集成、预提交钩子和 CI/CD 支持，旨在通过单一接口实现高性能的代码检查和格式化。
