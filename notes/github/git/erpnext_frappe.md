### [frappe erpnext](https://github.com/frappe/erpnext)

**ERPNext核心内容总结**  

**项目功能**  
ERPNext是一款开源ERP系统，整合企业核心业务功能，包括会计、订单管理、制造、资产管理、项目管理等，提供一站式解决方案。  

**主要特性**  
- **会计**：统一管理现金流，支持交易记录、财务报告分析。  
- **订单管理**：跟踪库存、管理客户/供应商、订单履约。  
- **制造**：优化生产流程，支持物料追踪、产能规划、外包管理。  
- **资产管理**：集中管理设备、IT基础设施等资产全生命周期。  
- **项目管理**：跟踪任务、工时和项目进度，确保按时按预算交付。  

**技术基础**  
基于**Frappe框架**（Python/JavaScript）和**Frappe UI**（Vue组件库），提供现代用户界面和扩展能力。  

**使用方法**  
1. **托管部署**：通过[Frappe Cloud](https://frappecloud.com)一键部署，无需自行维护。  
2. **自托管**：  
   - **Docker**：运行`git clone frappe/frappe_docker`后执行`docker compose up -d`，默认账号密码为`Administrator/admin`。  
   - **本地开发**：使用`bench`工具创建站点并安装ERPNext应用，通过`bench start`启动服务。
