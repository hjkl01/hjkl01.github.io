### [rayenghanmi RyTuneX](https://github.com/rayenghanmi/RyTuneX)  ![GitHub Repo stars](https://img.shields.io/github/stars/rayenghanmi/RyTuneX?style=social)

RyTuneX 是一款专为 Windows 10 (20H1+) 和 Windows 11 设计的系统优化工具，旨在提升性能、隐私及可用性。核心功能包括：

1. **应用移除**：卸载预装应用以释放系统资源。
2. **系统优化**：通过基础与高级选项减少资源占用，提升运行效率。
3. **隐私增强**：禁用遥测和数据收集功能。
4. **功能管理**：灵活调整特定的 Windows 系统功能。
5. **系统信息**：直观查看硬件及操作系统概况。

该工具基于 WinUI 3 框架构建，提供现代化且支持多语言的界面。
