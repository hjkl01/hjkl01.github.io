### [moby moby](https://github.com/moby/moby)  ![GitHub Repo stars](https://img.shields.io/github/stars/moby/moby?style=social)

Moby 是由 Docker 创建的开源项目，旨在通过提供模块化、灵活的容器化工具套件来加速软件容器化。项目包含容器构建、注册、编排和运行时等组件，可作为构建自定义容器系统的“乐高积木”。它面向工程师和爱好者，支持基于容器进行实验、修改和构建，不提供商业支持。Docker 将其作为产品上游，采用 Apache License 2.0 协议。
