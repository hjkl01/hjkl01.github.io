### [mergestat mergestat](https://github.com/mergestat/mergestat)  ![GitHub Repo stars](https://img.shields.io/github/stars/mergestat/mergestat?style=social)

MergeStat 允许对 Git 仓库及相关数据源（如 GitHub API）执行 SQL 查询，支持分析源代码的历史记录和具体内容。项目支持通过 Docker Compose 本地部署，提供管理界面以添加仓库和同步数据。使用 GitHub API 功能或访问私有仓库需提供个人访问令牌（PAT）。另有 CLI 工具 mergestat-lite 用于针对本地 Git 仓库进行 SQL 查询。
