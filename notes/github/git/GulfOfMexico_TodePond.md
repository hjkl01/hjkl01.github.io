### [TodePond GulfOfMexico](https://github.com/TodePond/GulfOfMexico)  ![GitHub Repo stars](https://img.shields.io/github/stars/TodePond/GulfOfMexico?style=social)

Gulf of Mexico（原名 DreamBerd）是一个讽刺性的编程语言。语句结尾必须加感叹号，逻辑非运算符为分号；无传统循环结构，使用 `when` 监控变量变更或 `previous/next` 访问变量历史与未来；数组索引从 -1 开始；算术运算依赖空格缩进；支持四种声明组合及 Unicode 字符和数字命名。语言包含多项戏谑功能，如类仅允许单实例、除零返回 undefined、AI 自动补全需人工完成等。目前官方编译器已失效，存在第三方部分实现。
