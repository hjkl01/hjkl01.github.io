### [ClashMetaForAndroid](https://github.com/MetaCubeX/ClashMetaForAndroid) ![GitHub Repo stars](https://img.shields.io/github/stars/MetaCubeX/ClashMetaForAndroid?style=social)

Clash Meta for Android 是 Clash.Meta 的安卓图形界面客户端，用于在 Android 设备上运行和管理 Clash.Meta 服务。

项目核心功能：
- 提供 Clash.Meta 在安卓上的图形化操作界面；
- 支持启动、停止、切换 Clash.Meta 服务状态；
- 支持通过 `clash://` 或 `clashmeta://` 链接导入配置文件；
- 支持自动化控制，可通过指定 Intent 操作服务；
- 支持 Android 5.0 及以上系统，推荐 Android 7.0 及以上；
- 支持 `armeabi-v7a`、`arm64-v8a`、`x86`、`x86_64` 架构。

项目还包含完整的安卓构建方式、签名配置、自定义应用包名支持，以及依赖更新、预发布和正式发布的自动化维护流程。
