### [regolith-labs ore](https://github.com/regolith-labs/ore)  ![GitHub Repo stars](https://img.shields.io/github/stars/regolith-labs/ore?style=social)

ORE 是一个加密货币挖矿协议，主要功能包括：

1. **挖矿**：配置自动化任务、领取 ORE 或 SOL 奖励、部署 SOL 占位及重置游戏轮次。
2. **质押**：支持 ORE 存入与提取、领取质押收益及获取 Seeker 代币。
3. **管理**：执行买卖埋藏交易、包装国库 SOL、重设管理员权限及更新交易费率。
4. **状态管理**：追踪自动化配置、游戏轮次、矿工状态、质押活动及代币铸造、销毁与托管。
