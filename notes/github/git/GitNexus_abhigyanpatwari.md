### [GitNexus](https://github.com/abhigyanpatwari/GitNexus) ![GitHub Repo stars](https://img.shields.io/github/stars/abhigyanpatwari/GitNexus?style=social)

**总结项目功能**

GitNexus 将代码库索引为知识图谱，提供命令行工具和多代理控制平台供 AI 交互，配备网页界面支持图谱浏览和对话。主要功能包括影响分析、搜索、上下文识别、变更检测、重命名、Cypher 查询和自动生成 wiki。

GitNexus 将任意代码库索引为完整知识图谱（依赖、调用链、聚类、执行流程等），通过 CLI+MCP 为 AI 代码助手提供深度上下文，支持影响分析、变更检测、搜索、上下文查看、批量重命名、Cypher 查询和自动生成代码 Wiki；同时提供纯浏览器 Web UI 进行图谱可视化与聊天探索，可本地或桥接到 CLI 使用。
