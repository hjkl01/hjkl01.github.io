### [gogs gogs](https://github.com/gogs/gogs)  ![GitHub Repo stars](https://img.shields.io/github/stars/gogs/gogs?style=social)

Gogs 是一个简单、稳定且可扩展的自托管 Git 服务，使用 Go 语言编写，提供跨平台独立二进制文件，硬件要求极低。

主要功能包括：
1. **仓库管理**：支持 SSH、HTTP 和 HTTPS 协议访问，提供用户、组织及仓库管理。
2. **协作功能**：支持 Issue、Pull Request、Wiki、受保护分支及代码合并协作。
3. **集成扩展**：支持 Git Hooks、Git LFS、Webhooks（Slack、Discord 等）、Git 部署密钥及在线文件编辑。
4. **安全认证**：支持 SMTP、LDAP、反向代理、GitHub 登录及双因素认证（2FA）。
5. **数据与本地化**：支持 PostgreSQL、MySQL、SQLite3 等数据库，提供 30 多种语言本地化。
