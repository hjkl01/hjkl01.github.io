### [NVIDIA nvidia-container-toolkit](https://github.com/NVIDIA/nvidia-container-toolkit)  ![GitHub Repo stars](https://img.shields.io/github/stars/NVIDIA/nvidia-container-toolkit?style=social)

NVIDIA Container Toolkit 允许用户构建和运行 GPU 加速容器。该工具包包含容器运行时库和实用工具，可自动配置容器以利用 NVIDIA GPU。使用时无需在主机系统安装 CUDA Toolkit，但需安装相应的 NVIDIA 驱动程序。
