### [apitable apitable](https://github.com/apitable/apitable)  ![GitHub Repo stars](https://img.shields.io/github/stars/apitable/apitable?style=social)

APITable 是一款开源的可视化数据库与电子表格工具，支持私有化部署和云端使用。
核心功能：
1. 实时协作：支持多人同时编辑，基于 OT 算法。
2. 高性能界面：Canvas 渲染，支持 10 万 + 数据行，提供 7 种视图。
3. API 优先：全栈 API 访问，支持类 SQL 查询（DQL）及嵌入。
4. 灵活关联：支持无限跨表链接。
5. 权限管理：行列级权限、文件夹权限，企业版含 SSO/SAML。
6. 扩展性：内置自动化机器人、BI 仪表盘，支持小部件系统。
技术栈：TypeScript/Java，支持 Docker 部署。
适用场景：项目管理、CRM、ERP、低代码平台及企业数据基础设施。
