### [hacdias webdav](https://github.com/hacdias/webdav)  ![GitHub Repo stars](https://img.shields.io/github/stars/hacdias/webdav?style=social)

这是一个用 Go 编写的简单独立 WebDAV 服务器。支持二进制、Docker 等多种部署方式，配置灵活（YAML/JSON/TOML）。提供用户认证（明文/加密/环境变量）、细粒度权限控制、TLS 加密、CORS 及日志管理。支持集成 Fail2Ban 防御暴力破解，并兼容 systemd 服务配置及反向代理部署。
