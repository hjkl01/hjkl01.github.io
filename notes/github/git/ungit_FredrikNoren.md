### [FredrikNoren ungit](https://github.com/FredrikNoren/ungit)  ![GitHub Repo stars](https://img.shields.io/github/stars/FredrikNoren/ungit?style=social)

Ungit 是一款基于 Web 的跨平台 Git 图形化界面工具，旨在降低 Git 的使用门槛。它提供简洁直观的 UI，支持在本地或云端通过浏览器访问，无需额外安装桌面客户端。主要功能包括自动刷新仓库状态、集成 GitHub 和 Gerrit、支持 PGP 提交签名、配置外部合并工具，以及与 Atom、Brackets、VSCode 等编辑器的插件集成。项目基于 Node.js 构建，支持通过配置文件或命令行参数进行自定义设置。
