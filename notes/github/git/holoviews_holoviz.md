### [holoviz holoviews](https://github.com/holoviz/holoviews)  ![GitHub Repo stars](https://img.shields.io/github/stars/holoviz/holoviews?style=social)

HoloViews 是一个开源 Python 库，旨在简化数据分析和可视化过程。其核心理念是“注释数据而非绘制数据”，允许用户用少量代码让数据自行可视化，从而专注于数据探索与表达。该库兼容 Linux、Windows 和 Mac 系统，与 Jupyter Notebook 及 JupyterLab 无缝集成。
