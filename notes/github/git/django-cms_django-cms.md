### [django-cms django-cms](https://github.com/django-cms/django-cms)  ![GitHub Repo stars](https://img.shields.io/github/stars/django-cms/django-cms?style=social)

django CMS 是一个基于 Django 的轻量级开源企业级内容管理系统。其功能包括：支持复杂发布工作流；具备强大的国际化（i18n）和多站点支持；提供前端（内联）编辑；拥有灵活的占位符和插件系统；集成多种富文本编辑器；支持内容版本控制、编辑工作流及无头模式；提供 SEO 友好的分层页面树；支持可扩展的导航和应用集成（apphooks）。
