### [arzzen git-quick-stats](https://github.com/arzzen/git-quick-stats)  ![GitHub Repo stars](https://img.shields.io/github/stars/arzzen/git-quick-stats?style=social)

`git-quick-stats` 是一个用于访问 Git 仓库各类统计信息的简单高效工具。支持交互式菜单和命令行模式，可展示提交、贡献者、分支等维度的统计信息，包括按日/周/月/年/小时/时区及作者的细分数据。支持变更日志、分支树、贡献热力图展示及 CSV/JSON 输出。可通过环境变量自定义筛选范围、统计选项及显示主题，兼容 Linux、macOS、Windows 及 Docker 环境。
