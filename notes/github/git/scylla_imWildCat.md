### [imWildCat scylla](https://github.com/imWildCat/scylla)  ![GitHub Repo stars](https://img.shields.io/github/stars/imWildCat/scylla?style=social)

Scylla 是一款智能代理池项目，旨在辅助从互联网提取内容。其核心功能包括：自动爬取和验证代理 IP；提供 JSON API 和 Web 界面供用户查询代理列表（支持按国家、匿名性、协议等筛选）及系统统计；内置 HTTP 转发代理服务，可自动随机选择可用代理；支持通过少量代码集成 Scrapy 和 requests；支持无头浏览器爬取；提供一键启动和 Docker 部署方案。
