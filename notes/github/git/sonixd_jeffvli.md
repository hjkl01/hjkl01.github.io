### [jeffvli sonixd](https://github.com/jeffvli/sonixd)  ![GitHub Repo stars](https://img.shields.io/github/stars/jeffvli/sonixd?style=social)

Sonixd 是一款跨平台桌面音乐客户端，支持连接 Subsonic-API 及 Jellyfin 兼容的音乐服务器。主要功能包括 HTML5 音频播放（支持交叉淡入淡出与无间隙播放）、播放列表与队列管理、拖拽操作、系统媒体键控制及多主题皮肤。目前项目已停止新功能开发，进入维护模式，后续将由 Feishin 重写。
