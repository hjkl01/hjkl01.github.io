### [NeogitOrg neogit](https://github.com/NeogitOrg/neogit)  ![GitHub Repo stars](https://img.shields.io/github/stars/NeogitOrg/neogit?style=social)

Neogit 是一款为 Neovim 设计的 Git 操作界面插件，灵感源自 Magit。它允许用户在编辑器内通过标签页、分屏或悬浮窗直接管理分支、提交、合并、变基及存储等操作。插件支持集成 telescope、diffview 等工具，提供高度可配置项，包括状态实时刷新、Git 事件监听及自定义快捷键等功能。
