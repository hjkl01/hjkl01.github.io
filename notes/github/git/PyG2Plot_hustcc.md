### [hustcc PyG2Plot](https://github.com/hustcc/PyG2Plot)  ![GitHub Repo stars](https://img.shields.io/github/stars/hustcc/PyG2Plot?style=social)

PyG2Plot 是 AntV G2Plot 的 Python3 绑定库，基于图形语法允许用户通过少量代码制作交互式统计图表。项目提供 Plot 类用于实例化图表并配置数据与选项，支持将图表渲染为 HTML 文件、HTML 字符串，或在 Jupyter Notebook 及 Jupyter Lab 环境中直接预览。此外，还支持通过 JS API 注入 JavaScript 回调以实现自定义样式或逻辑。
