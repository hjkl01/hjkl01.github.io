### [gitui-org gitui](https://github.com/gitui-org/gitui)  ![GitHub Repo stars](https://img.shields.io/github/stars/gitui-org/gitui?style=social)

GitUI 是一款基于 Rust 编写的终端 Git 管理工具，旨在提供终端内类似图形界面的操作体验。它支持全键盘控制的代码审查、提交、暂存、分支管理、远程同步、日志搜索及差分查看等功能。相比同类工具，GitUI 在处理大型仓库时速度更快、内存占用更低且运行稳定，不会发生冻结或崩溃。项目开源免费，支持自定义主题和快捷键，目前处于 Beta 版本。
