### [Danily07 Translumo](https://github.com/Danily07/Translumo)  ![GitHub Repo stars](https://img.shields.io/github/stars/Danily07/Translumo?style=social)

Translumo 是一款面向 PC 游戏及应用的高级实时屏幕翻译工具。它支持 Windows OCR、Tesseract、EasyOCR 多种引擎智能选择以优化识别精度，集成 DeepL、Google、Yandex 等主流翻译服务。项目经过低延迟优化，支持自定义屏幕捕获区域和热键设置，具备英、俄、日、中、韩等多种语言的识别与翻译功能。
