### [GraphiteEditor Graphite](https://github.com/GraphiteEditor/Graphite)  ![GitHub Repo stars](https://img.shields.io/github/stars/GraphiteEditor/Graphite?style=social)

Graphite 是一款免费开源的矢量与光栅图形引擎，目前处于 Alpha 阶段。它提供完全非破坏性的编辑工作流，结合基于图层的合成与基于节点的生成式设计。项目构建理念类似游戏引擎，旨在从矢量编辑器演变为全能图形工具箱。除支持矢量与光栅编辑外，其路线图涵盖照片编辑、动态图形、数字绘画、桌面出版及 VFX 合成等功能。
