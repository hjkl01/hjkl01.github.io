### [dateutil dateutil](https://github.com/dateutil/dateutil)  ![GitHub Repo stars](https://img.shields.io/github/stars/dateutil/dateutil?style=social)

dateutil 是 Python 标准 datetime 模块的扩展库，主要功能包括：
* 计算相对时间差（如“下个月”、“上次周一”等）；
* 基于 iCalendar 规范的灵活循环规则生成与解析；
* 通用且强大的日期字符串解析；
* 多种时区（tzinfo）实现及内置世界时区数据库支持；
* 支持多种历法的复活节日期计算。
