### [github gh-aw](https://github.com/github/gh-aw)  ![GitHub Repo stars](https://img.shields.io/github/stars/github/gh-aw?style=social)

GitHub Agentic Workflows 允许用户使用自然语言 Markdown 编写智能体工作流，并在 GitHub Actions 中运行。项目以安全为核心，默认采用只读权限，仅允许通过清洗后的安全输出进行写入，并实施沙盒执行、网络隔离、工具白名单及人工审批等多层防护。配套提供 Agent Workflow Firewall 用于网络出口控制，以及 MCP Gateway 用于模型上下文协议的路由管理。
