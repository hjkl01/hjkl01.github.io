### [plankanban planka](https://github.com/plankanban/planka)  ![GitHub Repo stars](https://img.shields.io/github/stars/plankanban/planka?style=social)

PLANKA 是一款基于 Kanban 的开源项目管理协作工具。核心功能包括：支持创建项目、看板、列表和卡片的拖拽式协作界面；多用户操作实时同步更新；卡片描述支持丰富的 Markdown 编辑；提供超过 100 种可定制的通知渠道；集成 OpenID Connect 实现单点登录；具备完善的多语言国际化支持。项目支持自托管部署，源码公开，并提供社区版与 Pro 版授权。
