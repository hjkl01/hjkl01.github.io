### [max-sixty worktrunk](https://github.com/max-sixty/worktrunk)  ![GitHub Repo stars](https://img.shields.io/github/stars/max-sixty/worktrunk?style=social)

Worktrunk 是一个用于管理 Git 工作区（worktree）的命令行工具，专为并行运行 AI 代理设计。它简化了工作区的创建、切换、状态查看及清理流程，使其操作体验如同管理 Git 分支般便捷。核心功能涵盖 `switch`、`list`、`merge` 和 `remove` 命令。此外，还提供自动化钩子、LLM 生成提交消息、交互式选择器、构建缓存共享等特性，以优化多任务并行开发的工作流。
