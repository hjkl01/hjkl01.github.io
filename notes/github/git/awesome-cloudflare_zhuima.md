### [zhuima awesome-cloudflare](https://github.com/zhuima/awesome-cloudflare)  ![GitHub Repo stars](https://img.shields.io/github/stars/zhuima/awesome-cloudflare?style=social)

这是一个基于 Cloudflare 生态的开源工具精选集，专为独立开发者提供省心省时的工具支持。项目收录了涵盖图床、临时邮箱、博客系统、短链接、网站分析、加速代理、文件分享、监控及开发者工具等多个类别的开源项目。所有工具均依托 Cloudflare Workers、Pages、R2 等免费服务实现 Serverless 部署，旨在帮助用户提升开发效率并降低成本。
