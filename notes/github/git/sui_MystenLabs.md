### [MystenLabs sui](https://github.com/MystenLabs/sui)  ![GitHub Repo stars](https://img.shields.io/github/stars/MystenLabs/sui?style=social)

Sui 是一个基于 Rust 和 Move 语言的下一代智能合约平台，具备高吞吐量、低延迟及资产导向编程模型特性。它支持交易并行处理，对常见用例采用简化原语以降低延迟，由无权限管理者网络维护。原生代币 SUI 用于支付 gas 及质押。Sui 旨在成为 web3 的互联网级可编程区块链基础层，提供卓越的扩展性、性能与可用性。
