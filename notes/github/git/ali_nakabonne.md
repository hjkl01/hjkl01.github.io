### [nakabonne ali](https://github.com/nakabonne/ali)  ![GitHub Repo stars](https://img.shields.io/github/stars/nakabonne/ali?style=social)

`ali` 是一款支持实时分析的负载测试工具，灵感源自 `vegeta`。它内置终端用户界面（UI），允许在终端中进行实时测试与指标可视化。主要功能包括：

1. **实时负载测试**：可配置请求速率、持续时间、HTTP 方法、请求体、头部及 TLS 参数，支持无限时长测试。
2. **终端可视化分析**：提供延迟、百分位（50/90/95/99）、字节数及直方图等实时图表，支持鼠标交互（缩放、拖拽、切换视图）以监控测试进度。
3. **结果导出**：支持将测试结果导出至指定目录，便于后续数据处理。
