### [cert-manager cert-manager](https://github.com/cert-manager/cert-manager)  ![GitHub Repo stars](https://img.shields.io/github/stars/cert-manager/cert-manager?style=social)

cert-manager 是一款 Kubernetes 证书管理工具，它将证书和证书颁发者作为自定义资源类型集成到集群中。它支持从多种来源（如 Let's Encrypt/ACME、HashiCorp Vault、CyberArk 及本地颁发）自动获取、续期和使用 TLS 证书，并能主动在过期前进行续期，以保障证书有效性和服务稳定性。
