### [unhappychoice gitlogue](https://github.com/unhappychoice/gitlogue)  ![GitHub Repo stars](https://img.shields.io/github/stars/unhappychoice/gitlogue?style=social)

gitlogue 是一款基于 Rust 开发的终端 Git 提交回放工具，将 Git 历史记录转化为包含逼真打字动画、语法高亮和文件树过渡的视觉体验。它支持 29 种编程语言的语法高亮，提供屏幕保护模式、连续循环播放及特定提交回放功能，并允许查看工作区差异。用户可通过命令行调整播放速度、筛选作者或日期、自定义主题及忽略特定文件，适用于终端装饰、代码演变可视化、演示及内容创作。
