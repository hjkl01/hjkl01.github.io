### [k88hudson git-flight-rules](https://github.com/k88hudson/git-flight-rules)  ![GitHub Repo stars](https://img.shields.io/github/stars/k88hudson/git-flight-rules?style=social)

该项目是 Git 飞行规则指南，借鉴 NASA 宇航员手册理念，旨在为程序员提供 Git 操作失误修复及特定场景的解决方案。内容涵盖仓库管理、提交编辑、暂存区操作、分支管理、变基与合并、数据查找、调试及配置等核心领域。针对常见错误情境（如误删文件、历史重置、分支冲突等），项目提供了具体的命令行操作步骤，帮助用户快速解决 Git 使用中的问题并恢复项目状态。
