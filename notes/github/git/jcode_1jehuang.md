### [jcode](https://github.com/1jehuang/jcode) ![GitHub Repo stars](https://img.shields.io/github/stars/1jehuang/jcode?style=social)

jcode 是一款面向多会话协作的高性能终端编程代理工具，支持交互式 TUI、非交互运行、持久化服务/客户端模式、会话续接、语音输入。它内置多种模型/账号登录与供应商接入，支持本地及 OpenAI 兼容接口、MCP 配置、浏览器自动化、记忆系统、侧边栏渲染、多人 Swarm 协作，还提供高度可定制和自我开发能力，并针对低内存占用和快速启动做了优化。
