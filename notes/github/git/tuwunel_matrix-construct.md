### [matrix-construct tuwunel](https://github.com/matrix-construct/tuwunel)  ![GitHub Repo stars](https://img.shields.io/github/stars/matrix-construct/tuwunel?style=social)

Tuwunel 是一款基于 Rust 编写的高性能 Matrix 家庭服务器，旨在替代 Synapse。它完全实现 Matrix 规范，支持各类客户端、桥接和机器人，具备可扩展、低成本及企业级就绪特性。作为 conduwuit 的官方继任者，支持 Docker、二进制及多种系统包部署，可直接从 conduwuit 迁移，并计划支持 Synapse 迁移。
