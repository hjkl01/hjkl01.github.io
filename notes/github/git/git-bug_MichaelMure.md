### [MichaelMure git-bug](https://github.com/MichaelMure/git-bug)  ![GitHub Repo stars](https://img.shields.io/github/stars/MichaelMure/git-bug?style=social)

git-bug 是一个去中心化的离线优先问题追踪与管理工具，将问题、评论等作为对象直接嵌入 Git 仓库进行版本管理。其核心功能包括：

1.  **原生 Git 存储**：管理问题、用户及评论，保持仓库版本化且整洁。
2.  **分布式架构**：支持离线工作，利用 Git 机制进行无缝同步。
3.  **高性能**：实现毫秒级的问题列表与搜索。
4.  **第三方桥接**：可同步至 GitHub、GitLab 等平台。
5.  **多端交互**：支持命令行（CLI）、终端界面（TUI）及 Web 浏览器。
6.  **易于集成**：无需复杂配置即可在现有仓库中启用。
