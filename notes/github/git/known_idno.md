### [idno known](https://github.com/idno/known)  ![GitHub Repo stars](https://img.shields.io/github/stars/idno/known?style=social)

Idno 是一个基于 PHP 8.2+ 的开源社交发布平台，支持内容发布、Webmention、内容分发及 ActivityPub 协议，可实现私有博客与联邦社交网络间的数据同步。为确保 ActivityPub 推送等功能正常运行，平台需配置异步队列后台工作进程处理事件，建议使用 systemd 或 cron 守护。项目支持多种安装方式，采用 Apache 2.0 许可证。
