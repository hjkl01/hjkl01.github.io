### [VonChange utao](https://github.com/VonChange/utao)  ![GitHub Repo stars](https://img.shields.io/github/stars/VonChange/utao?style=social)

油桃 TV 是一款专为智能电视、投影及安卓盒子设计的第三方浏览器应用，旨在解决大屏投屏限制及视频观看成本问题。

核心功能：
1. **突破投屏限制**：通过内置浏览器直接访问爱奇艺、腾讯视频、B 站等视频平台网页版，支持使用手机或 PC 会员账号在电视端直接播放高清内容，无需额外购买电视端会员。
2. **遥控器适配**：优化了电视端操作，支持全屏播放、选集、倍速、进度调整及视频清晰度切换。
3. **直播观看**：支持 CCTV 官网及各大地方台直播。
4. **免费无广告**：软件本身不收费且无植入广告。

技术架构与兼容性：
- 基于 Firefox 插件机制实现，兼容 X5 或 Webview 内核。
- 支持安卓 4.1 及以上版本（推荐安卓 9 及以上）。

注意事项：
- 受视频平台防盗链机制及机器性能影响，低配置设备或旧系统版本可能存在卡顿或兼容性问题。
