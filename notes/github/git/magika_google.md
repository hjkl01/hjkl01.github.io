### [google magika](https://github.com/google/magika)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/magika?style=social)

Magika 是一款基于深度学习的 AI 驱动文件类型检测工具，采用轻量级优化模型（仅几 MB），能在单 CPU 上毫秒级精确识别约 200 多种文件内容类型，准确率达 99%。项目提供 Rust 命令行工具、Python API 及 JavaScript/TypeScript 绑定，支持批量处理、递归扫描及多种预测置信度模式。其推理时间约为 5ms/文件且不受文件大小影响，已集成于 Google 内部安全扫描及 VirusTotal 等平台，采用 Apache 2.0 开源协议。
