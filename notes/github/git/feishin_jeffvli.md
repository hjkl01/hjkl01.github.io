### [jeffvli feishin](https://github.com/jeffvli/feishin)  ![GitHub Repo stars](https://img.shields.io/github/stars/jeffvli/feishin?style=social)

Feishin 是 Sonixd 的重写版音乐播放器客户端，采用现代化 UI。支持 MPV 和 Web 两种播放器后端，可实现播放记录同步至服务器，支持智能歌单编辑（Navidrome）及同步/不同步歌词。兼容 Navidrome、Jellyfin 及 OpenSubsonic 协议的音乐服务器。提供桌面客户端（推荐）、网页版及 Docker 容器等多种部署方式。
