### [WordPress gutenberg](https://github.com/WordPress/gutenberg)  ![GitHub Repo stars](https://img.shields.io/github/stars/WordPress/gutenberg?style=social)

Gutenberg 是 WordPress 旨在革新站点构建与发布体验的块编辑器项目。它将段落、图片等内容元素封装为独立的“块”，用户可直观地添加、排列和组合这些块来创建富媒体页面，无需使用短代码或自定义 HTML。该项目提供插件供用户试用最新功能，支持开发者扩展和定制编辑器，并致力于优化协作及多语言发布流程。这是一个开源项目，欢迎各方参与贡献。
