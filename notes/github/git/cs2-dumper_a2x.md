### [a2x cs2-dumper](https://github.com/a2x/cs2-dumper)  ![GitHub Repo stars](https://img.shields.io/github/stars/a2x/cs2-dumper?style=social)

cs2-dumper 是一款针对《反恐精英 2》的外部偏移量与接口转储工具，支持 Windows 和 Linux 系统。基于 memflow 技术，通过读取游戏进程内存生成.cs、.hpp、.json、.rs 等文件。运行时需要游戏处于启动状态，支持通过命令行参数配置内存连接器（如 pcileech、kvm 等），部分连接器需管理员权限。
