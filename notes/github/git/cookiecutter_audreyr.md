### [audreyr cookiecutter](https://github.com/audreyr/cookiecutter)  ![GitHub Repo stars](https://img.shields.io/github/stars/audreyr/cookiecutter?style=social)

Cookiecutter 是一款跨平台命令行工具，用于通过项目模板快速生成项目结构（如 Python 包）。它支持从本地或远程仓库创建项目，允许用户通过配置文件自定义参数，并利用钩子脚本处理初始化任务。该工具无需 Python 知识即可使用，兼容多种编程语言和框架，同时也支持在 Python 代码中调用。
