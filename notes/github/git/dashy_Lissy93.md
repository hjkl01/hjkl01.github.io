### [Lissy93 dashy](https://github.com/Lissy93/dashy)  ![GitHub Repo stars](https://img.shields.io/github/stars/Lissy93/dashy?style=social)

Dashy 是一个用于组织自托管服务的仪表盘工具，帮助用户从单一位置访问和管理所有服务。核心功能包括：

- **服务聚合**：支持多页面、自定义布局及动态内容小部件展示。
- **状态监控**：实时检测应用在线状态、响应时间及状态码。
- **交互体验**：内置即时搜索、键盘快捷键及多种应用打开方式（新标签、弹窗等）。
- **高度定制**：支持丰富主题、多图标源（Font-Awesome、Favicon 等）、自定义 CSS 及多语言切换。
- **安全认证**：支持基本认证、SSO（Keycloak）、多用户及细粒度权限控制。
- **视图模式**：提供默认主页、工作区视图及极简浏览器启动页。
- **配置管理**：支持 YAML 文件或 UI 编辑器配置，提供端到端加密的云端备份与同步。
- **部署灵活**：支持 Docker、源码运行及主流云平台一键部署。
- **开源隐私**：100% 免费开源，注重隐私保护。
