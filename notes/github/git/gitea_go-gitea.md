### [go-gitea gitea](https://github.com/go-gitea/gitea)  ![GitHub Repo stars](https://img.shields.io/github/stars/go-gitea/gitea?style=social)

Gitea 是一个使用 Go 语言编写的自托管 Git 服务软件，致力于提供易于设置且快速的 Git 服务部署方案。它支持 Linux、macOS、Windows 及多种处理器架构。主要功能包括代码版本控制、仓库管理、问题追踪、拉取请求、里程碑与发布管理、组织协作及持续集成动作（Actions）。项目提供官方 API 接口、CLI 工具、SDK 及 Action 运行器，支持第三方扩展与多语言本地化，采用 MIT 协议开源。
