### [vite-plus](https://github.com/voidzero-dev/vite-plus) ![GitHub Repo stars](https://img.shields.io/github/stars/voidzero-dev/vite-plus?style=social)

Vite+ 是一体化的 Web 开发工具链入口，零配置整合 Vite、Vitest、Oxlint、Oxfmt、Rolldown、tsdown 和任务调度。它提供 Node 版本管理、依赖安装、开发服务器、格式化/校验/类型检查、测试、生产构建、库打包、单体仓库任务缓存与调度、项目创建与迁移等功能，并通过统一的 vite.config.ts 集中配置。还自动封装包管理器操作及常用命令，覆盖从创建到发布的完整开发生命周期。
