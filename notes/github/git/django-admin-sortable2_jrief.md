### [jrief django-admin-sortable2](https://github.com/jrief/django-admin-sortable2)  ![GitHub Repo stars](https://img.shields.io/github/stars/jrief/django-admin-sortable2?style=social)

django-admin-sortable2 是一个 Django 扩展包，为 Django Admin 界面中的列表视图及 Stacked、Tabular 内联视图提供通用拖拽排序功能。

该项目提供简单的 Mixin 类，可轻松集成到现有的 ModelAdmin 或 Inline 类中，无需修改模型基类。2.0 版本改用 Sortable.js 替代 jQuery，并支持同时拖拽多个项目。
