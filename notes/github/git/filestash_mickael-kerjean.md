### [mickael-kerjean filestash](https://github.com/mickael-kerjean/filestash)  ![GitHub Repo stars](https://img.shields.io/github/stars/mickael-kerjean/filestash?style=social)

Filestash 是一款插件驱动型的通用文件管理平台，支持 FTP、SFTP、S3、WebDAV 等二十多种存储协议。其核心架构高度模块化，存储、认证、应用、主题等功能均为插件，支持灵活扩展与定制。平台提供 Web 客户端及 API 接口，内置支持摄影、科学、GIS、3D 等多种专业格式的预览与应用。此外，还提供工作流自动化、AI 语义搜索、智能文件夹、版本控制、审计及杀毒等高级功能，致力于提供极致的文件管理体验。
