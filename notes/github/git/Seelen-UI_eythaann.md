### [eythaann Seelen-UI](https://github.com/eythaann/Seelen-UI)  ![GitHub Repo stars](https://img.shields.io/github/stars/eythaann/Seelen-UI?style=social)

Seelen UI 是一款专为 Windows 系统打造的全功能可定制桌面环境工具，支持 70 多种语言。
核心功能：
- 个性化定制：支持调整主题、图标、菜单及界面元素以符合个人风格。
- 窗口平铺：内置平铺窗口管理器，自动排列窗口以优化多任务处理。
- 媒体控制：集成兼容主流音乐播放器的媒体模块，支持后台播放控制。
- 应用启动：提供类 Rofi 风格快速启动器，便于访问应用和执行命令。
- 简易配置：提供直观图形界面，轻松修改任务栏布局、主题等设置。
