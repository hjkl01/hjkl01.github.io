### [go-rod rod](https://github.com/go-rod/rod)  ![GitHub Repo stars](https://img.shields.io/github/stars/go-rod/rod?style=social)

Rod 是一个基于 DevTools Protocol 的 Go 语言库，专为 Web 自动化和网页抓取设计。它支持高层和低层开发，具备元素自动就绪等待、操作线程安全、链式上下文设计（支持超时或取消）等功能。项目提供高阶辅助函数，支持自动查找或下载浏览器，调试友好且支持远程监控无头浏览器。此外，它能正确处理嵌套 iframe 和 Shadow DOM，确保崩溃后无僵尸进程，并拥有 100% 的测试覆盖。
