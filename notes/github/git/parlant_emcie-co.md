### [emcie-co parlant](https://github.com/emcie-co/parlant)  ![GitHub Repo stars](https://img.shields.io/github/stars/emcie-co/parlant?style=social)

Parlant 是面向客户 AI 代理的对话控制层框架，旨在解决企业级交互中的上下文工程难题，确保对话的一致性、合规性与品牌规范。它通过代码定义规则、知识和工具，利用上下文引擎在每次对话回合动态筛选并注入最相关信息，避免系统提示词过载导致的注意力分散。

核心功能包括：
- **动态行为指南**：基于条件触发指令，确保每次仅加载相关规则。
- **关系管理**：支持指南间的依赖与排除，维持上下文聚焦。
- **多轮流程**：自适应的标准作业程序（SOP），灵活引导对话状态。
- **预设回复**：关键节点锁定预审批模板，消除幻觉风险。
- **条件化工具**：仅当匹配观察条件时调用外部 API 和工作流。
- **可解释性**：全链路 OpenTelemetry 追踪，记录所有决策日志。

该框架 LLM 中立，可集成 LangGraph 等现有工作流，适用于金融、保险等对可靠性要求极高的受监管领域。
