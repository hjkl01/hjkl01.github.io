UAD-ng（Universal Android Debloater Next Generation）是一款 Android 系统应用卸载工具，旨在移除设备上不必要和隐蔽的系统应用，以提升设备的隐私性、运行效率（能耗、速度、内存）及安全性（减少攻击面）。该工具不收集用户数据，仅连接 GitHub 获取包列表和更新信息。
