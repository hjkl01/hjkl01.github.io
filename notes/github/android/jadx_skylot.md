### [skylot jadx](https://github.com/skylot/jadx)  ![GitHub Repo stars](https://img.shields.io/github/stars/skylot/jadx?style=social)

JADX 是一款命令行和图形界面工具，用于从 Android Dex 和 Apk 文件生成 Java 源代码。
主要功能：
- 支持反编译 APK, dex, aar, aab, zip 等文件的 Dalvik 字节码至 Java 代码。
- 解码 AndroidManifest.xml 及其他资源文件。
- 包含内置反混淆器。
GUI 版本提供代码语法高亮、跳转定义、查找用法、全文搜索及 Smali 调试功能。
运行依赖：Java 11 或更高版本。
