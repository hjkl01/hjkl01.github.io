### [kivy buildozer](https://github.com/kivy/buildozer)  ![GitHub Repo stars](https://img.shields.io/github/stars/kivy/buildozer?style=social)

Buildozer 是由 Kivy 团队维护的开发工具，用于将 Python 应用程序打包为 Android、iOS、Windows、macOS 及 Linux 平台的可安装二进制包。开发者通过 `buildozer.spec` 配置文件定义应用需求，工具即可自动处理构建依赖并生成安装包。该工具支持 Kivy 框架但也可独立使用，提供调试、发布、部署等命令，并兼容 Docker 环境和 GitHub Actions 自动化构建。
