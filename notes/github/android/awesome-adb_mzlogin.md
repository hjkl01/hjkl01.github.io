### [mzlogin awesome-adb](https://github.com/mzlogin/awesome-adb)  ![GitHub Repo stars](https://img.shields.io/github/stars/mzlogin/awesome-adb?style=social)

本项目提供了 Android Debug Bridge (ADB) 的全面命令参考，适用于 Android 开发、测试及高级用户。核心功能涵盖设备连接管理、应用管理、文件传输、模拟按键输入、日志查看、设备信息查询、系统设置修改、实用工具（截图/录屏/重启）、刷机操作及安全配置命令。项目旨在提供全面的命令参考及常见问题解决方案。
