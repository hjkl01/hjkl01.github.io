### [MFDGaming ubuntu-in-termux](https://github.com/MFDGaming/ubuntu-in-termux)  ![GitHub Repo stars](https://img.shields.io/github/stars/MFDGaming/ubuntu-in-termux?style=social)

该脚本允许用户在不 root 设备的情况下，在 Termux 应用中安装并运行 Ubuntu。默认支持 Ubuntu 22.04，提供针对 x86/i*86 架构或旧版本（如 19.10）的分支选项，包含详细的安装与启动步骤，并附带常见错误（如内核过旧）的解决方案。
