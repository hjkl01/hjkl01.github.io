### [magisk-zapret2](https://github.com/youtubediscord/magisk-zapret2) ![GitHub Repo stars](https://img.shields.io/github/stars/youtubediscord/magisk-zapret2?style=social)

这是一个 Android 的 Magisk 模块，用于通过伪装网络流量绕过对 YouTube、Discord、Telegram 等服务的封锁，不是 VPN、无需第三方服务器，速度快。需 Root 权限，可选配控制应用进行启动/停止、自动启动、更新、日志查看，以及为不同服务选择绕过策略或使用预设。支持 Wi‑Fi 和移动网络，可把手机当热点共享已解封的网络。
