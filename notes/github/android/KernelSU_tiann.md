### [tiann KernelSU](https://github.com/tiann/KernelSU)  ![GitHub Repo stars](https://img.shields.io/github/stars/tiann/KernelSU?style=social)

KernelSU 是一款基于内核的 Android Root 解决方案。主要功能包括：基于内核的 `su` 及 Root 权限管理；支持基于 metamodules 的模块系统实现无系统修改；通过应用配置限制 Root 权限使用。官方支持 Android GKI 2.0 设备（内核 5.10+），兼容 arm64-v8a 和 x86_64 架构，亦支持旧内核及 WSA、ChromeOS 等环境。
