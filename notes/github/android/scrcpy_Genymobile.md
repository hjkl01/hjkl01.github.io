### [Genymobile scrcpy](https://github.com/Genymobile/scrcpy)  ![GitHub Repo stars](https://img.shields.io/github/stars/Genymobile/scrcpy?style=social)

scrcpy 是一款开源免费的跨平台工具，支持在 Linux、Windows 和 macOS 电脑上通过 USB 或网络镜像并控制 Android 设备（API 21+）。该工具无需 Root 权限，也不需要在设备上安装任何应用。其核心优势在于低延迟（35~70ms）、高帧率（30~120fps）、高画质及快速启动。主要功能涵盖音视频转发、屏幕录制、虚拟显示、双向剪贴板、摄像头镜像、HID 键盘鼠标模拟、游戏手柄支持及 OTG 模式等。
