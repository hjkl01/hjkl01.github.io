### [android nav3-recipes](https://github.com/android/nav3-recipes)  ![GitHub Repo stars](https://img.shields.io/github/stars/android/nav3-recipes?style=social)

本项目是 Jetpack Navigation 3 的代码示例库，提供实现常见导航用例的代码模板。内容涵盖基础 API 使用、深度链接、场景管理（支持 Dialog、BottomSheet、列表详情及 Material 3 自适应）、动画配置、返回栈行为（含多栈）、条件导航、架构模块化（支持 Hilt/Koin）、ViewModel 交互、结果返回及与 Fragment/View 的互操作。示例中的模式未来可能升级为库的内置功能。
