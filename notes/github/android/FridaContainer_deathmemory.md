### [deathmemory FridaContainer](https://github.com/deathmemory/FridaContainer)  ![GitHub Repo stars](https://img.shields.io/github/stars/deathmemory/FridaContainer?style=social)

FridaContainer 是一个整合了常用 Frida 脚本的逆向工程提效工具，主要功能如下：

1. **Android 支持**：提供一键去反调试、堆栈打印、Dump dex、SSL Pinning 绕过（含 cronet bypass）、Hook JNI 及方法追踪等功能。
2. **iOS 支持**：支持函数地址获取与模糊查找、堆栈打印、UI 结构 Dump 及常见数据类型转换。
3. **跨平台通用**：包含 FCCommon 模块，提供堆栈打印、模块信息获取、寄存器读取及模块 Dump 等通用方法。
4. **开发便捷**：编译后支持 IDE（如 PyCharm）代码补全，支持实时编译与模块化业务脚本开发。
