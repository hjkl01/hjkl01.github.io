### [nbs-system naxsi](https://github.com/nbs-system/naxsi)  ![GitHub Repo stars](https://img.shields.io/github/stars/nbs-system/naxsi?style=social)

Naxsi 是一款基于 Nginx 的 Web 应用防火墙（WAF）模块，旨在防御 XSS 和 SQL 注入。它采用默认拒绝策略，通过匹配简单规则识别恶意请求，支持自动学习生成白名单。与传统依赖特征库的 WAF 不同，它能有效防御未知攻击模式。原项目已停止维护，建议迁移至新维护版本。兼容类 Unix 系统及任意 Nginx 版本。
