### [0xJacky nginx-ui](https://github.com/0xJacky/nginx-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/0xJacky/nginx-ui?style=social)

Nginx UI 是一款基于 Web 的 Nginx 服务器管理工具，使用 Go 和 Vue 开发，提供单文件可执行版本。主要功能包括：

1.  **系统监控**：实时统计 CPU、内存、负载及磁盘使用情况。
2.  **配置管理**：支持在线编辑（内置编辑器与 LLM 代码补全）、自动备份、版本对比恢复、配置自动测试与重载。
3.  **AI 辅助**：集成多模型 ChatGPT 助手及 MCP 协议接口，辅助配置理解与优化。
4.  **证书管理**：一键部署和自动续期 Let's Encrypt SSL 证书。
5.  **集群支持**：支持多节点配置镜像同步，便于多服务器环境管理。
6.  **运维工具**：包含在线日志查看、Web 终端、加密配置导出与恢复功能。

支持命令行、Systemd、Docker 等多种安装部署方式，兼容多操作系统。
