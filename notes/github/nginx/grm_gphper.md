### [gphper grm](https://github.com/gphper/grm)  ![GitHub Repo stars](https://img.shields.io/github/stars/gphper/grm?style=social)

该项目是一款基于 Go 和 Vue 的 Web 版 Redis 管理工具。支持直连和 SSH 连接管理及 DB 切换；支持 String、List、Set、ZSet、Hash、Stream 等数据类型的增删查改；具备用户鉴权、操作日志、LUA 脚本执行及 Redis 监控功能；支持编译独立二进制文件、命令行管理、Docker 部署及系统服务运行，并支持 Nginx 反向代理配置。
