### [likaia nginxpulse](https://github.com/likaia/nginxpulse)  ![GitHub Repo stars](https://img.shields.io/github/stars/likaia/nginxpulse?style=social)

NginxPulse 是一款轻量级 Nginx 访问日志分析与可视化面板。核心功能包括实时统计、PV 过滤、IP 归属地查询与客户端解析。后端基于 Go 语言，前端使用 Vue 3，数据存储采用 PostgreSQL。支持 Docker、Docker Compose、手动构建及单体部署模式，提供移动端访问入口。项目具备 IP 归属地本地库与远程 API 双重查询策略及缓存机制，支持日志数据定时自动清理与权限管理配置。
