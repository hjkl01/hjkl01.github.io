### [certimate](https://github.com/certimate-go/certimate) ![GitHub Repo stars](https://img.shields.io/github/stars/certimate-go/certimate?style=social)

Certimate 是一款开源自托管的 ACME SSL 证书管理工具，提供可视化工作流自动完成证书申请、部署、续期与监控；支持单域名/多域名/通配符/IP 证书、RSA/ECC、DNS-01 与 HTTP-01 验证、多种证书格式；对接 60+ DNS 服务商、110+ 部署目标、多个通知渠道及多家 ACME CA，轻量无依赖、易部署。
