### [trimstray nginx-admins-handbook](https://github.com/trimstray/nginx-admins-handbook)  ![GitHub Repo stars](https://img.shields.io/github/stars/trimstray/nginx-admins-handbook?style=social)

该项目是一个 Nginx 管理员手册，汇集了关于 Nginx 管理的基础知识、配置技巧、安全陷阱及最佳实践。内容涵盖 HTTP 与 SSL/TLS 协议基础、Nginx 配置语法、安装部署、监控调试、性能优化及安全加固规则。项目提供了详细的配置检查清单、反向代理与负载均衡指南、实用脚本工具及外部资源推荐，旨在帮助系统管理员和 Web 专家全面掌握 Nginx 配置规范，确保服务的安全性、稳定性及高性能。
