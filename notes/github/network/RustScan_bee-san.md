### [bee-san RustScan](https://github.com/bee-san/RustScan)  ![GitHub Repo stars](https://img.shields.io/github/stars/bee-san/RustScan?style=social)

RustScan 是一款基于 Rust 编写的现代化端口扫描器，核心功能如下：

1. **极速扫描**：最快可在 3 秒内扫描完所有 65535 个端口。
2. **脚本扩展**：内置脚本引擎，支持 Python、Lua、Shell 脚本，可自动将结果管道传输至 Nmap 或执行自定义任务。
3. **自适应学习**：根据扫描环境和用户习惯自动调优，持续改进性能。
4. **全面支持**：支持 IPv6、CIDR、文件输入等标准网络扫描功能，并致力于工具的可访问性设计。
