### [rathole](https://github.com/rathole-org/rathole) ![GitHub Repo stars](https://img.shields.io/github/stars/rathole-org/rathole?style=social)

rathole 是用 Rust 编写的安全稳定高性能内网穿透反向代理，可在有公网 IP 的服务器上将 NAT 后设备的 TCP/UDP 服务暴露到互联网。特点包括高吞吐低内存、小体积、强制服务级 token 认证，支持 TLS/Noise/WebSocket 加密传输与代理、热加载配置，适合多服务转发场景。
