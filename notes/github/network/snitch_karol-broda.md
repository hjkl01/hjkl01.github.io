### [karol-broda snitch](https://github.com/karol-broda/snitch)  ![GitHub Repo stars](https://img.shields.io/github/stars/karol-broda/snitch?style=social)

Snitch 是一个面向人类设计的网络连接检查工具，旨在提供比 `ss` 和 `netstat` 更友好的体验。它支持通过整洁的交互式 TUI 或格式化表格检查网络连接。主要功能包括：

- **多种模式**：支持交互式实时查看、一次性表格输出及 JSON 流式输出。
- **灵活过滤**：支持 TCP/UDP、协议状态（监听/已建立/其他）、IP 版本筛选，以及自定义键值过滤（如进程名、端口）。
- **丰富格式**：默认美化表格，支持纯文本、JSON、CSV 输出，支持 DNS 地址及服务名解析。
- **配置与主题**：支持多色主题定制、配置文件管理及环境变量设置，TUI 状态可跨会话保存。
- **跨平台**：兼容 Linux 和 macOS 系统。
