### [rev1si0n lamda](https://github.com/rev1si0n/lamda)  ![GitHub Repo stars](https://img.shields.io/github/stars/rev1si0n/lamda?style=social)

FIRERPA Android 是一款轻量级、无外部依赖的下一代 Android 自动化框架，专为 AI 代理设计。核心功能包括：
1. 系统支持：兼容 Android 6.0 至 16，支持模拟器、云手机及 Root 环境，可运行长驻服务。
2. API 生态：提供 160+ 接口，覆盖设备状态、系统应用控制、UI 自动化、OCR、图像匹配、文件 I/O 及 Shell 执行，并配套 Python SDK。
3. AI 集成：支持 MCP/Agent 工具调用及扩展，内置脚本加密与证书管理。
4. 远程运维：具备低延迟远程桌面、WebSocket 视频流、文件传输、触摸流及实时 UI 诊断功能。
5. 安全与部署：集成 ADB/SSH/SCP、代理/VPN/MITM 及逆向工具（Frida/IDA），支持分布式部署（Hub/FRP/OpenVPN）。
