### [bol-van zapret-win-bundle](https://github.com/bol-van/zapret-win-bundle)  ![GitHub Repo stars](https://img.shields.io/github/stars/bol-van/zapret-win-bundle?style=social)

该项目是适用于 Windows 系统的 DPI 绕过工具集合，整合了 zapret1 和 zapret2 内核，支持 Windows 7 至 Server 2025 及 ARM64 架构。核心功能通过 winws/winws2 组件拦截并修改网络流量以绕过深度包检测。项目包含最小化 Cygwin 环境、blockcheck 分析工具、示例策略脚本及服务管理脚本。注意事项：非一键式解决方案，需理解工作原理；windivert 驱动可能触发杀毒软件误报；特定版本需额外配置（如开启测试签名或安装 ESU 更新）。
