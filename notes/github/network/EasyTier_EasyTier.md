### [EasyTier EasyTier](https://github.com/EasyTier/EasyTier)  ![GitHub Repo stars](https://img.shields.io/github/stars/EasyTier/EasyTier?style=social)

EasyTier 是一款基于 Rust 构建的简单、安全、去中心化虚拟私有网络（VPN）解决方案。
核心功能：
1. 去中心化架构：节点对等，无需中心化服务，支持 P2P 直连与 NAT 穿透。
2. 跨平台支持：兼容 Win/MacOS/Linux/Android 及 X86/ARM/MIPS 架构。
3. 安全加密：采用 AES-GCM 或 WireGuard 加密，防止中间人攻击。
4. 高级特性：支持子网代理、智能路由选择（低延迟优先）、零拷贝传输及 UDP 丢包抵抗（KCP/QUIC）。
5. 管理便捷：提供 Web 控制台、命令行及客户端，支持 WireGuard 集成及一键服务注册。
