### [TecharoHQ anubis](https://github.com/TecharoHQ/anubis)  ![GitHub Repo stars](https://img.shields.io/github/stars/TecharoHQ/anubis?style=social)

Anubis 是一个轻量级的 Web AI 防火墙工具，通过向访问请求施加一种或多种挑战来验证用户，从而保护上游资源免受爬虫机器人（特别是 AI 公司的大量自动请求）的恶意攻击。它可作为 Cloudflare 的替代防护方案，适用于需要自行部署的场景。虽然该工具可能会拦截部分“好机器人”（如互联网档案馆），但支持通过配置白名单策略来允许特定可信机器人通过。
