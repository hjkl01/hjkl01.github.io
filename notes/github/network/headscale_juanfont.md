### [juanfont headscale](https://github.com/juanfont/headscale)  ![GitHub Repo stars](https://img.shields.io/github/stars/juanfont/headscale?style=social)

Headscale 是一个开源、自托管的 Tailscale 控制服务器实现。它基于 Wireguard 技术，负责管理 Tailscale 网络中的节点公钥交换、客户端 IP 分配、用户边界创建、机器共享及路由广告。该项目旨在为自托管者和爱好者提供替代 Tailscale 官方控制服务器的开源方案，适用于个人或小型组织的单一 Tailnet 网络环境。
