### [tunnelto](https://github.com/agrinman/tunnelto) ![GitHub Repo stars](https://img.shields.io/github/stars/agrinman/tunnelto?style=social)

**生成项目功能中文摘要**

该项目是使用Rust编写的tunnelto，功能是通过公共URL隧道暴露本地Web服务器，支持客户端开启隧道、多种配置选项及自托管。

tunnelto 是用 Rust 编写的内网穿透工具，可将本地运行的 Web 服务器通过公网 URL 暴露出去；提供命令行客户端，支持指定端口、子域名、认证、转发协议等参数，并可自行部署服务器端以建立隧道转发流量。
