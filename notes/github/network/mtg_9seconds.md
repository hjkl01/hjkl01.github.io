### [mtg](https://github.com/9seconds/mtg) ![GitHub Repo stars](https://img.shields.io/github/stars/9seconds/mtg?style=social)

mtg 是一个简洁高性能的 Telegram MTPROTO 代理，主打在受限环境中稳定连接 Telegram，支持域名前置与失败时真实网站回退、TLS流量伪装（Doppelganger）、防重放、资源高效、易部署；支持 Proxy Protocol v1/v2、单一密钥、无广告标签与管理界面，内置 SOCKS5 代理链、IP 黑名单/白名单、可作为 Go 库嵌入，并提供 statsd/Prometheus 监控指标与配置文件/命令行启动方式。
