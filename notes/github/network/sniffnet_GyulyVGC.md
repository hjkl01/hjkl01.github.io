### [GyulyVGC sniffnet](https://github.com/GyulyVGC/sniffnet)  ![GitHub Repo stars](https://img.shields.io/github/stars/GyulyVGC/sniffnet?style=social)

Sniffnet 是一款跨平台的互联网流量监控应用程序，界面直观且可靠。其核心功能包括：支持选择网络适配器实时监控网络流量及统计；识别 6000 多种服务、协议、木马和蠕虫；查询主机域名、ASN 及地理位置；支持本地网络连接识别；提供 PCAP 捕获文件的导入与导出；支持自定义通知、主题及最小化后台监控；兼容 Windows、macOS 和 Linux 系统。
