### [seanmonstar reqwest](https://github.com/seanmonstar/reqwest)  ![GitHub Repo stars](https://img.shields.io/github/stars/seanmonstar/reqwest?style=social)

reqwest 是一个功能齐全的 Rust HTTP 客户端库。它支持异步和阻塞两种客户端模式，能够处理纯文本、JSON、URL 编码及多部分表单数据。库内提供 Cookie 存储，支持 HTTP 代理和可自定义的重定向策略。默认使用 rustls 支持 HTTPS，也可选择系统原生 TLS，同时兼容 WebAssembly 环境。
