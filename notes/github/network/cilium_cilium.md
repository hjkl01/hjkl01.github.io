### [cilium cilium](https://github.com/cilium/cilium)  ![GitHub Repo stars](https://img.shields.io/github/stars/cilium/cilium?style=social)

Cilium 是基于 eBPF 的 Kubernetes 网络、可观测性和安全解决方案（CNI 插件）。它提供扁平化三层网络，支持跨集群互联（原生路由或 Overlay 模式）。核心功能包括：基于身份的 L3-L7 安全策略，替代 kube-proxy 实现分布式负载均衡，提供集成网关、带宽管理和服务网格能力。内置 Hubble 平台提供流量可视与故障诊断，支持 Cluster Mesh 实现跨集群统一身份连接。作为 CNCF 毕业项目，支持多架构及软件物料清单。
