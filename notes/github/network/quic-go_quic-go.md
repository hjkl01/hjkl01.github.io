### [quic-go quic-go](https://github.com/quic-go/quic-go)  ![GitHub Repo stars](https://img.shields.io/github/stars/quic-go/quic-go?style=social)

quic-go 是一个基于纯 Go 语言的 QUIC 协议实现，符合 RFC 9000/9001/9002 标准并支持 HTTP/3、QPACK 及 HTTP Datagrams。它实现了不可靠数据报扩展、DPLPMTUD、QUIC 版本 2、qlog 事件日志及流重置等扩展功能。该库性能高效，被广泛用于 Caddy、traefik、frp 等主流 Web 服务器、代理工具及分布式网络应用中。
