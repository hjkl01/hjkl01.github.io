### [wavetermdev waveterm](https://github.com/wavetermdev/waveterm)  ![GitHub Repo stars](https://img.shields.io/github/stars/wavetermdev/waveterm?style=social)

Wave Terminal 是一款面向 macOS、Linux 和 Windows 的开源 AI 集成终端。主要功能包括：

1.  **AI 助手**：具备上下文感知能力，可读取终端输出、执行文件操作，支持接入多种本地及云端 AI 模型。
2.  **持久化 SSH**：远程会话可抵御网络中断与重启，支持自动重连。
3.  **灵活界面**：支持拖放管理终端块、编辑器、浏览器及 AI 组件，提供一键全屏切换。
4.  **内置工具**：集成支持语法高亮的远程文件编辑器，及多格式文件内联预览系统。
5.  **安全与便捷**：使用系统后端安全存储密钥，提供 `wsh` 命令行系统管理工作空间及文件同步。
