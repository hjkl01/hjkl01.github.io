### [tonyantony300 alt-sendme](https://github.com/tonyantony300/alt-sendme)  ![GitHub Repo stars](https://img.shields.io/github/stars/tonyantony300/alt-sendme?style=social)

AltSendme 是一款免费开源的点对点文件传输工具。它利用先进的 P2P 网络技术（基于 Iroh 和 QUIC 协议），实现设备间的直接文件传输，无需经过云端服务器存储。支持端到端加密（TLS 1.3）及多语言界面，无需注册账号或个人信息，通过生成临时分享码发起传输。支持任意大小和格式的文件及目录，具备断点续传能力。适用于 Windows、macOS、Linux 平台，支持本地网络或跨互联网环境，确保传输安全、快速且隐私无忧。
