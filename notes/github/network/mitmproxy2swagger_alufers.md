### [alufers mitmproxy2swagger](https://github.com/alufers/mitmproxy2swagger)  ![GitHub Repo stars](https://img.shields.io/github/stars/alufers/mitmproxy2swagger?style=social)

mitmproxy2swagger 是一个命令行工具，通过监控应用程序的 HTTP 流量自动反向工程 REST API。它支持将 mitmproxy 捕获文件或浏览器 HAR 文件转换为 OpenAPI 3.0 规范，具备流量数据合并、路径筛选功能，并可选项地在生成的规范中添加请求和响应的示例数据及头部信息，最终输出 API 规格文档。
