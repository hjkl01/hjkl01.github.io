### [vernesong OpenClash](https://github.com/vernesong/OpenClash)  ![GitHub Repo stars](https://img.shields.io/github/stars/vernesong/OpenClash?style=social)

OpenClash 是一款运行在 OpenWrt 系统上的 Clash 客户端插件。它兼容 Shadowsocks、Vmess、Trojan、Snell 等多种代理协议，支持通过灵活规则配置实现策略代理。项目提供 Web 管理界面，支持配置订阅、服务器管理、规则设置、运行日志查看及策略组管理等功能。
