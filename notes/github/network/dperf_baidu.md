### [baidu dperf](https://github.com/baidu/dperf)  ![GitHub Repo stars](https://img.shields.io/github/stars/baidu/dperf?style=social)

dperf 是一款基于 DPDK 的高性能网络流量生成与负载测试工具。它能在单台 x86 服务器上实现千万级 HTTP 连接每秒、数百 Gbps 吞吐量及数十亿并发连接。项目提供详细的实时统计指标，可识别数据包丢失或错误。适用于四层负载均衡器和网关的负载与稳定性测试、云服务器网络性能基准测试、网卡及 CPU 包处理能力评估，也可作为高性能 HTTP 服务器或客户端使用。
