### [3x-ui](https://github.com/MHSanaei/3x-ui) ![GitHub Repo stars](https://img.shields.io/github/stars/MHSanaei/3x-ui?style=social)

3X-UI 是一个高级开源的 Web 控制面板，用于管理 Xray-core 服务器，提供友好的界面来配置和监控多种 VPN 与代理协议。作为 X-UI 的增强分支，它强调更高稳定性、更广协议支持和更多扩展功能。项目提供一键安装脚本与完整 Wiki 文档，并集成了 Iran/Russia 的 v2ray 路由规则资源。项目采用 GPL-3.0 许可证，作者明确声明仅建议个人使用，不应用于非法用途或生产环境。
