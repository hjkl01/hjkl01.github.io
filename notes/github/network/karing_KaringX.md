### [karing](https://github.com/KaringX/karing) ![GitHub Repo stars](https://img.shields.io/github/stars/KaringX/karing?style=social)

Karing 是基于 Flutter 的 sing-box 图形化代理工具，支持 Clash、V2ray/V2fly、Sing-box、Shadowsocks 等订阅与配置；可自动按路由规则选择节点，支持自定义规则组与节点组，内置 GeoIP/GeoSite/ACL 规则；提供多设备配置备份与同步（iCloud、局域网、WebDAV、ZIP 导入导出）；内置改版 sing-box 核心，并提供新手模式，支持多平台。
