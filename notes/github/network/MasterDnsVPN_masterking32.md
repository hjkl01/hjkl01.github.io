### [MasterDnsVPN](https://github.com/masterking32/MasterDnsVPN) ![GitHub Repo stars](https://img.shields.io/github/stars/masterking32/MasterDnsVPN?style=social)

MasterDnsVPN 是一个通过 DNS 查询与响应承载 TCP 流量的研究型高级 DNS 隧道/VPN，面向强审查与恶劣网络环境，采用轻量自定义协议与 ARQ 重传机制，支持多解析器并行、流量复制、智能健康检测与自动故障转移，提供 SOCKS5/SOCKS4 代理与可选本地 DNS 缓存，支持多种加密与压缩，具备低开销、高稳定、强抗丢包与强抗审查能力，并可在多平台客户端/服务器部署。
