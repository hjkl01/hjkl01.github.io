### [next-terminal](https://github.com/dushixiang/next-terminal) ![GitHub Repo stars](https://img.shields.io/github/stars/dushixiang/next-terminal?style=social)

Next Terminal 是一个面向企业 IT 环境的简洁、安全、易用的交互式审计系统，支持 RDP、SSH、VNC、Telnet、HTTP 等多种远程访问协议。其核心功能是帮助企业实现会话录制、审计追踪与合规报告。  
项目自 v2.0.0 起后端代码不再开源，使用前需遵守许可证条款，且建议在企业网络部署前先咨询 IT 管理员。
