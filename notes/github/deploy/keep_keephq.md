### [keephq keep](https://github.com/keephq/keep)  ![GitHub Repo stars](https://img.shields.io/github/stars/keephq/keep?style=social)

Keep 是一个开源的 AIOps 和警报管理平台，提供统一的警报与事件管理视图。它具备警报去重、过滤、关联和增强功能，支持广泛的监控、AI、数据库、通信及工单工具的双向集成。平台通过声明式 YAML 工作流实现自动化响应（类似 GitHub Actions），并提供 AI 驱动的关联与摘要能力。此外，Keep 支持企业级安全认证、灵活部署方案及高可用扩展。
