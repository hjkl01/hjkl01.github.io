### [aws aws-sdk-go-v2](https://github.com/aws/aws-sdk-go-v2)  ![GitHub Repo stars](https://img.shields.io/github/stars/aws/aws-sdk-go-v2?style=social)

AWS SDK for Go v2 是适用于 Go 编程语言的 AWS 软件开发工具包第二版，最低要求 Go 1.24 版本。它实现了 AWS 服务 API，允许开发者使用 Go 代码调用各类 AWS 服务（如 DynamoDB）。SDK 支持从环境变量、共享凭证及配置文件中加载配置，并提供客户端实例进行 API 请求。项目提供开发者指南、迁移指南及 API 参考文档，遵循 Apache 2.0 许可，支持通过 GitHub 进行 Bug 反馈、功能请求及社区代码贡献。
