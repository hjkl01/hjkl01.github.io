### [get-convex chef](https://github.com/get-convex/chef)  ![GitHub Repo stars](https://img.shields.io/github/stars/get-convex/chef?style=social)

Chef 是由 Convex 团队开发的 AI 全栈 Web 应用构建工具。它基于 Convex 响应式数据库构建，能够生成包含内置数据库、零配置认证、文件上传、实时 UI 及后台工作流的应用。项目支持托管服务及本地运行，核心架构包含客户端前端、服务端代理循环、CLI 工具及数据库模块。作为 bolt.diy 的分支，它利用 Convex API 实现 AI 代码生成，生产部署时需自行配置认证系统。
