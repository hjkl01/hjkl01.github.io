### [1Panel-dev 1Panel](https://github.com/1Panel-dev/1Panel)  ![GitHub Repo stars](https://img.shields.io/github/stars/1Panel-dev/1Panel?style=social)

1Panel 是一款现代化的开源 VPS 控制面板，通过直观的 Web 界面简化服务器管理。主要功能包括：一键部署 OpenClaw 代理及本地大模型（支持 GPU 监控）；一键网站部署（自动域名绑定与 SSL 配置）；提供精选应用市场（快速安装 Nextcloud 等开源应用）；集中管理服务器资源、文件、数据库、容器和服务；具备企业级安全防护（容器隔离、防火墙、审计日志）；支持智能备份与恢复（本地或云端存储）。项目提供社区版和 Pro 企业版。
