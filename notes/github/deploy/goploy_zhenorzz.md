### [zhenorzz goploy](https://github.com/zhenorzz/goploy)  ![GitHub Repo stars](https://img.shields.io/github/stars/zhenorzz/goploy?style=social)

Goploy 是一个 Web 部署系统工具，支持通过网页一键实现代码发布与回滚。核心功能包括项目构建、支持 Git/SVN/FTP/SFTP 协议、跨操作系统部署、RBAC 权限控制、全方位监控（HTTP/TCP/Ping/进程/脚本/服务器）、秒级定时任务、Xterm 终端及 LDAP 集成。后端基于 Go 开发，支持 Docker 部署。
