### [dagger dagger](https://github.com/dagger/dagger)  ![GitHub Repo stars](https://img.shields.io/github/stars/dagger/dagger?style=social)

Dagger 是一个自动化软件交付平台，能够可靠地构建、测试和部署任意代码库，支持本地、CI 或云端运行，仅依赖容器运行时。

核心特性：
- **可编程**：提供执行引擎、系统 API 及 8 种语言 SDK。
- **可重复**：基于容器和增量执行，自动缓存，确保跨环境一致性。
- **可观测**：内置 OpenTelemetry 追踪与日志，支持可视化调试。

所有操作类型安全且可组合。
