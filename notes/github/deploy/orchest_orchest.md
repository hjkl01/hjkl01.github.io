### [orchest orchest](https://github.com/orchest/orchest)  ![GitHub Repo stars](https://img.shields.io/github/stars/orchest/orchest?style=social)

Orchest 是一个数据流程编排工具，支持直接使用 Python、R 或 Julia 编写数据处理代码，无需框架或 YAML。它提供友好的可视化界面构建管道，支持 Notebook 和脚本编码，可运行任务子集或定时任务，定义跨机器依赖环境，管理服务生命周期，并支持 Git 版本控制。目前项目已不再积极开发，建议改用 Apache Airflow。
