### [KusionStack karpor](https://github.com/KusionStack/karpor)  ![GitHub Repo stars](https://img.shields.io/github/stars/KusionStack/karpor?style=social)

Karpor 是一款 Kubernetes 可视化工具，集成了智能搜索、洞察和 AI 能力，帮助用户实现多云环境下 Kubernetes 集群的可观测性与高效管理。

主要功能包括：
- **搜索**：自动同步多集群资源，支持灵活强大的查询功能。
- **洞察**：提供合规治理状态、资源逻辑拓扑视图及成本优化建议。
- **AI**：支持自然语言操作、上下文智能响应及 AIOps 自动化运维。

项目致力于打破 Kubernetes 生态的复杂性，是供应商中立、开发者友好的开源项目。
