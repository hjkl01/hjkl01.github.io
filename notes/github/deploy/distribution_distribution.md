### [distribution distribution](https://github.com/distribution/distribution)  ![GitHub Repo stars](https://img.shields.io/github/stars/distribution/distribution?style=social)

Distribution 是一个基于 OCI 分发规范的开源容器注册表实现工具集，核心功能为存储、打包、运输和分发容器镜像及其他内容。该项目旨在为构建大规模注册表或运行私有注册表提供简单、安全且可扩展的基础，核心组件包括符合规范的 Registry 实现、交互库及文档。它是 Docker Hub、GitHub Container Registry 等多个知名平台的核心库，支持通过 HTTP 与 OCI 客户端进行通信。
