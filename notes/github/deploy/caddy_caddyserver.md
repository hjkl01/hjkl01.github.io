### [caddyserver caddy](https://github.com/caddyserver/caddy)  ![GitHub Repo stars](https://img.shields.io/github/stars/caddyserver/caddy?style=social)

Caddy 是一款默认启用 TLS 的 Go 语言扩展服务器平台。它主要作为 HTTPS 服务器使用，支持自动证书管理（Let's Encrypt/ZeroSSL/私有 CA）、HTTP/1.1/2/3 协议、Caddyfile/JSON 及多种格式配置转换、动态 API 修改及模块化扩展。项目采用静态二进制构建无外部依赖，具有高内存安全性和生产级稳定性，适用于大规模网站托管及运行任意 Go 应用。
