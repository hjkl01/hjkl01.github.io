### [argoproj argo-workflows](https://github.com/argoproj/argo-workflows)  ![GitHub Repo stars](https://img.shields.io/github/stars/argoproj/argo-workflows?style=social)

Argo Workflows 是运行在 Kubernetes 上的开源容器原生工作流引擎，通过 CRD 实现并行作业编排。支持将步骤定义为容器，利用 DAG 或步骤序列构建工作流，具备可视化 UI、多存储支持、定时调度、重试与超时控制、参数化及条件逻辑。提供多语言 SDK，适用于机器学习、数据处理、CI/CD 及基础设施自动化等场景。作为 CNCF 毕业项目，拥有 REST API、Prometheus 监控及完善的资源调度与管理能力。
