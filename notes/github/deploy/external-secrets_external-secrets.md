### [external-secrets external-secrets](https://github.com/external-secrets/external-secrets)  ![GitHub Repo stars](https://img.shields.io/github/stars/external-secrets/external-secrets?style=social)

External Secrets Operator 是一个 Kubernetes 操作符，用于集成外部密钥管理系统（如 AWS Secrets Manager、HashiCorp Vault、Google Secrets Manager、Azure Key Vault 等）。它能从外部 API 读取信息，并自动将密钥值注入到 Kubernetes Secret 中。
