### [opencontainers distribution-spec](https://github.com/opencontainers/distribution-spec)  ![GitHub Repo stars](https://img.shields.io/github/stars/opencontainers/distribution-spec?style=social)

该项目定义了 OCI 内容分发规范的 API 协议，旨在标准化容器镜像及其他类型内容的分发流程。项目提供内容存储、检索和管理的标准接口，配套 Go 类型定义与注册表合规性测试工具，并支持扩展 API 的原型设计与测试。
