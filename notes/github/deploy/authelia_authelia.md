### [authelia authelia](https://github.com/authelia/authelia)  ![GitHub Repo stars](https://img.shields.io/github/stars/authelia/authelia?style=social)

Authelia 是一个开源的认证与授权服务器，通过 Web 门户提供双因素认证（2FA）和单点登录（SSO），并作为反向代理（如 Nginx、Traefik、Caddy）的辅助组件控制请求。它支持安全密钥、TOTP、推送通知及无密码认证等多种第二因素方式，提供基于子域名、用户、URI 等条件的细粒度访问控制。此外，Authelia 符合 OpenID Connect 1.0 和 OAuth 2.0 标准，支持密码重置、高可用部署，并兼容 Docker、Kubernetes 等多种环境。
