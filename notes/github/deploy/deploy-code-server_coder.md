### [coder deploy-code-server](https://github.com/coder/deploy-code-server)  ![GitHub Repo stars](https://img.shields.io/github/stars/coder/deploy-code-server?style=social)

该项目提供一键部署脚本和按钮，支持将 code-server 快速部署至多种云托管平台（如 AWS、DigitalOcean、Heroku 等）。项目包含虚拟机和容器两种部署模式：虚拟机部署支持持久化存储和软件安装；容器部署可关闭以节省费用，环境重置干净，但存储可能不持久。
