### [TelegramMessenger Telegram-iOS](https://github.com/TelegramMessenger/Telegram-iOS)  ![GitHub Repo stars](https://img.shields.io/github/stars/TelegramMessenger/Telegram-iOS?style=social)

这是一个 Telegram iOS 源码编译指南，旨在帮助开发者利用官方 API 和源码创建应用程序。内容涵盖开发者合规要求（api_id 获取、标识与安全规范）、编译环境搭建（Git 克隆、Xcode 配置）、项目生成与构建流程（含开发版、Appstore 及 IPA 打包），以及常见问题解答与构建技巧（如模拟器构建、版本校验）。
