### [harness drone](https://github.com/harness/drone)  ![GitHub Repo stars](https://img.shields.io/github/stars/harness/drone?style=social)

Harness Open Source 是一个开源开发平台，核心功能涵盖代码托管、自动化 DevOps 流水线、托管开发环境（Gitspaces）以及制品注册中心。作为 Drone 的演进版本，它提供端到端的开源 DevOps 解决方案。平台支持通过 Docker 或本地环境部署运行，提供 Web 用户界面、REST API 及命令行工具，协助团队实现完整的代码管理和开发运维流程。
