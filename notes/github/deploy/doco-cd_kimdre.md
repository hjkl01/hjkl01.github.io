### [kimdre doco-cd](https://github.com/kimdre/doco-cd)  ![GitHub Repo stars](https://img.shields.io/github/stars/kimdre/doco-cd?style=social)

doco-cd 是一款轻量级 GitOps 工具，支持自动部署和更新 Docker Compose 项目及 Swarm 堆栈。它通过轮询和 Webhooks 触发部署，基于 Go 构建，资源占用极低。支持多种 Git 提供商、外部密钥管理与 SOPS 加密，并提供通知及 Prometheus 监控功能，是 Portainer 或 ArgoCD 的简化替代方案。
