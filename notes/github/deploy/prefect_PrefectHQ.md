### [PrefectHQ prefect](https://github.com/PrefectHQ/prefect)  ![GitHub Repo stars](https://img.shields.io/github/stars/PrefectHQ/prefect?style=social)

Prefect 是一个 Python 工作流编排框架，用于构建数据管道。它可将脚本转化为生产级工作流，支持构建弹性、动态的数据流程，具备调度、缓存、重试、事件自动化及依赖管理等核心功能。工作流活动可通过自托管的 Prefect Server 或托管的 Prefect Cloud 进行监控，并支持部署至生产环境。此外，项目还提供轻量级的 prefect-client 用于客户端与 Prefect Cloud 或远程服务器通信。
