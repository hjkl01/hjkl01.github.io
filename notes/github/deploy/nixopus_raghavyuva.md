### [raghavyuva nixopus](https://github.com/raghavyuva/nixopus)  ![GitHub Repo stars](https://img.shields.io/github/stars/raghavyuva/nixopus?style=social)

Nixopus 是一个开源的自托管平台，旨在替代 Vercel、Heroku 和 Netlify，支持在 VPS 上进行应用部署与终端集成。目前处于 Alpha 预发布阶段，不建议用于生产环境。

核心功能包括：
- **一键部署**：无需配置文件或 SSH 命令即可部署应用。
- **GitHub 集成**：推送代码即可自动部署。
- **浏览器文件管理**：支持文件的拖拽、编辑和管理。
- **内置终端**：无需离开页面即可直接访问服务器终端。
- **实时监控**：实时查看 CPU、内存及磁盘使用情况。
- **自动 HTTPS**：自动为域名配置 SSL 证书。
- **代理管理**：集成 Caddy 反向代理进行流量路由。
- **智能警报**：支持通过 Slack、Discord 或邮件接收系统通知。
- **简易安装**：提供 CLI 命令快速安装，支持自定义域名或 IP 地址。
