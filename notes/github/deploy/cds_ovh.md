### [ovh cds](https://github.com/ovh/cds)  ![GitHub Repo stars](https://img.shields.io/github/stars/ovh/cds?style=social)

CDS 是一款企业级持续交付与 DevOps 自动化平台，提供 Web UI 和命令行工具（cdsctl）用于构建、运行和管理复杂工作流。它支持工作流即代码（YAML）、多 Git 平台集成、多环境部署及安全权限控制，并提供多租户隔离。平台具备高可用性、基于云基础设施的自动弹性伸缩、插件扩展、REST API/SDK 及内置监控功能。数据基于数据库存储，支持自托管部署。
