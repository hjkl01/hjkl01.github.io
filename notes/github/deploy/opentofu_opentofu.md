### [opentofu opentofu](https://github.com/opentofu/opentofu)  ![GitHub Repo stars](https://img.shields.io/github/stars/opentofu/opentofu?style=social)

OpenTofu 是一款开源基础设施管理工具，支持安全、高效地构建、变更和版本化基础设施，可管理主流云厂商及自定义解决方案。核心功能包括：

1. **基础设施即代码**：通过高级配置语法描述资源，实现版本控制、共享与复用。
2. **执行计划**：预先展示变更操作，避免意外影响基础设施。
3. **资源图**：构建资源依赖关系图，并行处理非依赖资源以提升构建效率。
4. **变更自动化**：以最小人工干预执行复杂变更，确保顺序明确并减少人为错误。
