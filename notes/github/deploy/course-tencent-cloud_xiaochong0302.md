### [xiaochong0302 course-tencent-cloud](https://github.com/xiaochong0302/course-tencent-cloud)  ![GitHub Repo stars](https://img.shields.io/github/stars/xiaochong0302/course-tencent-cloud?style=social)

酷瓜云课堂是一套基于腾讯云架构和 Phalcon 框架的开源知识付费与在线教育系统。核心功能涵盖视频点播、直播、专栏、问答、会员及积分。采用 Phalcon、Layui、Xunsearch、PHP、MySQL、Redis 等技术栈，提供演示系统、部署文档及社区支持，致力于提供完整的网课、网校及在线教育解决方案。
