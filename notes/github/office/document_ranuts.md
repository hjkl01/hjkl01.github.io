### [ranuts document](https://github.com/ranuts/document)  ![GitHub Repo stars](https://img.shields.io/github/stars/ranuts/document?style=social)

这是一个基于 OnlyOffice 的纯前端网页文档编辑器。无需后端服务器，所有文档处理均在浏览器本地完成，确保隐私安全。支持 DOCX、XLSX、PPTX 等多种格式的实时编辑，可从 URL 加载文档，支持 PWA 离线使用及多语言切换。项目采用 WebAssembly 实现格式转换，支持 Docker 部署，并提供文档预览组件库服务。
