### [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) ![GitHub Repo stars](https://img.shields.io/github/stars/Stirling-Tools/Stirling-PDF?style=social)

Stirling PDF 是一款功能强大的开源 PDF 编辑平台，支持在桌面端、浏览器或私有服务器部署。它提供 50 多种 PDF 工具（如编辑、合并、拆分、签名、OCR、压缩等），支持无代码自动化工作流及 REST API 集成，具备企业级安全特性（SSO、审计），界面支持 40 多种语言，确保用户无需将文档发送至外部服务即可本地处理文件。
