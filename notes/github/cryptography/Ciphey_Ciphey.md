### [Ciphey Ciphey](https://github.com/Ciphey/Ciphey)  ![GitHub Repo stars](https://img.shields.io/github/stars/Ciphey/Ciphey?style=social)

Ciphey 是一款完全自动化的解密、解码与破解工具，基于自然语言处理（NLP）和人工智能技术自动识别加密类型。用户无需知晓具体加密方式，输入密文即可快速获取明文。项目支持 50 多种编码、经典密码、现代加密及哈希算法，通常在 3 秒内完成解密。核心采用 C++ 编写以确保高性能，提供命令行及 API 接口，适用于网络安全、CTF 竞赛及快速验证密文等场景。
