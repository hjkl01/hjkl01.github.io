### [FiloSottile age](https://github.com/FiloSottile/age)  ![GitHub Repo stars](https://img.shields.io/github/stars/FiloSottile/age?style=social)

age 是一个简单、现代且安全的文件加密工具、文件格式规范及 Go 库。它支持密钥生成，并可使用 age 或 SSH 公钥、多接收者及密码短语进行文件加密与解密。项目具备密钥小巧、无配置选项、UNIX 风格可组合性，支持后量子加密、硬件令牌插件及加密文件元数据检查。此外，它提供 Rust 和 TypeScript 等语言的互操作实现，兼容多种操作系统平台。
