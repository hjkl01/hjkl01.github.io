### [plane](https://github.com/makeplane/plane) ![GitHub Repo stars](https://img.shields.io/github/stars/makeplane/plane?style=social)

Plane：开源项目管理工具。追踪事务、管理周期与产品路线图。

核心功能：
- 工作项：创建/管理任务。支持富文本、文件上传、子属性与关联。
- 周期：管理团队冲刺。利用燃尽图追踪进度。
- 模块：拆分复杂项目。
- 视图：自定义过滤条件。保存并共享结果。
- 页面：记录想法。集成AI与富文本。笔记转行动项。
- 分析：获取实时洞察。可视化趋势。

部署方式：
- Plane Cloud (云端)
- 自建托管 (Docker / Kubernetes)

技术栈：
React Router, Django, Node JS。
