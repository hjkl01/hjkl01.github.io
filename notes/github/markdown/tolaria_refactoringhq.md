### [tolaria](https://github.com/refactoringhq/tolaria) ![GitHub Repo stars](https://img.shields.io/github/stars/refactoringhq/tolaria?style=social)

Tolaria 是一款支持 macOS、Windows、Linux 的桌面应用，用于管理 **Markdown 知识库**。其核心功能与定位包括：

- 以纯 Markdown 文件管理笔记，可用于个人知识管理、第二大脑、企业文档整理及 AI 助手记忆与流程存储。  
- **文件优先 + Git 优先**：每个知识库都是 Git 仓库，支持完整版本历史和任意远程仓库，不依赖官方服务器。  
- **离线优先、零锁定**：无需账号/订阅/云服务，数据始终本地可用，可随时迁移到其他工具。  
- 基于标准格式（Markdown + YAML frontmatter），避免专有数据结构。  
- 支持“类型”作为导航与检索辅助，而非强制字段校验。  
- **AI 友好**：可配合 Claude Code、Codex CLI、Gemini CLI 等使用，也可与任意编辑器/AI 工具协作。  
- **键盘优先**设计，面向高频操作用户。  
- 开源免费（AGPL-3.0-or-later），基于 Tauri + React + TypeScript 构建。  
- 提供 Homebrew 与发布页安装方式，并可克隆官方入门仓库快速上手。
