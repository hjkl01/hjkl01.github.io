### [jesse-ai jesse](https://github.com/jesse-ai/jesse)  ![GitHub Repo stars](https://img.shields.io/github/stars/jesse-ai/jesse?style=social)

Jesse 是一个先进的加密货币量化交易框架，旨在简化交易策略的研发、回测、优化及实盘交易。它支持使用简洁的 Python 语法定义策略，内置 300 多种技术指标，提供无未来偏见的精准回测功能，支持现货、合约（含杠杆与做空）的实盘与模拟交易。此外，项目还具备性能评估、风险管理、调试工具、AI 辅助优化（JesseGPT）等功能，并采用完全自托管架构以确保数据隐私安全。
