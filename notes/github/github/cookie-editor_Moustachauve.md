### [Moustachauve cookie-editor](https://github.com/Moustachauve/cookie-editor)  ![GitHub Repo stars](https://img.shields.io/github/stars/Moustachauve/cookie-editor?style=social)

Cookie-Editor 是一款浏览器扩展，支持用户高效创建、编辑和删除当前标签页的 Cookie。主要功能包括管理单个 Cookie 以及一键批量删除当前页面的所有 Cookie。界面简洁，适用于网页开发、测试及隐私管理，支持主流桌面及移动端浏览器。
