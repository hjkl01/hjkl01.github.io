### [toeverything AFFiNE](https://github.com/toeverything/AFFiNE)  ![GitHub Repo stars](https://img.shields.io/github/stars/toeverything/AFFiNE?style=social)

AFFiNE 是一款开源、隐私优先且本地优先的跨平台一体化工作空间，是 Notion 和 Miro 的替代方案。该项目实现了文档、画布和表格的深度融合，支持在无边画布上自由编排富文本、便签、数据库、网页嵌入等多种构建模块。核心功能包括多模态 AI 辅助创作（写作、演示、思维导图、原型设计）、本地数据拥有权与跨平台实时协作，以及支持用户自托管和二次开发。适用于知识管理、笔记、演示制作及数字资产管理。
