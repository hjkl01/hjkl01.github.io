### [TabbyML tabby](https://github.com/TabbyML/tabby)  ![GitHub Repo stars](https://img.shields.io/github/stars/TabbyML/tabby?style=social)

Tabby 是一个开源的自托管 AI 编程助手，提供 GitHub Copilot 的替代方案。主要功能包括：
- 独立部署：无需数据库或云服务，支持消费级 GPU，可通过 Docker 快速运行。
- 开发集成：提供 OpenAPI 接口，支持 VSCode、Vim、IntelliJ 等 IDE 扩展，易于接入现有基础设施。
- 代码辅助：具备代码补全、聊天助手、仓库上下文理解及 Answer Engine 团队知识库功能。
- 企业特性：内置团队协作管理、Git 仓库集成（GitHub/GitLab）、REST API 及 Admin 管理界面。
