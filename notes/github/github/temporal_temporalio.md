### [temporalio temporal](https://github.com/temporalio/temporal)  ![GitHub Repo stars](https://img.shields.io/github/stars/temporalio/temporal?style=social)

Temporal 是一个耐久的执行平台，帮助开发者构建可扩展且高可靠的应用程序。服务器以稳健方式执行工作流，自动处理间歇性故障并重试失败操作。该项目包含 Temporal 服务器源代码，支持多语言实现工作流、活动和 Worker，并提供本地部署、CLI 工具及 Web UI 用于运行、管理和监控。
