### [hysteria](https://github.com/apernet/hysteria) ![GitHub Repo stars](https://img.shields.io/github/stars/apernet/hysteria?style=social)

Hysteria 2 是一款强大、极速且抗审查的代理工具。它支持多种工作模式（如 SOCKS5、HTTP 代理、TCP/UDP 转发、Linux TProxy、TUN 等），基于定制 QUIC 协议，在不稳定和高丢包网络中也能保持高性能；其流量可伪装为标准 HTTP/3，降低被识别和封锁的风险。项目具备跨平台与多架构支持，便于在不同环境部署和使用；同时提供自定义认证、流量统计和访问控制等能力，方便与现有基础设施集成，并配有完善文档、开发规范及活跃社区支持。
