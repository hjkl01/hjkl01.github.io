### [pytest](https://github.com/pytest-dev/pytest) ![GitHub Repo stars](https://img.shields.io/github/stars/pytest-dev/pytest?style=social)

pytest 是一个 Python 测试框架，易于编写小型测试，可扩展支持复杂的应用和库功能测试。

主要功能：

- 详细展示 assert 失败信息
- 自动发现测试模块和函数
- 模块化 fixtures 管理测试资源
- 支持运行 unittest 测试套件
- 需 Python 3.10+ 或 PyPy3
- 丰富插件架构，支持 1300+ 外部插件
