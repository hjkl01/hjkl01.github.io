### [TNTwise REAL-Video-Enhancer](https://github.com/TNTwise/REAL-Video-Enhancer)  ![GitHub Repo stars](https://img.shields.io/github/stars/TNTwise/REAL-Video-Enhancer?style=social)

REAL Video Enhancer 是一款跨平台视频增强工具，支持 Windows、Linux 和 macOS 系统。核心功能包括视频帧插值（如提升帧率）和视频超分辨率上采样（如提升分辨率）。软件具备场景变化检测、渲染预览及 Discord RPC 支持。提供 TensorRT、PyTorch 和 NCNN 等多种加速后端，兼容多种主流 AI 模型。支持通过 Steam、Flathub 下载或源码构建。
