### [commaai openpilot](https://github.com/commaai/openpilot)  ![GitHub Repo stars](https://img.shields.io/github/stars/commaai/openpilot?style=social)

openpilot 是一款机器人操作系统，旨在升级 300 多种支持车型的驾驶辅助系统。

主要功能与特性：
1. **车辆适配**：支持 275 款以上车型，需配合兼容硬件设备（如 comma 3X）及汽车线束使用。
2. **软件分支**：提供稳定版、测试版及开发版，支持开源社区贡献与自定义开发。
3. **安全机制**：遵循 ISO26262 安全标准，具备完善的软硬件在环测试体系。
4. **数据管理**：默认上传驾驶数据（如摄像头、CAN、GPS 等）用于模型训练，用户可配置关闭非必要数据收集。
5. **许可声明**：采用 MIT 协议开源，明确为 Alpha 质量研究软件，非正式产品，用户需自行遵守当地法律法规。
