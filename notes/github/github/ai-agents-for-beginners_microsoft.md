### [microsoft ai-agents-for-beginners](https://github.com/microsoft/ai-agents-for-beginners)  ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/ai-agents-for-beginners?style=social)

本项目是一个面向初学者的 AI Agents 构建教学课程，旨在教授用户从零开始开发 AI Agents 的核心技能。课程基于 Microsoft Agent Framework 和 Azure AI Foundry Agent Service V2 技术栈，提供包含文本、Python 代码、视频及扩展资源的系列教学内容，涵盖基础概念、代理框架、多种设计模式（如工具使用、RAG、规划、多智能体）、可信性、生产部署及协议使用等。项目支持 50 多种语言，并通过 Microsoft Foundry Discord 社区提供技术支持与交流。
