Freqtrade 是一款基于 Python 编写的免费开源加密货币交易机器人，支持 Windows、macOS 和 Linux 系统。它兼容多种主流现货及期货交易所，可通过 Telegram 或内置 WebUI 进行控制。核心功能包括策略回测、基于机器学习的策略优化、干运行模拟、资产白/黑名单筛选、资金管理及盈亏法币显示。项目采用 SQLite 持久化存储，提供完整的命令行工具与自动化交易能力。
