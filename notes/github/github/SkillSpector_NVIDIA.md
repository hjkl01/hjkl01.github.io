### [SkillSpector](https://github.com/NVIDIA/SkillSpector) ![GitHub Repo stars](https://img.shields.io/github/stars/NVIDIA/SkillSpector?style=social)

AI 代理技能安全扫描器。检测 64 种漏洞模式（16 类别）。支持 Git 仓库、URL、ZIP 文件、目录或单文件输入。静态分析 + 可选 LLM 语义分析。实时 OSV.dev CVE 查询，离线自动回退。多格式输出（终端、JSON、Markdown、SARIF）。0-100 风险评分 + 建议。CLI 工具 + Python API，Docker 支持。
