### [runatlantis atlantis](https://github.com/runatlantis/atlantis)  ![GitHub Repo stars](https://img.shields.io/github/stars/runatlantis/atlantis?style=social)

Atlantis 是一个自托管的 Terraform Pull Request 自动化工具。它通过 webhook 监听 PR 事件，远程执行 Terraform 的 plan、import 和 apply 命令，并将结果评论在 PR 上。其功能包括让团队查看 Terraform 变更、支持非运维人员协作以及标准化 Terraform 工作流。
