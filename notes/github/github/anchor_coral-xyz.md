### [coral-xyz anchor](https://github.com/coral-xyz/anchor)  ![GitHub Repo stars](https://img.shields.io/github/stars/coral-xyz/anchor?style=social)

Anchor 是 Solana 智能合约开发框架，提供 Rust eDSL 用于编写程序，支持 IDL 规范生成及基于 IDL 的 TypeScript 客户端自动生成，并提供 CLI 和工作区管理工具。核心组件包括 Rust 库、客户端及 CLI 工具，旨在简化 Solana 程序的开发流程。需注意 API 处于活跃开发中可能变更，且代码未经审计。
