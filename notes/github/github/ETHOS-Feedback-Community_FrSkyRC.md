### [FrSkyRC ETHOS-Feedback-Community](https://github.com/FrSkyRC/ETHOS-Feedback-Community)  ![GitHub Repo stars](https://img.shields.io/github/stars/FrSkyRC/ETHOS-Feedback-Community?style=social)

本项目是 FrSky Ethos 系统的反馈与支持资源库，旨在收集用户建议与软件缺陷以推动系统优化。核心功能包括：

1. **问题反馈管理**：通过 GitHub 接收用户 Bug 报告，规范提交内容（固件版本、复现步骤、媒体文件等）以辅助开发修复。
2. **知识库支持**：提供 Ethos Wiki 链接，涵盖文件存储、固件刷写、TD ISRM 更新及仿真器信息。
3. **多语言手册**：提供中、英、法、德、意、西、葡等语言的 Ethos 用户手册 PDF 下载资源。
