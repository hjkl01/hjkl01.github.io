### [StackChan](https://github.com/m5stack/StackChan) ![GitHub Repo stars](https://img.shields.io/github/stars/m5stack/StackChan?style=social)

StackChan 开源项目汇总了与该机器人相关的完整资源，包括机器人固件、遥控器固件、移动端应用（iOS/Android）和服务器代码。  
该项目围绕一款由 M5Stack 与社区共创的 AI 桌面机器人展开，主控基于 CoreS3（ESP32-S3），支持 Wi-Fi/BLE，集成触摸屏、摄像头、传感器、IMU、扬声器、双麦克风、microSD 等；机身侧提供电池、双舵机、多组 RGB 灯、红外、三分区触控和 NFC。  
出厂固件提供 AI Agent、表情动画、ESP-NOW 遥控、在线应用下载、手机端视频查看与远程控制、OTA 升级等能力；同时支持 Arduino、UiFlow2 等二次开发，并可扩展 M5Stack 生态模块以实现自定义功能。  
此外，仓库更新可能略晚于已发布的固件和移动应用版本。
