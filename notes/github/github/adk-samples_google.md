### [google adk-samples](https://github.com/google/adk-samples)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/adk-samples?style=social)

该项目是 Agent Development Kit (ADK) 的样本仓库，提供基于 ADK 构建的即用型智能体集合，旨在加速开发流程。支持 Python、TypeScript、Go 和 Java 语言，包含多种覆盖简单对话到复杂多智能体工作流的场景示例（如客服、研究、写作等）。项目仅供演示，非 Google 官方支持产品，不可用于生产环境。
