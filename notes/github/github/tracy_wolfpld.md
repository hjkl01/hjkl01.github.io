### [wolfpld tracy](https://github.com/wolfpld/tracy)  ![GitHub Repo stars](https://img.shields.io/github/stars/wolfpld/tracy?style=social)

Tracy Profiler 是一款面向游戏及其他应用程序的实时性能分析工具，具备纳秒级分辨率与远程遥测功能，支持混合帧与采样分析。核心功能包括 CPU 分析（支持 C/C++/Lua/Python 等语言）、GPU 分析（支持 OpenGL/Vulkan/Direct3D 等主流图形 API）、内存分配与锁监控、上下文切换分析，以及自动将截图关联至捕获帧。
