### [lfnovo open-notebook](https://github.com/lfnovo/open-notebook)  ![GitHub Repo stars](https://img.shields.io/github/stars/lfnovo/open-notebook?style=social)

Open Notebook 是一款开源、注重隐私的 Google Notebook LM 替代方案，支持 100% 本地部署以实现数据自主可控。核心功能包括支持 16 多种 AI 模型（如 OpenAI、Ollama 等）、多模态内容管理（PDF、音视频、网页等）、专业多角色播客生成、智能搜索及上下文 AI 聊天。项目提供完整 REST API，支持 Docker 快速部署，具备开源可定制、无厂商锁定及灵活成本控制的特性。
