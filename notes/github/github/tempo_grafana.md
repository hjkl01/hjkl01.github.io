### [grafana tempo](https://github.com/grafana/tempo)  ![GitHub Repo stars](https://img.shields.io/github/stars/grafana/tempo?style=social)

Grafana Tempo 是一款开源、易用且支持高扩展的分布式追踪后端系统。它成本效益高，仅需对象存储即可运行，并深度集成于 Grafana、Prometheus 和 Loki 生态。主要功能包括：支持 Jaeger、Zipkin、Kafka 及 OpenTelemetry 标准，可缓冲并写入多种存储介质；提供 Traces Drilldown UI，实现无需复杂查询的直观追踪数据分析；实现 TraceQL 查询语言，支持基于追踪数据的指标创建与聚合查询。
