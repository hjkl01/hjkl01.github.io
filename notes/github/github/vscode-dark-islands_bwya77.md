### [vscode-dark-islands](https://github.com/bwya77/vscode-dark-islands) ![GitHub Repo stars](https://img.shields.io/github/stars/bwya77/vscode-dark-islands?style=social)

Islands Dark 是一个受 easemate IDE 启发的 VS Code 深色主题项目，由“配色主题 + CSS 界面定制”两部分组成，核心功能是把 VS Code 改造成“悬浮玻璃面板”风格的界面。

主要功能包括：深色画布背景、玻璃质感边框、全局圆角、药丸形活动栏与滚动条、标签页悬停渐显关闭按钮、面包屑与状态栏悬停高亮、侧边栏/滚动条/状态栏平滑动画、图标颜色匹配发光效果，以及对 JS/TS、Python、Go、Rust、HTML/CSS、JSON、YAML、Markdown 等语言的暖色语法高亮支持。默认字体搭配为编辑器 IBM Plex Mono、终端 FiraCode Nerd Font Mono，并配合 Bear Sans UI 用于界面文本。

项目提供一键脚本安装（macOS/Linux/Windows）、手动安装和 Nix Flake 运行方式；安装脚本可自动安装主题扩展与 Custom UI Style、备份并应用设置、启用样式并重载。还支持通过 CSS 变量快速自定义圆角、间距、面板层次和颜色，并提供故障排查与卸载脚本以恢复原配置。
