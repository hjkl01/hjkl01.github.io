### [Cybersecurity-Projects](https://github.com/CarterPerez-dev/Cybersecurity-Projects) ![GitHub Repo stars](https://img.shields.io/github/stars/CarterPerez-dev/Cybersecurity-Projects?style=social)

这是一个网络安全学习项目库，汇集了 70 个分层实践项目、10 条认证成长路线和配套学习资源，帮助学习者从零基础逐步成长到高级安全工程能力。

项目按难度分为 Foundations、Beginner、Intermediate、Advanced 四个层级，覆盖密码学、端口扫描、DNS、漏洞扫描、流量分析、日志与 SIEM、API 安全、密钥与凭证管理、DLP、容器安全、威胁检测、蜜罐、漏洞挖掘、恶意软件分析、云安全合规、后量子加密等方向。部分项目提供完整源码，适合通过实战学习安全原理、工具开发和攻防技术。

此外，仓库还提供面向 SOC 分析师、渗透测试工程师、安全工程师等岗位的认证路线图，以及工具、课程、认证、社区和安全框架等学习资源，定位为一个系统化的网络安全实战与职业发展导航库。
