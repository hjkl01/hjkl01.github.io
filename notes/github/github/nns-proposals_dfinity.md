### [dfinity nns-proposals](https://github.com/dfinity/nns-proposals)  ![GitHub Repo stars](https://img.shields.io/github/stars/dfinity/nns-proposals?style=social)

本项目用于存储互联网计算机（Internet Computer）网络神经系统（NNS）的提案及相关支持材料。NNS 负责以去中心化方式治理 IC 网络，涵盖协议升级、节点运营、子网配置、经济参数及网络安全等职能。治理机制基于用户持有的“神经元”对提案进行投票，以决定采纳或拒绝。提案主题涵盖神经元管理、汇率、网络经济、治理、节点管理、参与者管理、子网管理、网络 Canister 管理、KYC 及节点奖励等。
