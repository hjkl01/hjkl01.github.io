### [freemocap](https://github.com/freemocap/freemocap) ![GitHub Repo stars](https://img.shields.io/github/stars/freemocap/freemocap?style=social)

FreeMoCap 是一个免费开源、软硬件无关、低成本且研究级的人体动作捕捉系统与平台，面向去中心化的科研、教育和训练场景。项目支持通过 `pip install freemocap` 快速安装，并可通过命令行启动图形界面进行使用；也支持从源码（Conda/Python 环境）安装运行。官方提供完整文档与新手教程，并鼓励社区贡献与 Discord 交流。项目采用 AGPLv3 许可证。
