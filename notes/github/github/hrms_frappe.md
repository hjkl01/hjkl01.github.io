### [frappe hrms](https://github.com/frappe/hrms)  ![GitHub Repo stars](https://img.shields.io/github/stars/frappe/hrms?style=social)

Frappe HR 是一款开源、现代化且易用的全功能人力资源与薪酬管理软件，提供完整的人力资源管理系统（HRMS）解决方案，包含员工管理、入职、休假、薪酬、税务等十余个模块。

核心功能涵盖：
1. **员工全生命周期**：管理从入职、晋升、调岗到离职面谈的全过程。
2. **休假与考勤**：配置假期政策、自动同步区域假期、支持地理位置打卡及考勤追踪。
3. **费用报销与预支**：管理员工预支款和费用报销，支持多级审批工作流，并与 ERPNext 会计集成。
4. **绩效管理**：跟踪目标、对齐关键结果领域（KRA）、支持员工自评及绩效评估周期。
5. **薪酬与税务**：创建薪资结构、配置所得税级、运行标准薪酬、查看工资条收入明细。
6. **移动应用**：支持移动端审批休假、打卡及访问员工档案。

项目基于 Frappe Framework 和 Frappe UI 构建，支持 Docker 及本地部署。
