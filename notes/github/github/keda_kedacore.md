### [kedacore keda](https://github.com/kedacore/keda)  ![GitHub Repo stars](https://img.shields.io/github/stars/kedacore/keda?style=social)

KEDA 是一个基于 Kubernetes 的事件驱动自动扩缩容项目。它允许用户通过 Kubernetes 自定义资源定义（CRD）为事件驱动的工作负载配置细粒度的自动扩缩容规则，支持扩缩容至零。KEDA 作为 Kubernetes Metrics Server 运行，原生集成 Kubernetes 组件（如 HPA），无外部依赖，适用于云和边缘环境部署。
