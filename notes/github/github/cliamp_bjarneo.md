### [cliamp](https://github.com/bjarneo/cliamp) ![GitHub Repo stars](https://img.shields.io/github/stars/bjarneo/cliamp?style=social)

cliamp 是一款复古终端音乐播放器（灵感来自 Winamp），支持播放本地文件、网络流、播客，以及 YouTube、YouTube Music、SoundCloud、Bilibili、Spotify、网易云音乐、小宇宙、Navidrome、Plex、Jellyfin 等多平台音源。  
核心功能包括频谱可视化、参数均衡器、播放列表管理、在线电台浏览与搜索（3 万+ 电台）、远程媒体服务配置向导（`cliamp setup`）、主题与歌词支持、SSH 串流、IPC 远程控制、无头守护进程模式、媒体控制、Lua 插件扩展等。  
项目提供多种安装方式（脚本、Homebrew、AUR、预编译包、源码构建），并可通过可选依赖 ffmpeg 与 yt-dlp 扩展更多音频格式和流媒体站点支持。
