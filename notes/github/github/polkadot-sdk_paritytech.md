### [paritytech polkadot-sdk](https://github.com/paritytech/polkadot-sdk)  ![GitHub Repo stars](https://img.shields.io/github/stars/paritytech/polkadot-sdk?style=social)

Polkadot SDK 是为构建 Polkadot 多链区块链网络及平行链提供核心组件的开源项目，旨在实现不同区块链之间的安全互操作和信息共享。项目整合了 Substrate、FRAME、Cumulus 和 XCM 等模块，支持 WASM 和 PolkaVM 构建。此外，提供快速启动脚本、开发文档、依赖管理工具（psvm）以及季度稳定版本发布与维护。
