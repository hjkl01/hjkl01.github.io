### [spree spree](https://github.com/spree/spree)  ![GitHub Repo stars](https://img.shields.io/github/stars/spree/spree?style=social)

Spree Commerce 是一个面向快速增长企业的开源 API 优先电商开放平台。

**核心功能：**
1.  **API 与 SDK**：类 Stripe 风格，快速、简单且安全。
2.  **管理后台**：全新构建，基于 Tailwind CSS，支持基于角色的权限管理。
3.  **定价与促销**：内置灵活定价引擎，支持区域/B2B/批发定价、客户组、礼品卡及促销管理。
4.  **产品与目录**：支持元字段、CSV 导入导出、数字产品及批量操作。
5.  **结账与支付**：支持 Stripe、Adyen、数字钱包及先买后付等多种支付方式。
6.  **集成与扩展**：提供事件总线、Webhooks 及第三方服务（如 Google Analytics、Klaviyo）集成。

**适用场景：**
原生支持 B2B 电商、多站点、多区域、多供应商市场及多租户电商。

**企业版：**
提供增强的安全性（加密、SSO、PCI DSS 合规）及专用模块（多供应商、B2B、多租户），包含专属技术支持。
