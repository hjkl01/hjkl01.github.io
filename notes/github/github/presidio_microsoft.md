### [presidio](https://github.com/microsoft/presidio) ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/presidio?style=social)

Presidio：数据保护与脱敏SDK。

核心功能：
- 文本与图像PII（个人身份信息）快速识别与匿名化。
- 上下文感知，支持多语言。
- 预定义或自定义PII识别器（NER、正则、规则、校验和）。
- 支持接入外部PII检测模型。
- 图像文本脱敏（标准图像及DICOM医疗图像）。
- 跨平台部署（Python、PySpark、Docker、Kubernetes）。
- 插件化，高度可定制工作流。
