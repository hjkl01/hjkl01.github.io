### [gateway](https://github.com/envoyproxy/gateway) ![GitHub Repo stars](https://img.shields.io/github/stars/envoyproxy/gateway?style=social)

Envoy Gateway 是一个开源项目，用于将 Envoy Proxy 作为独立应用或 Kubernetes 应用网关进行统一管理。它基于 Gateway API 资源，动态地对受管的 Envoy 代理进行部署与配置。项目同时提供快速上手文档、目标与路线图、版本兼容矩阵，以及完善的社区协作、贡献开发和安全漏洞报告机制。
