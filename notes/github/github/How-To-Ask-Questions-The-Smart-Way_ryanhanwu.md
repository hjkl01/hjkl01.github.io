### [ryanhanwu How-To-Ask-Questions-The-Smart-Way](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way)  ![GitHub Repo stars](https://img.shields.io/github/stars/ryanhanwu/How-To-Ask-Questions-The-Smart-Way?style=social)

本项目是一份指导用户如何向技术社区（特别是开源和黑客社区）聪明提问的指南。其核心内容包括：

1.  **提问前准备**：指导用户在提问前应自行搜索、查阅文档、常见问题（FAQ）并尝试解决问题，同时表明已做的努力。
2.  **提问规范**：教授如何选择合适渠道（如邮件列表、论坛）、撰写清晰具体的标题、准确描述问题症状及环境，并提供精简的测试代码。
3.  **社区礼仪**：解释黑客文化规范，包括如何解读回应（如 RTFM 和 STFW 的含义）及如何处理无礼言论。
4.  **案例分析**：通过“好问题”与“蠢问题”的对比，明确有效沟通的标准及避免的行为。

该指南旨在帮助用户以尊重他人时间的方式寻求帮助，提高获得有效技术支持的概率，并维护良好的社区互动关系。
