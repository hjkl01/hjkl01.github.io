### [tokio-rs tokio](https://github.com/tokio-rs/tokio)  ![GitHub Repo stars](https://img.shields.io/github/stars/tokio-rs/tokio?style=social)

Tokio 是用于 Rust 编程语言的事件驱动、非阻塞 I/O 运行时平台，旨在构建可靠、异步且精简的应用程序。其核心功能包括基于操作系统事件队列的 Reactor 模型、多线程工作窃取任务调度器以及异步 TCP 和 UDP 套接字。项目具备零成本抽象的高性能、Rust 所有权保证的可靠性及良好的可扩展性，是 Rust 异步生态的基础设施。
