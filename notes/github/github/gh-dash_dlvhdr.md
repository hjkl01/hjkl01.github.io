### [dlvhdr gh-dash](https://github.com/dlvhdr/gh-dash)  ![GitHub Repo stars](https://img.shields.io/github/stars/dlvhdr/gh-dash?style=social)

gh-dash 是一款基于终端的 GitHub 管理工具，提供丰富的终端用户界面（TUI）以提升工作流效率。其主要功能包括：支持按仓库自定义 PR 和 Issue 视图板块；提供可覆盖的 Vim 风格快捷键；支持自定义操作以满足特定工作流需求；允许通过 YAML 配置文件控制所有设置；支持在终端内执行代码差异查看、评论、检出、推送和更新等 GitHub 常用操作。
