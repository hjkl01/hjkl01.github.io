### [projectcalico calico](https://github.com/projectcalico/calico)  ![GitHub Repo stars](https://img.shields.io/github/stars/projectcalico/calico?style=social)

Calico 是由 Tigera 维护的开源容器网络与安全解决方案，也是目前最广泛采用的容器网络方案之一。它支持 Kubernetes 集群及多云、裸机、虚拟机等多种环境，提供 eBPF、标准 Linux 和 VPP 等多种数据平面选择。其核心功能包括高可扩展的网络架构、优化的性能、细粒度访问控制、WireGuard 加密以及完整的 Kubernetes 网络策略支持，旨在为容器应用提供安全、灵活且高效的网络连接能力。
