### [ava-labs avalanchego](https://github.com/ava-labs/avalanchego)  ![GitHub Repo stars](https://img.shields.io/github/stars/ava-labs/avalanchego?style=social)

AvalancheGo 是 Avalanche 区块链网络的 Go 节点实现，支持高吞吐量与快速交易。项目提供连接主网、Fuji 测试网及本地测试网的功能，并支持源码构建、APT 仓库、二进制包及 Docker 等多种部署方式。节点需经历启动引导以同步网络状态，兼容 Linux 及 macOS 平台，遵循 Avalanche 网络版本规范。
