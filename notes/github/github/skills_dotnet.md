### [skills](https://github.com/dotnet/skills) ![GitHub Repo stars](https://img.shields.io/github/stars/dotnet/skills?style=social)

该项目是 .NET 团队维护的“Agent Skills”插件集合与市场，面向编码智能体，提供一套遵循 agentskills.io 标准的核心技能与自定义代理能力，并可查看各插件准确率与效率趋势仪表盘。

核心功能覆盖多个 .NET 开发场景，包括：
- 通用 .NET 编码任务  
- 数据访问与 Entity Framework  
- 性能诊断、调试与事故分析  
- MSBuild 与构建失败排查、性能优化、代码质量与现代化  
- NuGet 包与依赖管理、现代化升级  
- .NET 项目跨版本迁移与升级（框架、语言特性、兼容性）  
- .NET MAUI 开发环境配置与故障排查  
- .NET AI/ML（技术选型、LLM 集成、Agent 工作流、RAG、MCP、ML.NET）  
- 模板引擎（模板发现、项目脚手架、模板编写）  
- .NET 测试执行、诊断、筛选与迁移（含 MSTest）  
- ASP.NET Core（中间件、端点、实时通信、API 模式）  
- Blazor 组件开发与交互模式  
- .NET 11 新 API 与语言特性支持  

项目支持在 Copilot CLI、Claude Code、VS Code（预览）、Cursor 和 Codex CLI 中安装与使用这些插件/技能。
