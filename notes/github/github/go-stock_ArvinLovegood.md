### [go-stock](https://github.com/ArvinLovegood/go-stock) ![GitHub Repo stars](https://img.shields.io/github/stars/ArvinLovegood/go-stock?style=social)

基于Wails和NaiveUI开发的AI股票分析工具。支持A股、港股、美股实时行情与K线分析，提供多周期K线图表及技术指标分析。市场情绪分析、个股情绪分析、带频率权重的情感分析。AI智能选股、AI助手、自定义提问模板、多轮对话。资金流向、龙虎榜、行业排名、股票异动、机构研报、公司公告、财经日历功能。基金净值和持仓数据。支持OpenAI、Ollama、LMStudio、DeepSeek、硅基流动等多种大模型接入。Windows/MacOS跨平台，提供自动更新和多级赞助VIP服务。
