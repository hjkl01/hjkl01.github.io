### [saleor saleor-platform](https://github.com/saleor/saleor-platform)  ![GitHub Repo stars](https://img.shields.io/github/stars/saleor/saleor-platform?style=social)

Saleor Platform 是一个基于 Docker 的本地开发环境，用于通过单一仓库运行所有 Saleor 核心服务。它集成了 GraphQL API、Dashboard、Mailpit、Jaeger 及必要的数据库组件。项目仅支持本地开发，不支持生产部署。用户可通过 Docker Compose 快速搭建开发环境、执行数据迁移、初始化数据库并启动应用服务。
