### [mealie-recipes mealie](https://github.com/mealie-recipes/mealie)  ![GitHub Repo stars](https://img.shields.io/github/stars/mealie-recipes/mealie?style=social)

Mealie 是一款自托管的食谱管理、餐食规划和购物清单应用。主要功能包括：通过 URL 自动导入或手动添加食谱、制定每周用餐计划、生成按超市区域分类的购物清单、按自定义标准创建烹饪书、支持 Docker 部署以及提供 35 多种语言本地化。此外，项目还开放 API 供第三方应用交互。
