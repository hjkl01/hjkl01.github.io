### [hermes-webui](https://github.com/nesquena/hermes-webui) ![GitHub Repo stars](https://img.shields.io/github/stars/nesquena/hermes-webui?style=social)

Hermes WebUI 是 Hermes Agent 的轻量级自托管网页界面，在浏览器中提供几乎与命令行 1:1 对等的使用体验，无需构建流程、框架或打包器，仅依赖 Python 和原生 JavaScript。它让用户通过网页安全访问长期运行在服务器上的智能代理，并复用现有 Hermes Agent、模型配置与运行环境。

项目核心功能包括：实时流式聊天、多模型与多提供商支持、消息编辑重生成、任务取消、工具调用与子代理活动展示、Mermaid 图表渲染、上下文与 Token 用量显示；会话管理支持创建、搜索、归档、置顶、项目分组、标签、导入导出，以及桥接 CLI 会话；内置工作区文件浏览器，可预览、编辑、创建、删除和重命名文件，显示 Git 分支与变更状态；支持语音输入、个人配置档切换、主题与皮肤定制、斜杠命令、移动端适配，以及密码和 Passkey 身份验证等安全能力。

相较于普通 AI 工具，Hermes 的特点是具备跨会话持久记忆、自托管定时任务、多消息平台接入、自我积累与复用技能、兼容多模型提供商，并可编排其他编码代理协作。项目还提供自动发现配置、一键启动、SSH 隧道远程访问、Docker 部署、完整测试体系，以及清晰的前后端分层架构。
