### [ethereum EIPs](https://github.com/ethereum/EIPs)  ![GitHub Repo stars](https://img.shields.io/github/stars/ethereum/EIPs?style=social)

本项目是以太坊改进提案（EIPs）的官方仓库，旨在标准化和记录以太坊及其生态的高质量文档，记录并跟踪以太坊的改进提案。

主要功能包括：
1. 分类管理：涵盖核心协议、网络、接口、应用层标准（ERCs 已独立）、元及信息类 EIP。
2. 流程规范：要求提案需经社区讨论达成共识，遵循 EIP-1 标准提交。
3. 自动化验证：通过工具对合并请求进行格式、拼写及规范性校验，合格后方可合并。
4. 状态展示：提供官方状态页面集中查询和展示所有 EIP 的状态。
