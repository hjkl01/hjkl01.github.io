### [Anarios return-youtube-dislike](https://github.com/Anarios/return-youtube-dislike)  ![GitHub Repo stars](https://img.shields.io/github/stars/Anarios/return-youtube-dislike?style=social)

Return YouTube Dislike 是一款开源浏览器扩展及脚本，适用于 Chrome、Firefox 等浏览器。该项目旨在恢复 YouTube 被移除的视频“不喜欢”计数，通过后端聚合爬取数据与用户估算来实现，并开放了公共 API 供第三方使用。
