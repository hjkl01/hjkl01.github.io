### [browseros-ai BrowserOS](https://github.com/browseros-ai/BrowserOS)  ![GitHub Repo stars](https://img.shields.io/github/stars/browseros-ai/BrowserOS?style=social)

BrowserOS 是一款开源的 Chromium 浏览器分支，原生集成 AI 智能体功能。项目主打隐私优先，支持用户自行配置 API 密钥或使用本地模型（如 Ollama），确保浏览数据保留在本地。它保留了 Chrome 的界面和扩展兼容性，提供工作流可视化构建、定时任务、MCP 服务器集成及多模型侧边栏对比等功能，并内置广告拦截器。支持 macOS、Windows 和 Linux 平台，采用 AGPL-3.0 协议开源。
