### [oceanbase miniob](https://github.com/oceanbase/miniob)  ![GitHub Repo stars](https://img.shields.io/github/stars/oceanbase/miniob?style=social)

MiniOB 是 OceanBase 团队开发的数据库内核实现学习项目，用于帮助用户从零基础掌握数据库原理与工程能力。项目实现了网络交互、SQL 解析、查询执行、存储引擎（B+ Tree）、事务管理（MVCC）、日志及元数据管理等核心模块，并简化了并发与复杂事务机制。它提供由浅入深的实训题目、配套文档教程及本地与在线开发环境，同时也是 OceanBase 大赛的初赛实训平台。
