### [qiaomu-anything-to-notebooklm](https://github.com/joeseesun/qiaomu-anything-to-notebooklm) ![GitHub Repo stars](https://img.shields.io/github/stars/joeseesun/qiaomu-anything-to-notebooklm?style=social)

这是一个基于 Claude Code Skill 的“多源内容智能处理器”，可将网页、社交媒体、播客/视频、电子书、Office 文档、图片音频等 15+ 类型内容自动获取并上传到 NotebookLM，再一键生成播客、PPT、思维导图、Quiz、报告、视频、信息图、闪卡等输出。

项目核心能力包括：  
1. **智能识别输入类型**（链接、文件、搜索词等）并自动选择处理链路；  
2. **自动化内容处理流程**（抓取→转换→上传→生成→导出）；  
3. **付费墙自动绕过**（6 层级联策略，覆盖 300+ 新闻站点）；  
4. **多源混合整合**（可把多种来源内容合并生成综合结果）；  
5. **深度分析模式**（三轮递进提问，输出结构化分析 JSON）；  
6. **扩展集成能力**（支持微信公众号抓取 MCP、飞书文档写入、播客转写 API）。

整体定位是：用自然语言把“任何内容”快速转成“任意知识产品格式”，适合学习、整理、汇报与知识再加工。
