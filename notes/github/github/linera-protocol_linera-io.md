### [linera-io linera-protocol](https://github.com/linera-io/linera-protocol)  ![GitHub Repo stars](https://img.shields.io/github/stars/linera-io/linera-protocol?style=social)

Linera 是一个专为高可扩展性、安全性和低延迟 Web3 应用设计的去中心化区块链基础设施。项目基于 Rust 构建，采用微链架构，支持单个钱包管理多个微链并实现链间转账。核心功能涵盖基础定义、数据执行、链与存储抽象、节点同步协议及 RPC 通信，并提供 Rust SDK 用于开发运行于 Wasm 虚拟机上的应用程序。此外，项目包含命令行工具（CLI），支持本地测试网络搭建、钱包管理、转账及余额查询等功能。
