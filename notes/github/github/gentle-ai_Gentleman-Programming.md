### [gentle-ai](https://github.com/Gentleman-Programming/gentle-ai) ![GitHub Repo stars](https://img.shields.io/github/stars/Gentleman-Programming/gentle-ai?style=social)

Gentle-AI 是一个面向 AI 编程代理的“生态配置器”，不是安装器。它用于增强你已安装的代理（如 Claude Code、OpenCode、Cursor、Copilot、Codex 等），为其统一提供：持久化记忆（Engram）、规范驱动开发（SDD）工作流、技能注册与调用、MCP 工具接入、教学型人格与安全优先权限、以及按 SDD 阶段分配不同模型的能力。

项目核心价值是把“只会写代码的聊天代理”升级为“有记忆、有流程、有工具协作、可委派子代理、可分阶段用不同模型”的工程化开发助手；并通过委派触发规则控制复杂任务，避免上下文混乱和开发失控。

主要能力包括：支持 13+ 主流代理生态、自动化安装/同步与升级、项目级初始化（如 `/sdd-init` 与技能注册刷新）、配置自动备份（压缩/去重/保留最近 5 份/可 pin）、OpenCode 的 SDD 分阶段模型配置、跨会话记忆检索与管理（Engram），以及可选社区插件集成。整体定位是为 AI 编程代理提供统一、可持续、可扩展的生产级协作框架。
