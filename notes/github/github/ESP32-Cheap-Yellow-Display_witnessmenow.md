该项目是 ESP32“廉价黄显示屏”（CYD）的社区支持与文档平台。CYD 硬件集成 ESP32 芯片（含 WiFi/蓝牙）、2.8 英寸电阻触摸屏、SD 卡槽及 USB 接口。仓库提供硬件设置指南、基础及替代显示库代码示例、ESPHome 支持、引脚定义、故障排除、硬件改装、3D 打印模型等资源。旨在弥补官方支持不足，协助用户快速完成基于该显示模块的嵌入式项目开发。
