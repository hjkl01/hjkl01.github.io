### [facebook create-react-app](https://github.com/facebook/create-react-app)  ![GitHub Repo stars](https://img.shields.io/github/stars/facebook/create-react-app?style=social)

Create React App 是一个零配置创建 React 应用的项目（目前已弃用）。它预置构建、开发和测试环境，无需配置 webpack、Babel 等工具，支持 React、JSX、ES6、TypeScript 等。提供本地开发、测试及生产构建功能。适用于学习 React 或构建单页应用，但新项目建议迁移至其他框架。
