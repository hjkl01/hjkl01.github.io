### [ccxt ccxt](https://github.com/ccxt/ccxt)  ![GitHub Repo stars](https://img.shields.io/github/stars/ccxt/ccxt?style=social)

CCXT 是一个支持 JavaScript/TypeScript、Python、C#、PHP 和 Go 的加密货币交易 API 库。它提供统一接口连接全球 100 多家加密货币交易所和支付服务，支持 REST 和 WebSocket 协议，涵盖市场数据查询及账户交易管理等公共与私有 API 功能。项目提供归一化数据以简化跨交易所分析、套利、算法交易、策略回测及量化机器人开发，支持中心化与去中心化交易所，采用 MIT 开源协议。
