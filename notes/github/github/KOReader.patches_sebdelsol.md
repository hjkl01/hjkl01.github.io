### [sebdelsol KOReader.patches](https://github.com/sebdelsol/KOReader.patches)

**核心内容总结：**  
该项目为KOReader电子书阅读器提供一系列用户补丁，用于增强功能和自定义体验。主要功能包括：  
- **界面优化**：隐藏文件夹下划线、自定义文件夹封面、调整菜单大小、改进状态栏显示（如章节标记、颜色自定义、紧凑模式优化）。  
- **交互改进**：在文件管理器标题栏显示信息、屏幕保护程序中添加章节信息、支持状态栏预设切换。  
- **实用工具**：检查KOReader版本兼容性（guard.lua）、自动更新补丁（需v2025.04.107以上版本）。  
- **设置位置**：所有选项均通过KOReader内置设置菜单调整，例如“设置→UI字体”“设置→状态栏→进度条”等。  

**使用方法**：  
1. 通过[用户指南](https://koreader.rocks/user_guide/#L2-userpatches)安装补丁。  
2. 根据补丁说明，在对应设置路径中启用或配置功能。  
3. 部分补丁（如更新功能）需满足KOReader最低版本要求。
