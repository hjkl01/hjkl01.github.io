### [ironcalc IronCalc](https://github.com/ironcalc/IronCalc)  ![GitHub Repo stars](https://img.shields.io/github/stars/ironcalc/IronCalc?style=social)

IronCalc 是一个基于 Rust 开发的现代化电子表格引擎及工具集。它支持 XLSX 文件的读写、公式计算及电子表格数据处理，可集成至 Python、JavaScript、Node.js 等多种编程语言。项目支持终端、桌面及 Web 应用等多种运行形态，目前处于开发中，支持 Docker 部署，采用 MIT 或 Apache 2.0 协议开源。
