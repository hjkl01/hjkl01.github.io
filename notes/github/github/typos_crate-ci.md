### [crate-ci typos](https://github.com/crate-ci/typos)  ![GitHub Repo stars](https://img.shields.io/github/stars/crate-ci/typos?style=social)

typos 是一款源码拼写检查工具，用于查找并修正源代码中的拼写错误。它具有速度快、误报率低的特点，适用于单体仓库及 Pull Request 检查。工具支持通过命令行直接修复更改、生成差异报告，并提供配置文件以自定义忽略规则（如人名、缩写或特定文件）。此外，typos 支持集成至 GitHub Actions、pre-commit、VS Code 等多种开发工作流，并能通过标准输入输出、JSON 格式等方式实现自定义集成。
