### [ethereum-optimism optimism](https://github.com/ethereum-optimism/optimism)  ![GitHub Repo stars](https://img.shields.io/github/stars/ethereum-optimism/optimism?style=social)

Optimism 是一个扩展以太坊技术的项目，旨在构建可扩展的区块链网络及去中心化经济和治理系统。其核心成果为 OP Stack，一套开源软件栈，为 OP Mainnet 和 Base 等区块链提供基础设施。该项目仓库包含 Rollup 共识层客户端、数据提交器、智能合约以及测试、部署和管理工具。Optimism 遵循“影响=利润”原则，致力于通过开放源代码促进以太坊生态系统的协作与发展。
