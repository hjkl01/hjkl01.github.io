### [teslamate](https://github.com/teslamate-org/teslamate) ![GitHub Repo stars](https://img.shields.io/github/stars/teslamate-org/teslamate?style=social)

TeslaMate 是一款基于 Elixir 开发、使用 Postgres 存储数据并结合 Grafana 进行可视化的特斯拉自托管数据记录器。其核心功能包括高精度驾驶与充电数据记录、自动地址查找、地理围栏、多车支持、充电费用追踪及吸血鬼能耗监控。此外，它支持通过 MQTT 轻松集成 Home Assistant、Node-RED 和 Telegram，并提供电池健康、里程统计等丰富的仪表盘视图，同时具备从 TeslaFi 导入数据的功能。
