### [gaboolic rime-frost](https://github.com/gaboolic/rime-frost)  ![GitHub Repo stars](https://img.shields.io/github/stars/gaboolic/rime-frost?style=social)

白霜拼音是基于雾凇拼音优化的 Rime 输入法词库方案。

- **词库优化**：清理不健康及冷僻词，基于 7.4 亿高质量语料重统计字频词频，支持全拼与双拼输入。
- **功能扩展**：支持 Lua 辅助码，提供符号、日期时间、农历、计算器及大小写数字等快捷输入指令。
- **多端兼容**：适用于 Windows、Mac、Linux、Android、iOS 等主流操作系统及输入法框架。
