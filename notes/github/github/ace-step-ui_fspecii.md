### [ace-step-ui](https://github.com/fspecii/ace-step-ui) ![GitHub Repo stars](https://img.shields.io/github/stars/fspecii/ace-step-ui?style=social)

ACE-Step UI 是一个与 ACE-Step 1.5 深度集成的开源本地化 AI 音乐生成界面，定位为 Suno/Udio 的免费替代方案。它支持在本地 GPU 上生成最长 4 分钟以上的完整歌曲（含人声/歌词）和纯音乐，并提供自定义 BPM、调式、拍号、时长、风格标签、批量生成、AI 增强与 Thinking 模式等能力。项目还具备参考音频风格迁移、音频重绘、种子复现、推理步数控制、歌词编辑与提示词模板等专业功能。

在使用体验上，提供类 Spotify 的现代化界面（深浅色模式）、底部播放器、实时生成进度、曲库搜索管理、收藏与歌单、局域网多设备访问。内置工具包括音频编辑（裁剪/淡入淡出/效果）、Demucs 人声与伴奏分离、基于素材的视频生成、自动渐变封面生成。整体采用 React + TypeScript 前端、Express + SQLite 后端，数据本地存储，强调免费、隐私、本地运行和完全可控。
