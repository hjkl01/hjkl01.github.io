### [DearVa Everywhere](https://github.com/DearVa/Everywhere)  ![GitHub Repo stars](https://img.shields.io/github/stars/DearVa/Everywhere?style=social)

Everywhere 是一款上下文感知的交互式 AI 助手桌面应用。它支持通过快捷键在任何屏幕界面即时获取 AI 辅助，无需截图、复制或切换应用。项目支持 Windows 和 macOS 平台，集成了多种主流大语言模型（如 OpenAI、Claude、DeepSeek 等）及系统工具（如浏览器、文件系统、终端等）。核心功能涵盖错误排查、网页摘要、即时翻译及邮件润色，并提供现代化的磨砂玻璃 UI 界面。
