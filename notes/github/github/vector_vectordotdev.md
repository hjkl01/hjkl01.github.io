### [vectordotdev vector](https://github.com/vectordotdev/vector)  ![GitHub Repo stars](https://img.shields.io/github/stars/vectordotdev/vector?style=social)

Vector 是一个由 Datadog 维护的高性能开源可观测性数据管道，基于 Rust 构建，主打可靠性与端到端处理能力。它支持 Agent 和 Aggregator 部署模式，可统一收集、转换并路由日志与指标数据。Vector 旨在通过统一数据处理流程降低可观测性成本、提升数据质量与安全性，其性能与正确性表现优于 Filebeat、Logstash 等同类工具。
