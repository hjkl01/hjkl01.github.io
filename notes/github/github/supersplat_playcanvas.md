### [supersplat](https://github.com/playcanvas/supersplat) ![GitHub Repo stars](https://img.shields.io/github/stars/playcanvas/supersplat?style=social)

SuperSplat Editor 是一个免费开源的 3D Gaussian Splats（高斯点云）编辑工具，可在浏览器中直接使用，无需下载安装。其核心功能包括：查看、编辑、优化与发布 3D Gaussian Splats。项目提供在线可用版本，并支持本地开发（基于 Node.js 18+，可自动重建并刷新查看修改）。此外，编辑器支持多语言本地化，可通过新增语言文件与配置进行扩展，并可通过参数切换语言进行翻译测试。
