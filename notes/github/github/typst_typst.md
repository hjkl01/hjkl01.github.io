### [typst typst](https://github.com/typst/typst)  ![GitHub Repo stars](https://img.shields.io/github/stars/typst/typst?style=social)

Typst 是一款新兴的基于标记的排版系统，旨在提供与 LaTeX 相当的强大功能，同时具备更易学易用的特性。核心功能包括：
- 内置常用格式化标记及灵活函数；
- 集成脚本系统，支持数学公式排版与参考文献管理；
- 采用增量编译技术，实现快速编译与即时预览；
- 提供友好的错误提示；
- 包含本地命令行编译工具（CLI）及在线协作编辑器支持。
