### [easy-vibe](https://github.com/datawhalechina/easy-vibe) ![GitHub Repo stars](https://img.shields.io/github/stars/datawhalechina/easy-vibe?style=social)

Easy-Vibe 是一个面向 AI 时代的开源编程学习项目，核心目标是教用户通过“对话描述需求”快速构建真实应用（会说话就会做应用）。

项目提供分层学习路径与完整实践体系：
- **Stage 1（入门）**：从零开始学习 AI 编程、需求与创意验证、产品原型搭建、AI 能力接入，并完成完整项目迭代。
- **Stage 2（全栈实战）**：覆盖前端设计到开发、后端与数据库、API、部署、支付（如 Stripe），并通过 SaaS、在线考试系统等项目实现端到端交付。
- **Stage 3（进阶）**：聚焦 Claude Code、MCP、Skills、Agent 团队、Spec Coding 等 AI-Native 开发流程，以及微信小程序、Android、iOS、PWA、Electron、浏览器插件、VS Code 插件等跨平台开发。
- **附录知识库**：提供计算机基础、前后端、AI 原理、工程实践等 9 大领域、80+ 交互主题，帮助补齐概念与方法。

项目特点是**新手友好、可视化强、强调实操**：通过学习地图、步骤化教程、沉浸式演示和交互内容，帮助不同背景用户（小白、学生、产品经理、开发者）从想法验证到产品上线形成完整能力。项目支持 10 种语言，并可本地运行（npm install / npm run dev）。
