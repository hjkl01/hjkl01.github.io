### [goreleaser](https://github.com/goreleaser/goreleaser) ![GitHub Repo stars](https://img.shields.io/github/stars/goreleaser/goreleaser?style=social)

GoReleaser 是一个用于简化软件发布工程的工具，帮助开发者处理发布流程中的复杂工作，让团队专注于软件开发本身。它支持在本地机器和 CI/CD 系统中使用，提供完整在线文档，并面向多语言生态（如 Go、Rust、Zig、TypeScript、Python）。项目具备活跃社区与支持渠道（Discord、X、Telegram、GitHub Discussions），采用 MIT 许可证，并通过贡献指南、行为准则和赞助体系持续维护与发展。
