### [tracel-ai burn](https://github.com/tracel-ai/burn)  ![GitHub Repo stars](https://img.shields.io/github/stars/tracel-ai/burn?style=social)

Burn 是一个基于 Rust 的下一代张量库与深度学习框架，强调灵活性、效率与可移植性。它支持多种硬件后端（如 CUDA、ROCm、Metal、WebGPU 及 CPU），允许灵活切换与组合后端以优化性能。核心功能包括自动微分、内核融合、多设备路由及远程执行，覆盖模型训练与推理全流程。此外，Burn 支持浏览器（WebAssembly）和嵌入式设备（no_std）部署，兼容 ONNX 及 PyTorch 模型导入，并提供终端训练监控仪表板。
