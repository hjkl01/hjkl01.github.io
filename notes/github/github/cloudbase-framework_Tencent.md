### [Tencent cloudbase-framework](https://github.com/Tencent/cloudbase-framework)  ![GitHub Repo stars](https://img.shields.io/github/stars/Tencent/cloudbase-framework?style=social)

云开发 CloudBase Framework 是云开发官方出品的前后端一体化部署工具。它基于 Serverless 架构，无需改动代码即可一键托管部署前后端应用，具备访问加速、弹性扩容及免运维特性。项目通过插件机制兼容多种应用框架（如 Node.js、Vue、Go、Java 等）及云资源（如云函数、云托管、云数据库等），支持配置自动检测与生成。开发者可通过 CLI 快速初始化项目，并利用内置丰富的模板示例加速应用部署。
