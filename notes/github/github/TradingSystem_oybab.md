### [oybab TradingSystem](https://github.com/oybab/TradingSystem)  ![GitHub Repo stars](https://img.shields.io/github/stars/oybab/TradingSystem?style=social)

Oybab Trading System 是一款开源的跨平台交易系统，支持多语言和多种终端设备（Windows 及移动端）。它适用于个人交易管理、餐饮、零售、超市、酒店等多种场景，免费向最终用户开放，并支持二次开发与系统集成。系统包含服务器端及 PC、平板、手机客户端，兼容热敏打印机、扫码器、读卡器等外设，支持多国语言界面及内部多语言管理。
