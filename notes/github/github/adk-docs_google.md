### [google adk-docs](https://github.com/google/adk-docs)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/adk-docs?style=social)

Agent Development Kit (ADK) 是一款开源、代码优先的工具包，用于构建、评估和部署灵活的 AI 代理系统。它支持 Python、TypeScript、Go 和 Java 等多种编程语言，具备模型无关和部署无关的特性。核心功能包括丰富的工具生态、模块化多智能体编排、工作流跟踪监控以及云原生部署能力，并提供 AI 辅助编码支持以加速开发。
