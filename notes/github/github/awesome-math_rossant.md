### [rossant awesome-math](https://github.com/rossant/awesome-math)  ![GitHub Repo stars](https://img.shields.io/github/stars/rossant/awesome-math?style=social)

这是一个精心整理的数学资源精选列表项目。项目收录了学习平台、工具、书籍、视频、论文及会议信息等通用资源，以及基础数学、数论、代数、几何、拓扑、分析、概率统计等各大分支的专业资料。资源涵盖在线课程、讲义、软件及学术文献，大部分可免费获取，旨在为数学学习者及研究者提供全面的学习支持与检索平台。
