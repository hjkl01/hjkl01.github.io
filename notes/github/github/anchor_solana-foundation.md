### [solana-foundation anchor](https://github.com/solana-foundation/anchor)  ![GitHub Repo stars](https://img.shields.io/github/stars/solana-foundation/anchor?style=social)

Anchor 是 Solana 智能合约开发框架，提供 Rust 嵌入式领域特定语言 (eDSL) 用于编写程序，支持接口描述语言 (IDL) 规范，并可通过 TypeScript 包根据 IDL 生成客户端。框架还提供 CLI 工具和工作区管理以支持完整应用的开发，核心组件涵盖 anchor-lang、anchor-spl、anchor-client 及 TypeScript 包，是目前最流行的 Solana 程序开发框架。
