### [EbookFoundation free-programming-books](https://github.com/EbookFoundation/free-programming-books)  ![GitHub Repo stars](https://img.shields.io/github/stars/EbookFoundation/free-programming-books?style=social)

这是一个由免费电子书基金会维护的开源项目，致力于整理全球多语言的免费编程学习资源。项目提供了按语言和主题分类的免费书籍、速查表、课程、交互式教程、竞赛题集、播客及在线编程环境。它支持多语言检索，并允许社区成员通过 GitHub 协作贡献资源、更新文档或进行翻译。
