### [solidusio solidus](https://github.com/solidusio/solidus)  ![GitHub Repo stars](https://img.shields.io/github/stars/solidusio/solidus?style=social)

Solidus 是一个基于 Ruby on Rails 的免费开源电子商务平台，为 Spree 的分支。它提供完整的电商解决方案，涵盖 RESTful API、后台管理界面、核心模型及邮件服务等。项目采用模块化设计，支持直接使用全套组件或仅核心模块配合自定义前端。Solidus 支持多种数据库和 Docker 环境，具备沙箱测试功能，并拥有丰富的扩展生态与活跃社区，旨在赋予用户对在线商店的完全控制权。
