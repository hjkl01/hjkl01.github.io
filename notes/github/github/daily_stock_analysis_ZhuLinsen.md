### [daily_stock_analysis](https://github.com/ZhuLinsen/daily_stock_analysis) ![GitHub Repo stars](https://img.shields.io/github/stars/ZhuLinsen/daily_stock_analysis?style=social)

这是一个基于 AI 大模型的股票智能分析系统，面向 A 股、港股、美股和 ETF 的自选股日常分析。项目可自动汇总行情、K 线、技术指标、资金流、筹码、新闻、公告和基本面数据，生成包含结论评分、趋势判断、买卖点、风险警报、利好催化和操作清单的「决策仪表盘」，并支持大盘复盘。

系统提供 Web/桌面工作台与 API/Bot 能力，支持手动分析、任务监控、历史报告、回测、持仓管理、配置管理、智能导入（图片/表格/剪贴板）和股票信息补全。内置 Agent 策略问股，支持多轮对话与多种分析策略（如均线、趋势、缠论、波浪、题材、事件驱动等）。

部署方式灵活，可通过 GitHub Actions、Docker 或本地定时运行，实现工作日自动分析；结果可推送到企业微信、飞书、Telegram、Discord、Slack 和邮箱。系统兼容多种 AI 模型与多类行情/搜索/舆情数据源，适合构建自动化的个人或团队投研分析流程。
