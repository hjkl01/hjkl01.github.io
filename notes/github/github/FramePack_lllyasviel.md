### [lllyasviel FramePack](https://github.com/lllyasviel/FramePack)  ![GitHub Repo stars](https://img.shields.io/github/stars/lllyasviel/FramePack?style=social)

FramePack 是一款基于下一帧预测架构的视频扩散模型及其官方桌面软件。该项目通过压缩输入上下文为固定长度，实现生成负载与视频长度无关，支持在 6GB 显存的笔记本 GPU 上运行 13B 大模型。核心功能包括渐进式视频生成（逐帧或分段）、抗漂移处理及实时预览反馈。软件提供图形界面，支持图像生成视频和纯文本生成视频，需 Nvidia RTX 30/40/50 系列显卡及 Windows 或 Linux 操作系统。
