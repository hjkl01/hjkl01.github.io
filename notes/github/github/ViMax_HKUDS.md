### [ViMax](https://github.com/HKUDS/ViMax) ![GitHub Repo stars](https://img.shields.io/github/stars/HKUDS/ViMax?style=social)

ViMax 是一个多智能体自动化视频生成框架，目标是把“创意/小说/剧本”一站式转成完整视频。它通过“导演+编剧+制片+视频生成”协同流程，自动完成长文本脚本拆解、分镜与镜头规划、角色与场景提取、参考图选择、图像生成与一致性校验、镜头并行生成、时间线拼接，最终输出成片。

项目核心功能包括：
- **Idea2Video**：把简短想法自动扩展为完整视频故事并生成视频。  
- **Novel2Video**：将整本小说智能压缩改编为分集/分场景视频内容。  
- **Script2Video**：按用户给定剧本直接生成视频，支持较高创作自由度。  
- **AutoCameo**：上传人物/宠物照片，自动将其稳定融入剧情视频中。  

其重点能力是解决 AI 视频常见问题（片段过短、角色和场景不一致、缺少叙事结构），并支持音画协同、长视频连续性控制和更高效的端到端生产流程。
