### [django-oscar django-oscar](https://github.com/django-oscar/django-oscar)  ![GitHub Repo stars](https://img.shields.io/github/stars/django-oscar/django-oscar?style=social)

Oscar 是一个基于 Django 的领域驱动电子商务框架，核心功能高度可定制，适用于从 B2C 到复杂 B2B 的各类电商需求。项目提供沙箱站点以加速开发，并拥有丰富的官方及社区扩展，涵盖 RESTful API、多种支付网关集成（如 PayPal、Adyen）、账户管理、实体店整合、发票生成等功能。Oscar 采用 BSD 许可证开源，支持长期版本更新，拥有活跃社区并提供商业支持服务。
