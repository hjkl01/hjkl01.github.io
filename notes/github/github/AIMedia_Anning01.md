### [Anning01 AIMedia](https://github.com/Anning01/AIMedia)  ![GitHub Repo stars](https://img.shields.io/github/stars/Anning01/AIMedia?style=social)

AIMedia 是一款全自动托管 AI 媒体软件，采用 Django 后端与 PySide6 桌面端架构。核心功能包括：自动抓取抖音、微博等热点新闻；利用 AI 自动生成新闻文案与配图；一键发布至今日头条、微信公众号等多平台。系统支持任务调度、配置管理及支付登录集成，但属于工程级项目，需自行部署后端服务及打包桌面应用。
