### [ax](https://github.com/google/ax) ![GitHub Repo stars](https://img.shields.io/github/stars/google/ax?style=social)

AX（Agent Executor）是一个开源、自托管的分布式智能体运行时，用于在本地或远程环境中统一调度智能体、技能和工具的执行。它的核心能力是：通过单控制器与事件日志实现一致性管理、可审计调用链、故障自动恢复与会话续跑，适合长时运行和分布式部署的智能体应用。

项目主要功能包括：
- 分布式运行：控制器、智能体、技能、工具可隔离部署与通信；
- 可恢复执行：支持中断后续跑、断线补事件、从检查点分叉新会话；
- 执行治理：所有用户与智能体调用统一经过控制器，便于策略控制与审计；
- 可移植与可扩展：可本地运行，也可在 Kubernetes（推荐 Agent Substrate）生产部署；
- 高可定制：与具体模型、框架、harness 解耦，支持接入自定义远程智能体（含 A2A、ADK 等）；
- 内置能力：提供 CLI（exec/serve/fork/trace）、内置 Gemini 智能体、技能系统、需人工确认的 Bash 工具调用。

定位上，AX不是托管服务、不是智能体开发框架、也不是特定模型或特定业务智能体；它提供的是可靠的“执行与编排基础层”。目前项目仍处于早期快速迭代阶段，接口和协议在稳定版前可能有较大变更。
