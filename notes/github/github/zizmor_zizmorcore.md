### [zizmor](https://github.com/zizmorcore/zizmor) ![GitHub Repo stars](https://img.shields.io/github/stars/zizmorcore/zizmor?style=social)

zizmor 是一个面向 GitHub Actions 的静态分析工具，用于审计 CI/CD 工作流中的常见安全风险。它可检测的问题包括：模板注入导致的攻击者可控代码执行、凭据意外持久化与泄露、Runner 权限和凭据授予范围过大、伪造提交与易混淆的 git 引用等。项目提供完整文档，包含安装说明、快速上手和详细使用指南；采用 MIT 许可证。
