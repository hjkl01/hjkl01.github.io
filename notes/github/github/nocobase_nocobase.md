### [nocobase nocobase](https://github.com/nocobase/nocobase)

NocoBase 是一个 AI 驱动的无代码开发平台，支持通过数据模型而非表单/表格构建应用，提供高度可扩展性和 AI 协作能力。核心功能包括：

1. **数据模型驱动**：分离数据结构与界面，支持多块操作、多数据源（主数据库/外部数据库/API）。
2. **AI 能力集成**：可定义翻译员、分析师等 AI 角色，实现界面与流程中的 AI-人类协作。
3. **所见即所得**：一键切换使用模式与配置模式，页面类似 Notion 的拖拽式编辑。
4. **插件扩展**：所有功能以插件形式存在，支持自定义扩展页面、块、API 等。

**使用方法**：提供三种安装方式：
- Docker 部署（推荐，无需编码）
- 通过 create-nocobase-app CLI 安装（支持低代码开发）
- Git 克隆源码安装（适合体验未发布版本或参与开发）
