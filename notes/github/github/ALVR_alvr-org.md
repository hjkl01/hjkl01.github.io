### [alvr-org ALVR](https://github.com/alvr-org/ALVR)  ![GitHub Repo stars](https://img.shields.io/github/stars/alvr-org/ALVR?style=social)

ALVR（Air Light VR）是一款通过 Wi-Fi 将 PC 端的 VR 游戏串流至独立 VR 头显的开源软件。它支持 Quest 系列、Pico、Vision Pro 等多种头显设备，PC 端兼容 Windows 和 Linux 系统。运行需要高性能 GPU、SteamVR 环境及稳定的网络配置。软件遵循 MIT 协议，不收集任何个人数据。
