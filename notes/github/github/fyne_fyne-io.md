### [fyne-io fyne](https://github.com/fyne-io/fyne)  ![GitHub Repo stars](https://img.shields.io/github/stars/fyne-io/fyne?style=social)

Fyne 是一个使用 Go 语言编写的跨平台 UI 工具包和应用 API，允许开发者通过单一代码库构建同时运行在桌面和移动设备上的应用程序。项目提供丰富的 Widget 组件和易于上手的开发流程，支持通过 fyne 命令行工具进行应用安装、打包（涵盖 Android 和 iOS）及发布到应用商店。Fyne 应用无需预装依赖库即可运行，具备良好的便携性，并配有设置管理等辅助工具以完善桌面体验。
