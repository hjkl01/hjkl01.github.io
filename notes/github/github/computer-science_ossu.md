### [ossu computer-science](https://github.com/ossu/computer-science)  ![GitHub Repo stars](https://img.shields.io/github/stars/ossu/computer-science?style=social)

Open Source Society University (OSSU) 项目提供了一条完整的、免费的计算机科学自学教育路径，旨在帮助学习者获得等同于传统大学本科水平的计算机科学知识。课程体系基于 CS 2013 标准，分为入门、核心必修、高级选修及最终项目四个阶段，涵盖编程、数学、系统、理论、安全与伦理等核心领域。课程资源精选自哈佛、MIT 等全球顶尖名校的公开在线资料，大部分免费。项目还建立了全球学习者社区，协助学习者规划学习进度并完成最终项目验证。
