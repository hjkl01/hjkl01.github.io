### [wangyuan389 mall-cook](https://github.com/wangyuan389/mall-cook)  ![GitHub Repo stars](https://img.shields.io/github/stars/wangyuan389/mall-cook?style=social)

这是一个开源的可视化低代码商城搭建平台（Mall-Cook/云搭）。支持零代码构建 H5、小程序等多端商城应用，提供可视化页面编辑器、物料配置、Schema 生成及模板管理功能。项目基于 Vue 和 uni-app 开发，包含 Node.js 服务端，支持组件标准化接入、自定义页面及 Tab 设置，采用 Monorepo 结构，旨在提供便捷、专业的多场景商城解决方案。
