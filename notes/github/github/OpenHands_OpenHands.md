### [OpenHands OpenHands](https://github.com/OpenHands/OpenHands)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenHands/OpenHands?style=social)

OpenHands 是一个 AI 驱动的开发平台，主要包含以下功能组件：

1. **SDK**：可组合的 Python 库，支持定义和运行智能体。
2. **CLI**：命令行工具，支持多种大模型，提供类似 Claude Code 的交互体验。
3. **本地 GUI**：可在笔记本电脑上运行智能体的图形界面，包含 REST API。
4. **Cloud**：云端托管版本，支持多用户、权限管理及 Slack、Jira 等第三方集成。
5. **Enterprise**：企业级私有化部署方案，支持 Kubernetes 及提供商业支持。

项目核心代码遵循 MIT 协议开源（企业目录除外）。
