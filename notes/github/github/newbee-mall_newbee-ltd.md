### [newbee-ltd newbee-mall](https://github.com/newbee-ltd/newbee-mall)  ![GitHub Repo stars](https://img.shields.io/github/stars/newbee-ltd/newbee-mall?style=social)

newbee-mall 是一套基于 Spring Boot 的电商系统，包含前台商城系统及后台管理系统。
前台功能模块：首页门户、商品分类、新品上线、轮播推荐、商品搜索与展示、购物车、订单结算与流程、个人订单管理、会员中心及帮助中心。
后台功能模块：数据面板、轮播图管理、商品管理、订单管理、会员管理、分类管理及系统设置。
项目支持多种 Spring Boot 版本分支，并提供微服务版、前后端分离版（Vue2/Vue3）及 Go 语言版本。
