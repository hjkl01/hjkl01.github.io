### [pythops bluetui](https://github.com/pythops/bluetui)  ![GitHub Repo stars](https://img.shields.io/github/stars/pythops/bluetui?style=social)

bluetui 是一款运行于 Linux 系统的蓝牙管理终端用户界面（TUI）工具。它依赖 bluez 服务，支持通过包管理器、Cargo 或源码编译安装。核心功能包括管理蓝牙适配器（开关电源、配对、发现）以及管理已配对和新设备（扫描、连接/断开、信任、收藏、重命名）。用户可通过配置文件自定义界面布局及快捷键绑定。项目采用 GPLv3 协议开源。
