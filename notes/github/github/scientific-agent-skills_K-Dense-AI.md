### [scientific-agent-skills](https://github.com/K-Dense-AI/scientific-agent-skills) ![GitHub Repo stars](https://img.shields.io/github/stars/K-Dense-AI/scientific-agent-skills?style=social)

Scientific Agent Skills 是一个面向支持 Agent Skills 开放标准的 AI 代理（如 Cursor、Claude Code、Codex 等）的科研技能库，提供 138 个可直接使用的科学研究技能，用于把通用 AI 代理升级为可执行复杂多步骤流程的“AI 科研助手”。

项目核心功能包括：统一访问 100+ 科学与金融数据库（含 78+ 公共数据库）、70+ 优化的 Python 科学计算技能、9 个科研平台集成技能、30+ 分析与科研沟通工具、10+ 研究与临床决策工具。覆盖生物信息学、药物发现、蛋白质组学、临床研究、医学影像、机器学习、材料与物理、工程仿真、地理空间、实验室自动化、多组学、蛋白设计、科研写作与方法学等领域。

每个技能提供完整文档（SKILL.md）、代码示例、最佳实践、集成指南与参考资料；支持通过 npx 或 gh skill 快速安装，代理可自动发现并调用相关技能。项目采用 MIT 许可（但单个技能可能有独立许可），持续维护并支持社区贡献，同时强调安全审查与按需安装。
