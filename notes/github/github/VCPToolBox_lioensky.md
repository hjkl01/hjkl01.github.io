### [lioensky VCPToolBox](https://github.com/lioensky/VCPToolBox)  ![GitHub Repo stars](https://img.shields.io/github/stars/lioensky/VCPToolBox?style=social)

VCP (Variable & Command Protocol) 是一款新一代 AI 能力增强与进化中间层，致力于构建通用系统以赋能 AI 实现能力跃迁、记忆进化与群体智能涌现。

1. **分布式架构**：支持星型网络拓扑，主服务器通过 WebSocket 管理分布式节点（如 GPU、文件服务器），实现跨服务器文件访问与算力弹性扩展。
2. **记忆与认知系统**：底层数据层使用 Rust 重写，核心包含 TagMemo“浪潮”算法（V5 优化 RAG）、VCP 元思考系统（递归思维链）、Magi 三贤者决策核心及 AgentDream 梦系统（内省与记忆重构）。
3. **插件生态**：集成 300+ 官方插件，支持 6 种协议类型及 MCP 兼容端口，提供动态工具注入、零代码逻辑实现及插件间协作能力。
4. **Agent 主观能动性**：赋予 AI 自主时间规划、自巡航模式、跨 Agent 通讯、社区参与及内省进化能力，实现无需持续指令的主动交互。
5. **交互与管理**：提供 Web 管理面板与 VCPChat 官方前端，支持高级变量替换、系统提示词模块化工程及 GUI 自动生成，构建人机协同进化平台。
6. **安全与部署**：涉及硬件底层权限，强调 API 安全性，支持 Node.js、Python 及 Docker 部署，采用 CC BY-NC-SA 4.0 许可证，需专业用户谨慎部署。
