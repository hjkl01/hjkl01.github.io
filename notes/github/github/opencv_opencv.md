## OpenCV 功能总结

**OpenCV**（Open Source Computer Vision Library）是一个开源计算机视觉库，主要功能包括：

**核心功能：**
- 图像处理与变换
- 目标检测与识别
- 人脸检测与分析
- 视频分析与运动跟踪
- 特征提取与匹配
- 相机校准与3D重建
- 机器学习与深度学习支持

**支持特性：**
- 多语言API（C++、Python、Java等）
- 跨平台运行（Windows、Linux、macOS、Android、iOS）
- GPU加速计算支持
- 丰富的算法库与示例代码

**应用场景：**
- 实时视频监控
- 自动驾驶
- 医学图像分析
- 工业检测
- 增强现实
- 图像检索

OpenCV是计算机视觉领域最广泛使用的开源库之一，提供了从基础图像处理到高级视觉算法的完整工具链。
