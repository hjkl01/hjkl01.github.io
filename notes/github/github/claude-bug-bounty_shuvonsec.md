### [claude-bug-bounty](https://github.com/shuvonsec/claude-bug-bounty) ![GitHub Repo stars](https://img.shields.io/github/stars/shuvonsec/claude-bug-bounty?style=social)

AI 驱动漏洞赏金工具。终端运行。

支持独立 CLI (Ollama/Groq/DeepSeek) 或 Claude Code 插件。

侦察：枚举子域、探测主机、爬取 URL。扫攻击面。
搜寻：测试 Web2 (IDOR/XSS/SQLi) 与 Web3 智能合约漏洞。找漏洞。
验证：七问门槛过滤弱点。杀误报。
报告：生成 HackerOne/Bugcrowd 模板。换赏金。
记忆：跨会话学习模式。存技巧。

安装：运行 `install.sh`。执行 `bughunter recon/hunt`。
