### [czl9707 gh-space-shooter](https://github.com/czl9707/gh-space-shooter)  ![GitHub Repo stars](https://img.shields.io/github/stars/czl9707/gh-space-shooter?style=social)

该项目将 GitHub 贡献图转换为太空射击游戏动画。支持通过 Web 界面、GitHub Action 或命令行工具生成 GIF/WebP 格式动画，将贡献记录映射为敌人，飞船随用户编码历史进行战斗。功能涵盖自定义攻击策略、动画帧率、输出路径及数据导出，支持将动画直接嵌入 README 或通过 Action 自动更新。使用需配置 GitHub 个人访问令牌。
