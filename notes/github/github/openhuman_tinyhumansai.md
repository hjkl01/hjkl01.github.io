### [openhuman](https://github.com/tinyhumansai/openhuman) ![GitHub Repo stars](https://img.shields.io/github/stars/tinyhumansai/openhuman?style=social)

OpenHuman 是一款开源的个人 AI 智能体桌面平台，主打“本地优先 + 托管能力结合”：在本机保存记忆树、Obsidian 风格知识库、工作区配置与运行状态，同时按需使用官方托管服务（登录、模型路由、搜索代理、集成/OAuth 等）。

项目核心功能包括：提供简洁的 UI 优先体验和可交互桌面吉祥物；支持 118+ 第三方工具一键 OAuth 集成（如 Gmail、Notion、GitHub、Slack 等）；每 20 分钟自动拉取并同步外部数据；将文档/邮件/聊天等内容压缩整理为本地 SQLite 记忆树与 Markdown 知识库；内置网页搜索、抓取、代码工具链、语音能力与模型路由；支持可选本地模型（Ollama）；通过 TokenJuice 在保留信息的前提下显著降低 token 成本与延迟（最高可达约 80%）。

整体定位是让智能体在“几分钟而非几周”内获得你的上下文，形成可持续记忆与跨工作流协同能力，并在易用性、集成广度、自动上下文构建和本地数据掌控方面区别于其他 Agent 框架。
