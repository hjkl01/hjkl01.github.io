### [crossplane crossplane](https://github.com/crossplane/crossplane)

Crossplane 是一个由 Cloud Native Computing Foundation（CNCF）支持的开源框架，用于构建无需编写代码的云原生控制平面。其核心功能包括：  
1. **高度可扩展的后端**：支持在任意环境中编排应用和基础设施。  
2. **高度可配置的前端**：允许用户自定义声明式 API 的 schema。  
3. **多云兼容**：适用于跨云或混合云场景。  

**使用方法**：通过官方文档提供的安装指南和资源快速入门（如 [Get Started Docs]）。  

**主要特性**：  
- 声明式 API 设计  
- 社区驱动开发，支持通过 Slack、Twitter 等渠道参与贡献  
- 提供版本管理及路线图（GitHub 项目板）  
- Apache 2.0 开源许可证  

项目维护多个版本（如 v1.20 至 v2.4），并定期更新发布。
