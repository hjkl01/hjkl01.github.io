### [CapSoftware Cap](https://github.com/CapSoftware/Cap)  ![GitHub Repo stars](https://img.shields.io/github/stars/CapSoftware/Cap?style=social)

Cap 是一款开源视频消息工具，定位为 Loom 的开源替代品，支持快速录制、编辑和分享视频。项目采用 Monorepo 架构，包含基于 Tauri 的桌面端应用和基于 Next.js 的 Web 应用。支持多种自托管部署方式（如 Docker Compose、Railway），使用 MySQL 数据库，并集成 Tinybird 进行数据分析。
