### [OpenBMB ChatDev](https://github.com/OpenBMB/ChatDev)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenBMB/ChatDev?style=social)

ChatDev 2.0 (DevAll) 是一个零代码多智能体编排平台，旨在实现“开发一切”。用户无需编写代码，通过简单配置即可定义智能体、工作流和任务，快速构建并执行定制化的多智能体系统。平台支持数据可视化、3D 生成、深度研究、游戏开发及视频教学等多种复杂场景。提供 Web 可视化控制台、Python SDK 及 Docker 部署方式，支持自定义节点与工具，具备高度可扩展性。
