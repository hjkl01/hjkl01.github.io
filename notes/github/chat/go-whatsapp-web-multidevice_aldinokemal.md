### [aldinokemal go-whatsapp-web-multidevice](https://github.com/aldinokemal/go-whatsapp-web-multidevice)  ![GitHub Repo stars](https://img.shields.io/github/stars/aldinokemal/go-whatsapp-web-multidevice?style=social)

这是一个基于 Go 语言开发的 WhatsApp 多设备 Web 客户端工具，支持高效内存使用及多架构部署。

1. 多账户管理：支持同一实例连接和管理多个 WhatsApp 账户。
2. 消息交互：通过 REST API 或 MCP 协议发送文本、媒体、视频、贴纸、联系人等消息，支持自动提及、自动回复、自动已读及媒体压缩转换。
3. 对象管理：提供群组创建与成员管理、联系人/聊天列表操作、用户状态发布及隐私设置。
4. 系统集成：支持 Webhook 事件推送（含过滤与签名验证）、Chatwoot 客服集成及 n8n 工作流插件。
5. 部署配置：支持 Docker、二进制文件部署，具备 Basic Auth、子路径部署及调试模式配置能力。
