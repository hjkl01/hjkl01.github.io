### [quackduck devzat](https://github.com/quackduck/devzat)  ![GitHub Repo stars](https://img.shields.io/github/stars/quackduck/devzat?style=social)

Devzat 是一个基于 SSH 协议的在线聊天服务器，允许用户通过 SSH 客户端在任意设备（含移动端）上连接，无需安装额外应用。其核心功能包括多房间与私聊、Markdown 及代码语法高亮、表情符号替换、内置小游戏（井字棋、猜单词）及用户个性化设置（昵称、颜色、时区）。项目采用 Go 语言开发，支持自托管部署，并提供 Slack/Discord 桥接及插件 API 以实现消息互通与功能扩展。
