### [tinode chat](https://github.com/tinode/chat)  ![GitHub Repo stars](https://img.shields.io/github/stars/tinode/chat?style=social)

Tinode 是一个全栈开源即时通讯（IM）服务器项目。后端采用 Go 语言编写，提供 Android、iOS、Web 客户端及多语言 gRPC 接口支持。核心功能包括一对一和群组聊天、音视频通话、频道订阅、多设备消息同步、富文本消息（含图片、文件）、精细化权限控制及用户管理。系统支持 MySQL、PostgreSQL、MongoDB 等多种数据库，具备分片集群、容错及对象存储扩展能力。该项目旨在构建现代化、去中心化的即时通讯平台，作为 XMPP 的替代方案，目前处于 Beta 版本。
