### [chanify chanify](https://github.com/chanify/chanify)  ![GitHub Repo stars](https://img.shields.io/github/stars/chanify/chanify?style=social)

Chanify 是一款安全、简洁的跨平台通知推送工具，支持通过 API 向各类客户端发送消息。主要功能包括：

1.  **多格式支持**：支持发送文本、链接、图片、音频、文件及自定义操作指令。
2.  **分布式架构**：支持自建节点服务器（无状态/有状态模式），注重隐私保护。
3.  **多端客户端**：提供 iOS、macOS 原生应用，Windows 客户端及 Chrome 浏览器扩展。
4.  **丰富接口**：提供 HTTP API 及命令行工具，兼容 Python、Ruby、NodeJS、PHP 等多种编程语言。
5.  **灵活部署与扩展**：支持 Docker 容器化部署、Lua 脚本扩展（Webhook）及 YAML 配置，包含令牌安全与注册权限控制。
