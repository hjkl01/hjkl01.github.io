### [nickrunning wechat-selkies](https://github.com/nickrunning/wechat-selkies)  ![GitHub Repo stars](https://img.shields.io/github/stars/nickrunning/wechat-selkies?style=social)

WeChat Selkies 是一个基于 Docker 的微信/QQ Linux 客户端容器化项目，利用 Selkies WebRTC 技术实现浏览器访问微信/QQ，无需本地安装。支持配置与数据持久化、中文字体及输入法、图片复制、文件传输、GPU 硬件加速及多架构兼容，并可配置客户端自动启动。此外，支持在容器内安装第三方应用并自动集成桌面快捷方式，适用于服务器部署和远程办公。
