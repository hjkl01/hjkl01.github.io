### [huan docker-wechat](https://github.com/huan/docker-wechat)  ![GitHub Repo stars](https://img.shields.io/github/stars/huan/docker-wechat?style=social)

DoChat（盒装微信）是一个基于 Docker 和 Wine 的项目，用于在 Linux 系统上运行 Windows 版微信客户端。用户只需执行一条命令即可启动，支持中文输入显示、图片粘贴、多版本微信选择及屏幕 DPI 缩放配置。项目支持登录数据与文件持久化，但用户需注意微信账号可能存在封禁风险。
