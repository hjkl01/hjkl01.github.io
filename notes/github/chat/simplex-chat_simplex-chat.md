### [simplex-chat simplex-chat](https://github.com/simplex-chat/simplex-chat)  ![GitHub Repo stars](https://img.shields.io/github/stars/simplex-chat/simplex-chat?style=social)

SimpleX Chat 是一款设计完全私密的即时通讯平台，核心特点是没有任何用户标识符。主要功能与特性如下：

1. **零标识符隐私**：不分配任何用户标识符（包括随机数），彻底隐藏用户身份、联系人及通信元数据（如通话时间）。
2. **高级加密安全**：采用双棘轮端到端加密及抗量子密钥交换，支持 TLS 1.2/1.3 及 Tor 匿名访问，确保通信内容安全。
3. **数据本地化控制**：所有用户数据（消息、联系人、群组）仅存储在客户端，服务器仅作中继不持久化存储消息。
4. **跨平台应用**：提供 Android、iOS 移动应用及 Linux、macOS、Windows 终端（CLI）版本。
5. **丰富通讯能力**：支持一对一聊天、群组、语音/视频通话、大文件传输及自毁消息。
6. **抗垃圾与反滥用**：无标识符机制防止未经请求的联系，有效抵御骚扰。
7. **去中心化架构**：基于客户端 - 服务器网络，用户可自建中继服务器或加入公共网络。
8. **开源协议**：协议与核心代码开源，支持第三方开发及社区协作。
