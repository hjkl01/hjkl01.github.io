### [WhiskeySockets Baileys](https://github.com/WhiskeySockets/Baileys)  ![GitHub Repo stars](https://img.shields.io/github/stars/WhiskeySockets/Baileys?style=social)

Baileys 是一个基于 WebSockets 的 TypeScript 库，用于与 WhatsApp Web API 进行交互。该项目非 WhatsApp 官方产品，采用 MIT 协议开源。使用者需自行确保合规，严禁用于垃圾信息发送、自动化群发或违反服务条款的行为。
