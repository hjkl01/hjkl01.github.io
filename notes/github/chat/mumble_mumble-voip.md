### [mumble-voip mumble](https://github.com/mumble-voip/mumble)  ![GitHub Repo stars](https://img.shields.io/github/stars/mumble-voip/mumble?style=social)

Mumble 是一款开源、低延迟且高质量的跨平台语音聊天软件。它由客户端和服务器端（mumble-server）组成，支持 Windows、Linux、macOS、FreeBSD 等操作系统。项目功能包括实时语音通信、插件扩展、多语言翻译、服务器配置管理（含 Docker 镜像支持）以及 OpenGL 游戏内覆盖。
