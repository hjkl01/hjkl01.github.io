### [gunnarmorling 1brc](https://github.com/gunnarmorling/1brc)  ![GitHub Repo stars](https://img.shields.io/github/stars/gunnarmorling/1brc?style=social)

该项目名为“十亿行挑战”（1BRC），旨在探索现代 Java 处理十亿行文本数据聚合的性能极限。核心任务是编写 Java 程序读取包含气象站温度测量值的文本文件，计算每个站点温度的最小值、平均值和最大值，并按特定格式输出。规则限制仅使用 Java 及标准库，禁止外部依赖和构建时计算，鼓励通过多线程、SIMD、内存映射及 GC 调优等技术优化执行速度。项目提供统一硬件环境进行排行榜评比，以促进 Java 高性能计算技术的交流与学习。
