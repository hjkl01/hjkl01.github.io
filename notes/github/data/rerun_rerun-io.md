### [rerun-io rerun](https://github.com/rerun-io/rerun)  ![GitHub Repo stars](https://img.shields.io/github/stars/rerun-io/rerun?style=social)

Rerun 是一个时间感知的多模态数据栈和可视化工具，用于建模、摄入、存储、查询和查看机器人风格数据（包括图像、点云、文本、张量等）。它提供 C++、Python 和 Rust SDK，支持将数据实时流式传输至 Rerun 查看器或保存为 .rrd 文件。其核心功能包括基于时间轴的数据可视化与回溯调试、多模态数据查询（DataFrame API），适用于机器人、空间 AI、仿真、工业处理及计算机视觉等场景，帮助用户调试算法、分析数据演变及提取训练数据集。
