### [bruin-data ingestr](https://github.com/bruin-data/ingestr)  ![GitHub Repo stars](https://img.shields.io/github/stars/bruin-data/ingestr?style=social)

ingestr 是一款无需编写代码的命令行工具，支持将数据从任意源传输到任意目的地。它通过简单的命令行标志执行数据摄取，无需管理后端，支持追加、合并或删除插入等增量加载模式。项目支持广泛的数据库（如 PostgreSQL、BigQuery 等）和平台（如 Stripe、Shopify 等），其中部分数据库和平台可用作数据源，多种数据库可用作目的地。
