### [dlt-hub dlt](https://github.com/dlt-hub/dlt)  ![GitHub Repo stars](https://img.shields.io/github/stars/dlt-hub/dlt?style=social)

dlt 是一个开源 Python 库，旨在自动化数据加载任务。它能从 REST API、SQL 数据库、云存储等多种源提取数据并加载至各类目标存储。dlt 提供自动模式推断、数据归一化、增量加载及模式演化功能，支持数据转换与管道检查。该库灵活可扩展，可在 Airflow、Serverless 等任何 Python 环境中部署，并专为 LLM 工作流设计。
