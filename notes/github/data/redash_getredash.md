### [getredash redash](https://github.com/getredash/redash)  ![GitHub Repo stars](https://img.shields.io/github/stars/getredash/redash?style=social)

Redash 是一个基于浏览器的数据查询与可视化工具，旨在让用户轻松探索、查询、可视化及共享来自任意数据源的数据。其核心功能包括：支持 SQL 和 NoSQL 查询编辑（含自动补全与模式浏览器）；提供拖拽式可视化制作及仪表盘组合；支持查询与图表的共享协作、定时刷新及数据变更警报；提供完整的 REST API；原生支持 35 种以上常见数据库和平台并支持扩展。该项目易于上手，无需复杂技能即可帮助用户进行数据驱动决策。
