### [pola-rs polars](https://github.com/pola-rs/polars)  ![GitHub Repo stars](https://img.shields.io/github/stars/pola-rs/polars?style=social)

Polars 是一个用 Rust 编写的高性能 DataFrame 查询引擎，提供 Python、Rust、Node.js、R 及 SQL 多语言前端。它支持懒加载与即时执行模式，利用流式处理技术可处理超出内存的大数据集。项目具备查询优化、多线程、SIMD 加速及强大的表达式 API 特性，采用 Apache Arrow 列式格式，以速度极快、轻量易用为核心优势。
