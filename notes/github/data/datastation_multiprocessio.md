### [multiprocessio datastation](https://github.com/multiprocessio/datastation)  ![GitHub Repo stars](https://img.shields.io/github/stars/multiprocessio/datastation?style=social)

DataStation 是一款面向开发者的开源跨平台数据 IDE，支持 Windows、macOS 和 Linux。它支持连接 20 多种数据库（如 PostgreSQL、MySQL、Elasticsearch）和 13 种文件格式，可构建图表、表格和报告。用户可使用 Python、JavaScript、R、SQL 等语言编写脚本对数据进行清洗、关联和操作，并支持数据导出及 HTTP 面板功能。
