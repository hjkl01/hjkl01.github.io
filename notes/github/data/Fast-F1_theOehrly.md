### [theOehrly Fast-F1](https://github.com/theOehrly/Fast-F1)  ![GitHub Repo stars](https://img.shields.io/github/stars/theOehrly/Fast-F1?style=social)

FastF1 是一款 Python 包，用于获取和分析一级方程式赛车（F1）的赛程、赛果、计时数据及遥测信息。它兼容 Ergast API 以访问当前及历史数据，将数据封装为扩展的 Pandas DataFrame 并提供自定义处理函数以便操作。此外，项目集成 Matplotlib 支持数据可视化，并实现 API 请求缓存以优化脚本运行性能。
