### [zhu-xlab GlobalBuildingAtlas](https://github.com/zhu-xlab/GlobalBuildingAtlas)  ![GitHub Repo stars](https://img.shields.io/github/stars/zhu-xlab/GlobalBuildingAtlas?style=social)

GlobalBuildingAtlas 项目提供全球 Level of Detail 1 (LoD1) 建筑数据集及相关工具。
1. 数据内容：基于卫星影像和机器学习生成全球建筑物多边形（ODbL 及 CC BY-NC 4.0 许可）、LoD1 3D 模型及高度图，覆盖所有国家和地区。
2. 数据获取：支持 WFS 服务、Web 可视化查看及通过 HuggingFace 和 mediaTUM 下载。
3. 开发工具：提供建筑多边形生成、高度估算、模型构建及绘图代码（Python）。
4. 技术规范：多边形采用 EPSG:3857 坐标系；涉及混合许可证，用户需确保使用合规。
