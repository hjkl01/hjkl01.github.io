### [ibis-project ibis](https://github.com/ibis-project/ibis)  ![GitHub Repo stars](https://img.shields.io/github/stars/ibis-project/ibis?style=social)

Ibis 是一个可移植的 Python 数据框库，提供统一的数据操作 API，支持超过 20 种数据后端（如 DuckDB、BigQuery、Snowflake 等）。它能将 Python 表达式编译为后端的原生语言（SQL 或 DataFrame 代码），支持惰性表达式、交互式探索及 Python 与 SQL 混用。用户只需更改一行代码，即可在不同后端间切换，实现从本地迭代到远程部署的灵活转换。
