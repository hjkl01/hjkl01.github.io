### [gaoyifan china-operator-ip](https://github.com/gaoyifan/china-operator-ip)  ![GitHub Repo stars](https://img.shields.io/github/stars/gaoyifan/china-operator-ip?style=social)

本项目是一个按中国网络运营商分类的 IP 地址库，利用 BGP 和 ASN 数据分析生成，分类准确度高于仅依赖 WHOIS 数据库的传统库。收录了电信、移动、联通、教育网、科技网、鹏博士及谷歌中国等运营商的 IPv4 和 IPv6 地址段。数据每日自动更新，提供 CIDR 格式列表，可通过 Git 或 Web 镜像获取，适用于 DNS 分域解析、运营商流量分流及网络代理工具规则集配置等场景。
