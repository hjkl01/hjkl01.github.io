### [altair-viz altair](https://github.com/altair-viz/altair)  ![GitHub Repo stars](https://img.shields.io/github/stars/altair-viz/altair?style=social)

Vega-Altair 是基于 Vega-Lite 规范的 Python 声明式统计可视化库。它提供简洁一致的 API，能以少量代码生成美观图表，支持声明式的可视化与交互语法（如联动筛选）。图表可展示于 JupyterLab、VS Code 等平台，并支持导出为 PNG、SVG、HTML 或 JSON 格式。
