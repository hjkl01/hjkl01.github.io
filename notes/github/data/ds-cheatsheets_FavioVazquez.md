本项目是一个数据科学速查表集合，汇总了涵盖业务科学、Python 与 R 语言编程、数学统计、大数据处理、机器学习与深度学习、SQL、数据可视化以及 Git 等通用工具的参考资料。所有资源以 PDF 形式提供，旨在为数据科学家和开发者提供全面、便捷的学习与查阅指南。
