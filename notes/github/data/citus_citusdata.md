### [citusdata citus](https://github.com/citusdata/citus)  ![GitHub Repo stars](https://img.shields.io/github/stars/citusdata/citus?style=social)

Citus 是一款 100% 开源的 PostgreSQL 扩展，将 PostgreSQL 转换为支持水平扩展的分布式数据库。核心功能涵盖分布式表（分片存储以聚合资源）、引用表（全节点复制优化连接与读取）、分布式查询引擎（并行化处理查询）、列式存储（压缩与加速扫描）以及基于模式的分片。Citus 支持无缝扩展、高可用部署，完全兼容 PostgreSQL 功能与生态。适用于高并发交易、实时分析、时序数据处理、多租户应用及微服务等场景，可轻松支撑从单节点到 PB 级数据的规模。
