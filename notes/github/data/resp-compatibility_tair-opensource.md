### [tair-opensource resp-compatibility](https://github.com/tair-opensource/resp-compatibility)  ![GitHub Repo stars](https://img.shields.io/github/stars/tair-opensource/resp-compatibility?style=social)

resp-compatibility 是一款用于测试类 Redis 数据库与不同 Redis 版本（如 6.0、7.0 等）兼容性的 Python 命令行工具。它支持单机和集群模式，可执行命令测试、生成 HTML 报告，并支持自定义测试用例，主要用于数据库迁移前的兼容性验证。
