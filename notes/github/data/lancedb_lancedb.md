### [lancedb lancedb](https://github.com/lancedb/lancedb)  ![GitHub Repo stars](https://img.shields.io/github/stars/lancedb/lancedb?style=social)

LanceDB 是面向 AI/ML 应用的多模态数据平台（多模态 AI 湖仓），基于 Lance 列式格式，提供快速、可扩展的生产级向量搜索解决方案。

核心功能包括：
1. **高性能搜索**：支持毫秒级向量搜索（数十亿向量）、全文搜索及 SQL 查询。
2. **多模态管理**：存储、索引和过滤向量、元数据及文本、图像、视频、点云等多模态数据。
3. **高级特性**：零复制技术、自动数据版本管理、向量索引构建支持 GPU 加速。
4. **灵活部署**：100% 开源本地部署或云/企业级无服务器服务，无供应商锁定。
5. **广泛生态集成**：提供 Python、Rust、TypeScript SDK 及 REST API，无缝兼容 LangChain、LlamaIndex、Pandas 等工具。
