### [FortAwesome Font-Awesome](https://github.com/FortAwesome/Font-Awesome)  ![GitHub Repo stars](https://img.shields.io/github/stars/FortAwesome/Font-Awesome?style=social)

Font Awesome 是互联网广泛使用的图标库和工具包，服务于设计师、开发者及内容创作者。当前主版本为 V7，V6 为长期支持版。项目提供文档、升级指南及贡献规范。该库开源免费，图标、字体和代码分别遵循 CC BY 4.0、SIL OFL 1.1 和 MIT 协议，支持商业及开源项目使用。
