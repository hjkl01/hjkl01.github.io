### [chaitin SafeLine](https://github.com/chaitin/SafeLine)  ![GitHub Repo stars](https://img.shields.io/github/stars/chaitin/SafeLine?style=social)

SafeLine 是一款自托管的 Web 应用防火墙（WAF），通过反向代理机制过滤和监控 HTTP/HTTPS 流量，保护 Web 应用免受攻击。

主要功能包括：
- **攻击防御**：拦截 SQL 注入、XSS、命令注入、RCE、路径遍历等多种 Web 攻击。
- **速率限制**：防御 DoS 攻击、暴力破解及异常流量。
- **反机器人挑战**：自动区分人机，拦截恶意爬虫。
- **身份验证挑战**：支持设置访问密码，限制未授权访问。
- **动态保护**：实时加密 HTML 和 JS 代码。
- **访问控制**：支持基于 IP 的 Web 访问控制列表。
