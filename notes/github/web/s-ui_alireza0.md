### [alireza0 s-ui](https://github.com/alireza0/s-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/alireza0/s-ui?style=social)

S-UI 是一款基于 SagerNet/Sing-Box 的高级 Web 管理面板，支持 Linux、Windows 及 macOS 平台。主要功能包括：
1. 多协议支持：涵盖 VLESS、VMess、Trojan、Shadowsocks、Hysteria、TUIC 等多种协议。
2. 用户管理：支持多语言、多客户端及入出站配置，可设置流量限制与过期时间。
3. 流量监控：提供高级流量路由界面，实时统计客户端流量、系统状态及在线情况。
4. 服务集成：支持订阅链接生成（link/json/clash）、API 接口及 HTTPS/SSL 证书配置。
5. 部署灵活：支持脚本安装、Docker 容器化部署及手动编译运行。
