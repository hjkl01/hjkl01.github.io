### [PGYER codefever](https://github.com/PGYER/codefever)  ![GitHub Repo stars](https://img.shields.io/github/stars/PGYER/codefever?style=social)

CodeFever Community 是一个完全开源的 Git 代码托管服务，永久免费且无仓库及使用数量限制。项目轻量高效，仅需 1 核 CPU/1G 内存即可运行，核心功能涵盖代码版本对比、分支与 Tag 管理、合并请求、分支保护、多人协作权限设置、Webhook 集成及超级管理员后台。支持中英双语，提供 Docker 及源码安装方式，基于 MIT 协议发布。
