### [myide see](https://github.com/myide/see)  ![GitHub Repo stars](https://img.shields.io/github/stars/myide/see?style=social)

See SQL 审核平台是一个 SQL 审核与管理平台，集成 Inception、SQLAdvisor 和 SOAR 引擎。功能涵盖多数据中心数据库管理、SQL 语法检测与执行回滚、定时工单、表结构与数据查询导出。提供基于算法的 SQL 优化建议，具备基于 RBAC 的权限体系及自定义审批流程（支持双人确认）。此外，还包含管理员个性化设置、SSO 登录、邮件通知、数据报表及自动化 API 文档功能。
