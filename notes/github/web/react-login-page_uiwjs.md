### [uiwjs react-login-page](https://github.com/uiwjs/react-login-page)  ![GitHub Repo stars](https://img.shields.io/github/stars/uiwjs/react-login-page?style=social)

react-login-page 是一个基于 React 的登录页面组件库，提供封装好的基础组件以便快速构建和定制登录页。它支持灵活的 API，允许通过 index 参数控制控件顺序，通过 visible 属性控制显示隐藏。核心组件包括 Logo、Block、Input、Textarea、Select、Button 及 Render 容器，并提供 useStore 钩子进行状态管理，帮助开发者简化登录表单的开发流程。
