### [mui material-ui](https://github.com/mui/material-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/mui/material-ui?style=social)

Material UI 是一款功能全面的 React 组件库，独立实现了 Google 的 Material Design 系统设计。该库经过十余年开发和众多贡献者的验证，可靠性高。其核心功能由 MUI X 套件扩展，以支持高级用例。仓库内包含 Joy UI 组件库，但处于实验性开发暂停状态，建议新项目优先使用 Material UI。项目提供文档、示例和付费模板，遵循 MIT 开源协议。
