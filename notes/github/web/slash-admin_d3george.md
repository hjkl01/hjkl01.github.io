### [d3george slash-admin](https://github.com/d3george/slash-admin)  ![GitHub Repo stars](https://img.shields.io/github/stars/d3george/slash-admin?style=social)

Slash Admin 是一个基于 React 19、Vite、shadcn/ui 和 TypeScript 构建的现代管理后台模板。主要功能包括：响应式设计适配多端、灵活嵌套路由配置、基于角色的访问控制、多语言国际化、可定制主题样式、内置用户及权限管理模块，以及集成 Zustand 状态管理、React-Query 数据获取和 MSW 数据模拟解决方案。
