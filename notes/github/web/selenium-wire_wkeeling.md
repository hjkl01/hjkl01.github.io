### [wkeeling selenium-wire](https://github.com/wkeeling/selenium-wire)  ![GitHub Repo stars](https://img.shields.io/github/stars/wkeeling/selenium-wire?style=social)

Selenium Wire 是 Selenium 的 Python 扩展库，用于扩展浏览器自动化功能，允许用户访问、检查和修改浏览器底层的 HTTP/HTTPS 网络请求及响应。

主要功能：
- **请求监控**：捕获所有网络请求的 URL、状态码、Header、Body 及 WebSocket 消息详情。
- **拦截修改**：支持动态拦截请求和响应，修改 Header、参数及内容。
- **流量控制**：支持阻断特定请求、伪造响应。
- **数据存储**：支持 HAR 格式导出，提供磁盘或内存存储模式。
- **网络配置**：支持代理（HTTP/SOCKS）、SSL 证书配置及 HTTPS 流量解密。
- **兼容性**：兼容 Python 3.7+ 及 Selenium 4.0+，支持 Chrome、Firefox、Edge 等浏览器。
