### [newpanjing simpleui](https://github.com/newpanjing/simpleui)  ![GitHub Repo stars](https://img.shields.io/github/stars/newpanjing/simpleui?style=social)

SimpleUI 是一款基于 Element-UI 和 Vue 的 Django Admin 主题包。它重构了大部分后台页面，提供更简洁友好的管理界面。核心特性包括：内置 28 种主题、支持多标签页、100% 兼容原生 Admin 无需改代码、支持中英文国际化。配置简单，仅需在 settings.py 中添加应用即可生效，兼容最新版 Django 和 Python。此外还提供功能更强的专业版（Simple Pro）。
