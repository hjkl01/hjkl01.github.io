### [adrianhajdin zoom-clone](https://github.com/adrianhajdin/zoom-clone)  ![GitHub Repo stars](https://img.shields.io/github/stars/adrianhajdin/zoom-clone?style=social)

本项目是一个基于 Next.js 和 TypeScript 构建的 Zoom 视频会议克隆应用，核心功能如下：

- **用户认证**：集成 Clerk，支持通过社交账号或邮箱密码安全登录。
- **会议创建与加入**：可快速启动新会议、预约未来会议、生成个人房间链接，或通过链接加入他人会议。
- **会议控制**：提供录音、屏幕共享、音视频静音/取消、参与者管理（置顶、拉黑、视频权限）、表情反应及界面布局切换等功能。
- **会议管理**：支持查看历史会议列表及回放已录制的会议内容。
- **基础体验**：具备实时安全交互能力，采用响应式设计以适配不同屏幕设备。
