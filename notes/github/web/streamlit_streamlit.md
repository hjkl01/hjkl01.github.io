### [streamlit streamlit](https://github.com/streamlit/streamlit)  ![GitHub Repo stars](https://img.shields.io/github/stars/streamlit/streamlit?style=social)

Streamlit 是一款开源 Python 库，允许开发者快速将 Python 脚本转换为交互式 Web 应用，无需编写 HTML、CSS 或 JavaScript。它支持构建数据仪表盘、生成报告或创建聊天应用，具备代码简单、实时预览和快速原型开发的特性。项目提供丰富的功能组件（如输入控件、数据表格、图表、布局等）及多页面应用支持，并可通过社区云平台免费部署与分享。Streamlit 遵循 Apache 2.0 协议，完全免费开源。
