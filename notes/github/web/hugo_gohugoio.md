### [gohugoio hugo](https://github.com/gohugoio/hugo)  ![GitHub Repo stars](https://img.shields.io/github/stars/gohugoio/hugo?style=social)

Hugo 是一个用 Go 语言编写的快速、灵活的静态网站生成器，能在数秒内渲染完整站点，适用于企业官网、文档、博客等多种类型网站。它具备高级模板系统、多语言支持、强大的分类系统及内置开发服务器（支持即时预览）。项目提供高效的资产管道，支持图像处理、JavaScript、Sass 及 Tailwind CSS 等前端资源编译优化。通过 Hugo Modules 可共享内容、资源和主题。提供标准版与扩展版（含 Sass 编译及云存储部署支持）。
