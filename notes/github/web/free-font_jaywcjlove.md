### [jaywcjlove free-font](https://github.com/jaywcjlove/free-font)  ![GitHub Repo stars](https://img.shields.io/github/stars/jaywcjlove/free-font?style=social)

这是一个收录商用免费字体的开源项目，主要收集汉字及部分英文开源字体。旨在为《字帖宝宝》等项目提供字体资源支持，方便用户下载和使用。项目支持字体分类管理、自动生成预览封面及网站构建，提供本地开发和 Docker 部署方式。同时明确各字体授权协议（如免费商用、OFL、GPL 等），并提醒用户注意字体版权风险及文件上传大小限制。
