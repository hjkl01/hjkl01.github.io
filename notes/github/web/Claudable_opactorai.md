### [opactorai Claudable](https://github.com/opactorai/Claudable)  ![GitHub Repo stars](https://img.shields.io/github/stars/opactorai/Claudable?style=social)

Claudable 是一个基于 Next.js 的开源 Web 应用构建工具，通过集成 Claude Code、Cursor CLI 等 AI 代理能力，实现自然语言驱动的开发流程。

核心功能：
- **自然语言生成代码**：用户描述需求即可自动生成生产级 Next.js 代码。
- **实时预览**：支持热重载，AI 构建过程中可即时查看应用效果。
- **一键部署与集成**：无缝部署至 Vercel，自动集成 GitHub 版本控制及 Supabase 数据库（含认证）。
- **多 AI 代理兼容**：支持 Claude Code、Codex CLI、Cursor CLI、Qwen Code、Z.AI GLM-4.6 等多种工具。
- **桌面应用**：提供基于 Electron 的桌面客户端，支持 Mac、Windows 和 Linux。
- **免费开源**：构建工具无额外订阅费用，采用 MIT 协议。

技术栈：Next.js、Supabase、Vercel、Tailwind CSS、shadcn/ui。
