### [parcel-bundler parcel](https://github.com/parcel-bundler/parcel)  ![GitHub Repo stars](https://img.shields.io/github/stars/parcel-bundler/parcel?style=social)

Parcel 是一款零配置的 Web 构建工具，核心功能如下：
1. **开箱即用**：支持 HTML、CSS、JavaScript 及多种媒体资源，内置带热更新和错误诊断的开发服务器，无需配置。
2. **极速构建**：基于 Rust 编写编译器，支持多线程并行构建与智能缓存，显著提升速度。
3. **自动优化**：生产模式下自动执行代码压缩、树摇、图片处理、内容哈希及代码分割。
4. **多目标适配**：自动转换代码以适配现代和遗留浏览器环境，原生支持 JSX 和 TypeScript 编译。
5. **高度可扩展**：虽默认零配置，但可通过插件系统和配置灵活扩展，适应从初创到大型生产项目的各种需求。
