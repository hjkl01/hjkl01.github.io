### [lissy93 dashy](https://github.com/lissy93/dashy)  ![GitHub Repo stars](https://img.shields.io/github/stars/lissy93/dashy?style=social)

Dashy 是一款用于组织和管理自托管服务的个人主页仪表盘工具，旨在帮助用户从单一位置便捷访问所有服务。

主要功能：
- **服务聚合**：支持多页面布局，可快速分类和访问各类自托管应用。
- **状态监控**：实时检测服务在线状态、响应时间及 HTTP 状态码。
- **高度定制**：内置多种主题，支持自定义 CSS、图标（如 FontAwesome、Favicon）、布局及文本。
- **便捷交互**：支持即时搜索、自定义快捷键、多种应用打开方式及极简/工作区多视图模式。
- **安全配置**：提供多用户认证（含 SSO 支持）、端到端加密云备份及多语言支持。
- **灵活部署**：支持 Docker、源码安装及主流云服务商（如 Netlify、Vercel 等）一键部署。

项目完全免费且开源。
