### [scrapinghub dateparser](https://github.com/scrapinghub/dateparser)  ![GitHub Repo stars](https://img.shields.io/github/stars/scrapinghub/dateparser?style=social)

dateparser 是一个 Python 库，用于解析人类可读的日期文本。它支持几乎所有现有日期格式（如绝对日期、相对日期、时间戳），兼容 200 多种语言并支持语言自动检测。该库支持非格里历日历系统、时区识别、长文本日期搜索及时间跨度检测，并允许通过设置自定义行为。适用于数据抓取、日志处理、格式转换及自然语言交互等场景。
