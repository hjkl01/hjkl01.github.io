### [xiangyuecn AreaCity-JsSpider-StatsGov](https://github.com/xiangyuecn/AreaCity-JsSpider-StatsGov)  ![GitHub Repo stars](https://img.shields.io/github/stars/xiangyuecn/AreaCity-JsSpider-StatsGov?style=social)

该项目提供中国省市区镇四级行政区划数据的采集、处理、展示及查询服务。
1. **数据资源**：整合统计局、民政部及主流地图数据，提供省市区镇四级名称、编号、拼音标注，以及省市区三级坐标与边界范围（GCJ-02 坐标系），兼容直筒子市等特殊情况。
2. **在线工具**：支持数据在线预览、生成 JSON/多级联动 JS 代码、ECharts 地图四级下钻测试。
3. **转换入库**：提供格式转换工具，支持 CSV 数据导入 MySQL/SQL Server 等数据库，及转成 SQL、SHP、GeoJSON 格式，支持坐标系转换。
4. **地理查询**：提供高性能 Java 开源查询工具，支持 HTTP 接口，实现坐标反查城市及边界数据检索。
5. **采集维护**：提供浏览器采集脚本源码，支持自主更新行政区划及边界数据。
