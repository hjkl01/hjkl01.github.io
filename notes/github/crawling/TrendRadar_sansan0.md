### [sansan0 TrendRadar](https://github.com/sansan0/TrendRadar)  ![GitHub Repo stars](https://img.shields.io/github/stars/sansan0/TrendRadar?style=social)

TrendRadar 是一款轻量级热点资讯聚合与推送助手，支持快速部署并实现精准新闻监控。

1. **多平台聚合**：监控知乎、抖音、微博等 10+ 主流平台热搜，支持 RSS/Atom 订阅源抓取。
2. **精准筛选**：支持自定义关键词（含正则）、必须词、过滤词及全局过滤，过滤无效信息。
3. **AI 分析**：集成大模型进行热点趋势分析、情感识别、多语言翻译，支持 MCP 协议深度对话。
4. **多渠道推送**：支持飞书、钉钉、企业微信、Telegram、邮件、Bark、Slack 及通用 Webhook。
5. **灵活部署**：支持 GitHub Actions、Docker 及本地运行，提供可视化配置编辑器与灵活调度系统。
6. **多端存储**：支持本地 SQLite 或远程云存储（S3 兼容），保障数据安全。

提供当日汇总、当前榜单、增量监控三种模式，适合投资者、自媒体人及行业关注者使用。
