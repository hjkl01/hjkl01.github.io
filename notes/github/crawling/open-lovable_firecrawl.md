### [firecrawl open-lovable](https://github.com/firecrawl/open-lovable)  ![GitHub Repo stars](https://img.shields.io/github/stars/firecrawl/open-lovable?style=social)

Open Lovable 是一个开源项目，支持通过 AI 聊天对话即时构建 React 应用程序。它是 Firecrawl 团队制作的示例应用，旨在演示类似 Lovable.dev 的云解决方案。项目支持多种大语言模型（如 Gemini、OpenAI 等），可配置 Vercel 或 E2B 作为代码沙箱环境，并支持使用 Morph API 实现快速编辑。
