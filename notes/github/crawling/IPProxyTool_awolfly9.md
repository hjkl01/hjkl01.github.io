### [awolfly9 IPProxyTool](https://github.com/awolfly9/IPProxyTool)  ![GitHub Repo stars](https://img.shields.io/github/stars/awolfly9/IPProxyTool?style=social)

IPProxyTool 是一个基于 Scrapy 和 Flask 的免费代理 IP 管理与服务工具。主要功能包括：
1. 代理抓取：使用爬虫从多个网站收集免费代理 IP。
2. 验证过滤：检测 IP 可用性、HTTPS 支持及匿名度，筛选有效 IP 存入 MySQL 数据库。
3. 接口服务：提供 HTTP 接口，支持按条件查询、删除及插入代理数据。
4. 部署支持：支持自定义抓取和验证规则，并提供 Docker 部署方式。
