### [camofox-browser](https://github.com/jo-inc/camofox-browser) ![GitHub Repo stars](https://img.shields.io/github/stars/jo-inc/camofox-browser?style=social)

camofox-browser 是一个面向 AI 代理的反检测浏览器服务，基于 Camoufox 提供 Firefox 级指纹伪装与反爬能力，并通过 REST API 让代理可稳定地打开页面、获取可访问性快照、按元素引用点击/输入/滚动/导航。它还支持 Cookie 导入、会话隔离与持久化、代理与地理位置联动、YouTube 字幕提取、搜索宏、截图与下载捕获、VNC 交互登录、OpenAPI 文档、结构化抽取、会话追踪和结构化日志，便于在自动化场景中可靠访问真实网站。
