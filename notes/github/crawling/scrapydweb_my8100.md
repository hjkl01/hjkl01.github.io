### [my8100 scrapydweb](https://github.com/my8100/scrapydweb)  ![GitHub Repo stars](https://img.shields.io/github/stars/my8100/scrapydweb?style=social)

ScrapydWeb 是一款管理 Scrapyd 集群的 Web 应用，支持 Scrapy 日志分析与可视化。

核心功能包括：
- **集群管理**：支持所有 Scrapyd JSON API，提供节点分组、过滤及多节点一键执行命令。
- **日志分析**：提供统计收集、进度可视化及日志分类。
- **增强特性**：包含自动打包、集成 LogParser、定时任务、监控告警、移动端 UI 及 Web UI 基础认证。

使用前需确保所有主机已安装并运行 Scrapyd 服务。
