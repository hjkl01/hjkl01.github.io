### [marticliment WingetUI](https://github.com/marticliment/WingetUI)  ![GitHub Repo stars](https://img.shields.io/github/stars/marticliment/WingetUI?style=social)

UniGetUI 是一款面向 Windows 10/11 的开源图形界面管理工具，整合了 WinGet、Scoop、Chocolatey、Pip、Npm 等多种命令行包管理器。它提供直观的操作界面，支持一键下载、安装、更新和卸载软件，支持批量操作、自动更新提醒及自定义安装选项。此外，它还具备包详情查看、列表导出导入备份恢复等功能，方便用户统一管理和迁移软件环境。
