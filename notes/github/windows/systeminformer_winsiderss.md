### [systeminformer](https://github.com/winsiderss/systeminformer) ![GitHub Repo stars](https://img.shields.io/github/stars/winsiderss/systeminformer?style=social)

System Informer 是一款免费、功能强大的多用途系统工具，适用于 Windows 10 及以上（32/64 位）。它的核心功能包括：  
- 实时、详细地监控系统活动，并通过高亮显示关键信息；  
- 通过图表与统计快速定位资源占用过高或失控的进程；  
- 查找被哪些进程占用文件，解决无法编辑或删除文件的问题；  
- 查看具有活动网络连接的程序，并可按需关闭连接；  
- 提供磁盘访问的实时信息；  
- 支持查看详细调用栈（含内核模式、WOW64、.NET）；  
- 超越 services.msc，支持创建、编辑和控制系统服务；  
- 体积小、便携、无需安装；  
- 采用 MIT 许可证，属于 100% 自由软件。
