### [Raphire Win11Debloat](https://github.com/Raphire/Win11Debloat)  ![GitHub Repo stars](https://img.shields.io/github/stars/Raphire/Win11Debloat?style=social)

Win11Debloat 是一款轻量级 PowerShell 脚本，用于快速清理和优化 Windows 系统。核心功能涵盖：
1. **卸载应用**：移除预装臃肿软件。
2. **隐私与安全**：禁用遥测、广告、跟踪及隐私相关服务。
3. **AI 管控**：禁用 Copilot、Recall 等内置 AI 功能。
4. **系统调优**：调整更新策略、关闭快速启动、优化鼠标及网络设置。
5. **UI 定制**：修改开始菜单、任务栏、资源管理器及界面视觉效果。
6. **高级部署**：支持多用户应用更改及 Sysprep 系统镜像模式。

所有更改基本可逆，支持命令行操作及通过微软商店恢复应用。
