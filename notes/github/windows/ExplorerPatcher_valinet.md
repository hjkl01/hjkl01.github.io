### [valinet ExplorerPatcher](https://github.com/valinet/ExplorerPatcher)  ![GitHub Repo stars](https://img.shields.io/github/stars/valinet/ExplorerPatcher?style=social)

ExplorerPatcher 是一款增强 Windows 工作环境的工具，核心功能是将任务栏、开始菜单及 Alt+Tab 窗口切换器样式恢复为 Windows 10 风格。该软件支持通过任务栏属性面板进行配置，提供内置更新与多种卸载方式，并兼容 Intel、AMD 及 ARM64 等不同处理器架构。
