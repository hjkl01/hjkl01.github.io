### [massgravel Microsoft-Activation-Scripts](https://github.com/massgravel/Microsoft-Activation-Scripts)  ![GitHub Repo stars](https://img.shields.io/github/stars/massgravel/Microsoft-Activation-Scripts?style=social)

Microsoft Activation Scripts (MAS) 是一款开源的 Windows 和 Office 激活工具，支持 HWID、Ohook、TSforge 及在线 KMS 等多种激活方法，并提供高级故障排除功能。用户可通过 PowerShell 命令或直接运行脚本文件，激活 Windows 8.1/10/11 系统及其扩展安全更新（ESU）。
