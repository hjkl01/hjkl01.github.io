### [luodaoyi kms-server](https://github.com/luodaoyi/kms-server)  ![GitHub Repo stars](https://img.shields.io/github/stars/luodaoyi/kms-server?style=social)

该项目是一个基于 vlmcsd 的 KMS 服务器 Docker 镜像，用于 Windows 和 Office 批量授权版本的激活服务。
1. 部署方式：提供 Docker 镜像（绑定 1688 端口）及公网服务地址（kms.luody.info）。
2. 激活指导：包含 Windows 和 Office 的激活命令教程，指导配置服务器地址并执行激活。
3. 密钥资源：提供 Windows 7 至 11 及 Server 系列的 KMS 通用密钥（GVLK）对照表，用于解决密钥配置问题。
