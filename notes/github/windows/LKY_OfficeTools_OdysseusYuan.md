### [OdysseusYuan LKY_OfficeTools](https://github.com/OdysseusYuan/LKY_OfficeTools)  ![GitHub Repo stars](https://img.shields.io/github/stars/OdysseusYuan/LKY_OfficeTools?style=social)

LKY Office Tools 是一款绿色开源工具，可一键自动化下载、安装及激活最新版 Microsoft Office。支持 Word、PPT、Excel 等组件的自助安装，自动识别系统架构下载适配版本，采用正版激活模式且不篡改系统文件。支持在用户同意下自动升级冗余版本。适用于 Windows 10 (1703) 及以上系统，仅限学习交流，禁止商业化。
