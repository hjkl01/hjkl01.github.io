### [PowerToys](https://github.com/microsoft/PowerToys) ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/PowerToys?style=social)

Microsoft PowerToys 是 Windows 系统实用工具集，帮助用户自定义系统、简化日常任务。

核心功能包括：

- **快速启动与搜索**：PowerToys Run、Command Palette
- **窗口管理**：FancyZones（窗口布局）、Always On Top（置顶）、Crop And Lock、Workspaces（工作区）
- **文件工具**：PowerRename（批量重命名）、File Locksmith（文件占用检查）、File Explorer Add-ons（预览增强）、Image Resizer（图片调整）
- **系统工具**：Environment Variables（环境变量）、Hosts File Editor、Registry Preview、Awake（保持唤醒）
- **效率工具**：Advanced Paste（粘贴增强）、Text Extractor（文字识别）、Quick Accent（快捷重音）、Keyboard Manager、Shortcut Guide
- **鼠标键盘**：Mouse Without Borders（多设备共享鼠标）、Mouse Utilities、屏幕标尺、ZoomIt
- **其他**：Color Picker（取色器）、Peek（预览）、PowerDisplay、Light Switch

安装方式：GitHub 下载、Microsoft Store、WinGet (winget install Microsoft.PowerToys)
