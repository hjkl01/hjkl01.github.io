### [BluePointLilac ContextMenuManager](https://github.com/BluePointLilac/ContextMenuManager)  ![GitHub Repo stars](https://img.shields.io/github/stars/BluePointLilac/ContextMenuManager?style=social)

ContextMenuManager 是一款纯粹的 Windows 右键菜单管理程序。它支持在 Win10、8.1、8、7、Vista 系统上启用或禁用文件、文件夹、新建、发送到、打开方式、WinX 等场景的右键菜单项。用户可修改菜单项名称、图标，导航至注册表或文件位置，执行永久删除或自定义添加菜单命令。程序兼容 32/64 位系统，支持高分屏及多语言显示。
