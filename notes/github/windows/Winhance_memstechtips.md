### [memstechtips Winhance](https://github.com/memstechtips/Winhance)  ![GitHub Repo stars](https://img.shields.io/github/stars/memstechtips/Winhance?style=social)

Winhance 是一款基于 C# 的 Windows 10/11 增强工具，旨在无需重装系统的情况下精简、优化和自定义系统体验。核心功能包括：

1. **应用管理**：一键管理 Windows 内置应用（安装/卸载），并通过 WinGet 安装外部第三方软件。
2. **系统优化**：提供隐私、游戏性能、系统更新、电源计划及通知等设置的优化选项。
3. **界面定制**：支持更改系统主题、任务栏、开始菜单及资源管理器外观。
4. **高级工具**：支持基于当前系统创建自定义 Windows ISO 镜像及生成无人值守安装配置文件。
5. **配置管理**：允许保存和优化后的系统设置配置文件，以便在其他系统快速还原。
