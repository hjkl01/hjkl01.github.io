### [glzr-io glazewm](https://github.com/glzr-io/glazewm)  ![GitHub Repo stars](https://img.shields.io/github/stars/glzr-io/glazewm?style=social)

GlazeWM 是一款面向 Windows 的平铺式窗口管理器，灵感源自 i3wm。它允许用户通过键盘命令动态组织窗口并调整布局。

主要功能包括：
- **布局管理**：支持多显示器、自定义工作区及调整窗口内外间隙。
- **灵活配置**：使用 YAML 文件配置快捷键、窗口规则、绑定模式及默认行为。
- **自动化规则**：可针对特定进程、标题或类名自动执行移动、悬浮或忽略等操作。
- **系统集成**：可选集成 Zebar 状态栏，支持 Windows 11 窗口边框特效。
- **安装便捷**：提供独立安装包及 Winget、Chocolatey、Scoop 等多种包管理器支持。
