### [snowie2000 mactype](https://github.com/snowie2000/mactype)  ![GitHub Repo stars](https://img.shields.io/github/stars/snowie2000/mactype?style=social)

MacType 是一款 Windows 字体渲染优化工具，旨在提升系统字体显示效果。

核心功能：
1. 支持 Windows 11、多显示器、彩色字体及 DirectWrite 渲染优化。
2. 提供托盘应用、服务模式及注册表模式配置，降低 CPU 占用。
3. 支持简繁体中文、英文、韩文本地化。
4. 更新 FreeType 引擎，兼容 CET 技术。

注意事项：可能与杀毒软件冲突，Office 2013 不兼容，WPS 需特殊处理，升级前建议备份配置。
