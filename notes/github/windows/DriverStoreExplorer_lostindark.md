### [DriverStoreExplorer](https://github.com/lostindark/DriverStoreExplorer) ![GitHub Repo stars](https://img.shields.io/github/stars/lostindark/DriverStoreExplorer?style=social)

**总结 Windows 驱动管理工具**

该工具用于管理 Windows DriverStore，支持查看列表、添加/安装、删除、导出备份及清理旧驱动，涵盖联机和脱机操作，提供多种API接口，支持批量操作、搜索及多语言界面。

Driver Store Explorer（RAPR）是一款面向高级用户的 Windows 驱动仓库管理工具，可浏览并列出第三方驱动包信息，安装新驱动，删除单个或批量驱动（含强制删除），导出备份驱动，智能识别清理旧/未用驱动；支持本机或离线镜像、DISM/PNPUtil/原生 API，多选批处理、实时搜索与 CSV 导出，并提供多语言可定制界面。
