### [Atlas-OS Atlas](https://github.com/Atlas-OS/Atlas)  ![GitHub Repo stars](https://img.shields.io/github/stars/Atlas-OS/Atlas?style=social)

AtlasOS 是一个开源的 Windows 优化项目，通过透明的 Playbooks 脚本系统（基于 AME Wizard）对系统进行修改，旨在提升性能、隐私和可用性。其核心功能包括移除系统遥测数据、优化系统响应速度、精简无用预装应用、禁用广告以及允许用户自定义安全设置（如 Windows Defender 和更新）。该项目不直接分发修改版 ISO 镜像，符合微软使用条款，且主要组件代码开源。
