### [keon algorithms](https://github.com/keon/algorithms)  ![GitHub Repo stars](https://img.shields.io/github/stars/keon/algorithms?style=social)

这是一个基于 Python 3 的数据结构与算法实现库，提供简洁、清洁且文档完善的代码，主要用于学习与参考。

主要功能包括：
1. **数据结构**：涵盖树（AVL、BST、Trie 等）、堆、链表、栈、队列、哈希表及图结构。
2. **算法**：包含排序、搜索、图算法（最短路径、遍历等）、动态规划、回溯、位运算、数学及字符串处理。
3. **代码特点**：每个文件自包含，附带文档字符串、类型提示和复杂度说明，模块独立清晰。
4. **使用方式**：支持通过 pip 安装，可直接导入模块使用。
