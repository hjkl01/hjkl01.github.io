### [rendercv rendercv](https://github.com/rendercv/rendercv)  ![GitHub Repo stars](https://img.shields.io/github/stars/rendercv/rendercv?style=social)

RenderCV 是一款面向学术人员和工程师的简历生成器。用户通过 YAML 文件编写简历内容，即可生成排版完美的 PDF。核心功能包括：文本化存储支持版本控制，自动处理专业排版，提供多种预设主题及高度自定义设计选项，支持 JSON Schema 辅助编写与严格验证，支持多语言配置。需 Python 3.12+ 环境。
