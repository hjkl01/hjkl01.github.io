### [LandSandBoat server](https://github.com/LandSandBoat/server)  ![GitHub Repo stars](https://img.shields.io/github/stars/LandSandBoat/server?style=social)

LandSandBoat 是一个《最终幻想 XI》（FFXI）的开源服务器模拟器。项目提供快速启动指南、常见问题解答及功能列表等 Wiki 文档，支持通过 GitHub Issues 反馈游戏问题与 Bug，通过 Discussions 进行技术交流。项目采用 GNU GPL v3 协议。
