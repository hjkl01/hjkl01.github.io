### [1995chen dnf](https://github.com/1995chen/dnf)  ![GitHub Repo stars](https://img.shields.io/github/stars/1995chen/dnf?style=social)

该项目是一个基于 Docker 的《地下城与勇士》（DNF）服务端部署方案，基于 CentOS 镜像整合游戏服务端、MySQL 数据库及初始化脚本，实现在 Linux 服务器上快速启动游戏服务。

主要功能：
1. **快速部署**：通过环境变量和脚本自动化初始化，支持 Docker 容器化运行。
2. **服务端功能**：支持多区域部署、假人功能及登录器插件扩展。
3. **运维管理**：提供 Web 管理页面，支持进程监控、日志查看及服务重启。
4. **客户端配置**：提供网关设置、补丁安装及登录器生成的完整指南。
5. **使用限制**：仅供技术学习交流，严禁用于正式商业开服。
