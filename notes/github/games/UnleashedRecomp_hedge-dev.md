### [hedge-dev UnleashedRecomp](https://github.com/hedge-dev/UnleashedRecomp)  ![GitHub Repo stars](https://img.shields.io/github/stars/hedge-dev/UnleashedRecomp?style=social)

Unleashed Recompiled 是一款将 Xbox 360 版《索尼克释放》移植到 PC 的非官方项目，通过静态反编译技术将原始代码转换为 C++ 和 HLSL，支持 Windows 和 Linux 系统。项目核心功能包括：支持高分辨率、超宽屏及高帧率（默认 60 FPS）；利用新渲染器和异步着色器编译显著提升性能；内置高质量选项菜单、成就系统、自定义本地化及手柄图标支持；兼容 Hedge Mod Manager 模组。安装时需提供用户自有的合法 Xbox 360 游戏文件（仅限美版和欧版），项目不包含任何游戏资源。
