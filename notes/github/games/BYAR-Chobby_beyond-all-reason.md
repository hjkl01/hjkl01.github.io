### [beyond-all-reason BYAR-Chobby](https://github.com/beyond-all-reason/BYAR-Chobby)  ![GitHub Repo stars](https://img.shields.io/github/stars/beyond-all-reason/BYAR-Chobby?style=social)

BYAR-Chobby 是面向 Chobby 启动器的《Beyond All Reason》游戏模组，支持 Windows 和 Linux 平台。主要功能包括提供启动器下载、搭建本地大厅开发环境以修改和测试 BYAR 专属功能、支持本地游戏对战调试、提供新版本构建与发布流程指南，并接受社区贡献与反馈。
