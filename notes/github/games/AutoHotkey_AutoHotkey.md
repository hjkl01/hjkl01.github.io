### [AutoHotkey AutoHotkey](https://github.com/AutoHotkey/AutoHotkey)  ![GitHub Repo stars](https://img.shields.io/github/stars/AutoHotkey/AutoHotkey?style=social)

AutoHotkey 是一款免费开源的宏创建与自动化工具，允许用户通过自定义脚本语言自动化重复任务并定义键盘快捷键（热键）。软件支持 Windows XP 及以上版本（涵盖 Win32 和 x64 平台），提供 v1 和 v2 两个版本，其中 v1 已停止官方维护但由社区支持。开发者可使用 Visual Studio 或 VS Code 进行编译和调试，支持多种构建配置，主要技术支持与开发交流由社区论坛提供。
