### [bevyengine bevy](https://github.com/bevyengine/bevy)  ![GitHub Repo stars](https://img.shields.io/github/stars/bevyengine/bevy?style=social)

Bevy 是一个基于 Rust 编写的免费开源游戏引擎，支持完整的 2D 和 3D 功能。采用数据导向架构和实体组件系统（ECS）范式，设计简单且模块灵活，支持按需使用或替换功能。引擎注重性能与开发效率，支持并行处理及快速编译。目前处于早期开发阶段，API 定期更新，采用 MIT 与 Apache-2.0 双重许可。
