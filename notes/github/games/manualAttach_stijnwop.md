### [stijnwop manualAttach](https://github.com/stijnwop/manualAttach)  ![GitHub Repo stars](https://img.shields.io/github/stars/stijnwop/manualAttach?style=social)

ManualAttach 是《模拟农场 25》的模组，旨在增强农具连接的真实感，主要功能包括：

1. **手动连接农具**：强制农具在连接时保持下降状态，禁止大多数情况下从车内切换农具，需玩家下车手动连接（前置装载机、挖掘机、叉车等特定设备除外）。
2. **手动控制附件**：支持按键连接或断开动力输出轴（PTO）及连接软管；软管未连接会导致液压、灯光和刹车功能失效。
3. **扩展支持**：通过 XML 属性（isManual/isAuto）允许开发者自定义农具的连接模式为手动、标准或自动。
