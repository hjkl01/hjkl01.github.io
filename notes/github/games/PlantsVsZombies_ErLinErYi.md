### [ErLinErYi PlantsVsZombies](https://github.com/ErLinErYi/PlantsVsZombies)  ![GitHub Repo stars](https://img.shields.io/github/stars/ErLinErYi/PlantsVsZombies?style=social)

本项目是基于 Cocos2d-X v3.16 引擎和 C++ 开发的植物大战僵尸模仿游戏。包含 104 个关卡、14 种僵尸和 13 种植物，动画由 Dragonbones/Spine 制作。相比原版，游戏拥有更高分辨率、画质、刷新率，难度稍高且含新模式。项目仅供学习交流，禁止商业用途。运行需 Visual Studio 2017/2019 编译，资源文件需另行获取。
