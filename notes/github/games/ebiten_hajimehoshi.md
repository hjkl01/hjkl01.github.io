### [hajimehoshi ebiten](https://github.com/hajimehoshi/ebiten)  ![GitHub Repo stars](https://img.shields.io/github/stars/hajimehoshi/ebiten?style=social)

Ebitengine（前身为 Ebiten）是一款基于 Go 语言的开源 2D 游戏引擎。它提供简洁的 API，允许开发者快速开发并跨平台部署 2D 游戏。核心功能涵盖 2D 图形渲染（支持几何变换、着色器、文本渲染、离屏渲染等）、输入控制（支持鼠标、键盘、手柄、触摸）以及音频播放。支持的平台包括 Windows、macOS、Linux、Android、iOS、WebAssembly 和 Nintendo Switch 等。
