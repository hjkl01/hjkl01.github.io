### [game1024 OpenSpeedy](https://github.com/game1024/OpenSpeedy)  ![GitHub Repo stars](https://img.shields.io/github/stars/game1024/OpenSpeedy?style=social)

OpenSpeedy 是一款开源免费的 Windows 游戏加速工具，用于突破游戏帧率限制并提供流畅的加速体验。其核心功能包括自定义加速倍率、兼容多种游戏引擎、低系统资源占用，同时支持 x86 和 x64 进程。该技术通过 Hook 系统时间函数（如 Sleep、GetTickCount 等）在 Ring3 级别实现加速，不修改系统内核。使用方法简单，启动后选择目标游戏进程即可调整速度。需注意该工具仅供学习研究，在线游戏中使用可能导致账号封禁或游戏物理引擎异常。
