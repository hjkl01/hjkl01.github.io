### [azerothcore azerothcore-wotlk](https://github.com/azerothcore/azerothcore-wotlk)  ![GitHub Repo stars](https://img.shields.io/github/stars/azerothcore/azerothcore-wotlk?style=social)

AzerothCore 是一个基于 C++ 的开源游戏服务器框架，专为托管《魔兽世界》（World of Warcraft）3.3.5a 版本的 MMORPG 设计。其核心目标是提供高稳定性、高度还原原版（Blizzlike）的游戏体验，并通过模块化架构支持深度自定义。该项目由社区驱动，既可作为搭建私人服务器的基础平台，也可作为开发者学习游戏服务器架构、C++ 及 SQL 的技术资源。
