### [argosopentech argos-translate](https://github.com/argosopentech/argos-translate)  ![GitHub Repo stars](https://img.shields.io/github/stars/argosopentech/argos-translate?style=social)

Argos Translate 是一款基于 Python 的开源离线翻译库，底层采用 OpenNMT 技术。它支持作为 Python 库、命令行工具或图形界面应用使用，核心功能包括安装语言模型包，并能在无直接翻译模型的语言对之间通过中间语言自动转换。该项目是 LibreTranslate 的基础，支持多种语言，具备 GPU 加速能力，并提供 HTML 及文件翻译扩展库支持。
