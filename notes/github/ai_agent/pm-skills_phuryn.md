### [pm-skills](https://github.com/phuryn/pm-skills) ![GitHub Repo stars](https://img.shields.io/github/stars/phuryn/pm-skills?style=social)

PM Skills Marketplace：AI产品管理操作系统。

**核心功能：** 68个PM技能 + 42个工作流，9个插件，覆盖产品全生命周期。

**主要插件：**

1. **产品发现** — 构思、假设识别与测试、实验设计、用户访谈、机会解决方案树
2. **产品策略** — 愿景、商业模式画布、价值主张、定价、竞争分析(SWOT/PESTLE/波特五力)
3. **产品执行** — PRD、OKR、路线图、冲刺计划、回顾、发布说明、干系人管理、优先级框架
4. **市场调研** — 用户画像、分群、旅程图、市场规模(TAM/SAM/SOM)、竞品分析、反馈情感分析
5. **数据分析** — 自然语言生成SQL、队列分析、A/B测试分析
6. **市场进入** — 海滩头细分、ICP、增长飞轮、GTM策略、竞品战斗卡
7. **营销增长** — 营销想法、定位、价值主张语句、产品命名、北极星指标
8. **工具包** — PM简历审查、NDA起草、隐私政策、语法检查
9. **AI代码交付** — AI构建应用的文档化、安全审计、性能审计、测试覆盖映射

**集成方式：** 支持Claude Code、Claude Cowork、Codex CLI、其他AI助手(Gemini CLI/OpenCode/Cursor/Kiro)

**使用方式：** `/命令名` 触发工作流，技能自动加载，命令可串联形成完整PM工作流。
