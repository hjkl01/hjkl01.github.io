### [yikart AiToEarn](https://github.com/yikart/AiToEarn)  ![GitHub Repo stars](https://img.shields.io/github/stars/yikart/AiToEarn?style=social)

AiToEarn 是一款开源的全栈 AI 驱动内容增长与变现平台，旨在帮助创作者、品牌及企业实现内容的创建、发布、互动与盈利。

核心功能：
1. **多平台分发**：支持一键发布至抖音、小红书、微信、TikTok、YouTube、Instagram 等全球主流平台，具备内容同步、日历调度及历史内容重编辑功能。
2. **AI 自动化运营**：内置 All In Agent 助手，辅助内容创作与发布，支持线下业务（如餐饮、零售）转化为线上推广任务。
3. **数据洞察与用户挖掘**：提供热点趋势发现、品牌监控、评论关键词搜索（精准挖掘高转化用户）及跨平台数据分析仪表盘。
4. **AI 内容生成**：支持自动生成文案、标题、图片、视频及标签，兼容多种主流 AI 模型（如 Seedance、Kling 等），提供智能改写与翻译。
5. **内容变现**：计划上线内容市场，支持创作者内容交易与品牌采购。

部署方式：支持 Docker 一键部署（含前后端及数据库服务），提供 Electron 桌面客户端及 MCP 服务集成。
