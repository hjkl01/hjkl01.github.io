### [google-agentic-commerce AP2](https://github.com/google-agentic-commerce/AP2)  ![GitHub Repo stars](https://img.shields.io/github/stars/google-agentic-commerce/AP2?style=social)

该项目是 Agent Payments Protocol (AP2) 的代码示例与演示仓库，旨在展示代理支付协议的核心组件。它包含 Python 和 Android 环境下的多种场景代码，演示代理间的支付交互流程。项目支持 Google API Key 或 Vertex AI 认证，并提供核心协议类型定义供直接安装。通过运行脚本可快速启动场景，允许开发者使用任意工具构建代理应用。
