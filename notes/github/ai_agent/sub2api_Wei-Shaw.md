### [Wei-Shaw sub2api](https://github.com/Wei-Shaw/sub2api)  ![GitHub Repo stars](https://img.shields.io/github/stars/Wei-Shaw/sub2api?style=social)

Sub2API 是一个 AI API 网关平台，用于管理和分发 AI 产品订阅的 API 配额。用户可通过平台生成的 API Key 访问上游 AI 服务，平台负责身份验证、计费、负载均衡和请求转发。主要功能包括多账户管理、API Key 分发、精确计费、智能调度、并发控制、限流设置和管理员仪表盘。技术栈包括 Go、Vue、PostgreSQL 和 Redis，支持脚本安装、Docker 部署和源码构建。
