### [jlowin fastmcp](https://github.com/jlowin/fastmcp)  ![GitHub Repo stars](https://img.shields.io/github/stars/jlowin/fastmcp?style=social)

FastMCP 是一个基于 Model Context Protocol (MCP) 的 Python 框架，用于连接大语言模型（LLM）与工具和数据，支持从原型到生产的全流程开发。

核心功能包括：
1. **Servers**：自动将 Python 函数转换为 MCP 工具、资源和提示，处理 Schema、验证及文档。
2. **Apps**：提供在对话中直接渲染的工具交互式 UI。
3. **Clients**：连接任何本地或远程 MCP 服务器，管理传输、认证及协议生命周期。

框架内置最佳实践，自动化处理技术细节，使开发者专注于业务逻辑。
