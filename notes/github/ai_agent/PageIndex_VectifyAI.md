### [VectifyAI PageIndex](https://github.com/VectifyAI/PageIndex)  ![GitHub Repo stars](https://img.shields.io/github/stars/VectifyAI/PageIndex?style=social)

PageIndex 是一个无向量数据库、基于推理的 RAG 系统。它构建文档层次化树结构索引，利用 LLM 进行推理检索以模拟人类专家导航，无需传统向量搜索或文本分块。系统支持 PDF 图像（无需 OCR）及 Markdown 输入，具备高可解释性和追踪性，适用于金融、法律等长篇专业文档分析。提供本地部署、云端平台及 API/MCP 集成，在 FinanceBench 基准测试中达到 98.7% 准确率。
