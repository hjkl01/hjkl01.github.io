### [agent-toolkit-for-aws](https://github.com/aws/agent-toolkit-for-aws) ![GitHub Repo stars](https://img.shields.io/github/stars/aws/agent-toolkit-for-aws?style=social)

AWS Agent Toolkit 旨在为 AI 编程代理（如 Claude Code、Codex、Cursor 和 Kiro）提供工具、知识和安全护栏，以辅助其在 AWS 上构建、部署和管理应用程序。该项目通过插件和技能包的形式，涵盖核心 AWS 服务操作、AI 代理开发、数据分析及 DevSecOps 等场景，并利用 AWS MCP Server 提供全面的 API 访问、沙箱脚本执行、实时文档检索及企业级审计监控功能。
