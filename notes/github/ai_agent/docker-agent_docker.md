### [docker docker-agent](https://github.com/docker/docker-agent)  ![GitHub Repo stars](https://img.shields.io/github/stars/docker/docker-agent?style=social)

Docker Agent 是一个 Docker CLI 插件，用声明式 YAML 配置创建、运行和分享 AI 代理，无需编码；支持多智能体协作、丰富工具生态与 MCP 服务器、跨多家模型提供商、内置思考/待办/记忆工具、可插拔 RAG 检索，并可将代理打包推送到 OCI 仓库、随处拉取运行。
