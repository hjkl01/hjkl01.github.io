### [ArchiveBox ArchiveBox](https://github.com/ArchiveBox/ArchiveBox)  ![GitHub Repo stars](https://img.shields.io/github/stars/ArchiveBox/ArchiveBox?style=social)

ArchiveBox 是一款开源的自托管网络存档工具，旨在永久保存互联网内容。它支持从多种来源（URL、浏览器书签、历史记录、RSS、浏览器扩展等）导入网页，并将其转换为标准且耐久的格式（HTML、PDF、PNG、JSON、TXT、WARC 等）。项目提供命令行（CLI）、Web 管理界面、REST API 及 Python API，支持定时任务、标签管理和日志查看。所有数据以本地文件（SQLite、普通文件）形式存储，确保长期可读且不依赖软件本身即可访问。适用于个人备份、法律证据保存、研究资料归档及组织数据管理，支持 Docker、pip 等多种安装部署方式。
