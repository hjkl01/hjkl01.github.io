### [AzuraCast AzuraCast](https://github.com/AzuraCast/AzuraCast)  ![GitHub Repo stars](https://img.shields.io/github/stars/AzuraCast/AzuraCast?style=social)

AzuraCast 是一个自托管、一站式的网络电台管理套件。它提供简易安装程序和直观强大的 Web 界面，用户可在几分钟内搭建功能完整的网络电台。该系统适用于各种规模和类型的电台，可在廉价 VPS 主机上运行，电台的所有方面均通过 Web 界面进行便捷管理。项目为基于开源软件构建的免费软件，采用 AGPL-3.0 许可证，目前处于 Beta 测试阶段。
