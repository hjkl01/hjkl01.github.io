### [seerr-team seerr](https://github.com/seerr-team/seerr)  ![GitHub Repo stars](https://img.shields.io/github/stars/seerr-team/seerr?style=social)

Seerr 是一款免费开源的媒体库请求管理软件，支持 Jellyfin、Plex 和 Emby 媒体服务器，并可集成 Sonarr 和 Radarr 服务。核心功能包括用户认证与权限管理、数据库支持（PostgreSQL/SQLite）、媒体库扫描、自定义请求系统（支持电影和剧集）、请求审批管理、通知推送、移动端适配以及媒体订阅和黑名单功能。项目支持从 Overseerr 或 Jellyseerr 迁移，并提供 API 文档。
