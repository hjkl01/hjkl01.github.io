### [ihmily StreamCap](https://github.com/ihmily/StreamCap)  ![GitHub Repo stars](https://img.shields.io/github/stars/ihmily/StreamCap?style=social)

StreamCap 是一个基于 FFmpeg 和 StreamGet 的多平台直播流录制客户端，覆盖 40+ 国内外主流直播平台。支持 Windows、MacOS、Web 及 Docker 部署。核心功能包括批量录制、循环监控（开播即录）、定时任务、多种输出格式支持（如 ts、flv、mp4 等）、录制完成后自动转码以及直播状态消息推送。
