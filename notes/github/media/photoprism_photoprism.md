### [photoprism photoprism](https://github.com/photoprism/photoprism)  ![GitHub Repo stars](https://img.shields.io/github/stars/photoprism/photoprism?style=social)

PhotoPrism 是一款基于人工智能的照片管理应用，支持在本地、私有服务器或云环境中运行。其核心功能包括自动标记和搜索照片、支持多种图片和视频格式、提供跨平台的类原生应用体验、智能人脸识别、地图浏览、元数据提取、WebDAV 同步等。PhotoPrism 注重隐私保护，承诺不会出售用户数据，并提供完整的文档和社区支持。用户可通过多种方式安装和部署，包括 Docker 和 tar.gz 包。项目通过会员支持持续开发，支持者可获得额外功能和专属服务。
