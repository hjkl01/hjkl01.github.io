### [openscreen](https://github.com/siddharthvaddem/openscreen) ![GitHub Repo stars](https://img.shields.io/github/stars/siddharthvaddem/openscreen?style=social)

OpenScreen 是一个免费开源的 Screen Studio 简化替代品，用于制作产品演示与操作讲解视频。支持录制全屏或窗口、麦克风与系统音频；自动/手动缩放并可调持续时间与位置；裁剪与截取片段、分段变速；添加文字/箭头/图片标注；设置壁纸/纯色/渐变/自定义背景；平移缩放带运动模糊；导出不同画幅与分辨率。适用于个人与商业用途，可自由修改与分发。
