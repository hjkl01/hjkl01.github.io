### [mltframework shotcut](https://github.com/mltframework/shotcut)  ![GitHub Repo stars](https://img.shields.io/github/stars/mltframework/shotcut?style=social)

Shotcut 是一款免费、开源且跨平台的视频编辑软件。它基于 MLT 多媒体创作框架，集成 Qt 6、FFmpeg、Frei0r 等核心库，支持多种音视频格式的编辑与处理。项目支持 Windows、Linux 和 macOS 系统，提供官方预编译安装包及源码构建方式，采用 GPLv3 许可协议，并支持通过社区平台进行翻译。
