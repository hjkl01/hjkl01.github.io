### [MoonTechLab LunaTV](https://github.com/MoonTechLab/LunaTV)  ![GitHub Repo stars](https://img.shields.io/github/stars/MoonTechLab/LunaTV?style=social)

MoonTV 是一个基于 Next.js 14、Tailwind CSS 和 TypeScript 构建的跨平台影视聚合播放器。核心功能包括多源聚合搜索、流畅在线播放（集成 HLS.js & ArtPlayer）、收藏与播放记录多端同步（支持 Kvrocks/Redis/Upstash）、PWA 离线缓存及响应式布局。项目采用容器化部署，部署后为空壳架构，需用户自行配置资源站点接口与分类，无内置播放源。仅供个人学习使用，建议设置密码保护并遵守当地法律法规。
