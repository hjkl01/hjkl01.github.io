### [Free-TV IPTV](https://github.com/Free-TV/IPTV)  ![GitHub Repo stars](https://img.shields.io/github/stars/Free-TV/IPTV?style=social)

该项目提供全球免费电视频道的 M3U 播放列表，收录通过地面广播及互联网服务（如 Pluto TV、Plex TV）免费提供的官方频道。用户可用 IPTV 播放器引用链接观看。项目坚持质量优先，仅收录高清、无付费的主流公共频道，排除成人、宗教及政治内容。播放列表由脚本生成，用户需通过 Pull Request 修改 .md 源文件来贡献或更新频道。
