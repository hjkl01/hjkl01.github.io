### [5rahim seanime](https://github.com/5rahim/seanime)  ![GitHub Repo stars](https://img.shields.io/github/stars/5rahim/seanime?style=social)

Seanime 是一款跨平台媒体服务器，提供 Web 界面及 Windows、Linux、macOS 桌面客户端，用于管理本地动漫与漫画库、流式播放动漫及阅读漫画。

核心功能包括：
1. **库与内容管理**：支持本地文件快速扫描、AniList 集成、自定义来源及内置漫画阅读器。
2. **播放与流媒体**：内置播放器支持字幕、转码及外部设备联动；支持 Torrent 搜索、下载、流式播放及在线源观看。
3. **自动化与扩展**：提供扩展市场安装插件，支持自动下载新剧集、播放列表管理。
4. **体验优化**：支持 UI 自定义、离线模式、新番追踪及 Discord 状态同步。

项目声明不托管任何媒体内容，用户需自行负责资源合法性。
