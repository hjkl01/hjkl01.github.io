### [YouROK TorrServer](https://github.com/YouROK/TorrServer)  ![GitHub Repo stars](https://img.shields.io/github/stars/YouROK/TorrServer?style=social)

TorrServer 是一款用于在线流媒体播放种子文件的工具，无需预先下载即可通过 HTTP 协议缓存和传输数据。支持 Windows、Linux、macOS 及 Docker 等多平台部署，提供现代化 Web 界面和 API 集成，支持跨设备访问及本地/远程服务器模式。功能扩展包括 DLNA 服务器、WebDAV、FUSE 挂载、身份验证、IP 黑白名单及代理设置等。
