### [Recordly](https://github.com/webadderallorg/Recordly) ![GitHub Repo stars](https://img.shields.io/github/stars/webadderallorg/Recordly?style=social)

Recordly 是一款开源跨平台桌面应用（macOS / Windows / Linux），用于快速完成**屏幕录制 + 演示视频编辑 + 导出**的一体化工作流，面向产品演示、操作讲解和教程视频制作。

核心功能包括：录制整屏或窗口、采集麦克风与系统音频、时间线拖拽编辑（裁剪、缩放、变速、标注、附加音轨、裁切）、自动缩放建议与光标美化（平滑、模糊、点击动效等）、动态摄像头气泡叠加、背景与画框样式设计（壁纸/纯色/渐变/圆角/阴影/模糊/边距）、导出 MP4/GIF（可调质量、帧率、尺寸、循环）以及 `.recordly` 项目文件保存与继续编辑。

项目还提供社区扩展系统与 Marketplace，可安装扩展增强能力（如点击音效、设备边框、浏览器样机、壁纸、渲染钩子等）。整体定位是让用户无需额外动效设计工具，也能在一个免费应用中产出精致演示视频。
