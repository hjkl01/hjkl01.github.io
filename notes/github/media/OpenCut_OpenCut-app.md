### [OpenCut](https://github.com/OpenCut-app/OpenCut) ![GitHub Repo stars](https://img.shields.io/github/stars/OpenCut-app/OpenCut?style=social)

OpenCut 是一款免费开源的视频编辑器，支持 Web、桌面和移动端。该项目目前正基于 Rust 核心进行重写，旨在实现跨平台统一代码库、插件优先架构、AI 代理支持的 MCP 服务器、无头模式（自动化/批量渲染）以及内置脚本功能。当前稳定版本仍为经典版（opencut-classic），新架构版本正在开发中。
