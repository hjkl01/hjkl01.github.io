### [pure-admin vue-pure-admin](https://github.com/pure-admin/vue-pure-admin)  ![GitHub Repo stars](https://img.shields.io/github/stars/pure-admin/vue-pure-admin?style=social)

vue-pure-admin 是一款基于 Vue3、Vite、Element-Plus、TypeScript、Pinia 和 Tailwindcss 开发的开源免费中后台管理系统模版。项目遵循 ESM 规范，支持国际化、暗色风格及移动端适配。提供精简版、Tauri 版和 Electron 版，并配套 CLI 脚手架与 Docker 支持，适合实际项目快速开发。
