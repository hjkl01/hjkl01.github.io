### [evanw esbuild](https://github.com/evanw/esbuild)  ![GitHub Repo stars](https://img.shields.io/github/stars/evanw/esbuild?style=social)

esbuild 是一款极速的 JavaScript 打包工具，旨在大幅提升构建性能。其核心功能包括：无需缓存即可实现极致构建速度；内置支持 JavaScript、CSS、TypeScript 和 JSX；支持 ESM 和 CommonJS 模块打包；具备代码树摇、压缩、源码映射及 CSS Modules 功能；提供简洁的 CLI、JS 和 Go API；支持本地服务器、监听模式和插件。
