### [nvm-sh nvm](https://github.com/nvm-sh/nvm)  ![GitHub Repo stars](https://img.shields.io/github/stars/nvm-sh/nvm?style=social)

nvm 是 Node.js 的版本管理工具，允许用户通过命令行快速安装、切换和管理不同版本的 Node.js。它支持在 POSIX 兼容 Shell（如 bash、zsh、fish）上运行，适用于 Unix、macOS 和 Windows WSL 环境。主要功能包括：安装指定版本、设置默认版本、查看版本列表、根据 .nvmrc 文件自动匹配项目所需版本、迁移全局 npm 包以及安装 LTS 版本。此外，nvm 还支持 Docker 环境部署、自定义镜像源及 Bash 自动补全。
