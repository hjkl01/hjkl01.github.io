### [rolldown rolldown](https://github.com/rolldown/rolldown)  ![GitHub Repo stars](https://img.shields.io/github/stars/rolldown/rolldown?style=social)

Rolldown 是一个由 Rust 编写的 JavaScript/TypeScript 打包器，旨在成为 Vite 的下一代打包解决方案。它兼容 Rollup 的 API 和插件接口，同时在功能范围上更接近 esbuild。项目目前处于发布候选（RC）阶段，内置压缩功能尚在 Alpha 状态，由 VoidZero Inc. 开发维护。
