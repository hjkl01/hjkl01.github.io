### [yakami129 VirtualWife](https://github.com/yakami129/VirtualWife)  ![GitHub Repo stars](https://img.shields.io/github/stars/yakami129/VirtualWife?style=social)

VirtualWife 是一个虚拟数字人项目，旨在打造拥有“灵魂”的数字伴侣，作为恋爱导师、心理咨询师融入人类生活以解决情感需求。项目支持一键 Docker 跨平台（Linux/Windows/MacOS）部署，提供 Web 配置界面。核心功能包括：自定义角色设定与 VRM 模型更换、文字驱动表情与动作、长短期记忆功能、多 LLM 模型切换（支持私有化模型）、中文语音对话（支持 Edge 与 Bert-VITS2 切换）、B 站直播支持以及流式传输加速响应。
