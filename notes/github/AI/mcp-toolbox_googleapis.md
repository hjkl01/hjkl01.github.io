### [mcp-toolbox](https://github.com/googleapis/mcp-toolbox) ![GitHub Repo stars](https://img.shields.io/github/stars/googleapis/mcp-toolbox?style=social)

MCP Toolbox for Databases 是开源的 MCP 服务器与工具框架，连接 AI 代理、IDE 和应用到企业数据库。它提供可直接使用的预置数据库工具（如列出表、执行 SQL），支持多种数据库；同时允许通过配置文件安全构建自定义工具、结构化查询、语义搜索与 NL2SQL。具备连接池、集成认证与 OpenTelemetry 监控，并提供多语言 SDK 便于集成到应用与各类 AI 框架。还支持工具测试 UI、遥测导出和将工具集生成可分发的 Agent Skills。
