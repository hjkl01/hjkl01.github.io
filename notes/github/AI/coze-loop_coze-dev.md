### [coze-dev coze-loop](https://github.com/coze-dev/coze-loop)  ![GitHub Repo stars](https://img.shields.io/github/stars/coze-dev/coze-loop?style=social)

Coze Loop 是面向开发者的 AI Agent 全生命周期管理与运营平台级解决方案。它提供从开发、调试、评估到监控的一站式支持，核心功能涵盖：Prompt 开发（可视化调试、版本管理、多模型对比）、自动化评估（多维测试输出质量）、全程可观测性（记录执行全流程及异常）以及多模型集成（支持 OpenAI、火山引擎等）。项目采用开源模式，支持 Docker 和 Kubernetes 部署，旨在帮助开发者高效降低 AI Agent 构建与运营的技术门槛。
