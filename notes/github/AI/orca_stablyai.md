### [orca](https://github.com/stablyai/orca) ![GitHub Repo stars](https://img.shields.io/github/stars/stablyai/orca?style=social)

Orca 是一款跨平台（macOS / Windows / Linux）的 AI 编程协同工作台，用于在多个仓库中并行编排和管理各类 CLI AI 代理。它以 Git worktree 为核心，让每个功能分支独立运行，支持多代理并行终端（标签页/分屏）、内置代码审查与提交、GitHub（PR/Issue/Actions）联动、SSH 远程运行、通知提醒等能力。

项目特点包括：无需登录（可直接使用自有 Claude Code/Codex/Grok 等订阅）、支持几乎所有 CLI 代理、可拖拽文件给代理、可注释 AI 生成差异、支持账号切换与用量追踪、原生搜索、丰富仓库内容预览，以及 Orca CLI 自动化能力；并提供移动端配套应用（iOS/Android）以便在手机上控制代理。
