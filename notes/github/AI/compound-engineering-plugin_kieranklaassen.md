### [kieranklaassen compound-engineering-plugin](https://github.com/kieranklaassen/compound-engineering-plugin)  ![GitHub Repo stars](https://img.shields.io/github/stars/kieranklaassen/compound-engineering-plugin?style=social)

该项目是一个 AI 编程助手插件市场，核心提供“工程复合”（Compound Engineering）插件。主要功能如下：

1.  **多平台支持**：支持 Claude Code、Cursor 及 OpenCode、Codex、Gemini、Copilot、Kiro、Windsurf 等多种开发工具。提供 CLI 工具，可一键将插件转换为不同平台格式并安装。
2.  **标准化工作流**：提供计划（plan）、执行（work）、审查（review）、沉淀（compound）四个核心命令。通过高强度的规划与审查，将经验沉淀为可复用的知识库，实现技术债的递减，使后续开发更顺畅。
3.  **配置同步**：支持将用户的个人技能、命令及 MCP 服务器配置从 Claude Code 自动同步至其他支持的 AI 工具，保持跨工具体验一致。

核心理念是每一轮工程工作都应降低下一轮工作的难度，而非增加复杂性。
