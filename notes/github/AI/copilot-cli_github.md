### [github copilot-cli](https://github.com/github/copilot-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/github/copilot-cli?style=social)

GitHub Copilot CLI 是一款将 AI 编程助手直接集成到终端的命令行工具，核心功能总结如下：

*   **终端原生开发**：允许用户在命令行中通过自然语言对话直接与 AI 协作，进行代码构建、编辑、调试和重构。
*   **GitHub 深度集成**：支持使用自然语言访问 GitHub 仓库、Issues 和 Pull Requests，无需上下文切换。
*   **智能代理能力**：具备规划并执行复杂任务的 AI 代理功能，可充当代码智能协作者。
*   **MCP 扩展支持**：默认搭载 GitHub MCP 服务器，并支持通过自定义 MCP 服务器扩展功能。
*   **操作可控性**：所有操作在执行前均可预览，需用户明确确认。
*   **LSP 增强功能**：支持配置语言服务器（LSP），提供代码跳转、悬停提示和诊断等智能辅助。
*   **跨平台支持**：兼容 Linux、macOS 和 Windows 系统，支持多种安装方式（如 Homebrew、npm、Winget 等）。
*   **实验性功能**：支持启用实验模式（如自动飞行 Autopilot 模式）以访问开发中的新特性。
