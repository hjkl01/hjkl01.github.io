### [NevaMind-AI memU](https://github.com/NevaMind-AI/memU)  ![GitHub Repo stars](https://img.shields.io/github/stars/NevaMind-AI/memU?style=social)

memU 是一个专为 24/7 主动式 AI 智能体设计的开源内存框架，旨在降低长期运行 LLM 的成本并实现智能体的持续进化。

**核心功能：**
- **主动记忆**：7x24 小时后台运行，持续捕捉用户意图，无需指令即可预测需求并主动行动。
- **文件系统化存储**：将记忆组织为分类、条目和链接，支持结构化导航、持久化存储和跨文件引用。
- **意图捕捉**：自动理解并记住跨会话的用户目标、偏好及上下文。
- **成本效率**：通过缓存洞察和避免冗余调用，显著减少 Token 消耗。
- **双模式检索**：支持快速 RAG 上下文加载和深度 LLM 意图推理，兼顾实时建议与复杂预测。
- **多模态与灵活集成**：支持文本、图片、文档等多种数据，兼容 OpenAI、OpenRouter 等多种模型提供商。
- **灵活部署**：支持通过 pip 自托管或云端 API 服务。

适用于信息推荐、邮件管理、交易监控及自我改进智能体等生产场景。
