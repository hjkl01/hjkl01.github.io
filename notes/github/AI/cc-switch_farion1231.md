### [farion1231 cc-switch](https://github.com/farion1231/cc-switch)  ![GitHub Repo stars](https://img.shields.io/github/stars/farion1231/cc-switch?style=social)

CC Switch 是一款基于 Tauri 2 构建的跨平台桌面管理器，用于统一管理 Claude Code、Codex、Gemini CLI、OpenCode 和 OpenClaw 五大 AI 命令行工具。

核心功能包括：
1. **配置管理**：内置 50+ 预设 Provider，提供可视化界面一键导入和切换 API，无需手动编辑配置文件。
2. **统一协作**：跨应用统一管理 MCP 服务器、Prompts 和 Skills，支持双向同步与一键安装。
3. **快捷操作**：支持系统托盘快速切换 Provider，无需打开主界面；提供本地代理与故障转移功能。
4. **数据同步**：支持 Dropbox、OneDrive 等多平台云同步，本地数据基于 SQLite 存储并自动备份。
5. **辅助工具**：提供使用量仪表盘、成本追踪、会话历史管理及会话恢复功能。
6. **跨平台支持**：原生支持 Windows、macOS 和 Linux 系统。
