### [linshenkx prompt-optimizer](https://github.com/linshenkx/prompt-optimizer)  ![GitHub Repo stars](https://img.shields.io/github/stars/linshenkx/prompt-optimizer?style=social)

Prompt Optimizer 是一款强大的 AI 提示词优化工具，旨在帮助用户编写更高质量的提示词以提升 AI 输出效果。

核心功能：
- 智能优化：支持一键优化、多轮迭代，区分系统提示词与用户提示词优化。
- 多模型集成：兼容 OpenAI、Gemini、DeepSeek 等主流大模型，支持文生图（T2I）和图生图（I2I）。
- 测试调试：提供原始与优化提示词实时对比、上下文变量管理、多轮会话测试及工具调用。
- 多端部署：支持 Web 应用、桌面应用、Chrome 插件和 Docker 部署，兼容 MCP 协议。
- 安全隐私：纯客户端处理，数据不经过中间服务器，支持访问密码保护。
