### [HKUDS DeepCode](https://github.com/HKUDS/DeepCode)  ![GitHub Repo stars](https://img.shields.io/github/stars/HKUDS/DeepCode?style=social)

DeepCode 是一个基于多智能体系统的开源代码生成平台，核心功能包括：

1.  **自动化代码实现**：支持将研究论文转换为可执行代码（Paper2Code），以及将自然语言描述转化为前端或后端代码（Text2Web/Text2Backend）。
2.  **多智能体架构**：由中央协调代理与意图理解、文档解析、代码规划、代码生成等专用代理协同工作，实现复杂开发任务的自主编排。
3.  **多平台交互**：提供命令行（CLI）、现代化 Web 界面、Docker 容器化部署及飞书聊天机器人（nanobot）集成。
4.  **高级技术特性**：集成 CodeRAG 系统、基于 MCP 协议的工具链（搜索、文件系统、代码执行等）以及自动化测试与文档生成。
5.  **卓越性能**：在 PaperBench 基准测试中超越人类专家和主流商业代码代理，显著提升科研复现与开发效率。
