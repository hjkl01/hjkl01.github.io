### [Canner WrenAI](https://github.com/Canner/WrenAI)  ![GitHub Repo stars](https://img.shields.io/github/stars/Canner/WrenAI?style=social)

Wren AI 是一款开源生成式商业智能（GenBI）智能体，支持通过自然语言查询各类数据库并生成商业洞察。主要功能包括：

1. **自然语言查询**：支持多语言提问，自动生成精确 SQL 语句及回答，降低技术门槛。
2. **智能分析**：AI 自动撰写数据总结、生成图表及业务报告。
3. **语义层架构**：通过 MDL 模型编码架构与指标，确保大模型输出准确且可控。
4. **API 嵌入**：提供 API 接口，支持将查询与图表生成功能嵌入自定义应用。

项目兼容 PostgreSQL、BigQuery、Snowflake 等多种数据源，支持 OpenAI、DeepSeek、Ollama 等主流大语言模型，提供本地部署与云端服务两种使用方式。
