### [context-hub](https://github.com/andrewyng/context-hub) ![GitHub Repo stars](https://img.shields.io/github/stars/andrewyng/context-hub?style=social)

Context Hub 是面向编码智能体的文档中枢，提供可搜索、版本化、按语言区分的精选 API/技能文档。通过命令行搜索和获取文档，支持增量获取特定文件或完整内容；可为文档添加本地注释并跨会话保留，自动在下次获取时呈现；支持对文档进行投票反馈，帮助维护者改进内容，从而让智能体随着任务持续变聪明。
