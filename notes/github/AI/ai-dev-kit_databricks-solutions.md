### [databricks-solutions ai-dev-kit](https://github.com/databricks-solutions/ai-dev-kit)  ![GitHub Repo stars](https://img.shields.io/github/stars/databricks-solutions/ai-dev-kit?style=social)

Databricks AI Dev Kit 是一个用于增强 Databricks 平台 AI 驱动开发（vibe coding）的工具套件。它通过向 AI 编程助手（如 Claude Code、Cursor、Windsurf 等）提供可信数据源，帮助其更快、更智能地构建应用。

核心功能与组件包括：
1. **开发支持**：支持开发 Spark 声明式管道、Databricks 作业、AI/BI 仪表盘、Unity Catalog、知识助手、MLflow 实验、模型服务及全栈 Web 应用。
2. **工具组件**：包含核心 Python 库、MCP 服务器（提供 50+ 工具）、最佳实践技能文档及可视化 Web 构建器。
3. **快速集成**：支持一键安装至现有项目，兼容多种 AI 编码环境，无需复杂配置即可开始开发。
