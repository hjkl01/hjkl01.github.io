### [humanlayer humanlayer](https://github.com/humanlayer/humanlayer)  ![GitHub Repo stars](https://img.shields.io/github/stars/humanlayer/humanlayer?style=social)

CodeLayer 是一款基于 Claude Code 的开源 IDE，用于编排 AI 编程代理。它专为在大型复杂代码库中解决难题设计，核心功能包括键盘优先的高效工作流、高级上下文工程（支持团队规模化开发）以及多会话并行运行（支持多工作树和远程云节点）。该项目支持从个人笔记本到整个团队的 AI 开发扩展。
