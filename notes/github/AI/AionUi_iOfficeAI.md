### [iOfficeAI AionUi](https://github.com/iOfficeAI/AionUi)  ![GitHub Repo stars](https://img.shields.io/github/stars/iOfficeAI/AionUi?style=social)

AionUi 是一款免费开源的跨平台 AI 代理协作（Cowork）应用，允许 AI 代理在本地计算机上协同工作。

核心功能包括：
1. **内置 AI 代理**：零配置安装，支持文件读写、代码编写、网页浏览、图像生成及任务自动化。
2. **多模型兼容**：支持 20+ AI 平台（Gemini、Claude、OpenAI 等），可使用任意 API Key，同时自动整合本地 CLI 代理（如 Claude Code）。
3. **远程控制与自动化**：提供 WebUI 及 Telegram、钉钉、飞书接口，支持 Cron 定时任务实现 24/7 无人值守运行。
4. **办公辅助**：集成文件智能管理、Excel 数据分析、文档（PPT/Word/PDF/Markdown）生成及 12+ 专业助手。
5. **隐私与平台**：数据本地 SQLite 存储，支持 macOS、Windows 和 Linux 系统，完全开源。
