### [upstash context7](https://github.com/upstash/context7)  ![GitHub Repo stars](https://img.shields.io/github/stars/upstash/context7?style=social)

Context7 MCP 是一款由 Upstash 提供的 MCP 服务器工具，旨在为 AI 编程助手（LLM）提供最新、版本特定的库文档和代码示例。它解决了 LLM 因训练数据过时产生的 API 幻觉和代码陈旧问题。用户通过简单指令或规则设置，即可将实时官方文档注入模型上下文，辅助生成准确的代码和配置。支持 Cursor、Claude Code、Opencode 等多种开发客户端，可通过 NPM 安装，支持 API Key 及 OAuth 认证，并提供库解析与文档查询工具。
