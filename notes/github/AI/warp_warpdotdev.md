### [warp](https://github.com/warpdotdev/warp) ![GitHub Repo stars](https://img.shields.io/github/stars/warpdotdev/warp?style=social)

Warp 是一个从终端发展而来的“智能代理式开发环境”（Agentic Development Environment），支持使用内置编码代理，也支持接入第三方 CLI 代理（如 Claude Code、Codex、Gemini CLI 等）。项目开源了 Warp 客户端代码，欢迎社区贡献，并提供从提 Issue 到 PR 的协作流程与本地构建脚本（bootstrap/run/presubmit）。

项目配套了智能代理协作平台与工作流：可在 build.warp.dev 查看大量 Oz 代理自动执行问题分拣、规格撰写、代码实现与 PR 审查，并跟踪贡献者和功能进展。针对开源维护者，Warp 提供 Oz for OSS 合作计划，将同类自动化工作流用于议题管理、PR 审核、社区运营和贡献者协同。

许可证方面：`warpui_core` 和 `warpui` 使用 MIT 许可证，仓库其余代码使用 AGPL v3。OpenAI 是该新版开源仓库的创始赞助方，相关代理管理工作流由 GPT 模型驱动。
