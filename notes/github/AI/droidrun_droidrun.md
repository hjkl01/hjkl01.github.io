### [droidrun droidrun](https://github.com/droidrun/droidrun)  ![GitHub Repo stars](https://img.shields.io/github/stars/droidrun/droidrun?style=social)

DroidRun 是一个基于大语言模型（LLM）智能体控制 Android 和 iOS 设备的开源框架。它允许通过自然语言命令自动化手机交互，支持复杂多步骤任务规划。核心功能包括兼容多种 LLM 提供商（如 OpenAI、Anthropic 等）、提供命令行工具与可扩展 Python API、通过截图分析实现设备视觉理解以及执行追踪。适用于移动端自动化 UI 测试、重复任务处理、远程协助及用户工作流引导等场景。
