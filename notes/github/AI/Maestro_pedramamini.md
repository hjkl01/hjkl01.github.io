### [pedramamini Maestro](https://github.com/pedramamini/Maestro)  ![GitHub Repo stars](https://img.shields.io/github/stars/pedramamini/Maestro?style=social)

Maestro 是一款跨平台桌面应用，专为高效开发者设计，用于编排和管理多个 AI 代理及项目。

**核心功能：**

1.  **多代理并行管理**：支持 Claude Code、OpenAI Codex、OpenCode 等多种 AI 工具，允许并行运行无限量代理和终端会话，每个会话拥有独立工作区和上下文。
2.  **自动化任务执行**：通过 Auto Run 和 Playbooks 功能，基于文件系统自动批处理 Markdown 任务清单，支持循环运行和会话隔离，实现无人值守运行。
3.  **高效协作与集成**：支持多代理群聊协同讨论、Git Worktrees 实现并行隔离开发、内置 Web 服务器支持手机远程监控与控制。
4.  **键盘优先交互**：全键盘控制设计，支持快速操作、会话切换及专注模式管理，旨在提升工作流效率。
5.  **数据分析与追踪**：提供实时使用量仪表盘、文档知识图谱可视化、Token 成本追踪及历史会话导入与恢复。
6.  **命令行与扩展**：提供 CLI 工具用于无头操作和脚本集成，兼容现有 AI 工具的配置权限与 MCP 工具。
