### [vanna-ai vanna](https://github.com/vanna-ai/vanna)  ![GitHub Repo stars](https://img.shields.io/github/stars/vanna-ai/vanna?style=social)

Vanna 2.0 是一款将自然语言查询转化为 SQL 数据洞察的工具。核心功能包括：

1. **NL2SQL 生成**：自动将自然语言问题转换为 SQL，返回数据表、可视化图表及自然语言总结。
2. **企业级安全**：全链路用户感知权限，支持行级安全过滤、审计日志、速率限制及自定义身份认证。
3. **现代 Web 界面**：提供预构建的 `<vanna-chat>` 组件，支持流式实时响应（表格、图表、进度更新）。
4. **灵活集成**：兼容任意 LLM、数据库及认证系统，可无缝嵌入 FastAPI、React、Vue 等现有架构。
5. **生产级扩展**：支持自定义工具、生命周期钩子、LLM 中间件及内置可观测性追踪。
