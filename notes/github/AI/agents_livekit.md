### [livekit agents](https://github.com/livekit/agents)  ![GitHub Repo stars](https://img.shields.io/github/stars/livekit/agents?style=social)

LiveKit Agents 是一个 Python 框架，用于构建运行在服务器端的实时可编程参与者，旨在创建可听、可见、能理解的多模态语音代理。它支持灵活集成 STT、LLM、TTS 及实时 API 模型，提供内置任务调度、WebRTC 客户端支持、电话集成、语义停顿检测、MCP 工具支持及原生测试框架。项目完全开源，支持在自建服务器上运行包括 LiveKit 服务器在内的完整技术栈，并提供丰富的示例与开发、测试、生产等多种运行模式。
