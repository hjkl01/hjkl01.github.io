### [badlogic pi-mono](https://github.com/badlogic/pi-mono)  ![GitHub Repo stars](https://img.shields.io/github/stars/badlogic/pi-mono?style=social)

该项目是一个用于构建 AI 代理和管理大语言模型（LLM）部署的工具集（Pi Monorepo）。核心功能包括：提供统一的跨供应商 LLM API、支持工具调用与状态管理的代理运行时、交互式编程 CLI 代理、Slack 消息转发机器人、带差异渲染的终端 UI 库、AI 聊天 Web 组件，以及用于管理 GPU 上 vLLM 部署的命令行工具。
