### [cc-switch-cli](https://github.com/SaladDay/cc-switch-cli) ![GitHub Repo stars](https://img.shields.io/github/stars/SaladDay/cc-switch-cli?style=social)

CC-Switch CLI 是一个命令行管理工具，用于统一管理 Claude Code、Codex、Gemini、OpenCode、OpenClaw 的配置与生态。支持提供商切换与测速/健康检查、MCP 服务器管理与同步、系统提示词管理、技能库安装与启用、配置备份/导入导出与 WebDAV 同步、本地代理路由管理，以及环境变量/本地工具检查；提供交互式 TUI 与多语言支持。
