### [cheahjs free-llm-api-resources](https://github.com/cheahjs/free-llm-api-resources)  ![GitHub Repo stars](https://img.shields.io/github/stars/cheahjs/free-llm-api-resources?style=social)

本项目整理了一份提供免费或试用额度访问大语言模型（LLM）API 服务的资源列表。内容涵盖完全免费的服务商（如 OpenRouter、Google AI Studio、Groq 等）和提供试用信用额的平台（如 Fireworks、阿里云国际等）。列表详细记录了各平台支持的模型名称、使用限制（请求次数、Token 上限）及访问要求，旨在帮助用户低成本或免费地接入 LLM 服务。
