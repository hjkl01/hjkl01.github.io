### [simular-ai Agent-S](https://github.com/simular-ai/Agent-S)  ![GitHub Repo stars](https://img.shields.io/github/stars/simular-ai/Agent-S?style=social)

Agent S 是一款开源 GUI 代理框架，旨在让 AI 像人类一样自主操作电脑。其最新版本 Agent S3 在 OSWorld 基准测试中以 72.60% 的成绩首次超越人类水平，支持 Windows、macOS 和 Linux 系统。该框架提供 CLI 和 SDK 接口，兼容多种大语言模型及定位模型，并具备本地代码执行能力（支持 Python 和 Bash），允许 AI 结合图形界面操作与编程指令，自主完成数据处理、文件管理及系统自动化等复杂任务。
