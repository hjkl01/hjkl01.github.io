### [oh-my-claudecode](https://github.com/Yeachan-Heo/oh-my-claudecode) ![GitHub Repo stars](https://img.shields.io/github/stars/Yeachan-Heo/oh-my-claudecode?style=social)

**总结多智能体编排工具**

该项目支持Claude Code、多团队模式和自动驾驶，具备零配置、自然语言操作和任务并行化，使用tmux管理Codex、Gemini和Claude工作进程，支持技能学习、HUD显示和通知功能。

oh-my-claudecode 是一款为 Claude Code 提供零配置、多智能体编排的工具，通过自然语言即可驱动团队式流水线执行、自动并行化与持续验证修复；支持 tmux 启动 Claude/Codex/Gemini CLI 工作者协同；提供自动化模式（autopilot/ralph/ulw 等）、智能模型路由、19 类专用代理、技能学习与自动注入、HUD 状态监控、会话回放与成本分析，并集成通知与 OpenClaw 事件回调等运维功能。
