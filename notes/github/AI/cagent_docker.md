### [docker cagent](https://github.com/docker/cagent)  ![GitHub Repo stars](https://img.shields.io/github/stars/docker/cagent?style=social)

Docker Agent 是一款 Docker CLI 插件，允许无需编写代码即可通过声明式 YAML 配置创建、运行和共享智能 AI 代理。采用多代理架构，支持专业化团队自动协作与任务委派。核心功能包括丰富的工具生态（支持 MCP 服务器）、多 AI 模型兼容、高级推理与记忆工具、检索增强生成（RAG），以及将代理封装至 OCI 注册表进行分发。
