### [agentfield](https://github.com/Agent-Field/agentfield) ![GitHub Repo stars](https://img.shields.io/github/stars/Agent-Field/agentfield?style=social)

AgentField 是一个开源 AI 后端控制平面，让用 Python/Go/TypeScript 编写的智能体像 API 一样被调用并自动暴露为 REST 接口，提供路由与跨智能体调用、结构化 LLM 输出、共享内存、异步执行与人审暂停、可观测性与版本灰度发布等生产基础设施，并内置加密身份、访问策略和可验证审计凭证，支持部署与管理大规模、可追踪的 AI 代理系统。
