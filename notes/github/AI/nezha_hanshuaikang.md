### [nezha](https://github.com/hanshuaikang/nezha) ![GitHub Repo stars](https://img.shields.io/github/stars/hanshuaikang/nezha?style=social)

Nezha 是一款以 Agent 为核心的轻量桌面 IDE（约 7MB），面向并行 AI 编程场景，支持在多个项目中同时运行 Claude Code 和 Codex，并将任务管理、终端、代码编辑与 Git 工作流整合到同一界面。

核心功能包括：多项目工作区与一键任务切换、实时终端交互、会话自动发现与提醒、任务全生命周期可视化与会话回放恢复、内置轻量代码/Markdown 编辑器（含语法高亮与文件树）、原生 Git 集成（Diff、日志、分支管理、AI 生成提交信息），以及 Token 与工具调用的使用统计分析。  
其目标是减少在终端、编辑器、Git 客户端和日志之间频繁切换，帮助用户在多任务并行开发中更高效地追踪与协作。
