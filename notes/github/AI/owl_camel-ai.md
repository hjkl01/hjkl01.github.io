### [camel-ai owl](https://github.com/camel-ai/owl)  ![GitHub Repo stars](https://img.shields.io/github/stars/camel-ai/owl?style=social)

OWL（Optimized Workforce Learning for General Multi-Agent Assistance in Real-World Task Automation）是一个基于 CAMEL-AI 框架的前沿多智能体协作框架，旨在实现现实世界任务的自动化。该框架在 GAIA 基准测试中取得开源框架排名第一的成绩，平均分为 69.09。

核心功能包括：
1. **任务自动化**：支持在线搜索（多个引擎）、多模态处理（视频、图像、音频）、浏览器自动化（Playwright）、文档解析（PDF/Office）及代码执行。
2. **丰富工具集**：内置 Model Context Protocol (MCP) 及数十种专用工具包（如搜索、数学、GitHub、地图等），支持自定义工具配置。
3. **模型兼容**：支持多种大语言模型后端（OpenAI、Claude、Qwen 等），需具备工具调用或多模态理解能力。
4. **灵活部署**：提供本地安装、Docker 容器化及 Web 界面等多种交互与部署方式。
