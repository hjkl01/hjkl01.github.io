### [harry0703 MoneyPrinterTurbo](https://github.com/harry0703/MoneyPrinterTurbo)  ![GitHub Repo stars](https://img.shields.io/github/stars/harry0703/MoneyPrinterTurbo?style=social)

MoneyPrinterTurbo 是一个全自动短视频生成工具。用户只需提供视频主题或关键词，即可自动生成视频文案、素材、字幕、背景音乐及语音配音，并合成高清短视频。系统支持 Web 界面和 API 调用，兼容竖屏和横屏多种尺寸。支持批量生成、自定义文案、接入多种 AI 大模型及语音实时试听，支持使用高清无版权库或本地视频素材，兼容中英文文案。
