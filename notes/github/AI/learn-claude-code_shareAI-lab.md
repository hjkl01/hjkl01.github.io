### [shareAI-lab learn-claude-code](https://github.com/shareAI-lab/learn-claude-code)  ![GitHub Repo stars](https://img.shields.io/github/stars/shareAI-lab/learn-claude-code?style=social)

该项目是一个从零开始构建类似 Claude Code 的小型 AI 代码代理的学习项目，共包含 12 个循序渐进的课程。每个课程逐步添加一种机制，如工具使用、任务分解、上下文压缩、持久化存储、后台任务、团队协作、任务隔离等。项目提供 Python 示例代码、多语言文档和交互式学习平台，帮助开发者理解 AI 代理的核心运行模式和架构设计。
