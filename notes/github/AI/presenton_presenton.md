### [presenton](https://github.com/presenton/presenton) ![GitHub Repo stars](https://img.shields.io/github/stars/presenton/presenton?style=social)

Presenton 是一个开源的 AI 演示文稿生成平台（可替代 Gamma、Beautiful AI、Decktopus），支持本地自托管和桌面端使用，核心功能包括：

- 通过提示词或上传文档自动生成演示文稿，并提供 API 批量生成能力；
- 支持多种大模型与服务商（OpenAI、Gemini、Vertex、Azure OpenAI、Bedrock、Anthropic、Ollama、LM Studio 及 OpenAI 兼容接口等）；
- 支持多图像生成来源（如 DALL·E 3、Gemini Flash、Pexels、Pixabay、ComfyUI 等）；
- 支持自定义模板与主题（HTML + Tailwind CSS），并可从现有 PPT 生成模板；
- 导出为可完整编辑的 PPTX，以及 PDF；
- 支持本地运行、Docker 一键部署、GPU 加速、云端浏览器使用和团队 API 部署；
- 提供单实例管理员认证、可配置密钥与隐私控制，强调数据自主与无 SaaS 锁定；
- 项目基于 Apache 2.0 开源许可。
