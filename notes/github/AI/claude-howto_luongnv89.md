### [claude-howto](https://github.com/luongnv89/claude-howto) ![GitHub Repo stars](https://img.shields.io/github/stars/luongnv89/claude-howto?style=social)

**Summarizing project functions in Chinese**

项目涵盖Claude Code的全面功能指南，包括教程、模板和学习路径，支持斜杠命令、记忆、技能、子代理、MCP、钩子、插件、检查点等功能，另外提供高级功能和命令行界面（CLI）支持。

这是一个 Claude Code 学习与实践指南项目，提供结构化教程、可视化说明、复制即用的配置模板和分级学习路径，帮助用户掌握并组合使用斜杠命令、记忆、技能、子代理、MCP、钩子、插件、检查点、规划/思考模式与 CLI 等功能，构建自动化代码评审、文档生成、CI/CD、DevOps 等工作流。
