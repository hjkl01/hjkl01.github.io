### [rf-detr](https://github.com/roboflow/rf-detr) ![GitHub Repo stars](https://img.shields.io/github/stars/roboflow/rf-detr?style=social)

RF-DETR 是由 Roboflow 开发的一款基于 DINOv2 视觉 Transformer 骨干网络的实时目标检测架构。它支持统一 API 下的物体检测、实例分割和关键点检测（预览版），在 Microsoft COCO 和 RF100-VL 数据集上实现了准确率与延迟的最佳平衡。项目提供多种模型尺寸（从 Nano 到 2XLarge），包含开源的 Apache 2.0 许可证模型及需额外授权的 Plus 组件，并支持通过 Google Colab 或 Roboflow 平台进行模型微调训练。
