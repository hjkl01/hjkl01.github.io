### [agent-of-empires](https://github.com/njbrake/agent-of-empires) ![GitHub Repo stars](https://img.shields.io/github/stars/njbrake/agent-of-empires?style=social)

Agent of Empires 是基于 tmux 的终端会话管理器，用于在 Linux/macOS 上并行运行多个 AI 编码代理。支持多种 AI 工具，每个代理独立会话，可在不同 Git 分支（worktree）上工作，提供 TUI/CLI 管理、代理与终端切换、状态监控、差异查看与文件编辑、每仓库配置、工作区档案，以及可选 Docker 沙箱隔离。会话可持久运行，关闭界面不影响代理。
