### [fengshao1227 ccg-workflow](https://github.com/fengshao1227/ccg-workflow)  ![GitHub Repo stars](https://img.shields.io/github/stars/fengshao1227/ccg-workflow?style=social)

CCG 是一个由 Claude Code 统一编排的多模型协作开发系统，自动将前端任务路由给 Gemini、后端任务路由给 Codex，模型仅返回补丁由 Claude 审核应用，提升安全性与效率。它提供完整的多阶段开发流程与 27+ 斜杠命令，涵盖规划、执行、分析、测试、优化、代码审查与 Git 操作，并集成 OPSX 进行规格驱动开发与约束式计划，支持并行团队协作和项目上下文管理。
