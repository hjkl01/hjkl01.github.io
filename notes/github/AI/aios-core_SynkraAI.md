### [SynkraAI aios-core](https://github.com/SynkraAI/aios-core)  ![GitHub Repo stars](https://img.shields.io/github/stars/SynkraAI/aios-core?style=social)

Synkra AIOX 是一个 AI 驱动的通用智能体框架，专注于自动化软件开发及多领域任务处理。核心功能总结如下：

1. **CLI 优先架构**：以命令行为核心，全面兼容 Claude、Cursor、Codex、Gemini 等主流 IDE 与 CLI 工具，实现自动化集成与环境配置。
2. **多智能体协作**：内置 11 个专用智能体（涵盖分析、规划、架构、开发、测试等角色），采用“智能规划 + 上下文工程化开发”的两阶段模式，消除上下文丢失与规划不一致。
3. **自主开发引擎 (ADE)**：提供规范管道、执行引擎、QA 审查、自动恢复及记忆层等组件，支持从需求到代码的自主执行、审查与优化。
4. **Squads 扩展能力**：支持创建自定义智能体团队，可灵活扩展至创意写作、商业策略、教育等非软件技术领域。
5. **企业级支持**：提供 AIOX Pro 模块，具备持久化记忆、商业分析、企业工具集成（Jira、Notion 等）及分层配置管理能力。
