### [skillshare](https://github.com/runkids/skillshare) ![GitHub Repo stars](https://img.shields.io/github/stars/runkids/skillshare?style=social)

skillshare 是用于统一管理并同步各类 AI CLI 工具技能（skills）及规则、命令、提示词等文件资源的工具，提供单一来源目录，通过命令同步到 Claude、Codex 等 50+ 目标；支持从各类 Git 仓库安装与更新，内置安全审计防提示注入与数据外泄，支持项目/组织级共享与过滤控制，跨平台单一二进制离线使用，并提供 CLI 与 Web 仪表盘管理。
