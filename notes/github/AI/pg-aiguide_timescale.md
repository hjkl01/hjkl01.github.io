### [timescale pg-aiguide](https://github.com/timescale/pg-aiguide)  ![GitHub Repo stars](https://img.shields.io/github/stars/timescale/pg-aiguide?style=social)

pg-aiguide 是专为 AI 编码助手优化的 PostgreSQL 专家工具。它提供基于版本的官方手册语义搜索、AI 优化的最佳实践技能库（涵盖架构设计、索引策略、性能调优等）及扩展生态文档支持。支持作为公共 MCP 服务器或 Claude Code 插件使用，旨在通过注入深度 PostgreSQL 知识与验证模式，帮助 AI 生成更健壮、高性能且符合现代规范的数据库代码。
