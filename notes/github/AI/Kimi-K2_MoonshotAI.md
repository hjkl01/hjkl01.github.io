### [MoonshotAI Kimi-K2](https://github.com/MoonshotAI/Kimi-K2)  ![GitHub Repo stars](https://img.shields.io/github/stars/MoonshotAI/Kimi-K2?style=social)

Kimi K2 是月之暗面（Moonshot AI）推出的最先进混合专家（MoE）语言模型，总参数 1 万亿，激活参数 320 亿，上下文长度 128K。模型基于 Muon 优化器训练，在知识、推理、编码及智能体能力（工具调用、自主解决问题）方面表现卓越。提供 Base（基础模型）和 Instruct（对话指令版）两个变体，代码与数学任务性能领先行业。支持 API 调用及本地部署（兼容 vLLM、SGLang 等框架），模型权重开源，采用 Modified MIT 协议。
