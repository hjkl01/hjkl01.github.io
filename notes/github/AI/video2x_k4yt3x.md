### [k4yt3x video2x](https://github.com/k4yt3x/video2x)  ![GitHub Repo stars](https://img.shields.io/github/stars/k4yt3x/video2x?style=social)

Video2X 是一个基于机器学习的视频超分辨率（放大）和帧插值框架。项目采用 C/C++ 完全重写，支持 Windows 和 Linux 跨平台运行，拥有更高效的架构。核心功能包括视频滤波放大和帧率插值，支持 Anime4K v4、Real-ESRGAN、Real-CUGAN 和 RIFE 等多种模型，利用 ncnn 和 Vulkan 技术加速。新版本为 Windows 提供了图形用户界面和安装程序，处理过程中无需占用额外磁盘空间。
