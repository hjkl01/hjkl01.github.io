### [anthropics skills](https://github.com/anthropics/skills)  ![GitHub Repo stars](https://img.shields.io/github/stars/anthropics/skills?style=social)

这是一个 Anthropic 提供的 Claude 技能（Skills）官方实现仓库。项目核心功能是通过动态加载包含指令、脚本和资源的技能文件夹，使 Claude 能够以可重复的方式完成特定任务（如创意创作、技术开发、企业工作流等）。仓库包含技能示例、Agent Skills 规范、创建模板及部分文档处理源码，支持在 Claude Code、Claude.ai 及 API 中部署使用或自定义开发。内容主要用于演示和教育目的，供开发者参考和学习，具体实现行为可能因环境而异。
