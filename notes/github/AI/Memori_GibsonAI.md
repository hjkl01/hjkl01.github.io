### [GibsonAI Memori](https://github.com/GibsonAI/Memori)  ![GitHub Repo stars](https://img.shields.io/github/stars/GibsonAI/Memori?style=social)

Memori 是企业 AI 的记忆层基础设施，可无缝集成至现有软件架构。它兼容任意大语言模型、数据存储及开发框架，能自动持久化对话并在后台自动召回信息。核心功能包括实体与进程的归属标记、会话管理及高级记忆增强（如属性、偏好、关系等）。提供 TypeScript 和 Python SDK，支持多种主流大模型与框架。支持零配置快速接入，并通过 CLI 管理账户与配额。
