### [langchain-ai deepagents](https://github.com/langchain-ai/deepagents)  ![GitHub Repo stars](https://img.shields.io/github/stars/langchain-ai/deepagents?style=social)

Deep Agents 是一个基于 LangGraph 构建的“开箱即用”智能体（Agent）框架，旨在简化智能体的开发与部署。

**核心功能：**
*   **内置能力**：包含任务规划、文件读写与搜索、沙箱化 Shell 执行、子智能体委托及上下文自动管理。
*   **快速启动**：提供 Python SDK 和命令行工具（CLI），无需手动配置提示词、工具和上下文即可运行，支持多种 LLM 模型。
*   **高度可定制**：支持自定义工具、模型替换、提示词调整，兼容 LangChain 生态、MCP 协议及多种远程沙箱环境（如 Modal、Daytona）。
*   **生产就绪**：基于 LangGraph 架构，支持流式传输、持久化存储、检查点（Checkpointing）及人机协作流程。

项目采用 MIT 协议开源。
