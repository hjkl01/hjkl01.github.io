### [shareAI-lab Kode-cli](https://github.com/shareAI-lab/Kode-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/shareAI-lab/Kode-cli?style=social)

Kode 是一款运行在终端中的 AI 编程助手，核心功能如下：

1. **代码与任务处理**：理解代码库结构，直接编辑文件，运行 Shell 命令，自动化处理开发工作流。
2. **多模型协作**：支持 20+ AI 模型，可灵活切换、组合不同模型优势（如架构设计、代码实现、专家咨询），支持模型配置共享。
3. **智能代理系统**：支持子代理任务委托（@run-agent）和专家模型咨询（@ask-model），实现任务专业化分工。
4. **文档标准支持**：原生支持 AGENTS.md 规范，可自动生成和维护项目文档，兼容 .claude 工作流。
5. **智能交互体验**：提供上下文感知、模糊匹配的模型/代理/文件/命令补全系统，优化 Unix 命令提示。
6. **扩展与配置**：支持 MCP 服务器、Agent Skills 插件系统、自定义输出风格及技能，支持多模型配置文件。
7. **安全与部署**：默认不发送遥测，提供安全模式（权限审批）和 Linux 系统沙箱，支持跨平台（含 Windows 原生）及 Docker 部署。
