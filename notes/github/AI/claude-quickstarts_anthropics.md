### [anthropics claude-quickstarts](https://github.com/anthropics/claude-quickstarts)  ![GitHub Repo stars](https://img.shields.io/github/stars/anthropics/claude-quickstarts?style=social)

Claude Quickstarts 是一套帮助开发者快速基于 Claude API 构建应用程序的模板集合，提供可定制的基础项目。核心功能包括：
- 客服助手：结合知识库提供 AI 辅助客户支持。
- 金融数据分析师：支持通过聊天进行财务数据分析与可视化。
- 电脑使用演示：展示控制桌面电脑（含缩放）的能力。
- 浏览器工具 API 演示：基于 Playwright 实现网页自动化交互。
- 自主编码代理：双代理模式通过 Git 持久化进度构建完整应用。

开发者克隆仓库、配置 API 密钥即可使用各示例构建应用。
