### [crewAIInc crewAI](https://github.com/crewAIInc/crewAI)  ![GitHub Repo stars](https://img.shields.io/github/stars/crewAIInc/crewAI?style=social)

CrewAI 是一个基于 Python 的轻量级、高性能多智能体自动化编排框架，完全独立于 LangChain 构建。其核心功能包括：

1. **Crews**：支持多个具有自主权的智能体进行角色协作和任务委派，实现自主决策和动态交互。
2. **Flows**：提供企业级生产环境所需的事件驱动工作流控制，支持状态管理和条件分支，可与 Crews 无缝结合。
3. **企业级支持**：通过 AMP 套件提供统一控制平面、实时可观测性、高级安全性及灵活的部署选项（云/本地）。
4. **灵活性与兼容性**：支持多种大语言模型（含本地模型），允许从工作流到内部提示词的深度定制，兼容外部工具与 API。
5. **生态与授权**：开源（MIT License），拥有活跃社区和丰富文档，隐私保护型 Telemetry 可选。
