### [ZJU-LLMs Agent-Kernel](https://github.com/ZJU-LLMs/Agent-Kernel)  ![GitHub Repo stars](https://img.shields.io/github/stars/ZJU-LLMs/Agent-Kernel?style=social)

Agent-Kernel 是一个基于大语言模型的多智能体系统开发框架，旨在支持大规模社会仿真以探索集体智能扩展规律。其核心功能包括：支持智能体在运行时的动态增删与无限扩展、提供仿真过程中的实时干预能力、验证智能体行为及模型输出的可信性、实现跨场景代码复用。框架采用模块化微内核架构，具备高适应性、可配置性、可靠性与可重用性。项目提供独立与分布式两种部署模式，配套可视化 Web 控制面板，已成功应用于“宇宙 25”实验及校园生活仿真等场景。
