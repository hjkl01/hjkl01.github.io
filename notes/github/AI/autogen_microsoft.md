### [microsoft autogen](https://github.com/microsoft/autogen)  ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/autogen?style=social)

AutoGen 是一个用于构建多智能体 AI 应用的框架，支持智能体自主运行或与人类协作。主要功能包括：

1. **分层架构**：提供 Core API、AgentChat API 和 Extensions API，设计灵活可扩展，支持 Python 和 .NET。
2. **多智能体编排**：支持创建专家智能体协作处理复杂任务，可集成 MCP 服务器（如网络浏览）及代码执行能力。
3. **无代码开发**：提供 AutoGen Studio 图形界面，无需编写代码即可原型设计和运行多智能体工作流。
4. **模型与工具集成**：支持多种 LLM 客户端（如 OpenAI、Azure OpenAI）及扩展功能。
5. **开发者工具**：包含 AutoGen Bench 用于评估智能体性能。
6. **社区生态**：提供完善的文档、教程、技术支持及活跃的社区交流渠道。
