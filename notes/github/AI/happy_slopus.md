### [slopus happy](https://github.com/slopus/happy)  ![GitHub Repo stars](https://img.shields.io/github/stars/slopus/happy?style=social)

Happy Coder 是一款开源工具，提供针对 Claude Code 和 Codex 的移动端、Web 端及命令行客户端。它允许用户通过 `happy` 命令包裹原有指令，在保持代码端到端加密的同时，实现桌面与移动设备间的无缝控制切换。主要功能包括：远程监控 AI 编程进度、接收权限或错误通知、支持跨设备即时接管，并提供 iOS、Android 和 Web 应用访问。项目由应用客户端、CLI 接口、远程代理及后端同步服务组成。
