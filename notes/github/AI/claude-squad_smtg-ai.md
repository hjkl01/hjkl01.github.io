### [claude-squad](https://github.com/smtg-ai/claude-squad) ![GitHub Repo stars](https://img.shields.io/github/stars/smtg-ai/claude-squad?style=social)

Claude Squad 是一款终端 TUI 工具，用于在独立工作区中管理多个 AI 编程代理（如 Claude Code、Codex、Gemini、Aider 等），通过 tmux 会话和 git worktree 为每个任务隔离分支，支持同时处理多任务、后台执行、自动接受提示、查看/提交/切换变更，并在一个终端中创建、切换、暂停、恢复与删除会话。
