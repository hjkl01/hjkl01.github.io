### [alibaba OpenSandbox](https://github.com/alibaba/OpenSandbox)  ![GitHub Repo stars](https://img.shields.io/github/stars/alibaba/OpenSandbox?style=social)

OpenSandbox 是一个通用的 AI 应用沙箱平台，提供多语言 SDK、统一的沙箱 API 和 Docker/Kubernetes 运行环境，支持代码代理、GUI 代理、代理评估、AI 代码执行和强化学习训练等场景。主要功能包括多语言 SDK、沙箱协议、Docker 和 Kubernetes 运行时、网络策略、安全隔离和丰富的沙箱环境配置。
