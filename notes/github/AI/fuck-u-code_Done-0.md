### [Done-0 fuck-u-code](https://github.com/Done-0/fuck-u-code)  ![GitHub Repo stars](https://img.shields.io/github/stars/Done-0/fuck-u-code?style=social)

fuck-u-code 是一款命令行代码质量分析工具，支持 Go、JavaScript、Python 等 14 种编程语言。它基于 AST 解析进行离线静态分析，提供 0~100 的整体评分和单文件质量指数，涵盖复杂度、大小、注释、错误处理、命名、重复性和结构等七项检测指标。工具支持集成 AI 大模型进行代码审查，提供终端、Markdown、JSON、HTML 等多种输出格式，支持多语言配置，并兼容 MCP 协议以与 AI 编辑器（如 Claude Code、Cursor）集成调用。
