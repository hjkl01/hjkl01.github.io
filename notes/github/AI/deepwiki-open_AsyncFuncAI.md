### [AsyncFuncAI deepwiki-open](https://github.com/AsyncFuncAI/deepwiki-open)  ![GitHub Repo stars](https://img.shields.io/github/stars/AsyncFuncAI/deepwiki-open?style=social)

DeepWiki-Open 是一款自动生成 GitHub、GitLab 或 BitBucket 仓库交互式 Wiki 的工具。它能分析代码结构、生成文档与可视化图表，支持私有仓库访问。项目集成 AI 能力，提供基于 RAG 的问答（Ask）和深度研究（DeepResearch）功能，兼容多种大模型（如 Gemini、OpenAI、Ollama）及嵌入模型。支持 Docker 部署及手动配置，允许自定义 API 密钥与模型选择。
