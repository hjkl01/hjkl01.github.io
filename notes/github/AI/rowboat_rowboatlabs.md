### [rowboatlabs rowboat](https://github.com/rowboatlabs/rowboat)  ![GitHub Repo stars](https://img.shields.io/github/stars/rowboatlabs/rowboat?style=social)

Rowboat 是一款开源的本地优先 AI 同事工具，旨在通过构建知识图谱辅助用户高效工作。其核心功能包括：

1. **知识管理与集成**：连接 Gmail、Granola、Fireflies 等工作数据，在本地构建基于 Markdown 的持久化知识图谱，支持可视化编辑与审查。
2. **智能内容生成**：基于上下文生成会议简报、邮件草稿、文档及 PDF 演示文稿；支持语音备忘录自动总结关键信息。
3. **自动化后台代理**：自动执行重复性任务，如起草邮件回复、生成每日语音备忘录及项目更新。
4. **模型与工具扩展**：支持本地模型（Ollama）或自有云 API，通过 MCP 协议集成搜索、CRM、Slack 等外部工具。
5. **隐私与本地化**：所有数据以纯文本 Markdown 形式本地存储，确保无云端锁定且用户完全可控。
