### [ostris ai-toolkit](https://github.com/ostris/ai-toolkit)  ![GitHub Repo stars](https://img.shields.io/github/stars/ostris/ai-toolkit?style=social)

AI Toolkit 是由 Ostris 开发的面向消费级硬件（Nvidia GPU）的一站式扩散模型训练套件，支持图像与视频模型训练。项目提供 Web 图形界面（GUI）和命令行（CLI）操作模式，兼顾易用性与功能完整性。

主要功能特性：
1. **多模型支持**：涵盖 FLUX.1 (dev/schnell)、SDXL、SD 1.5、Wan 2.1、Kontext 及 OmniGen2 等。
2. **训练能力**：支持 LoRA、LoKr 及特定层微调，数据集具备自动缩放与分桶功能，无需手动预处理。
3. **部署环境**：支持本地 Linux/Windows 安装，并提供 RunPod 和 Modal 云平台的官方模板与配置。
4. **安全管理**：Web UI 支持任务全生命周期管理，并可通过令牌认证保护服务安全。
