### [project-nomad](https://github.com/Crosstalk-Solutions/project-nomad) ![GitHub Repo stars](https://img.shields.io/github/stars/Crosstalk-Solutions/project-nomad?style=social)

Project N.O.M.A.D. 是一个离线优先的自包含知识与教育服务器，通过命令中心管理并用 Docker 编排多种工具，提供本地 AI 聊天与知识库检索、离线维基与资料库、Khan Academy 课程学习、离线地图、数据处理工具、笔记、系统基准测试及便捷内容管理与初始配置，浏览器访问即可使用，安装后可在无网环境运行。
