### [trycua cua](https://github.com/trycua/cua)  ![GitHub Repo stars](https://img.shields.io/github/stars/trycua/cua?style=social)

这是一个构建、评估和部署计算机智能体（AI Agents）的综合平台，核心功能模块包括：

1. **CuaBot**：为智能体提供沙盒电脑操作环境，支持原生窗口、屏幕流媒体、共享剪贴板及音频，内置 Web 及移动端（iOS/Android）自动化支持。
2. **Cua Agent**：智能体开发框架，使 AI 能自主识别屏幕元素、执行点击操作并运行代码，提供控制桌面环境的 SDK 及隔离的代码执行环境。
3. **Cua-Bench**：提供计算机智能体的基准测试与强化学习环境，支持多任务评估及训练轨迹导出。
4. **Lume**：基于 Apple Silicon 的虚拟化解决方案，用于高效创建和管理 macOS 与 Linux 虚拟机。

平台提供 CLI 工具以快速集成各类 AI 助手，并配套完整的开发文档与社区支持。
