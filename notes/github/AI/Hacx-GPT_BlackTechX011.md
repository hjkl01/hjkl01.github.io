### [BlackTechX011 Hacx-GPT](https://github.com/BlackTechX011/Hacx-GPT)  ![GitHub Repo stars](https://img.shields.io/github/stars/BlackTechX011/Hacx-GPT?style=social)

HacxGPT-CLI 是一个开源的跨平台命令行工具，旨在提供无审查的 AI 对话体验。

**核心功能：**
*   **多模型支持**：兼容 HacxGPT、Groq、OpenRouter 等多个 API 提供商，用户需自备密钥。
*   **去审查框架**：通过高级系统提示词减少模型限制，实现自由交互。
*   **本地 API 引擎**：自研高性能引擎，零第三方 SDK 依赖，保障速度与安全性。
*   **隐私安全**：密钥本地存储且绑定设备硬件，不经过第三方服务器中转。
*   **会话管理**：支持对话保存、加载、列表查看及重置。
*   **自动化更新**：内置更新系统，支持在线自动升级。
*   **跨平台兼容**：完美支持 Linux、Windows、macOS 及 Termux。

该工具完全免费（基于用户自有密钥），适用于编程辅助、逻辑推理及个人 AI 探索，不包含自有模型，仅提供连接接口。
