### [rtk-ai rtk](https://github.com/rtk-ai/rtk)  ![GitHub Repo stars](https://img.shields.io/github/stars/rtk-ai/rtk?style=social)

RTK 是一个高性能的命令行代理工具，通过过滤和压缩命令输出，减少发送给大语言模型（LLM）的 token 数量，从而节省 60-90% 的 token 消耗。它支持多种开发常用命令（如 Git、测试工具、构建工具、容器命令等），提供智能过滤、聚合、截断和去重等策略，实现输出内容的精简。RTK 提供自动命令重写功能，可透明集成到开发流程中，无需修改原有命令，即可实现 token 节省。支持跨平台安装（macOS、Linux、Windows），并提供 token 节省统计和分析功能，帮助用户了解节省效果。
