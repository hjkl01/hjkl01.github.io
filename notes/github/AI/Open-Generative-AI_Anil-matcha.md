### [Open-Generative-AI](https://github.com/Anil-matcha/Open-Generative-AI) ![GitHub Repo stars](https://img.shields.io/github/stars/Anil-matcha/Open-Generative-AI?style=social)

Open Generative AI 是一个免费、开源、可自托管、无内容审查的 AI 创作平台，主打替代 Higgsfield/Freepik/Krea/Openart 等封闭式产品。项目集成 200+ 图像与视频模型，支持文生图、图生图、文生视频、图生视频、音频驱动口型同步，并提供 Image / Video / Lip Sync / Cinema / Workflow 五大工作室。

核心能力包括：最多 14 张参考图的多图输入、9 种口型同步模型、专业电影镜头参数控制、可视化节点工作流编排、历史记录与素材复用。平台既可在线免安装使用，也提供 Windows/macOS/Linux 桌面端；桌面端支持本地推理（sd.cpp）和远程 GPU 推理（Wan2GP）。整体架构基于 Next.js + React + Tailwind 的 monorepo，使用 Muapi.ai 作为统一模型 API 网关，代码 MIT 开源，可自由二次开发与扩展。
