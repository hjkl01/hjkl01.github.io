### [agent-framework](https://github.com/microsoft/agent-framework) ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/agent-framework?style=social)

**总结项目功能**

该项目是微软代理框架，支持多语言（.NET/Python），用于构建、编排和部署AI代理，包含图形工作流、开发界面、监控、中间件和多供应商支持。

微软 Agent Framework 是一个支持 Python 与 .NET 的多语言框架，用于构建、编排和部署 AI 代理，支持从简单聊天代理到图形化多代理工作流，提供数据流编排、流式处理、检查点、人类参与、时间回溯、可观测性（OpenTelemetry）、多模型提供方接入、可插拔中间件，以及用于开发调试的 DevUI 和实验性功能包。
