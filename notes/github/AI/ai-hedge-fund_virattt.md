### [virattt ai-hedge-fund](https://github.com/virattt/ai-hedge-fund)  ![GitHub Repo stars](https://img.shields.io/github/stars/virattt/ai-hedge-fund?style=social)

这是一个用于教育和研究目的的 AI 驱动对冲基金概念验证系统。系统通过模拟多位著名投资者及估值、情绪、基本面、技术、风险管理等角色的 AI 智能体协同工作，分析市场数据并生成交易决策信号，但不进行真实交易。项目支持命令行 (CLI) 和 Web 界面，具备回测功能，需配置大模型及财务数据 API 密钥运行，旨在探索 AI 在投资决策中的应用。
