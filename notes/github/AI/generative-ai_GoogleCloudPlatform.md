### [GoogleCloudPlatform generative-ai](https://github.com/GoogleCloudPlatform/generative-ai)  ![GitHub Repo stars](https://img.shields.io/github/stars/GoogleCloudPlatform/generative-ai?style=social)

该项目是一个基于 Google Cloud 的生成式 AI 示例和工具集，主要功能包括：

1. **Gemini 模型**：提供入门笔记本、用例、函数调用和示例应用，用于探索 Gemini 3.1 Pro 模型。
2. **Vertex AI Search**：展示如何快速构建网站和企业数据搜索引擎。
3. **RAG 与 Grounding**：提供基于 Vertex AI 的检索增强生成（RAG）和 Grounding 的示例和教程。
4. **图像生成与处理**：通过 Vertex AI Imagen API 实现图像生成、编辑、视觉描述和视觉问答。
5. **语音处理**：使用 Vertex AI Chirp API 实现语音相关功能。
6. **环境搭建**：提供在 Google Colab 和 Vertex AI Workbench 上配置 Google Cloud 和 Vertex AI Python SDK 的指南。
7. **学习资源**：包含博客、YouTube 播客等关于 Google Cloud 生成式 AI 的学习资料。
8. **相关项目**：链接多个与生成式 AI 相关的开源项目，涵盖代理开发、营销、开发者生产力、MLOps 等多个领域。
