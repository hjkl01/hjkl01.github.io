### [microsoft magentic-ui](https://github.com/microsoft/magentic-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/magentic-ui?style=social)

Magentic-UI 是一个以人为本的人工智能代理研究原型，旨在自动化复杂的网页浏览和编码任务。其核心特点是透明可控的交互机制，支持执行前计划展示、实时操作指导及敏感操作人工审批。主要功能涵盖人机协作（共同规划与执行）、长时任务监控、MCP 服务器扩展、文件上传及多种大语言模型支持（OpenAI、Azure、Ollama、Fara-7B 等）。基于 AutoGen 构建，提供 Web 界面和命令行工具，支持 Docker 部署，适用于需要人类监督的自动化场景。
