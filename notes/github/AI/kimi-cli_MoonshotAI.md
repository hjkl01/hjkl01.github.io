### [MoonshotAI kimi-cli](https://github.com/MoonshotAI/kimi-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/MoonshotAI/kimi-cli?style=social)

Kimi Code CLI 是一款运行于终端的 AI 代理，辅助完成软件开发任务与终端操作。它具备读取编辑代码、执行 Shell 命令、搜索获取网页内容及自主规划调整动作的能力。

核心功能特性：
1. **Shell 命令模式**：支持在 CLI 内部通过快捷键直接运行 Shell 命令。
2. **编辑器集成**：支持 VS Code 扩展，并可通过 Agent Client Protocol (ACP) 与 Zed、JetBrains 等 IDE 无缝集成。
3. **Zsh 增强**：提供专用插件，将 AI 代理能力融入 Shell 体验。
4. **MCP 支持**：支持 Model Context Protocol，可通过命令行或配置文件管理 MCP 服务器。
5. **开发构建**：提供代码格式化、测试及 Python 包与二进制文件的构建工具。
