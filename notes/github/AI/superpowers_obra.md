### [obra superpowers](https://github.com/obra/superpowers)  ![GitHub Repo stars](https://img.shields.io/github/stars/obra/superpowers?style=social)

Superpowers 是一个面向编码代理的完整软件开发工作流系统，基于一组可自动触发的“技能”构建。它引导代理在编码前通过对话梳理需求、确认设计方案，并制定符合 TDD、YAGNI 原则的实施计划。系统核心功能包括子代理驱动开发、测试驱动开发（RED-GREEN-REFACTOR 循环）、系统化调试、自动化代码审查及 Git 分支管理。该工具兼容 Claude Code、Cursor、Codex 和 OpenCode 等平台，旨在通过标准化流程确保代码质量与开发效率。
