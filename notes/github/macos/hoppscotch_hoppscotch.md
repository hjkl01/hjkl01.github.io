### [hoppscotch hoppscotch](https://github.com/hoppscotch/hoppscotch)  ![GitHub Repo stars](https://img.shields.io/github/stars/hoppscotch/hoppscotch?style=social)

Hoppscotch 是一个开源的轻量级 API 开发与测试生态系统。
- **多协议支持**：支持 HTTP 全方法（GET/POST 等）及 WebSocket、Socket.IO、MQTT、GraphQL 等协议的请求交互。
- **请求与响应管理**：提供实时请求发送、响应预览（JSON/HTML 等）、代码片段生成及 cURL 导入。
- **数据组织与同步**：支持集合（Collections）、历史记录、环境变量管理，可实现云端与本地数据同步。
- **自动化与测试**：支持预请求/后请求脚本（JavaScript）编写及响应状态测试。
- **团队协作**：支持多工作区、团队权限管理及跨设备数据同步。
- **辅助功能**：内置代理模式解决 CORS 问题，支持 PWA 安装、主题定制、键盘快捷键及多语言国际化。
- **多端覆盖**：提供 Web 端、桌面应用（Windows/Linux/macOS）及浏览器扩展。
