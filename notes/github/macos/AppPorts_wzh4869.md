### [wzh4869 AppPorts](https://github.com/wzh4869/AppPorts)  ![GitHub Repo stars](https://img.shields.io/github/stars/wzh4869/AppPorts?style=social)

AppPorts 是专为 macOS 设计的应用迁移与链接工具，旨在将大型应用移至外部存储以节省本地空间。

**核心功能：**
1. **应用迁移**：一键将多 GB 应用（如 Xcode、Logic Pro、游戏）迁移至外部驱动器（SSD、NAS 等），本地仅保留目录元数据。
2. **无缝兼容**：采用优化的 Contents 链接策略，Finder 无异常图标，完美支持启动台及系统菜单。
3. **安全机制**：自动识别并锁定系统应用，迁移前检查运行状态，支持随时一键还原至本地。
4. **用户体验**：支持暗色模式、多语言（含中文）、无障碍访问（VoiceOver/Braille）及应用快速搜索。

**系统要求：**
- macOS 14.0 (Sonoma) 或更新版本。
- 首次运行需授予“完全磁盘访问”权限。
