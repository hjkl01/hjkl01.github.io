### [majd ipatool](https://github.com/majd/ipatool)  ![GitHub Repo stars](https://img.shields.io/github/stars/majd/ipatool?style=social)

IPATool 是一款跨平台命令行工具，支持在 App Store 上搜索 iOS 应用并下载应用包（ipa 文件）。项目需 Apple ID 认证，兼容 Windows、Linux 及 macOS 系统。主要功能涵盖账户管理（登录、登出、查看信息、撤销凭证）、应用搜索、获取应用授权、列出可用版本、下载加密 ipa 包以及获取版本元数据。工具支持非交互式模式运行，并提供文本或 JSON 格式的输出选项。
