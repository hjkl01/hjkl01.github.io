### [qjfoidnh BaiduPCS-Go](https://github.com/qjfoidnh/BaiduPCS-Go)  ![GitHub Repo stars](https://img.shields.io/github/stars/qjfoidnh/BaiduPCS-Go?style=social)

本项目是仿 Linux shell 风格的百度网盘命令行客户端，支持 Windows、macOS、Linux 及移动设备。主要功能如下：

1. 帐号管理：支持多百度账号登录、切换、注销及配额查询。
2. 文件操作：支持网盘目录切换、文件列表、搜索、创建、删除、拷贝、移动、重命名及元信息获取。
3. 传输下载：支持断点续传、并发下载、128G 大文件上传及直链获取。
4. 转存与分享：支持分享链接转存（含密码）、分享创建/取消及管理。
5. 离线下载：支持 http/https/ftp/电驴/磁力链等协议。
6. 其他：支持回收站管理、通配符匹配、Tab 自动补齐及配置项修改。

注：本软件不提供超出官方客户端的下载提速服务。
