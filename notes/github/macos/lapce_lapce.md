### [lapce lapce](https://github.com/lapce/lapce)  ![GitHub Repo stars](https://img.shields.io/github/stars/lapce/lapce?style=social)

Lapce 是一款使用纯 Rust 编写的高性能代码编辑器，UI 基于 Floem 构建，渲染采用 wgpu。主要功能包括：内置 LSP 支持（提供代码补全、诊断等智能特性）；支持可切换的模态编辑（类 Vim 模式）；内置远程开发功能；支持编译为 WASI 格式的插件开发（如 C、Rust）；内置终端可直接执行命令。支持 Windows、Linux 和 macOS 平台，采用 Apache License 2.0 开源协议。
