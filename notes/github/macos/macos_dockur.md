### [dockur macos](https://github.com/dockur/macos)  ![GitHub Repo stars](https://img.shields.io/github/stars/dockur/macos?style=social)

该项目旨在在 Docker 容器中运行 macOS 操作系统。核心功能包括 KVM 硬件加速、Web 端远程查看器及系统镜像自动下载。支持通过 Docker Compose、CLI、Kubernetes 或 GitHub Codespaces 部署。用户可自定义 macOS 版本、磁盘大小、CPU 核数及内存，支持网络配置（桥接、macvlan、DHCP）、硬件直通（磁盘/USB）及主机文件共享。通过浏览器连接端口即可完成系统安装与设置，仅限在 Apple 硬件上运行。
