### [rio](https://github.com/raphamorim/rio) ![GitHub Repo stars](https://img.shields.io/github/stars/raphamorim/rio?style=social)

Rio Terminal 是一个现代化终端，目标是在多平台环境中运行。它支持作为桌面应用运行于 macOS、Linux 和 Windows，并计划进一步移植到 Web（当前渲染组件已具备基础）。项目提供完整的安装、配置与更新日志文档，支持社区赞助与多发行渠道打包维护。项目基于 Rust 开发，最低稳定 Rust 版本要求为 1.92.0。
