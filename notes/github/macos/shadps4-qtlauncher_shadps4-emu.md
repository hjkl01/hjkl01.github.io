### [shadps4-emu shadps4-qtlauncher](https://github.com/shadps4-emu/shadps4-qtlauncher)  ![GitHub Repo stars](https://img.shields.io/github/stars/shadps4-emu/shadps4-qtlauncher?style=social)

QtLauncher 是 shadPS4 PS4 模拟器的官方跨平台启动器，支持 Windows、Linux 及 macOS 系统。它提供模拟器启动与管理界面，支持键盘鼠标及游戏手柄（如 Xbox、DualShock）输入映射，允许自定义按键绑定并按游戏保存。内置实用工具包括 FPS 计数器、调试信息查看、全屏切换及 RenderDoc 捕获，用户可直接下载预编译版本。
