### [gfx-rs wgpu](https://github.com/gfx-rs/wgpu)  ![GitHub Repo stars](https://img.shields.io/github/stars/gfx-rs/wgpu?style=social)

`wgpu` 是一个跨平台的、安全的纯 Rust 图形 API，基于 WebGPU 标准构建。它原生支持 Vulkan、Metal、D3D12 和 OpenGL，并在 Web (Wasm) 环境下支持 WebGL2 和 WebGPU。该库是 Firefox、Servo 和 Deno 中 WebGPU 集成的核心，支持 Windows、Linux、macOS、iOS、Android 及 Web 平台，并提供 `wgpu-native` 以支持 C 及其他语言的绑定。
