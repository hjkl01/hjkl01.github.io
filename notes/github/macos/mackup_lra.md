### [lra mackup](https://github.com/lra/mackup)  ![GitHub Repo stars](https://img.shields.io/github/stars/lra/mackup?style=social)

Mackup 是一个跨平台（macOS/Linux）的应用程序配置同步与备份工具。它支持将配置文件存储于 Dropbox、iCloud、Google Drive 或 Git 等云存储。提供复制模式（备份/恢复）和链接模式（软链接同步，新版 macOS 不推荐）两种机制。支持数百种常用应用及自定义文件的配置管理，方便用户在不同设备间快速迁移环境并保持设置一致。
