### [gfx-rs wgpu-native](https://github.com/gfx-rs/wgpu-native)  ![GitHub Repo stars](https://img.shields.io/github/stars/gfx-rs/wgpu-native?style=social)

wgpu-native 是一个基于 Rust 实现的原生 WebGPU 库，核心依赖 wgpu-core。项目主要功能如下：
1. 提供 C 语言 FFI 绑定，支持多种编程语言（如 Python、.NET、Go、Java 等）的封装调用。
2. 提供 macOS、Windows 和 Linux 平台的预编译二进制文件。
3. 包含基础的 C 语言渲染与计算示例，用于演示库的使用。
