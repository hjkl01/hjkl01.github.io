### [cmushroom redis-pro](https://github.com/cmushroom/redis-pro)  ![GitHub Repo stars](https://img.shields.io/github/stars/cmushroom/redis-pro?style=social)

Redis Pro 是一款基于 SwiftUI 开发的 macOS Redis 图形化管理工具，支持 Redis 3.x 至 6.x 版本。主要功能包括客户端列表与断开管理、慢日志查看、配置更新、SSH 登录、批量删除及收藏管理。支持深色模式，可通过 Homebrew 或安装包安装。
