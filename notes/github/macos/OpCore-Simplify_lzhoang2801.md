### [lzhoang2801 OpCore-Simplify](https://github.com/lzhoang2801/OpCore-Simplify)  ![GitHub Repo stars](https://img.shields.io/github/stars/lzhoang2801/OpCore-Simplify?style=social)

OpCore Simplify 是一款专为 Hackintosh 设计的工具，旨在通过自动化设置简化 OpenCore EFI 的创建。

主要功能：
1. **硬件与系统兼容**：支持广泛的 Intel 及 AMD 硬件，涵盖 macOS High Sierra 至 Tahoe 26。
2. **自动配置生成**：根据硬件自动检测并添加 ACPI 补丁和 kexts（集成 SSDTTime）。
3. **资源自动更新**：构建前自动从官方源更新 OpenCorePkg 和 kexts。
4. **EFI 深度定制**：提供 CPU/GPU ID 欺骗、拓扑重建、SIP 禁用、NVRAM 配置及电源优化选项。
5. **跨平台构建**：支持 Windows、macOS 和 Linux 环境，一键生成 EFI 并辅助制作 USB 安装器。
