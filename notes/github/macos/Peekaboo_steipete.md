### [steipete Peekaboo](https://github.com/steipete/Peekaboo)  ![GitHub Repo stars](https://img.shields.io/github/stars/steipete/Peekaboo?style=social)

Peekaboo 是一款面向 macOS 15+ 的屏幕视觉自动化工具。它结合高精度屏幕捕捉与 AI 分析能力，允许用户通过自然语言指令执行点击、输入、滚动、窗口管理及菜单交互等 GUI 操作。项目提供 CLI 命令行工具和 MCP 服务器（兼容 Claude/Cursor），支持 OpenAI、Anthropic、Grok、Gemini 及本地 Ollama 等多种 AI 模型，并支持配置可复现的自动化工作流。使用前需授予屏幕录制和辅助功能权限。
