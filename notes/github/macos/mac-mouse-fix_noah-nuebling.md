### [noah-nuebling mac-mouse-fix](https://github.com/noah-nuebling/mac-mouse-fix)  ![GitHub Repo stars](https://img.shields.io/github/stars/noah-nuebling/mac-mouse-fix?style=social)

Mac Mouse Fix 是一款 macOS 鼠标增强工具，旨在将普通鼠标体验提升至媲美苹果触控板的水平。

**核心功能：**
*   **手势与滚动**：提供流畅的手势支持和平滑滚动，实现类触控板的浏览体验。
*   **按键自定义**：支持将键盘功能（亮度、音量、媒体控制）及系统导航操作映射到鼠标按键。
*   **窗口管理**：通过“点击拖动”功能实现窗口的快速平铺、缩放或全屏调整。
*   **兼容性**：macOS 11+ 支持最新版本（v3），旧系统使用 v2；兼容大多数鼠标（部分依赖专有协议的鼠标受限）。
*   **许可与安装**：v3 版试用 30 天后收费，v2 及以下永久免费；支持官网下载及 Homebrew 安装。
