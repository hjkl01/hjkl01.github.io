### [tw93 Mole](https://github.com/tw93/Mole)  ![GitHub Repo stars](https://img.shields.io/github/stars/tw93/Mole?style=social)

Mole 是一款面向 macOS 的命令行系统清理与优化工具，集深度清理、磁盘分析及性能监控于一体。主要功能包括：清除缓存、日志及浏览器残留；智能卸载应用及隐藏残留文件；可视化分析磁盘大文件；实时监控硬件性能指标；优化系统服务并重建缓存；清理开发项目构建产物；查找并移除安装程序文件。支持 Homebrew 安装，提供安全预览模式，可配合 Raycast 或 Alfred 快速启动。
