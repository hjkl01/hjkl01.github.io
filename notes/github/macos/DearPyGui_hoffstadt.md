### [hoffstadt DearPyGui](https://github.com/hoffstadt/DearPyGui)  ![GitHub Repo stars](https://img.shields.io/github/stars/hoffstadt/DearPyGui?style=social)

Dear PyGui 是一个基于 C/C++ 和 Dear ImGui 构建的现代、快速且强大的 Python GUI 框架。它利用 GPU 渲染，支持 Windows、Linux、macOS 及 Raspberry Pi 跨平台。核心功能包括高性能数据图表绘制（支持每秒 60 帧显示超百万数据点）、内置节点编辑器、画布绘图 API（支持自定义绘图与 2D 游戏）、完整的主题样式控制及异步操作支持。项目内置演示程序与开发者调试工具，采用 MIT 许可证。
