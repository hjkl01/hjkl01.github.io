### [kubernetes ingress-nginx](https://github.com/kubernetes/ingress-nginx)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubernetes/ingress-nginx?style=social)

本项目是基于 NGINX 的 Kubernetes Ingress 控制器，提供反向代理和负载均衡功能。项目已启动退休计划，仅维持至 2026 年 3 月的尽力而为维护，此后将停止发布新版本、修复漏洞及安全更新。现有部署和制品将持续可用。不建议新用户部署，推荐迁移至 Gateway API。此外，该项目不适用于多租户 Kubernetes 生产环境。
