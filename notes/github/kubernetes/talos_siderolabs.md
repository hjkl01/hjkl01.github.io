### [siderolabs talos](https://github.com/siderolabs/talos)  ![GitHub Repo stars](https://img.shields.io/github/stars/siderolabs/talos?style=social)

Talos Linux 是一个专为 Kubernetes 设计的现代操作系统，具备安全、不可变和精简的特性。所有系统管理均通过 API 执行，无需 Shell 或交互式控制台。其核心优势在于：通过最小化和加固降低攻击面并启用 mTLS 认证；利用不可变基础设施消除配置漂移并实现原子更新；简化架构并确保 Kubernetes 与 Linux 版本保持最新。该项目完全开源，适用于生产环境。
