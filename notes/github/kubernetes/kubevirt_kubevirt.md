### [kubevirt kubevirt](https://github.com/kubevirt/kubevirt)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubevirt/kubevirt?style=social)

KubeVirt 是 Kubernetes 的虚拟机管理附加组件，旨在为 Kubernetes 平台提供虚拟化解决方案。它通过自定义资源定义（CRD）扩展 Kubernetes API，新增 VM 等资源类型，允许使用 Kubernetes API 管理虚拟机资源。项目通过在集群中运行额外的控制器和代理，实现声明式地创建、调度、启动、停止和删除虚拟机的全生命周期管理功能。
