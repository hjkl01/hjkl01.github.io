### [helm helm](https://github.com/helm/helm)  ![GitHub Repo stars](https://img.shields.io/github/stars/helm/helm?style=social)

Helm 是 Kubernetes 的包管理工具，功能上类似 apt/yum/homebrew，旨在简化应用的安装、管理和部署。它通过管理预配置 Kubernetes 资源的 Chart 包（包含配置文件和模板）来操作资源。核心功能包括：查找和安装第三方 Chart、分享自定义应用、创建可重复构建的 Kubernetes 应用、智能管理清单文件及发布版本。Helm 通过渲染模板与 Kubernetes API 通信，支持在本地、CI/CD 等多种环境运行。
