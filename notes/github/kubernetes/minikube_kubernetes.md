### [kubernetes minikube](https://github.com/kubernetes/minikube)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubernetes/minikube?style=social)

minikube 是一款在 macOS、Linux 和 Windows 系统上部署本地 Kubernetes 集群的工具，旨在成为本地 Kubernetes 应用开发的最佳解决方案。

主要功能包括：
1. **核心功能**：支持运行最新稳定版 Kubernetes 及标准特性（如 LoadBalancer、持久化存储、Dashboard、多集群管理、容器运行时配置）。
2. **开发者特性**：提供 Addons 插件市场、GPU 支持（NVIDIA/AMD）、文件系统挂载及常用 CI 环境支持。
3. **运行方式**：支持命令行管理，同时兼容 GitHub Codespace 在线运行和本地 Dev Container。
