### [mherrmann helium](https://github.com/mherrmann/helium)  ![GitHub Repo stars](https://img.shields.io/github/stars/mherrmann/helium?style=social)

Helium 是一个基于 Python 的轻量级浏览器自动化库，支持 Chrome 和 Firefox。它底层封装 Selenium，但提供高级 API，允许直接通过页面可见文本标签定位元素，无需使用 ID、XPath 或 CSS 选择器，使脚本更短、易读且稳定。Helium 简化了 iFrames 嵌套、窗口管理及等待机制等繁琐操作，并支持与 Selenium 代码混合使用。安装仅需 Python 3 环境，通过 pip 即可部署。
