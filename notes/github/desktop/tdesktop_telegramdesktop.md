### [tdesktop](https://github.com/telegramdesktop/tdesktop) ![GitHub Repo stars](https://img.shields.io/github/stars/telegramdesktop/tdesktop?style=social)

**Summarizing Telegram Desktop功能**

Telegram Desktop是Telegram官方桌面客户端，提供跨平台即时通讯功能，支持消息发送、群组管理、多媒体共享及加密通信。源码包含Qt界面、网络模块及数据存储，实现高效稳定的消息同步和丰富用户交互。

该项目是 Telegram 官方桌面客户端的完整源代码与构建说明，基于 Telegram API 和 MTProto 安全协议，提供跨平台（Windows、macOS、Linux 等）桌面即时通讯客户端。
