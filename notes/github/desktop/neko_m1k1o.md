### [m1k1o neko](https://github.com/m1k1o/neko)  ![GitHub Repo stars](https://img.shields.io/github/stars/m1k1o/neko?style=social)

Neko 是一款基于 Docker 和 WebRTC 技术的自托管虚拟浏览器及远程流媒体平台。主要功能包括：

1. **隔离环境运行**：支持在 Docker 容器内运行完整浏览器、Linux 桌面环境（如 XFCE、KDE）及各类应用。
2. **多用户协作**：支持多人同时接入，实现实时屏幕共享、协同操控及音频同步，适用于观影派对、协作开发和远程教学。
3. **安全隐私**：提供持久化或一次性匿名浏览，敏感数据不落地本地，支持 Tor/VPN 及内网安全访问。
4. **流媒体优势**：相比传统远程桌面，具备更流畅的视频流、内置音频及原生多用户控制支持。
5. **扩展应用**：支持会话录制、RTMP 直播推流、API 嵌入 Web 应用及自动化浏览器任务。
