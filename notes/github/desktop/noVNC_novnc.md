### [novnc noVNC](https://github.com/novnc/noVNC)  ![GitHub Repo stars](https://img.shields.io/github/stars/novnc/noVNC?style=social)

noVNC 是一个基于 HTML 的 VNC 客户端库及应用程序，支持在现代浏览器（含移动端）运行。它遵循标准 VNC 协议并需 WebSocket 支持。功能包括多种 VNC 认证与编码、桌面缩放裁剪、鼠标与光标控制、剪贴板管理（支持 Unicode）、触摸手势及多语言翻译。提供快速启动、Snap 包安装及集成部署方案，可嵌入各类软件或服务中。
