### [input-leap](https://github.com/input-leap/input-leap) ![GitHub Repo stars](https://img.shields.io/github/stars/input-leap/input-leap?style=social)

Input Leap 为软件 KVM 切换器。单套键鼠控多机。鼠标移至屏幕边缘切换控制。

核心功能与特点：
- 跨平台共享键鼠（Windows、macOS、Linux、BSD）。
- 支持跨机剪贴板共享（Linux/Wayland 除外）。
- Barrier 活跃分支。保持极简，无臃肿功能。

使用限制：
- 所有机器皆需安装该软件。
- 不兼容 Synergy。
- Linux 暂无拖拽文件支持。
- 不支持 32 位 Windows 及旧版 macOS。
