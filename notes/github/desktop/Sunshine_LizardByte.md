### [LizardByte Sunshine](https://github.com/LizardByte/Sunshine)  ![GitHub Repo stars](https://img.shields.io/github/stars/LizardByte/Sunshine?style=social)

Sunshine 是一款专为 Moonlight 客户端设计的自托管游戏串流服务器。它提供低延迟的云游戏功能，支持 AMD、Intel 及 Nvidia 显卡的硬件编码和软件编码。项目配备 Web 界面用于配置和客户端配对，兼容 FreeBSD、Linux、macOS 和 Windows 操作系统，支持多种屏幕捕获技术并具备游戏手柄模拟功能，允许从各类设备连接。
