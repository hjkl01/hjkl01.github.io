### [FreeRDP FreeRDP](https://github.com/FreeRDP/FreeRDP)  ![GitHub Repo stars](https://img.shields.io/github/stars/FreeRDP/FreeRDP?style=social)

FreeRDP 是一个基于 Apache 许可证的免费开源远程桌面协议（RDP）实现。它支持跨平台编译与运行（涵盖 Windows、macOS、Linux、BSD 及多种 CPU 架构），允许用户在多种操作系统下使用 RDP 进行远程桌面连接，同时提供完整的 API 文档、安全政策、微软开放规范参考及编译指南。
