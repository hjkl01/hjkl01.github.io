### [wailsapp wails](https://github.com/wailsapp/wails)  ![GitHub Repo stars](https://img.shields.io/github/stars/wailsapp/wails?style=social)

Wails 是一个使用 Go 语言和 Web 技术构建桌面应用程序的框架。它能够将 Go 后端代码与 Web 前端界面打包成单个跨平台二进制文件，无需启动独立 Web 服务器。支持使用任意熟悉的 Web 技术构建 UI，提供 Go 与 JavaScript 间的无缝通信、原生对话框与菜单、深色模式及窗口特效。基于原生渲染引擎（非嵌入式浏览器），并配备 CLI 工具用于快速生成、编译和打包项目。
