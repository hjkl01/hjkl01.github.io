### [rustdesk rustdesk](https://github.com/rustdesk/rustdesk)  ![GitHub Repo stars](https://img.shields.io/github/stars/rustdesk/rustdesk?style=social)

RustDesk 是一款使用 Rust 语言编写的远程桌面解决方案。该软件开箱即用，无需额外配置，确保用户拥有完全的数据控制权并注重安全性。支持使用官方服务器、自建服务器或自定义中继服务器。核心功能包括远程控制、文件传输、剪贴板同步、音频及 TCP 隧道支持。兼容 Windows、Linux、macOS 及移动端平台，提供多语言界面，支持源码编译构建与 Docker 部署。
