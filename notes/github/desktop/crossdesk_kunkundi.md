### [kunkundi crossdesk](https://github.com/kunkundi/crossdesk)  ![GitHub Repo stars](https://img.shields.io/github/stars/kunkundi/crossdesk?style=social)

CrossDesk 是一款轻量级跨平台远程桌面软件，支持 Windows、Linux 及 macOS 系统。基于 MiniRTC 实时音视频传输库，提供 PC 客户端与 Web 客户端，支持通过设备 ID 和密码发起远程连接。项目支持 Docker 部署自托管服务器，可选启用 CUDA 硬件编解码，实现跨设备的实时远程控制。
