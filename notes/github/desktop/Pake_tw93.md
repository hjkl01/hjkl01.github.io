### [Pake](https://github.com/tw93/Pake) ![GitHub Repo stars](https://img.shields.io/github/stars/tw93/Pake?style=social)

Pake 是一个将任意网页一键打包为桌面应用的工具，支持 macOS、Windows 和 Linux。

核心功能：
- 基于 Rust 和 Tauri 构建，体积轻量，通常约 5MB，接近 Electron 应用的二十分之一
- 运行速度快、内存占用低
- 支持通过 CLI 一键打包，也支持在线构建，无需复杂配置
- 可自定义应用图标、窗口尺寸、标题栏等参数
- 支持快捷键、沉浸式窗口、拖拽、样式定制、去广告等功能
- 提供多种现成应用包下载，如 WeRead、Twitter、ChatGPT、Gemini、YouTube、DeepSeek 等

适用人群：
- 新手可直接下载现成应用包或使用在线构建
- 开发者可使用 CLI 将任意网站快速封装成桌面应用
- 高级用户可本地开发并进行更深入的样式和功能扩展
