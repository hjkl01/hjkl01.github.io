### [neutrinolabs xrdp](https://github.com/neutrinolabs/xrdp)  ![GitHub Repo stars](https://img.shields.io/github/stars/neutrinolabs/xrdp?style=social)

xrdp 是一个开源的 RDP 服务器，通过 Microsoft 远程桌面协议为远程机器提供图形化登录。

主要功能：
1. **远程桌面访问**：支持通过 RDP 连接 Linux 桌面，提供会话重连、窗口大小调整及 RDP/VNC 代理功能。
2. **资源重定向**：支持剪贴板双向传输（文本、图像、文件）、音频与麦克风重定向，以及本地驱动器挂载。
3. **客户端兼容**：兼容多种 RDP 客户端（如 Windows MSTSC、FreeRDP 等），支持跨平台访问（Windows、Mac、iOS、Android），传输默认采用 TLS 加密。
4. **平台支持**：主要针对 GNU/Linux 系统，成熟支持 x86 和 ARM 架构。
