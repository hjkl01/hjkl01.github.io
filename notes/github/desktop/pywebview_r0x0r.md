### [r0x0r pywebview](https://github.com/r0x0r/pywebview)  ![GitHub Repo stars](https://img.shields.io/github/stars/r0x0r/pywebview?style=social)

pywebview 是一个轻量级原生 Webview 封装库，用于在桌面应用的独立原生 GUI 窗口中显示 HTML 内容。它支持 Windows、macOS、Linux (GTK/QT) 及 Android 平台，利用原生 GUI 组件渲染，打包时不捆绑重型渲染器以保持体积小。核心功能包括内置 HTTP 服务器、Python 端 DOM 支持、窗口管理与事件系统、原生 GUI 元素（如菜单和对话框），以及 Javascript 与 Python 之间的双向通信。
