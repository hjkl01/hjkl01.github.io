### [pentagi](https://github.com/vxcontrol/pentagi) ![GitHub Repo stars](https://img.shields.io/github/stars/vxcontrol/pentagi?style=social)

PentAGI 是一款基于人工智能的自动化渗透测试平台，运行在隔离的 Docker 沙箱中，能自主规划并执行渗透测试流程，内置 20+ 专业安全工具，支持多代理协作、长短期记忆与知识图谱，集成浏览器与多种外部搜索引擎获取情报，生成详细漏洞报告；提供现代化 Web 界面、完善监控与日志、REST/GraphQL API、持久化存储与可扩展微服务架构，支持多家云/本地大模型与自托管部署。
