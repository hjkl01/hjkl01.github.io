### [google googletest](https://github.com/google/googletest)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/googletest?style=social)

GoogleTest 是 Google 的 C++ 单元测试框架，整合了 GoogleTest 与 GoogleMock 项目。它基于 xUnit 架构，支持自动测试发现、丰富的断言及自定义断言。核心功能包括死亡测试、致命与非致命失败处理，以及值和类型参数化测试。此外，项目提供多种测试运行选项，支持并行执行，遵循 C++17 标准。
