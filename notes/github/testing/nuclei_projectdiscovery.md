### [projectdiscovery nuclei](https://github.com/projectdiscovery/nuclei)  ![GitHub Repo stars](https://img.shields.io/github/stars/projectdiscovery/nuclei?style=social)

Nuclei 是一款现代、高性能的漏洞扫描器，利用简单的基于 YAML 的模板来设计和执行定制化的漏洞检测场景。

**核心功能：**
*   **模板化扫描**：使用 YAML 模板定义请求，支持高度定制化，模拟真实条件以最大程度减少误报。
*   **多协议支持**：支持 HTTP、TCP、DNS、SSL、WHOIS、JavaScript、Code 等多种协议扫描。
*   **高性能处理**：具备超快的并行扫描处理和请求聚类能力，支持批量扫描。
*   **丰富生态**：拥有由数千名安全专家贡献的模板库，支持自定义模板创作。
*   **灵活集成**：支持 CI/CD 流水线集成，可与 Jira、Splunk、GitHub、Elastic 及云平台集成。
*   **多功能扩展**：支持 DAST 模糊测试、Uncover 目标发现引擎及网络段扫描。
*   **多版本选择**：提供开源 CLI 工具版以及适用于企业级的云托管 Pro 版。
