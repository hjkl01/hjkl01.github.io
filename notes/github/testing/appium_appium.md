### [appium appium](https://github.com/appium/appium)  ![GitHub Repo stars](https://img.shields.io/github/stars/appium/appium?style=social)

Appium 是一个基于 WebDriver 协议的开源跨平台测试自动化框架，支持 iOS、Android、macOS、Windows 及 IoT 等平台上的原生、混合、移动 Web 和桌面应用自动化测试。无需重新编译应用即可运行，支持本地及云端部署。项目采用模块化架构，通过驱动支持特定平台，通过客户端库支持多种编程语言，并通过插件扩展服务器功能。
