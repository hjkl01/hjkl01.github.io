### [google oss-policies-info](https://github.com/google/oss-policies-info)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/oss-policies-info?style=social)

该仓库汇总了多个 Google 开源项目的支持政策，依据 opensource.google 官方原则总结支持矩阵（涵盖操作系统、运行时及语言版本），而非展示特定时点状态。需注意并非所有 Google 项目均采纳这些政策，且用户可通过提交 Issue 或 PR 修正矩阵错误。
